## 动画师，层，状态，过渡

一个**状态**就像是**Unity**动画师树中的一个航点。这就好比到达和起点之间的**过渡**。

通常，这些事情比“当 X 发生时从 A 到 B”更复杂(我说的是条件驱动的转换)。

有时逻辑会发挥作用，这可能取决于我们目前所处的**状态**。
