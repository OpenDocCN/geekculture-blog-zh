# K6 是什么？

k6 是一个以开发人员为中心的、可访问的、开源的负载测试工具，旨在使性能测试成为一种高效且愉快的体验。

使用 k6，您将能够更早地发现性能退化和问题，允许您构建可伸缩的弹性应用程序。
