# 营销是“我给你下了咒语”的歌。

当你试图说服某人做他以前从未想过的事情时，当你看到它起作用时，这有点神奇，但如果你在他的脑海中改变他以前真正想到的事情——操纵，那就更神奇了！

这是一个营销舞台，你是目标，所有人都向你开枪。你没有被强迫买不喜欢的东西，自由选择的是你，但情况总是这样吗？

![](img/a56e7578f8b53915e436a9bf54c2adf2.png)

The marketing arena is where you’re the target and all shoot on you

营销最基本的基本概念是人类需求，操纵人类的关键是首先理解需求、想要和需求之间的区别，因为它们就像最终导致购买决策的三个阶梯——消费。

**需求**是感觉被剥夺的状态，它们包括*生理需求*如食物、性和衣服、*社会需求*如感情和归属感、*个人需求如知识和*自尊。这些需求不是由营销人员创造的，它们只是人类构成的一个基本部分，有时被称为*欲望的原始形式*。

**想要**是人类需求的形式，因为它们是由文化、个性、你来自社会的个人信仰、宗教、种族、肤色、性别、国家等等塑造的。你可能需要食物，但想要一个巨无霸，这就解释了为满足需求而选择的物品。

当需求得到购买力和金钱的支持时，它就变成了需求，这是控制市场的两大市场力量之一。

既然如此，那么营销人员的手在哪里玩呢？

营销人员努力通过创造成功的合适的产品创意将需求转化为需要。

这一切都始于确定消费者寻求满足的需求，例如，有时顾客购买产品或服务仅仅是因为它感觉很好，这意味着一种情感需求，如果营销人员明智地考虑，说服他某个特定的产品不仅是他可以从众多产品中选择的好的替代品，而且是必须购买的，因为这是唯一合适的缺口，即使在最糟糕的经济条件下，这也会使生意兴隆。

一个小技巧..是的，营销就是这种小技巧。

使*想要* **取代** *需要*，并且**说服**客户*不可错过的*产品或服务的价值，并且**将**您的业务带到*下一个层次*。
