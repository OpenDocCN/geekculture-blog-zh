# 用人工智能对抗工作中的人类偏见

> 原文：<https://medium.com/geekculture/using-ai-to-fight-human-bias-at-work-caee2f18f0ff?source=collection_archive---------9----------------------->

作者:[利兹·梅尔顿](https://www.linkedin.com/in/elizabethmelton1/)和[格兰特·李维](https://www.linkedin.com/in/grantriewe/)

人工智能已经成为大多数新技术应用的焦点，并有望彻底改变我们与系统、数据和其他人的交互方式。许多人梦想有更好的解决方案，可以更好地做出决策，更清楚地看到问题，并在嘈杂的未知水域中开辟道路。然而，这些仍然是梦想；人工智能解决相当于人类认知的广泛问题的能力仍有待未来。

但狭义的问题(例如销售模式洞察)或离散边界的问题(例如处理语言或视觉图像)都在人工智能的范围内。这些是认知解决方案，因为我们已经训练系统像人类一样(或比人类更好)识别模式，但它们不一定是超越这一点的决策者。利用这些解决方案的内在能力应该是显而易见的下一步。机器天生能做什么人类纠结的事？

# 如果 NLP 可以帮助识别偏见呢？

没有人能否认人类保持高度专注的能力是有限的。虽然我们知道这是真的，但我们并没有改变这些限制影响我们工作的方式，特别是在电话会议、基于聊天的协作工作或最重要的绩效评估等现场互动中。

评论应该是客观的，但是所有人都会有偏见。虽然许多公司选择小组评审作为消除偏见和挑战现状的一种方式，但我们必须注意在这些会议上说了什么，这些评论是如何说的，以及这些评论的背景，以使这种评审结构达到预期的效果。同时，大多数人的注意力持续时间比评论本身要短。另外，升职取决于老板们对他们的直接下属的印象如何，[他们对员工成功的主观衡量](https://hbr.org/2017/04/how-gender-bias-corrupts-performance-reviews-and-what-to-do-about-it)，以及他们说服他人员工的成就值得奖励的能力。由于这些复合因素，公司文化中出现了偏误模式。

Combine those limitations with the fact that [reviews are often a breeding ground for subtle 一 and not-so-subtle 一 bias](https://www.impraise.com/blog/how-performance-reviews-are-reinforcing-gender-bias-5-steps-to-fight-against-it), and it begs the question: **Why aren’t we using technology to help?**

在人类注意力疲劳存在的地方，机器系统更胜一筹。与人类不同，计算机不会感到疲劳。随着自然语言处理(NLP)和会话人工智能(CAI)的发展，计算机可以实时识别有偏见的短语。这些技术要匹配人类的细微差别还有很长的路要走，但即使是现在，我们至少可以在像绩效评估这样重要的事情上标记出有问题的短语。有了植根于社会科学并基于地理、背景关系和文化标准化的正确输入，我们可以在整个组织中发现潜在的偏见。

在接下来的几个部分中，我们将看看未来的对话式人工智能工具如何减少偏见，并最终教会人们重新评估和重构他们的思维。

# 假设的审查过程

在检查我们假设的例子之前，我们认为定义我们所说的“绩效评估”是很重要的当我们提到绩效评估时，我们指的是大型咨询公司和高功能财富 500 强公司使用的共同评估结构。这种审查方法包括几个要素:

*   一个同行评审小组，为其直接下属的晋升展示案例。每个演讲者只有几分钟的时间来讲述每个案例。然后，在确定结果(绩效评定、晋升、奖金/薪酬等)之前，同事会提出澄清性问题以了解情况
*   主持会议室并指导流程结构的主持人
*   根据上下文、语言和其他因素提出质疑的指定偏见监督员
*   专家小组通常会在每次审查会议上听几十个案例，听了几个小时，然后会感到注意力疲劳。

这些审核流程旨在鼓励消除偏见的对话和平衡结果，但有这么多机会注入有偏见的评论，大多数审核周期仍会产生一些不公平的结果。

# 这项技术会是什么样子？

So, how would this work? We envision applying NLP and NLU to monitor and alert within conversations. Examples of similar services already exist 一 take Alexa, Siri, and Cogito 一 so we have the ability to monitor and engage with conversations. The difference with this technology is that it would be actively detecting comments or statements that imply bias.

许多组织在反偏见培训期间已经做了大量的准备工作，识别出特定于公司文化并与审核者/被审核者关系相关的有偏见的短语。为了应用这些前期工作，我们需要了解一些关于评审者和被评审者的人口统计信息；年龄、种族和性别是强有力的背景指标。了解了公司特定的问题短语、谁正在介绍以及谁正在介绍，我们就有了构建实时警报系统的所有数据元素。

在绩效评估环境中，我们将在下面的示例中看到，系统会在有问题的短语出现时标记出来，委员会负责人会停止对话。然后，委员会将评估评论，要求演讲者提供进一步的信息，只有在足够清晰的情况下才继续。一旦讨论结束，审查周期将继续，直到确定另一个短语。该系统用于在所有对话中始终保持警觉，并突出潜在的偏见，供每个人学习。

需要明确的是，这个偏见检测系统并不是独立地决定哪些陈述是有偏见的。它只是在寻找我们让它寻找的东西。换句话说，这个系统是一个助手，而不是决定者。只有人类能够有效地平衡陈述、个体和文化，以理解一个短语是否真的有偏见；文化和关系中有太多的细微差别，机器无法考虑。

我们预计有偏见和有问题的短语会随着时间的推移而改变，因此公司必须每年用新的术语和短语更新解决方案。虽然这听起来很乏味，但它可以作为一项富有成效的工作，涉及偏见指标、结果数据以及如何建设性地相互交流。

# 偏置检测正在发挥作用

让我们通过一个绩效评估示例来展示偏差检测工具是如何工作的。下面的例子是基于一个通用的过程，在这个过程中，评审者向一个由同行和高层领导组成的委员会提交被评审者的年度绩效，如前所述。

**主要参与者:**

*   **Sundar** (合伙人，男，归化的印度裔美国人，住在新泽西)，审核人
*   **蒂芙尼**(助理，女，黑人，美国人，住在加州)，由 Sundar 审核
*   **普雷斯顿**(助理，男，白人，美国人，住在纽约)，由 Sundar 审核

> **Sundar**:*蒂芙尼今年做得很好，她开始对客户表现出更多的独立性，但如果她能在团队中更多地发表意见，那就更好了*
> 
> **委员** : " *你对她直言有什么具体的反馈？她不代表她的工作流吗？*
> 
> Sundar::*嗯，不，她总是在工作流程的顶端。但她不像我团队中的另一位被评估者普雷斯顿那样参与小组讨论。*
> 
> **委员会主席**:*比较意见不一定是评价性的。她的参与度差距在哪里？*”
> 
> Sundar:“我想她只需要再来一两个周期来培养更多的自信。”

虽然这是一次短暂的交流，但蒂芙尼在她的评论中的形象仍存在一些问题。下面，我们强调了哪些语句可能有问题以及为什么有问题。

## Sundar: " *蒂芙尼今年做得很好，她的* ***开始对客户表现出更多的独立性*** *，但如果她在团队空间里更多地发言**那就太好了。"*

*Sundar 的评论中有两个争论点。首先是短语 ***“开始显示更多的独立性。”****

*It’s not abundantly clear what this means 一 is Tiffany running meetings on her own? Is she forming relationships with clients individually? There is an assumed-to-be understood expectation that is not explicitly expressed, so the comment becomes vague.*

*让我们假设在期望的公司里有一种文化理解。即便如此，根据我们所知的少数族裔女性在晋升前被要求反复证明自己的情况，这种评论仍然是有问题的。这两个方面都需要 Sundar 的进一步澄清和说明。*

*第二个问题与 ***“说出来”有关。****

*我们不知道蒂芙尼为什么没有说出来，她没有说出来的是什么，以及项目本身进展如何。也许蒂芙尼有保持沉默的理由，或者每当她试图说话时，有人在抢她的风头。我们也知道，在咨询文化中，傲慢、主导、咄咄逼人的风格被认为是规范，而忽略了其他有效的沟通风格，这是很常见的。此外，参与的评论与性别或种族差异的规范不一致有很高的相关性。因此，我们不清楚 Sundar 是在提供真正的评价性反馈，还是仅仅在描述蒂芙尼。*

## ***委员** : " *你对她直言有什么具体的反馈？她不代表她的工作流吗？*”*

*虽然委员会成员在寻求反馈方面做得很好，但他们忽略了“独立性”一词，因此委员会错过了参与和澄清的机会。*

> *哦，不，她总是在工作流程的最顶端。但她不像我团队中的其他被评估者 Preston 那样参与小组讨论。*

*在这一点上，Sundar 实际上并没有回答委员会成员的问题。我们没有得到更多关于蒂芙尼直言不讳的细节，Sundar 将她比作一个白人男性同事。比较评估可能是有益的，但在这种情况下，Sundar 没有表达委员会可以判断的任何比较要素。实质上，他不让委员会参与评估过程。*

> ***委员会主席**:*比较意见不一定是评价性的。她的参与度差距在哪里？**
> 
> ***Sundar**:*我想她只是* ***还需要一两个周期*** *来培养更多的自信*”*

*再次，Sundar 没有给出蒂芙尼为什么需要另一个周期来提升的具体理由。这种形式的偏见在少数族裔女性中很常见，她们不得不一次又一次地证明自己的价值，以获得与白人男性同事相同的绩效评估结果。他继续将蒂芙尼评估委员会排除在外。因此，蒂芙尼收到的评价是客观的，但 Sundar 没有足够的意识和压力来为他的评论提供评价证据。*

*在这个例子中，我们仍然看到委员会主席有足够的意识去质疑和询问。然而，Sundar 的回应排除了委员会参与蒂芙尼的评估。此外，只有委员会主席参与质疑职能，委员会的其他成员并未表现出对有问题的措辞的任何意识。所有这些都对蒂芙尼不利，其业绩可能会受到影响。*

# *现在，让我们来看看使用偏差检测工具时，这种对话会如何进行。*

> ***Sundar**:*蒂芙尼今年做得很好，她开始对客户表现出更多的独立性，但如果她能在团队中更多地发表意见，那就更好了**
> 
> ***偏差检测工具**:潜在问题短语:“开始显示更多独立性”。预警原因:确认偏差。背景:女性和少数族裔经常被要求在其他同事没有证明的地方再次证明这一点。*
> 
> ***偏差检测工具:**潜在问题短语:“大声说出来”。报警原因:说法模糊。语境:演讲不是参与的唯一衡量标准；个人有不同的参与方式。*
> 
> ***委员会主席:** " *好了暂停一下，有两个有问题的说法已经确定了。在我们继续之前，我们需要澄清和理解你所说的是真正的评估。Sundar，你能进一步解释你关于“开始表现出更多的独立性”的陈述，并提供蒂芙尼被要求做什么以及她如何做的具体例子吗？当你完成后，我们就可以在团队讨论室里更精确地讨论期望和表现。”**
> 
> *好吧，伟大的旗帜们，我的描述并不准确。让我做得更好。*

*偏见检测工具已经识别出这两个有问题的陈述，并将它们放在上下文中。理想的情况是，这一信息将提供给整个委员会。这样做的目的不是让 Sundar 感到羞耻，而是让人们意识到存在不清楚的陈述，为什么这种不清楚是相关的，并让委员会有机会确定他们是否需要调解和讨论。该工具只是根据它所拥有的培训环境来提供信息——它既不决定结果，也不评估演示者——它只是提供了一个一致的、持久的提醒功能。*

*在一次会议中的多次评审过程中(会议可能持续 6-8 小时，涉及数十名被评审者)，注意力和调解能力会随着疲劳的出现而减弱。人类根本无法持续保持这么长时间的警觉。这就是技术可以帮助我们识别代祷点的地方，因为它不会疲劳。*

# *偏差检测工具的其他应用*

*你可能会问自己，除了指出有偏见的短语之外，这个工具是否还有其他用处。虽然这是我们引入该系统的主要原因，但这并不是它能做的全部。请记住，这种 NLP/CAI 组合正在跟踪每一次对话——谁在发言、他们说了什么、其他人的反应等等。因此，这些数据是公司范围内分析的不可思议的基础。*

*使用偏差检测工具，您可以看到在大多数绩效评估会议中谁在发言。你可以理解谁挑战有偏见的短语，谁不挑战。你甚至可以确定某些类型的人在评论中被歪曲的频率是多还是少。所有这些信息让我们对对话中发生的事情有了一个全新的认识，让我们不仅可以根据绩效评估结果，还可以根据每次绩效评估本身的执行情况进行参与。*

# *现在怎么办？*

*不知疲倦的助手的创建是人工智能助手进化的逻辑步骤。客户服务聊天机器人的扩散也是类似的，尽管我们设想了其他参与策略，而不是常见的简单问答。我们认为这种调解对于改变组织内部的行为是必要的。虽然有专门的解决方案和咨询机构来执行这种分析，但技术的存在使这成为一种实时的、因而更有价值的工具。*

*关于我们提出的偏差检测和监控解决方案，将有一组必要的演进来在组织内保持持久性。第一组将发展解决方案，以包含由于检测到的结果而出现的新的表达或行为。组织将需要首先建立发展解决方案的能力，然后快速进行上游调解、指导和参与。*

*最终，我们会希望退出维护解决方案的工作，让它自己维护和发展。这需要努力解决伦理问题，围绕组织目标的问题，并创建挑战路径，以确保解决方案不会扼杀或过度支配人类行为。*

*科幻小说告诉我们要警惕人工智能，创造了一个机器变坏的神话，这已经演变成释放控制的潜在焦虑。在所有这些故事中，邪恶或流氓元素植根于在创造机器时缺乏深刻的意图和深思熟虑。当我们确实考虑将机器组织的目标和目的与人类组织的目标和目的相一致时，我们创造了仁慈的助手，允许我们挑战自我，并以最理想的方式成长。*
