# 带有自定义到期时间的 Laravel Sanctum

source: laravelnews

Laravel Sanctum 提供了一个简单的基于令牌的认证系统，适用于任何 spa、移动应用集成和 API。Sanctum 允许应用程序的每个用户为他们的帐户生成多个 API 令牌。这些令牌可以被授予指定令牌被允许执行哪些动作的能力/范围。
