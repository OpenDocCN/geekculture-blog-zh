作为技术专业的学生或专业人士，我们通常会在空闲时间做一些项目来娱乐、学习或制作文件夹。这些项目可大可小，但有时我们想在网上托管它们。

当我在做一些像 URL Shortener 和托管 API 这样的项目时，我想现场托管它们，这样我就可以使用它们或者向朋友或面试官展示它们。然而，像 AWS 和 Azure 这样的选项是要花钱的(我的一些需求超出了免费层)，如果可以避免的话，我不想花钱托管一个小项目。

因此，我开始寻找免费托管它们的选项。这里有一些我认为是最好的，我想与你分享，这样你也可以从中受益。

## [1。网络寿命](https://www.netlify.com/)

Netlify 是一个平台，可以让您将所有开发工具集成到一个工作流中。它们支持一系列框架、数据源和工具，如 React、Next.js、GitHub、SalesForce、Datadog 和 Snyk。

它们具有免费的启动层，允许您将静态或动态应用程序部署到它们的全球边缘网络，并具有版本控制功能。对于大多数项目来说，这应该足够好用了。

**特征简单地说:**

*   部署到 Netlify 的全球边缘网络
*   来自 Git 的应用程序和自动化构建和管道的版本控制
*   支持静态和动态资产
*   还支持无服务器功能
*   有利于协作

## [2。渲染](https://render.com/)

Render 是一个统一的云，通过免费的 TLS 证书、全球 CDN、DDoS 保护、专用网络和 Git 自动部署来构建和运行您的所有应用和网站。

它们支持托管 web 服务、静态站点、工人、cron 作业、Docker 容器、数据库等等。您可以免费托管静态网站、web 服务、Redis 和 PostgreSQL 数据库。

它还附带了一系列功能，如 CDN、每日自动备份、后台工作人员等。

**特性简单来说:**

*   有利于协作
*   为您管理 SSL 的自定义域
*   来自 Git 的自动化部署
*   大约 100 GB/月的高带宽

## [3。消防基地](https://firebase.google.com/)

Firebase 是一个应用程序开发平台，帮助您构建和开发用户喜爱的应用程序和游戏。由谷歌支持，全球数百万企业使用。

Firebase 免费提供许多功能，如 A/B 测试、身份验证、远程配置等。

**特性简单来说:**

*   实时数据库
*   测试实验室
*   谷歌云(可能会收费)
*   证明
*   A/B 测试
*   分析学
*   更大的

## [4。Heroku](https://www.heroku.com/)

Heroku 是一个云平台，允许公司构建、交付、监控和扩展应用程序。它们支持许多语言，比如 Node.js、Ruby、Java、Python 等等。

Heroku 的免费层允许您每月从 Git 和 Docker 部署高达 1000 dyno 小时。它还允许您使用自定义域、容器编排和自动操作系统补丁。

**特性简单来说:**

*   使用 Git 和 Docker 的部署
*   自定义域
*   容器编排
*   自动操作系统补丁

## [5。Github 页面](https://pages.github.com/)

如果没有好的 ol' GitHub 页面，这个列表是不完整的。这个工具允许你直接从 [GitHub 库](https://github.com/)托管你的项目。只需编辑、推送，您的更改就生效了。

每个帐户和组织获得一个网站和无限的项目网站。它还与 GitHub 完全集成，并允许您实现更多功能，如使用 Jekyll 写博客、自定义 URL 等。

**特性简单来说:**

*   自定义 URL
*   使用 Jekyll Markdown 写博客
*   与 GitHub 完全集成
