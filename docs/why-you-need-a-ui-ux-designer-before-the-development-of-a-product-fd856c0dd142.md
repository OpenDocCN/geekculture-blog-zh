# 为什么在开发产品之前你需要一个 UI/UX 设计师

> 原文：<https://medium.com/geekculture/why-you-need-a-ui-ux-designer-before-the-development-of-a-product-fd856c0dd142?source=collection_archive---------21----------------------->

在我们探索产品开发前需要 UI/UX 设计师背后的一些原因之前，我们需要知道谁是 UI 和 UX 设计师。

# 谁是 UI(用户界面)设计师？🎨

[用户界面设计师是设计数字产品界面视觉外观的人。他/她负责设计界面(可能是应用程序、网站、手表等)，重点是外观、风格和交互性。](https://careerfoundry.com/en/tutorials/ui-design-for-beginners/what-is-user-interface-design/)

UI 设计师的工作是确保产品界面美观和直观(意味着一个人可以轻松地浏览它，而不必过多考虑他们在做什么)。

你最喜爱的应用程序/网站的外观是用户界面设计师的职责...从颜色到排版，按钮，图标，布局...你在那里看到的一切。

> 所以基本上 UI 设计师是一个负责数字产品审美的人

# **UX(用户体验)设计师怎么样？🔍👨‍👧‍👧**

UX 设计师确保用户在与产品互动时有愉快的体验。他们负责提高用户满意度。

UX 设计师通过提高产品的可用性和可访问性来提高用户满意度，而用户界面设计师通过使产品的界面看起来和感觉起来让用户愉快来提高用户满意度。

> UX 设计师让产品易于使用，并在用户与产品互动时为他们找到最佳体验。

现在我们知道了这些人是谁，让我们在开发产品之前看看为什么需要他们。

# 为什么我们在产品开发前需要一个用户界面/UX 设计师

1.  **UI/UX 设计师降低开发成本**💸:未来可能发生的不必要成本可以在产品开发前的早期阶段看到并避免。UI/UX 设计师帮助创建产品的原型，并在开发之前将其发送给该产品的潜在用户进行测试，以查看它是否满足他们的需求，这有助于将项目保持在预算内。想象一下这样一种情况，一个产品突然被开发出来，然后它不能满足潜在用户的需求，这将花费开发人员的时间、精力和更多，并且对涉众来说也是更多的钱，因为他/她不得不再次支付开发产品的费用。你可以想象一遍又一遍地继续这个过程…那是昂贵的😤。*“如果你觉得好的设计很贵，那就要看看坏设计的成本。”拉尔夫·斯佩思博士*
2.  **UI/UX 设计师关注产品的用户**👪:他们确保用户满意，他们尽可能优化用户在与产品互动时的体验。你可以想象一个产品，用户在使用时感到沮丧和烦恼，他们不会再回来使用该产品，因为他们有过使用该产品的经历，这意味着它会影响业务，因为你会失去客户。但是在 UI/UX 设计师的帮助下，他/她确保用户对产品的体验感到满意。满意的用户会回来使用产品，而沮丧的用户不会。*“不理人的设计不理人”——*[*法兰克奇美罗*](https://medium.com/u/dcda1d2b2f7f?source=post_page-----fd856c0dd142--------------------------------) *。*
3.  **UI/UX 设计者指导软件开发者**👨🏼‍💻:在 UI/UX 设计器的帮助下，软件开发人员开发产品变得更加容易，因为他们为产品制定了指导方针，如字体、颜色、布局、尺寸、交互、流程等，因此开发人员不必在产品的可视化上花费时间，而是将主要精力放在产品的功能和逻辑上，这使得开发速度更快。在某些时候，UI/UX 设计师负责项目管理。
4.  **UI/UX 设计师有助于增加您的收入和客户忠诚度💵**:如果产品的用户感到满意，他们会坚持使用该产品，并向其他人推荐该产品，这将增加您的业务收入，因为该产品的用户越多，您获得的收入就越多。这只有在用户信任你的产品时才会发生。所以用户界面/UX 设计师必须确保用户对你的产品产生信任。好的设计是我们区别于竞争对手的最重要的方式—尹钟永。

这是我们在开发产品之前需要 UI/UX 设计师的更多原因，这只是其中的几个。总之，我们可以说一个 UI/UX 设计师是一个产品的建筑师，就像当你想建一座房子时，你需要一个建筑师来绘制建筑的蓝图，这将指导建筑者如何着手建造房子。UI/UX 设计师和数字产品也是如此。好了，有了这些，我希望你能在开发你的产品之前，找到一些你需要 UI/UX 设计师的理由。
