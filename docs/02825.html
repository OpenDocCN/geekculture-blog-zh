这是浪子系列的第二部分。您可能想在之后结账

  1. 在Ubuntu中安装浪子

先决条件:

  • ruby 2.5或以上版本
  • Ubuntu 16.04或以上

完成设置需要以下步骤。

  1. 安装鲁比和浪子。跟随这篇文章在Ubuntu中安装浪子

  3. 设置您的Android项目
  4. 使用Google控制台创建服务帐户
  5. 使用4中的Json初始化项目中的快速通道
