# 如何在 Python 和 R 中度量集中趋势

集中趋势度量是一种汇总度量，它试图用一个值来描述一组数据，该值代表数据分布的中间或中心。

有三种统计测量可用于测量集中趋势，这些测量是:-

1.  意思是一组数字的平均值。
2.  中位数，是一组数字的中心数。
3.  模式，是最常用的数字。
