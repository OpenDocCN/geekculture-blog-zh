## 1.稀有

r 是一种编程语言，可以在包括 Windows 和 MacOS 在内的各种平台上运行。它是一种开源语言。r 用于数据科学和机器学习。数据科学中的两个主要软件包是 R 和 Python。r 执行统计和图形功能。Python 可以帮助你建立真正的机器学习模型。两者都很强大，在行业内都有使用。但是，Python 是一种通用语言。大部分大数据和 AI 公司都把 Python 作为通用语言。伦敦 Museam 用 R 分析历史数据。

r 是一种用于统计计算和图形的编程语言和软件环境。这是一个 GNU 软件包。R 系统是一个由语言和运行时环境组成的编程环境。统计学家和数据挖掘者广泛使用 R 语言来开发统计软件和数据分析。R 语言是围绕各种各样的统计和图形技术而设计的，为数据操作、计算和图形显示提供了一组丰富的函数。它有一个全面的统计分析和图形软件包。

## 2.计算机编程语言

Python 是一种通用编程语言。它用于 web 开发，有大量可用的库和包。Python 被设计成具有高度可读性，并且它的语法易于学习和理解。Python 是初学者的绝佳语言，因为它简单易用。很多开发者在学习 C++或者 Java 之后，把 Python 作为第一语言。它还用于处理大量数据，因为 Python 是一种解释型语言。事实上，它可以在几天内学会，使它成为初学者的一个伟大的选择。

Python 是一种功能强大的编程语言，常用于机器学习。它被认为是一种高级语言，意味着它易于读写。对于仍在学习如何编码的初学者来说，这是一个巨大的好处。该语言还提供了大量用于数据处理和分析的库，这使得它成为数据科学家的热门选择。Python 也被认为是初学者的优秀语言，因为它非常容易学习。有大量的免费资源可供使用，比如 Python 初学者速成班和 DataCamp 上的 Python 教程。

## 3.（舞台上由人扮的）静态画面

Tableau 是我们在工作中使用的一个数据可视化工具，我已经使用它很长时间了。它实际上是为商业用户设计的，但它在创建快速可视化方面真的很棒，它非常适合了解数据，了解正在发生的事情，而不会陷入细节中。对于数据科学来说，它也是一个非常好的工具，因为它有很多用于执行统计分析和创建预测模型的功能。我已经能用 Tableau 做一些非常酷的东西，我希望能用它做更多。这对我来说是一个很好的工具，因为我在工作中有很多不同的职责，我做很多不同的事情。我做过很多数据可视化、很多预测建模，甚至一些传统的数据科学工作，我希望在未来做得更多。它是一个强大的工具，很直观，免费版可以让你做很多事情，所以我强烈推荐它。

Tableau 是一个优秀的可视化讲故事软件，允许用户创建交互式和有吸引力的图形来讲述数据驱动的故事。有了 Tableau，你将能够执行复杂的数据分析，创建可视化并在线发布。Tableau 是昂贵的专有软件的一个很好的替代品，并提供了广泛的功能。

## 4.斯堪的纳维亚航空公司

SAS 是商业分析领域的领导者。它非常专注于业务分析和数据管理。该软件几乎用于每个行业，并在大数据市场大受欢迎。它为用户提供了大量的统计模型和广泛的功能。然而，如果你想成为一名数据科学家，你最好看看这个列表中的其他选项。SAS 对于数据管理和访问数据库中的数据是一个很好的工具。它有一些可视化和数据挖掘的选项，但除此之外没有别的。对于希望以不同方式查看数据的商业用户来说，这是一个很好的软件，但是如果你希望用这个软件进行预测，你会失望的。

SAS 是由 SAS 研究所开发和营销的软件套件，用于高级分析、企业数据管理、商业智能、大数据和物联网(IoT)。SAS 软件广泛应用于世界各地的商业、政府和学术界。“SAS”是“统计分析系统”的缩写。该公司由吉姆·古德奈特和约翰·索尔于 1976 年创立。[1]首字母缩略词最初是“自动统计分析系统”的缩写。Goodnight 和 Sall 在北卡罗来纳州立大学使用 Fortran 和自数组语言编写了第一个 SAS 程序。这个缩写后来被更改为支持不带空格的首字母缩写。

## 5.矩阵实验室

Matlab 是一种用于统计、科学计算和工程的编程语言和软件环境。在学习工程或科学的学生中，这是一个行业标准。Matlab 是一种用于数据操作和建模以及计算模拟的编程语言。它主要用于分析数据和创建模型，但 Matlab 也用于工程和交互式应用，如视频游戏。Matlab 可以在各种行业中使用，包括航空航天、汽车、通信、数据存储、金融、医疗保健、工业自动化和软件。Matlab 还用于教育，有各种在线课程可以教人们如何使用 Matlab。

Matlab 是一种帮助解决复杂数学问题的编程语言。它是 MathWorks 的开源工具，可以免费获得。如果你是理科生，或者追求数据科学或分析，这是一门很好的语言。这是一个基于图形用户界面的工具，你可以用它广泛的功能来解决复杂的问题。
