# 什么是机器学习？

对于那些不熟悉机器学习的人，这里是 IBM 的正式定义:“机器学习是人工智能(AI)和计算机科学的一个分支，它专注于使用数据和算法来模仿人类学习的方式，逐步提高其准确性”(IBM Cloud Education，2020)。

换句话说，机器学习是一个专注于使用数据&数学算法和关系来模仿人类学习方式的领域。你有几乎 100%的机会在今天的某个时间点与机器学习系统进行过交互。这种系统的一个例子是推荐系统。一个推荐系统获得关于你喜欢和不喜欢的“反馈”数据，然后预测你更有可能阅读哪些文章(就 Medium 而言)。正如例子所示，你可以用机器学习做很多很酷的事情。

# 机器学习的应用

正如我在上一节中讨论的，机器学习有许多不同的应用。医生使用机器学习来诊断疾病。事实上，2019 年发表的一篇论文发现，支持向量机(SVM)能够以 96%的准确率进行癌症诊断(Sidey-Gibbons & Sidey-Gibbons，2019)。

如果你有一辆特斯拉，你很可能会使用“自动驾驶”系统。虽然这个系统远比许多机器学习系统先进，但它仍然是机器学习的一个应用(我知道你们很多人会认为这是一个深度学习系统，而不是机器学习系统；但是，回想一下，深度学习是机器学习的一个子领域)。

仅仅通过这两个例子，你就可能明白为什么机器学习是一个如此热门的领域。它有无限的应用，而且这些应用看起来很“酷”，非常吸引人。

# 机器学习是否言过其实？

我要说在某种程度上，机器学习被夸大了。我相信当很多人听到机器学习时，他们会想到像《终结者》这样的电影。就此而言，目前机器学习还没有能力像人类和其他生物一样智能。我还相信，当人们听到机器学习这个术语时，他们会立即认为这是关于部署高级智能算法来完成某项任务。然而，机器学习工程师的许多职责需要大量的数据准备。您还可以在其中加入模型测试和模型部署。本质上，有许多任务可能看起来不如在数据集上部署机器学习算法那样有吸引力。

当我参与机器学习项目时，我意识到机器学习不仅仅是知道什么是支持向量机。你必须知道如何准备你的数据，你必须知道在模型训练中使用什么技术，等等。在所有的魅力背后，有许多“不吸引人”的任务。然而，这并不是说一个人不应该追求机器学习。就我而言，机器学习是一个非常非常有趣的领域。它有很多应用，而且就我个人而言，我非常喜欢构建机器学习系统，即使有些任务很乏味。

# 简而言之…

简而言之，我认为机器学习是一个有趣的领域，但我确实认为它在某种程度上被夸大了。我不应该太惊讶，因为每当有一个“一代天才”(正如体育界的评估者所说)出现时，炒作就会扶摇直上。因此，我们对“一代人才”设定了一些无法实现的期望，然后最终认为“一代人才”是一个“泡沫”。

机器学习也经历了同样的阶段。虽然我们不看领域，说那是一个“半身像”，但我认为肯定有许多人相信领域不辜负炒作。然而，我相信你应该把炒作作为进入这个领域的动力，但随后要致力于完善你的手艺。毕竟，机器学习是一门艺术，而不是一门科学。
