# 加密货币的剖析

自加密货币创建以来所有确认的交易都存储在一个名为*区块链* 的公共账本[中，该账本利用加密技术来保持记录的真实性。交易发生在两个数字钱包之间，然后提交给区块链进行确认。](https://wazirx.com/blog/blockchain-explained/)

这个确认并将[交易添加到区块链](/coinmonks/defi-risks-explained-how-to-stay-safe-on-the-blockchain-7e4572658e69)的过程称为挖掘，它通常要求挖掘者运行一个验证软件来解决加密难题，以验证合法交易。

挖掘过程，也称为 [*工作证明*](https://wazirx.com/blog/proof-of-work-vs-proof-of-stake-whats-better/) 系统，通过生成并向首先解决该难题的矿工分发额外的加密货币来激励。然而，像比特币这样的大多数加密货币都有硬币数量的硬上限，因此使其成为通货紧缩的价值储存手段。

加密货币吸引了所有人的注意力。大品牌、名人、商人和投资者希望从这一大现象中获利。奇怪的是，世界各国已经意识到加密货币不仅仅是数字货币，而是一种合法的替代资产类别，可以用来实现投资组合的多样化。

当你听到有人刚刚成为加密货币百万富翁或亿万富翁时，如果你感到震惊，我们不会责怪你。有一些匿名的比特币亿万富翁；根据 [BitInfoCharts](https://bitinfocharts.com/top-100-richest-bitcoin-addresses.html) 的数据，截至 2021 年 1 月，25 个个人比特币名单持有超过 10 亿美元。

# 使用加密货币赚钱的 7 种方式

# **1。投资**

投资是一项长期战略，需要耐心和毅力。由于过程简单，这是最容易上手的用加密赚钱的方法之一，通常被称为买入并持有(HODL)策略。在尽职调查之后，您相信某个特定加密的用例在未来会变得非常广泛。

在这种情况下，你直接购买一些密码作为多样化投资组合的一部分，然后等待价格长期不可避免地上涨，这可能是有意义的。

**优点**:投资回报率高

一个直接让投资加密货币变得合理的统计数据是，2013 年投资比特币的 1000 美元如今将价值超过 40 万美元。

缺点:市场波动性很大，这意味着出现重大损失的风险很高。

# **2。交易**

这种用 crypto 赚钱的策略不适合没有经验的用户。投资是长期策略，而交易是短期策略，需要一些知识。分析上市资产表现的市场图表需要一些技术和分析技巧，以准确预测价格的涨跌。

**优点**:做多或做空并从摇摆交易中获利的能力。

**缺点:**与通常被认为规避风险的投资不同，[交易加密货币比投资风险更大](https://wazirx.com/blog/guide-margin-trading-cryptocurrency/)。建议使用技术诀窍。

# **3。采矿**

如果你[挖掘一个加密货币](https://wazirx.com/blog/what-is-bitcoin-mining/) y，你将获得新的硬币奖励。挖掘需要专业技术以及对专门硬件的初始投资。这是通过在采矿钻机上使用专门的软件来实现的——硬件越强大，成功验证交易的机会就越高。

**优点**:你在帮助加密货币社区发展的同时也在赚钱。

**缺点**:电费和硬件成本高。同样重要的是，不要忽视这一过程的复杂性。

# **4。运行主节点**

一个 [masternode](https://www.investopedia.com/terms/m/master-node-cryptocurrency.asp) 本质上是一个保留区块链实时副本的电脑钱包。它的功能是任何人都可以使用的服务器。不同的加密货币基于矿工和主节点持有者之间的奖励分成具有不同的赚钱激励结构。

**优点**:在加密领域获得奖励。

**缺点**:你的账户里必须有一定数量的硬币才能运行 masternode，这也需要一些技术知识

# **5。接受加密支付**:

是的，这似乎是用加密赚钱的最简单的方法。如果你拥有一家企业，你可能会考虑接受你喜欢的加密货币支付。你可以用它来支付其他东西，或者留着它，等待它升值。如果硬币升值，你可以获得可观的利润。

**优点**:更安全(对双方而言)，降低了欺诈的可能性。促进即时跨境交易

**缺点**:价格波动

# 6。高产农业:

[收益农业](https://wazirx.com/blog/a-beginners-guide-to-yield-farming-the-latest-on-the-defi-hype-train/)是以不同加密货币的形式，押注或出借比特币资产以赚取显著回报或奖励的活动。这是一种通过向 DeFi 平台临时出借加密资产来生成加密货币的技术，也称为流动性挖掘。换句话说，你让你的加密货币为你工作。虽然通过使用产量农业方法用加密货币赚钱很简单，但确定最成功的策略并不简单。

优点:包括潜在的收入，减少的学习曲线，以及入门的便利性和较少的要求。

**缺点:**因为是一个竞争激烈、快节奏的市场，结果不一定是恒定的。

# **7。赌注加密货币:**

由于运行 PoW 区块链所需的能量非常高，一些区块链提出了一种看起来更好的验证算法，称为“赌注证明(PoS)”。PoS 算法不是利用能源和硬件来运行复杂的加密难题，而是让用户锁定(标记)他们的令牌来验证加密。

锁定是锁定数字资产，使其在分散加密网络中起到验证器的作用，以确保网络的完整性、安全性和连续性。利益相关者保护他们的资产作为节点和确认块。赌注者获得新铸造的加密货币作为帮助保护网络的奖励。

**优点**:从加密货币中赚钱的一种更便宜的方法

**缺点**:价格波动是最大的危险。
