Web 3.0才刚刚建立

原文：https://medium.com/geekculture/web-3-0-is-just-being-built-7e99238713d?source=collection_archive---------14-----------------------#2021-11-06

为什么我们需要这种转变？

基于区块链的第三代万维网即将问世。这意味着它将是完全去中心化的，与现在的互联网相反，它是一个由各种实体控制的全球网络。

我们走了很长的路才走到这一步，现在是改变的时候了。

从Web 1.0到Web 3.0

互联网已经从最初的1.0版本发展到第二个版本，在1.0版本中，大多数人作为网络上提供的内容的消费者对这一新技术感到兴奋，在第二个版本中，更多的人作为内容的创造者参与进来。

第一个版本更多地基于文本和颜色，但在2000年后，web 2.0带来了图形设计，最大限度地提高了浏览器显示的2D功能。后来，3D设计也开始被使用，但是我们仍然在2D的浏览器窗口中看到它们。

Web 3.0带来了彻底的革命性变化。我们将拥有3D图形，但不是在2D浏览器上。通过使用虚拟现实眼镜，这将像一个游戏，你将完全沉浸其中。你会用手势和动作改变一切。它将是完全互动的，就像一个全新的世界。

分散的互联网

为什么我们需要这个3.0版本的万维网？因为我们需要互联网上的自由，因为我们需要让互联网更加直观。区块链技术可以通过去中心化让这成为可能。

互联网被认为是没有规则和条件的，但事情发生了很大变化，现在远非如此。请记住，在过去十年中，我们的数据收集是如何发展的，我们现在是如何在甚至没有阅读条款的情况下给予同意的，因为有太多的信息需要阅读。

更安全的互联网

Web 3.0目前正在构建中，即将彻底改变我们使用互联网的方式。为了更清楚地理解这意味着什么，你可以想象最简单的事情，比如登录一个网站。

因为web 3.0是基于区块链技术的，所以你可以像登录区块链一样登录，而不用写用户名和密码。

换句话说，您将使用钱包登录。这个钱包可以用于进入多个网站，也可以用于游戏到应用程序。如果您获得了什么，而不是使用支付处理器，收益将被转移到您的钱包。钱包的地址将代表帐户名称，但这并不意味着您不能选择昵称。密码将是钱包的密钥。这种类型的登录更加安全。

一个全新的时代

在新的互联网中，没有主要的管理机构或公司。人们可以参与其治理，并获得代币奖励。Web 2.0编程正在变得过时。一种构建网络的新方法正在开发中，它也可能使编程工作发生革命性的变化。

业内知名企业已经开始尝试迈出第一步。变化是不可避免的，因为变化就是进化。

如你所见，我们需要这种转变，因为这是互联网技术的正常发展。你准备好迎接web 3.0带来的新时代了吗？你应该担心，因为它比你可能意识到的更快地进入你的家。
