# 为什么 AI 是企业家的新朋友

商业启动过程的每个阶段都有一个人工智能工具

加里·卡斯帕罗夫从来没有机会。1997 年，国际象棋世界冠军[输给了由人工智能(AI)驱动的计算机深蓝](https://theconversation.com/twenty-years-on-from-deep-blue-vs-kasparov-how-a-chess-match-started-the-big-data-revolution-76882)。在比赛中，卡斯帕罗夫正在评估每秒三步棋。相当令人印象深刻。与此同时，深蓝正在策划超过 200 个*百万个*招式。

想象你现在是一名企业家。让深蓝的更聪明的表弟担任你的高级管理团队的顾问，你会获得什么样的优势？你会产生多少更可行的商业想法？你将这些想法转化为赚钱的产品或服务的速度有多快？你的决定最终会建立在冰冷、坚硬的数据上，而不是偏见和狭隘的思维上吗？

这不是一个幻想的场景。人工智能应用程序是为初创公司量身定制的，许多应用程序已经触手可及。这些技术的魔力——最明显的是[机器学习](https://en.wikipedia.org/wiki/Machine_learning)——与其说是关于*智能*，不如说是关于模式寻找和预测，这是创新的基石。这就是为什么有些人把人工智能称为“一种发明方法的发明”[——一种特别适合创新企业家的发明。](https://www.nber.org/papers/w9825)

人工智能可以改善创业过程的每一步，从发现机会到开发和利用机会。要求相当简单。只要小型企业拥有深入而清晰的客户交互数据集，并能够访问提供人工智能应用程序的基于云的平台，如亚马逊网络服务、微软 Azure 和谷歌云人工智能，他们就可以获得人工智能的好处。这些平台提供了灵活性和可扩展性；创业者可以在一段时间内免费试用机器学习服务，然后选择为特定项目支付费用，如构建虚拟助手或运行情感或图像分析项目(通常按计算时间的每小时收费)。

下面让我们仔细看看人工智能技术如何推动创业旅程的各个阶段。

# 建立创业团队

新生的创业企业往往缺乏人力资源专业知识。然而，在某些时候，他们不得不招募一个功能专家团队——在同学或家庭社交网络中找不到的人。人工智能在这方面尤其有前途。

人工智能公司可以帮助客户消除工作描述中的偏见语言，使用自然语言处理和机器学习筛选简历，并使用[聊天机器人](https://en.wikipedia.org/wiki/Chatbot)管理候选人的体验。Pymetrics 公司让公司员工参加一系列类似电子游戏的测试。然后，它将数据输入一个算法，建立一个适合特定角色的人的角色。候选人然后参加类似的测试，看看他们在行为和团队文化上是否匹配。

# 寻找创意

人工智能工具可以连接不同的信息，以确定市场需求或失败，这些需求或失败可以转化为新的创业机会。

例如，可以部署社交情感分析和自然语言处理来分析社交媒体和在线客户论坛，以确定某个产品或服务类别是否准备好进行颠覆。一个[情绪分析工具](https://blog.hubspot.com/service/sentiment-analysis-tools)是一个分析文本对话并评估每条信息背后的语气、意图和情绪的软件。本质上，这些微弱的信号可以被输入到一个算法中，以识别违反直觉的见解。

如果企业家被一个创业想法难住了，他们可以访问 [IdeasAI。](https://www.google.com/url?sa=t&rct=j&q=&esrc=s&source=web&cd=&cad=rja&uact=8&ved=2ahUKEwi7ptSams3xAhUOWs0KHUnCBMcQFjAAegQIBhAD&url=https%3A%2F%2Fideasai.net%2F&usg=AOvVaw0pDCFUhxK9NgZwT7lTTD0c)该网站利用深度学习产生新产品和商业创意。访问者“喜欢”或“不喜欢”一个想法，并在这个过程中训练算法和改进未来的想法。

# 设计和原型制作

人工智能可以通过提供更严格和无偏见的数据评估以及整合多种信息来源来扰乱和加速产品设计过程。一个适度的例子是 IntelligentX，一家英国初创公司[利用人工智能开发了一系列新的啤酒](https://digital.hbs.edu/platform-digit/submission/intelligentx-changing-the-world-one-beer-at-a-time/)。该公司首先部署了一个机器人来询问消费者对口味的偏好。然后，它将这些数据输入到一个算法中，这个算法被用来创建它的第一个啤酒产品系列。配方会根据持续的反馈不断调整。

像欧特克这样的公司正在为生成式设计开发人工智能驱动的工具。设计师或工程师将设计目标以及性能要求、材料和成本约束等参数输入到生成式设计机器学习软件中。该软件探索潜在的解决方案，快速生成设计方案，并从每次迭代中学习什么可行，什么不可行。该软件还通过 3d 打印为快速原型制作生成数字模型。

# 销售产品和服务

人工智能支持的销售工具可以通过[自动化线索参与和资格](https://exceed.ai/ai-sales-assistant/)以及[通知销售人员谁值得联系](https://105.incomaker.com/en)，将销售人员从更高价值的面向客户的任务中解放出来。或者，他们可以用基于社交信号处理和计算机视觉的人工智能系统来超越销售人员，这些系统可以解释行为线索，帮助销售人员评估客户对产品功能或价格的反应。或者他们可以用自然语言处理和深度神经网络创造的聊天机器人来取代销售人员。

人工智能还可以解锁可能隐藏在众目睽睽之下的营销信息。IBM 的 Watson Personality Insights 平台扫描一家初创公司的社交渠道，以创建目标客户的个性档案。然后，它可以确定有影响力的人或客户的同行群体中有可能成为公司冠军的成员。

# 扩展企业

当管理、设计和销售流程主要由人工智能应用驱动时，一旦新业务流行起来，扩大运营规模就变得容易得多。应对资源限制不再是增加员工，而是增加计算能力，以提供更多的人工智能功能。

后台职能、客户关系管理和其他日常任务的自动化意味着更多的人力时间用于更高附加值的工作。

如果聊天机器人被证明是有效的，他们可以很容易地被添加到。顶级销售人员甚至可以被克隆。有些公司使用对话式人工智能技术来分析客户组织中表现最佳的销售人员。他们使用这些数据来训练机器学习系统，并创建可以全天候回答买家问题的虚拟销售助理。

# 培养未来的企业家

研究人员正在使用人工智能来更好地理解创业的动力。在未来的几年里，我们可以期待在董事会、产品开发会议、创业推介和其他场合读到关于[促进我们理解创业和投资者决策的研究](https://link.springer.com/article/10.1007/s11187-019-00205-1)。这些研究将基于社会信号处理，分析视频剪辑和图像中的身体姿势、面部表情和声音行为线索。

一项已经发表的研究使用[机器学习来预测众筹活动的结果](https://www.google.com/url?sa=t&rct=j&q=&esrc=s&source=web&cd=&cad=rja&uact=8&ved=2ahUKEwi3sPeno8fxAhXELs0KHViGBksQFjABegQIBhAD&url=https%3A%2F%2Flink.springer.com%2Farticle%2F10.1007%2Fs11187-019-00218-w&usg=AOvVaw0ZZ5sUmgT4_-KSh6otESDJ)。研究人员利用 20188 次众筹活动中的文本、语音和视频相关元数据，建立了一个模型，该模型在解释活动成功或失败方面的准确率为 73%。

另一项研究使用基于人工智能的技术[分析企业家的“数字足迹”](https://www.sciencedirect.com/science/article/abs/pii/S2352673417300239)以获得对超级企业家个性特征的洞察。研究人员通过用计算机化的文本分析工具分析他们的推特信息，比较了 106 位最具影响力的商业领袖的性格特征。研究表明，超级明星经理在许多个性特征上比超级明星企业家更具企业家精神。

这并不是说使用人工智能对企业家或他们的团队来说很容易。不会的。人工智能技术需要一定程度的数据管理成熟度。基于云的人工智能应用提出了自己的挑战，其中主要是隐私、数据所有权和数据可移植性。

尽管存在这些问题，但想象人工智能时代是企业家最感兴趣的时代，肯定很有诱惑力。
