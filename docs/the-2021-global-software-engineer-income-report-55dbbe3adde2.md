近日，针对美国科技公司的数据收集网站 [Levels.fyi](https://www.levels.fyi/) 发布了《2021 年全球软件工程师收入报告》，该网站于 2017 年建立，拥有全球 1200 多个城市的 5 万个数据点。在报告中，Levels.fyi 根据级别和地点分析了各公司的薪酬。数据是权威的。
