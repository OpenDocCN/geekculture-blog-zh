# 1.铬 devo

Devo 是一个 chrome 扩展，可以在每个新标签上显示 GitHub 趋势、黑客新闻、产品搜索和许多不同平台的首页，不需要登录。

你可以改变卡片的位置，以不同的顺序显示你想要的任何平台。

这有助于我找到新的 GitHub 项目、产品搜索上的新产品和黑客新闻。我喜欢这个扩展，尤其是它的黑暗主题。
