# 什么是死海效应？

如果你关注我，你会知道我通常写软件工程。你会发现很多关于死海效应的讨论都与软件工程有关，但是死海效应并不是软件工程的专利。很多球队都可能发生这种情况。

死海效应是指糟糕的管理实践、臃肿的流程，或者你在团队中可能发现的任何其他有害特质，将好员工赶走，只留下“坏”员工。

这个名字是指死海，位于以色列和约旦之间的一大片水域。水只能通过蒸发离开死海。这留下了大量的盐，比海洋多八倍。这种积聚的结果是，死海无法维持生命。

*理念*是优秀的或备受追捧的员工不会容忍不良行为，因为机会对他们来说是无限的。所以留下的“盐”是那些助长或鼓励不良行为的人，或者是那些太无能而无法找到另一份工作的人。

留下的“坏”员工并不总是坏的——因此有了这句名言。我不同意那个想法。我认为它缺乏同理心和视角。有很多好的雇主出于非常正当的理由留在这种情况下的例子。以下是一些例子:

*   他们可能被困在专有工具中，不再拥有非常有市场价值或可转让的技能。
*   他们缺乏雄心壮志，因为他们被他们的组织所蒙蔽，认为这是对他们最好的。
*   他们相信，作为个人贡献者，他们可以单枪匹马地改变不良做法。
*   他们不想经历另找工作的麻烦。进入就业市场是很多的。准备简历，应聘新工作，准备面试，面试一大堆。

但是当“坏”员工真的很坏时，你的组织或部门将一事无成，也无法留住人才。

# 解决方案

![](img/3e724854ddca2f0f9f71cb5d2aca66e0.png)

## 对雇主而言

*   认真对待你从高绩效员工那里得到的反馈。问问他们的痛点是什么。问他们瓶颈在哪里。相信他们的反馈，因为你不是“在杂草中”
*   给你的团队注入一个学习过程或轮换。这可能是一个读书俱乐部，或者类似的东西，每个人都花一个月的一天来试验一个新的花哨的工具、过程或框架。接触会改变他们的头脑，促进创新。
*   更高的薪酬是留住人才最简单、最有效的方法，而且可以大幅提高。

## 对于员工

*   尽最大努力避免不可转移或不可出售的任务和责任。我知道这说起来容易做起来难。
*   为你的职业生涯制定一个长期计划。不要滑行。
*   很多组织都失败了。别以为你的不行。知道何时从沉船上跳下。

# 结论

死海效应是指糟糕的做法只会留下不称职的员工。如果雇主积极主动，这是可以避免的。
