# UX 设计师和前端开发人员的区别

> 原文：<https://medium.com/geekculture/the-differences-between-a-ux-designer-and-a-front-end-developer-452efa9df07d?source=collection_archive---------4----------------------->

UX 设计师和前端开发者的主要区别在于，前者负责设计网站的整体观感。另一方面，前端开发人员主要通过代码和图形来设计网站的 UI。如果你是一名设计师，清楚两者之间的区别是很重要的，这样你就可以相应地给客户提供建议。
