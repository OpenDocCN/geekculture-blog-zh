密码是保护我们账户的锁的钥匙:你的电子邮件，银行和投资账户，社交媒体，公司门户网站，以及更多使用基于密码的认证。他们都要求你创建一个安全的密码，有时会给你这样的规则:

*   最少 8 个字符
*   至少一位数字…
