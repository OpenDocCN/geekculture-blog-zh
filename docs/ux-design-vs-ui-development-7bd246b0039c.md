# UX 设计与用户界面开发

> 原文：<https://medium.com/geekculture/ux-design-vs-ui-development-7bd246b0039c?source=collection_archive---------7----------------------->

UI 设计和 UX 设计之间只有一线之隔。虽然这两个概念是相关的，但每一个都有不同的用途。UI 设计关注的是用户界面，更多的是一个艺术过程。相比之下，UX 设计更注重分析、构思、开发和交付。以下是两者之间的一些关键区别。为了理解这两者之间的区别，考虑下面的例子。
