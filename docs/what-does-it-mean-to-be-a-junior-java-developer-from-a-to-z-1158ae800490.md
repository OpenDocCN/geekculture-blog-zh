说到 it 行业，我们通常根据专业水平将开发人员分为三个等级:初级、中级和高级。或者四个……以防万一，你把编码实习生列为软件开发行业排名最低的“士兵”。

而如果你认为成为 Java 开发的专家主要靠你多年的编程经验，这就离事实再远不过了。实际上，初级开发人员的服务年限可能比中级甚至高级开发人员更长。这完全取决于你的技能水平。这就是为什么，我决定描绘一幅清晰的画面，描绘一个普通初级开发人员的样子，他拥有获得“初级”徽章所需的技能，你将获得的观点，以及你可以期望的工资水平。我们来钻研一下。

# 谁是初级开发者？

到底谁是初级 Java 开发人员？初级程序员和中级程序员有什么区别？初级 Java 开发人员是开发人员世界中的入门级专家，初级开发人员和中级开发人员之间有明显的差距。

不说明显的，但主要的区别是初级开发人员通常是“刚毕业”的程序员，他们需要学习这个新职业的许多东西。相比之下，mid 开发人员已经拥有 2 到 4 年的经验。中层已经可以独立工作，而低层需要高层或领导的监督。

也就是说，大三学生刚刚开始他们激动人心的职业道路，并熟悉现代软件开发的工作方式。您应该准备好从一些简单的任务开始，比如调试、为简单的特性编写代码等等。

接下来，我想更深入地探讨雇主对初级开发人员的期望，他们最需要的技能，以及他们的主要职责。

# 初级开发人员的主要职责是什么？

一般来说，初级开发人员应该能够:

*   编写和维护代码；
*   修复代码中的小 bug 和小错误；
*   从事工具/功能的内部(管理)工作；
*   了解代码库和项目的结构；
*   用 codebase 工作，不用做太复杂的事情；
*   致力于由高级软件开发人员设定的已定义功能、它们的一般概念和标记；
*   次要方面侧重后端；
*   协助项目规划的初始阶段，分析项目的技术要求；
*   参与测试的执行和记录；
*   与敏捷团队合作，参加会议；
*   准备报告、手册和其他文件；
*   从用户那里收集关于产品的信息。

换句话说，除了计算机知识，一个好的初级软件工程师在某种程度上应该是一个后端助手。初级开发人员可以定义常见的障碍，阐明工作流程，同时逐渐适应现有的工作条件。

起初，初级设计意味着在部门高级开发人员的监督下工作，但参与项目的所有阶段，从头脑风暴到最终测试。作为一名有能力的初级 Java 开发人员，您可能还会会见客户，了解他们的偏好并提供建议。

# 初级开发人员的基本技能

*   0-1.5 年的工作经验，对学习充满热情；
*   了解软件如何与硬件交互并依赖于硬件；
*   Java 语法、集合、多线程；
*   使用编码工具(Eclipse、IntelliJ IDEA 或 NetBeans)、版本控制系统和服务(GitHub、git lab)；
*   了解构建 web 项目的框架(Spring，Spring Boot)；
*   对象关系映射(Hibernate)；
*   楼宇自动化工具(Gradle、Maven)；
*   使用单元测试工具(JUnit，Mockito)。
*   其他常见编程语言的基础知识，如 JavaScript，以及 HTML5/CSS3 等技术；
*   编程和代码编写的基本实践经验；
*   数据库和操作系统知识；
*   计算机科学的基础知识(作为几乎所有软件开发专业的基础知识)；
*   快速学习新软件平台和技术的能力(对任何初级程序员来说都是相当重要的技能)；
*   遵循指示和在团队环境中工作的能力(另一项不应该被低估的技能，尽管它经常被低估)。

# 成为初级开发人员的几大理由

成为初级 Java 开发人员有几个好处。例如，初级开发人员经常在开发项目的所有阶段工作，因此他们获得了非常有用的经验，可以很容易地转移到以后的各个领域。通常，初级开发人员也可以利用通常允许远程工作的灵活时间表。当然，他们的职业是很有回报的。尽管我不认为经济补偿应该是主要原因，但我不能否认高薪是一些 Java 开发者的主要动机之一。根据热门的招聘网站，比如 Glassdoor、T2 和 T3，初级 Java 开发人员的年薪高达 75，000.00 英镑。而且，随着你职业的发展，你的薪水也会增加。

就职业发展而言，不提及初级程序员有真正的机会获得整个工作过程、任务和技术范围的应用经验是不合适的。而这种经历可能会帮助你最终成长为一名高级开发人员。试着保持开放的心态，倾听团队新的想法和任务方法。此外，对你的同龄人要积极友好，如果你有不明白的地方，不要害怕提问。

# 从哪里获得成为一名优秀的初级 Java 开发人员的技能？

好了，现在你几乎了解了初级 Java 开发人员的一切，从他们的观点到基本技能。但是，在哪里获得这些技能并获得 Java 开发人员的第一份工作呢？

*   [**代码馆**](https://codegym.cc/) 。这是理想的在线课程，可以帮助你从头开始学习 Java，并为你的第一次面试做好充分准备。该课程包括结构良好的课程，包含 1200 个即时验证任务，甚至还有一个游戏部分，可以让你编写一个完整的游戏。课程由四个 quest(Java 语法、内核、多线程、集合)组成，应该给任何初级开发者一个很好的进一步成长的基础。
*   [**Codewars**](https://www.codewars.com/) 。对于所有程序员来说，这是另一个有用的编码实践平台，允许您以互动、游戏化的方式学习各种编程语言(当然包括 Java)。通过在 kata(小型编码练习)上挑战自己，你将能够提高你的编码技能并改进你的编码风格。Codewars 也有一个强大的社区，你可以随时从你的同伴那里寻求支持。
*   [**Cyber-Dojo**](https://cyber-dojo.org/creator/home) 是又一个练习编程的在线场所。这是一个非常直观的基于网络的环境，不需要订阅。所有你需要进入一个现有的实践或创建一个新的是一个体面的互联网连接和浏览器。
*   [**code chef**](https://www.codechef.com/)**更像是一个国际社区而非线上课程。它允许进行全球编程竞赛，以便来自世界任何角落的初学者和专家可以评估他们的技能和对概念的理解。从 3 小时的短比赛到持续几天的比赛，任何偏好和技能都有完美的比赛。**
*   **[**完整的 Java 大师级**](https://www.udemy.com/course/java-programming-course/) 。是 Udemy 为新手提供的 Java 综合课程。该课程包括 80 小时的宝贵信息，包括示例、练习和测验。有了那个，这个课程性价比很高，不会掏空你的钱包。**
*   **[**Java 编程 250 步完全初学者**](https://www.simplivlearning.com/java/java-programming-for-complete-beginners-in-250-steps) 。顾名思义，非常适合没有编程背景的同学。该课程从头到尾揭示了一切，让您逐渐获得必要的 Java 知识。**
*   **[**杜克大学**](http://coursera.org/specializations/java-programming)Java 认证。这门课程是与杜克大学合作开设的，对于那些倾向于更学术的学习方法的人来说可能是理想的选择。它是为初学者设计的，将教会你核心的 Java 概念，并使你具备足够的知识来编写程序和解决复杂的问题。此外，你将获得任何软件工程师在未来都需要的基本技能。**

# **结论**

**最后，我想强调的是，有很多因素可以增加你找到第一份工作的机会。你应该战略性地集中精力，让自己保持动力，对未来的职业充满热情。如果事情进展顺利，你将有机会在完成课程后的一个月内找到第一份工作！但是，如果花费的时间比你预期的要长，不要担心。好事通常需要时间，你一定会得到回报。所以，今天就开始学习，尽快获得初级 Java 开发人员的职位！**
