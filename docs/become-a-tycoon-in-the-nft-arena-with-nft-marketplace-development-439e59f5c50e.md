Reddit 是一家美国内容评级和社交新闻聚合平台，去年 7 月推出了自己的 NFT 市场。在 4 个月的时间内，超过 300 万用户在平台上创建了一个金库，使 Reddit 成为 web3 竞技场上最大的[***【NFT 市场】***](https://bit.ly/3zPzrW8) 。Reddit 在如此短的时间内获得如此巨大的成功，显示了非功能性测试的价值和需求。随着 Opensea、Rarible 等成功的 NFT 市场在加密市场上取得巨大成功，Reddit 这样的新进入者在如此短的时间内成为轰动一时的热门，这对初露头角的加密企业家来说是一个令人鼓舞的迹象。因此，许多新的企业家开始冒险进入 NFT 市场发展，这是因为他们最大限度地满足了对非正规金融服务和 NFT 市场的需求。

➤ **用户资料:**所有用户，无论是创作者还是买家，都需要在平台上创建一个帐户才能访问。用户购买和出售的资产显示在用户简档上。

➤ **店面:**它是 NFT 市场最重要的特征之一，提供与 NFT 相关的重要信息和描述。店面包含的信息包括 NFTs 列表、价格、所有权、出价、艺术家等等。它还允许用户使用智能合约来检查 NFTs 的真实性。

➤ **虚拟展览:**就像实体展览一样，数字艺术家可以在 NFT 市场上以虚拟展览的形式展示他们的作品。

➤ **数字钱包:**NFT market place 上的集成数字钱包可用于平台上的交易，并存储 NFT 和私钥。

➤ **搜索和过滤:**NFT 市场上的搜索功能帮助用户无缝地找到他们想要的 NFT。通过过滤功能，用户可以通过奖品、所有权、利基等过滤器找到他们想要的 NFT。

➤ **清单:清单**特性使用户能够在平台上无缝地列出 NFT。

➤ **拍卖:**有了拍卖功能，非物质文化遗产的创造者可以通过拍卖出售他们的非物质文化遗产，让感兴趣的买家出价购买。

➤ **推送通知:**通过推送通知，用户可以收到关于新更新、事件、出价等的即时通知。

➤ **智能合约:**智能合约自动化了 NFT 市场上的所有活动，使平台更加直观。
