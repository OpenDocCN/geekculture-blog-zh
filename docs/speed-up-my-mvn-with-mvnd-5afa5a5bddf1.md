## mvnd 是新一代 mvn CLI

我相信我们 Java 开发人员对 maven 的态度总是矛盾的:我们用它来管理项目中的依赖关系，但也为它的速度或 XML 结构冗长的配置所困扰。

尽管如此，由于历史和惯性，仍然不可能切换到其他更快的构建工具。为了解渴，我想向您介绍一个更快的 maven 包装工具`maven-mvnd`。
