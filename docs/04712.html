如何使用React构建电子商务应用程序

原文：https://medium.com/geekculture/how-to-build-an-ecommerce-app-with-react-a040bec4089d?source=collection_archive---------6-----------------------#2021-07-02

在本教程中，我们将使用 ButterCMS 和 Snipcart 构建一个带有 React 的电子商务应用。我们将使用 Chakra UI ，一个模块化和可访问的组件库，来设计应用程序。
最后，我们将使用 Stormkit 部署我们的电子商务应用，Stormkit是一种类似于Vercel或Netlify的JavaScript应用的无服务器部署服务。
