# 为什么计算机专业和非计算机专业的学生应该练习竞争性编程？

> 原文：<https://medium.com/geekculture/why-should-a-cs-and-non-cs-student-practice-competitive-programming-9fa1451fcab3?source=collection_archive---------16----------------------->

![](img/56148af2f550290b631ae803e9f8eb70.png)

在这个高科技蓬勃发展的世界里，保持信息的更新是非常必要的。编码利用了一个创造性的过程，在这个过程中，计算机被“指导”如何执行不同的动作。这是通过代码或一组指令来完成的，计算机需要稍微思考一下编码。你可以开始问自己，如果你知道如何实现的话，你会对一个设备做什么样的改变。

CP 使一个人更有纪律，更快，更专注于批判性思考。它提高了一个人把握现实世界问题的能力，而不是更多地被认为是炒作。在当时，这对于教育系统来说是一个新奇的想法，并没有像现在这样受到重视。随着时间的推移，人们的想法从“编码很酷，所以试试吧”变成了“编码很酷，你可以靠它过上好日子！”。随着时间的推移，随着对 STEM(科学、技术、工程和数学)工作的需求预计在未来几十年内增长，如果你拥有正确的技能，那么会有很多高薪工作。竞争性编程是一项基于逻辑的任务，但同时也可以是一种创造性的出路。许多程序员活着就是为了创造东西的挑战和看到结果的回报。

![](img/4cfaa7e375df10d722351c21684698a5.png)

所以，不要对所涉及的术语或编码太复杂的想法感到恐慌。如果有正确的指导，学习编码会变得有趣和容易。做好行业准备或如何有效地合作，因为一个人必须与团队中的其他人一起完成相同的任务。CP 培养了一种意识，即如何评估团队成员的优势和劣势，并有效地在彼此之间划分责任，从而以完美的结论完成工作。

渐渐地，CP 为你提供了不可或缺的技能，如解决问题的方法，在压力下工作，管理时间和期限，尽量减少错误。要启动你在五大财富公司的职业生涯，学会将自己置于富有挑战性的环境中，将不同的部分和代码片段组合在一起，给生活带来一些东西。作为一名程序员，你拓展你的目标公司，申请你认为你可以胜任的职位，并且出色地完成工作，从而获得你梦想中的收入。它不仅考验一个人的耐心、奉献精神和一致性，而且非常重要的是，它促使你跳出思维定势，找到解决任何问题的最佳方案。

> *程序员犯的最大错误可能是灰心丧气和练习不够。或者只练习他们觉得自己擅长的问题。”*
> 
> **–德里克·基斯曼，又名 SnapDragon**

A 根据深入研究竞争性编程的调查，人们可以遵循一种基于项目的学习方法，在这种方法中，一旦个人知道了语言的语法和基本原理，他/她就可以创建多个具有挑战性的项目，在完成许多项目后，他们将认识到逻辑在开发中是如何工作的，这将有助于提高他们的编程能力。

![](img/ed5837a3e942383048b512fefe15cd73.png)

之后，你可以转向竞争性编程，你会意识到处理竞争性编程的问题现在更容易了，现在你可以做得很好。

*   接近 35%的开发人员认为自己是自学成才的。
*   三分之一的开发人员每天在工作或学校之外编码超过一个小时。
*   大多数科技招聘平台，如 Codeforces、CodinGame for Work 等，都在为基于技能的招聘铺平道路。
*   优秀程序员的高薪工作——无论他们的教育、专业或社会背景如何。
*   一旦你有了最基本的东西，是时候去做更大更好的东西了。
*   请记住，你还没有完全实现梦想，但是你已经准备好在旅程中迈出下一步。
