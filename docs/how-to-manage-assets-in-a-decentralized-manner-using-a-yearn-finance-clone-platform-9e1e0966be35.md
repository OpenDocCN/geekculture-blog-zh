**铁银行—** 在一个渴望金融克隆平台上，加密令牌的借贷变得很容易。因此，投资者可以在 DeFi 平台上存入一定金额作为抵押品。之后，他们可以用这个来借加密货币和 DeFi 代币。随后，买家还可以将虚拟资产转移到铁银行。此外，他们可以赚取更多的加密货币作为回报。通常，它们的总收益取决于以太坊智能合约中编码的条款和条件。

**滑点容忍按钮—** 用户可以保护自己免受极端市场波动的影响。他们可以单击滑动公差选项并固定某个百分比。这可以保护他们在下订单和确认订单时免受价格变化的影响。

他们获得了几个优势，如交易的即时处理，低汽油费，以及成功完成市场和限价订单。一般来说，它们的回报取决于流动性池的起伏和众多交易对的表现。
