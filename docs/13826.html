##将github中超过100MB的文件添加到。gitignore

github的默认设置不允许推送超过100MB的文件，提交和推送文件会比较麻烦。

超过100MB

find . -size +100M -ls

增加

find . -size +100M | sed -e 's/^\.\///' >> .gitignore

移除缓存

git rm -r --cached .
