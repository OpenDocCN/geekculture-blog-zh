Python 中的组合迭代器

`itertools`模块提供了几个组合迭代器。这些函数接受一个或多个输入可迭代对象，并返回一个迭代器，该迭代器提供这些可迭代对象元素的一组特定组合。

该部分的功能包括:

*   `product`
*   `permutations`
