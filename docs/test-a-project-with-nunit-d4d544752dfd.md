NUnit 是一个用于 Microsoft.NET 和 Mono 框架的免费开源单元测试框架。
它提供了几个特性，例如

*   并行运行测试。
*   做出多个断言。
*   支持数据驱动测试。
*   支持多种平台，包括 Xamarin、Compact…
