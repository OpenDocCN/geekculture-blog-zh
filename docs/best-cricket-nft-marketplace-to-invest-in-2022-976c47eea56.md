# 什么是不可兑现的代币？

不可兑现的代币是区块链上代表特定资产所有权的合约。在大多数情况下，这些资产是独一无二的、不可替代的，并且具有某种稀有性。

不可伪造的代币(通常缩写为 NFT)主要代表数字艺术或品牌收藏品或游戏资产。
