# 开发人员为什么使用两个显示器？

> 原文：<https://medium.com/geekculture/why-developers-use-two-monitors-2e21a7537ef9?source=collection_archive---------5----------------------->

开发人员为什么使用双显示器？答案很简单:它们让同时使用多个程序变得更容易。使用第二个监视器让开发者可以选择在第二个屏幕上显示一个程序，而另一个程序在主监视器上打开。这使得不同程序之间更容易共享数据，并节省空间。这不仅有助于您节省空间，还会让您看起来像个电脑高手。
