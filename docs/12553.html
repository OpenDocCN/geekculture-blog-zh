使用NumPy来加速您的代码。

NumPy矢量化在一次调用中对整个数组应用一个函数。例如:

a = np.array([1, 2, 3])
b = np.array([4, 5, 6])
u = np.sqrt(a)  # [1. 1.414 1.732]
v = a + b       # [5 7 9]
