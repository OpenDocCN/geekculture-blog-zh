什么是 [*Deno*](https://deno.land/) (读作 dee-no)？它是由 [Ryan Dahl](https://github.com/ry) *创建的 *JavaScript* 、 *TypeScript* 、和 *WebAssembly* 的现代运行时。*他还通过最初的开发创建和管理了 [*节点、js*](https://nodejs.org/en/) (JavaScript 运行时)。Deno 和 Node 都是基于 V8 JavaScript 引擎构建的。另外，Deno 使用 [*锈*](https://www.rust-lang.org/) 和 [*托基奥*](https://tokio.rs/) 。
