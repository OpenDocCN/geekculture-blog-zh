如果你的安装成功了，你可以到你的终端，简单地输入“go”，按回车键，它会给你一个可以使用的命令列表，这意味着 go 确实安装在你的系统上。

## 什么是包裹？

在文件的第一行，我们声明这个文件属于主包。包基本上是具有特定用途的代码的集合。您的项目总是在主包中启动，这是唯一可以执行的包。但是，主包可以从其他包中导入代码。你可以从互联网上下载其他人创建的软件包，帮助你更快更容易地完成某些事情，例如构建流媒体服务。

Go 在你安装的时候会为你安装基本包，“fmt”包就是其中之一。它所做的是将数据打印到控制台，以及其他一些事情。我们在第二行和 main 函数中导入了“fmt”包，我们使用“fmt”包的“Println”函数将“Hello World”打印到控制台。

你可以从网上下载别人写的包，go 会把它们安装在“go/pkg”文件夹里。

稍后，我们将创建我们自己的单独的包来完成特定的任务，并在我们的主包中导入和使用它们。

## 什么是函数？

函数基本上是做特定事情的代码块，可以根据需要随时调用，并且可以给定参数。

看下面的功能。你不必把它复制到你的代码中，只要看看它做了什么。它只是将两个数字“a”和“b”相加。

```
func addTwoNumbers(a, b int){
   fmt.Println(a + b)
}
```

您用“func”关键字声明一个函数，后跟函数名，在本例中是“addTwoNumbers”，因为它接受两个数字并将它们相加。在括号中，您声明参数，这些参数是实值(参数)的占位符，您可以在调用函数时赋予它，参数可以是您希望的任何类型。您必须声明这些参数的类型，在本例中是 int。

在函数体内，我们返回 a + b 的值。

让我们“调用”这个函数:

```
addTwoNumbers(7, 21)
```

这将在您的控制台上打印 28。

例如，如果您希望函数返回一个值，然后您可以将该值用于不同的函数，则必须在函数名和参数后声明要返回的类型:

```
func addTwoNumbers(a, b int) int{
   return a + b
}
```

所以，你可以通过写下函数名并在括号内给它你想要的值来调用你的函数。

## 变量

编程中的变量类似于数学中的变量:它将一个值存储在一个关联的名称中，以便您可以使用该名称来引用该值。它很有用，因为如果您经常使用该变量，您可以通过简单地更改一行代码来快速更改其值。

您可以像这样在 Go 中声明一个变量:

```
var a = 10
```

Go 现在知道你的变量叫“a”，值是 10，类型是 int。但是什么是 int 呢？

在计算机科学中，有某些值类型。Int 代表整数，如 10、2 或 19032。基本上没有小数点的数字。

小数点为 3.29、9.1 或 948.2124 的数字称为浮点数。在围棋中，有 float32 和 float64 两种类型。这指的是所使用的位。Float 32 编译起来更快，但是 float64 存储浮点数要精确得多。

另一种值类型叫做 string，基本上就是文本的意思。字符串的值放在引号中。所以声明一个字符串变量看起来像这样:

```
var name = "Alejandro"
```

如果你真的很懒，你可以使用简写版本来声明和初始化一个变量，如下所示:

```
name := "Alejandro"
```

它做的事情和上面一样，但是你不需要 var 关键字。

## 创建您自己的包

到目前为止，我们只声明了主包——我们的代码从这里开始——并导入了“fmt”包来将内容打印到控制台。但是我们也可以创建我们自己的包来做一些事情并包含一些函数，然后我们可以将这些函数导入到我们的主包中。

我们将创建一个新的包，它包含一个根据立方体的边来计算立方体质量的函数。快速提醒:公式很简单:

```
side * side * side
```

在您的“测试项目”文件夹中，为您的新包创建一个名为“mathcalc”的新文件夹。在这个文件夹中，创建一个名为“masscube.go”的文件。在这个文件中，声明包名为“masscube”。包名应该始终与其所在的文件夹名相同，并且一个文件夹中只能有一个包。如果你试图在一个文件夹中创建几个包，Go 会报错。接下来，创建一个名为“MassCube”的函数，它接受一个 int 类型的参数“side ”,并返回一个 int 值。在函数内部，您只需返回立方体质量的公式:

你应该意识到的一件重要的事情是，如果你在一个包里写一个函数或者一个变量，以大写或者小写字母开始，这很重要。区别在于以大写字母开头的值是导出值，这意味着您可以在包外使用它们。在我们的例子中，因为我们将在主包中使用 mathcalc 包中的“MassCube”函数，所以我们需要用大写字母来写它。以小写字母开头的值是未导出的值，您只能在声明它们的包中使用它们。

```
package mathcalc func MassCube(side int) int { return side * side * side}
```

由于 Go 安装包来自互联网的“go/pkg”文件夹，它会自动寻找当你导入一个包。

为了告诉 Go 在哪里可以找到您刚刚创建的包，您必须创建一个模块。在您的终端中，导航到您的项目目录并执行以下命令:

```
go mod init test-project
```

现在您可以使用刚刚创建的包了。

再次打开 main.go 文件，通过写入“name your project/name your package”导入“mathcalc”包，并删除 main 函数中的内容，因为我们不再需要它了:

```
package mainimport "test-project/mathcalc"func main() {
}
```

在 main 函数中，您现在可以调用 Println 函数中的 MassCube 函数，方法是写下包名、点号，然后写下函数名:

```
package mainimport "test-project/mathcalc"func main() {
    fmt.Println(mathcalc.MassCube(3))
}
```

转到您的终端，再次运行代码。它将打印出 27，这是 3 乘以 3 乘以 3(记住:函数 Masscube 刚刚计算了一个值的 3 次方—值*值*值)。

包是很棒的，因为你可以用这种方式把你的代码分成更小的部分，并且保持你的项目结构化。此外，它们还使得从项目外部访问您编写的代码成为可能。
