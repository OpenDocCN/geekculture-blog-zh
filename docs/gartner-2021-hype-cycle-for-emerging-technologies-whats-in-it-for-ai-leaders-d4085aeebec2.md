# Gartner 2021 新兴技术炒作周期。人工智能领导者有什么好处？

## Gartner 的 2021 年新兴技术炒作周期已经结束，因此这是一个深入研究该报告并思考我们作为公司的人工智能战略的好时机。你可以在这里找到完整报告[的简要总结](https://www.gartner.com/smarterwithgartner/3-themes-surface-in-the-2021-hype-cycle-for-emerging-technologies)。

像每年一样，让我们从一些假设开始，每个人在解释这个炒作周期时都应该知道，特别是在将周期的图形表示与过去几年进行比较时:

*   炒作周期中出现的所有技术都被认为处于成熟的早期阶段。
*   如果某项特定的技术没有被重点介绍，并不一定意味着它们不会产生重大影响。这可能意味着完全相反。一些技术从炒作周期中消失的一个原因可能是，它们不再是“新兴”的，而是足够成熟，足以成为业务和 IT 的关键，已经证明了其积极的影响。正如我们将在本文后面看到的，这些技术中的一些在特定的炒作周期中有所涉及。

对于 2021 年，Gartner 将新兴技术分为三个主题，它们在整个宣传周期中使用。这些主题与**建立信任、增长和变革**相关。我真的认为这三个主题非常符合每个人工智能领导者应该拥有的首要任务:通过创造可靠、道德、透明和可解释的(**信任**)产品、服务或流程变化，交付对业务产生深远影响的大规模人工智能解决方案(**增长**)。

## 与信任相关的人工智能新兴技术

*   **同态加密**是一种加密形式，允许对数据执行计算操作，而无需先解密。对于人工智能驱动的公司来说，这为通过共享数据鼓励数据驱动的经济以及通过能够在不损害隐私的情况下纳入外部数据在其算法中获得更准确的结果打开了大门。[来自 BSC](https://www.bsc.es/news/bsc-news/bsc-executes-the-first-time-big-encrypted-neural-networks-using-intel-optane-persistent-memory-and) (巴塞罗那超级计算中心)等一级机构的最新研究成果打开了将这种技术应用于大型加密神经网络的大门。

## 与增长相关的人工智能新兴技术

*   **人工智能驱动的创新**是指利用人工智能创造产品和服务。虽然 Gartner 将其归入增长类别，但在我看来，它与这三者有关。通过人工智能进行创新需要变革和信任，确保底层人工智能技术能够带来成果，并证明这些成果能够影响公司的 P & L。此外，从道德和法律角度来看，新的人工智能驱动的产品和服务必须是值得信赖的。根据我的经验，人工智能驱动的创新计划的成功取决于端到端的业务和数据技术方法:

1.  恰当地构建要解决的商业机会，探索社会和市场趋势以及与深入理解消费者驱动因素和竞争框架相关的现有服务。
2.  将业务问题转化为数据问题。在这个阶段，通过全面的数据图确定数据源并决定要遵循的算法策略是很重要的。
3.  为我们正在定义的产品/服务定义正确的 UX/用户界面。
4.  定义管理服务的组织结构，定义大规模交付服务的角色、职责和任务。

*   **量子毫升。**虽然量子计算及其在 ML 中的应用被大肆宣传，但即使是 Gartner 也承认，目前还没有明确的证据表明在机器学习中使用量子计算技术有所改善。该领域的真正进步将需要通过同时从两个角度研究该问题来弥合当前量子硬件和 ML 之间的差距:设计能够最好地实现新的有前途的机器学习算法的量子硬件。这一领域的挑战之一是找到合适的人才，他们在机器学习和量子硬件设计和实现方面拥有跨学科知识。就主流采用而言，Gartner 将 Quantum ML 定位在十年以上的时间框架内。
*   **生成式人工智能**非常简单地说，就是一套算法，可以生成类似于用来训练它们的数据。 [OpenAI 在 2021 年宣布了两个多模态神经网络，包括 WALL-E](https://towardsdatascience.com/openais-dall-e-and-clip-101-a-brief-introduction-3a4367280d4e) ，这有助于促进生殖人工智能的普及。虽然这种人工智能的创造性用途背后有很多宣传，但它也为未来其他相关研究领域打开了大门，例如药物发现。从社会角度来看，生殖人工智能也提出了重大挑战，正如 OpenAI 在他们的博客中提到的[:他们“计划分析像 DALL E 这样的模型如何与社会问题相关联[…]，模型输出中偏见的可能性，以及这项技术隐含的长期伦理挑战。俗话说，一张图片胜过千言万语，我们应该非常认真地对待像这样的工具在未来如何影响错误信息的传播。](https://openai.com/blog/dall-e/)

## 与变革相关的人工智能新兴技术

*   **人工智能增强的设计**和**人工智能增强的软件工程**都与生成性人工智能以及人工智能对计算机前工作的影响有关，特别是软件开发和网页设计。由于 GPT-X 或 OpenAI 的 Codex 等算法的发布，我们看到了围绕这两项技术的大量宣传，这些算法适合像 GitHub 的 Copilot 这样的解决方案。2021 年 6 月中旬，OpenAI 的首席执行官萨姆·奥尔特曼(Sam Altman)发布了一条推文，他在推文中声称，人工智能将对电脑前的工作产生更大的影响，比现实世界中的工作快得多:

Sam Altman’s tweet about AI disruption in computer-assisted work

一如既往，这些技术并非没有挑战。从他们可能在一些低级编码和 UX 任务中造成的破坏，到训练这些人工智能算法可能带来的[法律影响](https://www.technollama.co.uk/is-githubs-copilot-potentially-infringing-copyright)。

*   **物理通知型人工智能**是一种不仅从数字训练数据中学习，而且能够适应物理环境的人工智能。虽然人工智能越来越擅长解决数字世界中的问题，但现实世界的互动带来了更大的挑战，需要结合实时传感和与环境的互动，我们可以期待在这一领域的大量投资。

作为对这些人工智能新兴技术的补充， **Gartner 的人工智能宣传周期，2021** 优先级矩阵使我们能够列出一系列人工智能技术，根据 Gartner 的说法，这些技术将在未来 2-5 年内产生变革性影响:

*   复合人工智能
*   计算机视觉
*   决策智能
*   深度学习
*   边缘人工智能
*   生成人工智能
*   以人为中心的人工智能
*   智能应用
*   机器学习

有趣的是，Gartner 并未将 NLP 和 Transformer 技术包括在此列表中，预计在 5-10 年内不会产生变革性影响。这种延迟的原因有很多，包括少数民族语言的 [NLP 算法的开发](https://towardsdatascience.com/how-can-nlp-impact-the-future-of-minority-languages-555b0fc80bd0)或该算法面临的[伦理问题和偏见](https://www.theverge.com/21346343/gpt-3-explainer-openai-examples-errors-agi-potential)。

最后，有趣的是看到社会挑战如何成为人工智能新兴技术被采用的关键。随着负责任的人工智能变得越来越受欢迎，我预计这种趋势将在未来继续增长，正如 Gartner 自己指出的那样，将它作为创新触发器纳入其 Gartner 的人工智能宣传周期，2021 年。
