作为一名开发人员意味着要在终端上花费大量的时间，尤其是基于 Linux 的终端。你可能害怕这个最小的黑色文本窗口，但是它是你成为一个高效开发人员的最有力的盟友。

终端(也叫控制台，也叫命令提示符)随处可见。您会发现它们被集成到您的代码编辑器中，如 Visual Studio 代码，或者作为 GitHub 或 AWS 等服务的 CLI(命令行界面)。

与拖动鼠标指针在 GUI 上点击相比，能够在终端中随意发出命令可以节省您更多的时间和精力。有趣的是，开始并不那么令人畏惧。

让我们看看必须知道的命令…

# 1.了解你和“残疾人”的关系

`pwd`命令代表*当前工作目录。它告诉你你在文件系统中的哪个文件夹/目录。把这想象成你在谷歌地图上的— *查找我的位置*服务。下面是它的使用方法:只需在终端中的光标处键入`pwd`,按回车键，它就会输出当前目录的路径。*

```
$ pwd
```

# 2.用“ls”列出文件和目录

`ls`是一个非常强大的命令，可以列出一个目录的所有内容。您可以使用不同的标志来选择要显示的信息。

当您在没有任何选项的情况下使用它时，它会显示您所在的当前目录中所有未隐藏文件和目录的名称。

```
$ ls
```

添加`-l`标志和`ls`将会以表格的形式列出你当前目录中所有未隐藏的内容

```
$ ls -l
```

现在，如果我们想显示所有隐藏的文件和目录，我们使用`-a`标志。如果我们将它与`-l`标志一起使用，它会以漂亮的表格格式列出所有内容，包括隐藏的内容

*我强烈推荐您使用这种格式的* `*ls*` *命令，来列出所有目录的内容。*

```
$ls -al
```

> 在 Linux/mac 中，隐藏的文件和目录名称以`.`开头，例如`.my-secret-directory`是一个隐藏的目录。

# 3.用“cd”改变目录

`cd`代表更改目录。你可能已经猜到了，它帮助我们在目录中移动。搭个便车！

当您使用不带任何标志的 cd 时，它会将您带到您的主目录/根目录。例如，如果我在我的 mac 上尝试这个命令，它会把我带到我的主目录`/Users/shashank`

```
$ cd
```

要进入一个特定的目录，你可以使用`cd`后跟目录名

```
$ cd my-directory
```

您还可以进行多次跳转，并使用路径进入嵌套在另一个目录(子目录)中的目录。例如，如果我想跳转到我的项目文件夹，我会这样做

```
$ cd shashank/projects
```

# 4.懂点“.”和双点“..”

单点`.`是当前目录的别名，双点`..`是父目录的别名。

因此，如果你做了一个`$ cd .`，你仍然会留在当前目录。然而，单个点别名对其他命令非常有用，比如`open`

```
$ open .
```

这将在当前目录下启动默认的文件浏览器。如果您在 mac 上，这将启动 finder，显示当前目录的内容。

现在如果你使用

```
$ cd ..
```

这将带你到一个更高的级别或你当前目录的父目录。

> 是的，当前目录和父目录的别名`*.*`和`*..*`也被隐藏。当您使用带有`*-a*`标志的`*ls*`时，您会看到它们被列出

# 5.用“mkdir”创建目录

`mkdir`代表制作目录。要创建一个目录，使用带有目录名的`mkdir`

```
$ mkdir my-new-directory
```

您也可以使用长路径来创建目录，只要该路径存在，直到新目录存在。例如，您可以像这样创建`my-new directroy`:

```
$ mkdir /users/shashank/my-new-directory
```

如果目录结构 `/users/shashank`已经存在，这将起作用。

# 6.克里特岛一个新的文件与“触摸”

`touch`让您快速创建一个空文件或多个文件。当您需要创建没有任何内容的文件来测试日志等时，此命令非常有用。要使用它，您可以使用`touch`，后跟带扩展名的文件名。

```
$ touch log.txt dump.csv 
```

# 7.用“>”和“> >”将内容添加到文件

`>`允许您覆盖内容并将其推入文件。例如，您可以将内容从`file1`推送到`file2`。如果文件 2 已经有任何内容，它的内容将被覆盖

```
$ file1 > file2
```

但是，如果您想将内容附加到现有文件而不覆盖，请使用`>>`

```
$ file1 >> file2
```

# 8.用“cat”查看文件内容

在终端上显示文件内容的最快方法是使用`cat`命令。类似于`touch`你可以提供多个文件名。

```
$ cat file1 file2
```

# 9.用“mv”重命名或移动

`mv`用于重命名文件/目录或将文件/目录从一个地方移动到另一个地方

移动文件

```
$ mv file1 my-directory
```

要重命名目录或文件，请提供现有的文件/目录名，后跟新的文件名/目录名。

```
$ file1 newfile
```

> 如果第二个参数是一个新文件夹，`mv`命令将第一个参数中的文件/目录移动到第二个参数中的文件夹。否则，它将重命名该文件。

# 10.删除带有“rm”和“rmdir”的文件和文件夹

你可以使用`rm`命令删除一个文件。

```
$ rm filename
```

如果文件有写保护，在删除之前会提示您确认。如果它不受保护，该命令会继续删除该文件

使用`rmdir`命令删除一个空文件夹

```
$ rmdir my-directory
```

*危险区域:*使用带有标志`r`和`f`的`rm`命令删除包含文件的文件夹

```
$ rm -rf my-directory
```

# 11.使用通配符的力量

*通配符*(也称为元字符)是代表其他字符的符号或特殊字符。您可以将它们与任何命令一起使用，如`ls`或`rm`来列出或删除符合给定标准的文件。

*   星号`(*)`–匹配一个或多个出现的任何字符，包括无字符。
*   问号`(?)`–代表或匹配任何字符的一个出现。
*   方括号内的字符`([ ])`–匹配方括号内出现的任何字符。可以使用不同类型的字符(字母数字字符):数字、字母、其他特殊字符等。

这里我们列出了所有扩展名为`.txt`的文件

```
$ ls -al *.txt
```

以下示例匹配名称以`l`开头、后跟任意单个字符并以`st.sh`(后缀)结尾的所有文件。

```
$ ls l?st.sh
```

下面我们匹配所有文件名以`l`开头，后跟方括号中的任何字符，但以`st.sh`结尾的文件。

```
$ ls l[abdcio]st.sh
```

> 注意:在使用通配符`rm`或`rmdir`时要格外小心。这可能是灾难性的

# 10.用“清除”清理终端窗口

有了所有的命令和输入，终端很快变得混乱不堪。`clear`言出必行；它清理屏幕，并把你的光标移到顶部，这样你就有了一个新的干净的区域来启动更多的命令！

```
$ clear
```

> 当您使用 clear 命令时，并没有全部丢失，如果您想回顾以前的命令和输出，只需向上滚动，它就在那里。

# 11.使用“man”命令列出任何命令的手册

使用`man`命令打印您可以在终端中运行的任何命令的完整手册。它给出了使用选项、标志、错误代码等细节。*拿不准的时候* `*man*` *吧！*

这里 man 命令将为`ls`命令打印整个手册

```
$ man ls
```

# **奖励:理解 TAB 自动完成和箭头键历史的力量**

这些不是命令，而是减少终端输入的捷径。

键入文件名或目录名时，使用 TAB 键进行自动完成。您可以键入单个字符并按 TAB 键。您可以在以字符开头的所有选项之间循环

此外，您可以使用上箭头键和/或下箭头键在最近使用的命令的历史记录中循环。
