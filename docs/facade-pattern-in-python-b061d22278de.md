设计模式:外观

大家好，欢迎阅读另一篇关于结构设计模式的文章。在本文中，我们将回顾 facade 模式，并看看它的 Python 实现。让我们先简单介绍一下。

什么是门面模式？门面模式是一种技术，其目的是将系统的复杂性“隐藏”在一个简单且大多数时间用户友好的应用程序编程接口(API)之后。您可以将 facade 模式视为在复杂的底层系统之上实现的抽象层。
