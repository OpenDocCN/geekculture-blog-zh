# 什么是函数式编程？

> 原文：<https://medium.com/geekculture/what-is-functional-programming-679f9a92e19d?source=collection_archive---------17----------------------->

函数式编程的核心思想是减少对多态和循环的依赖，而是使用高阶函数进行抽象，如映射、过滤和减少迭代函数。

Swift 是一种多范式语言，这意味着它可以符合一种以上的范式，如面向对象、函数式、命令式和块结构语言。
