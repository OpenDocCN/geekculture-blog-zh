## 何时应该使用它们

![](img/172deeb631d5c6283914f3d2ac41e280.png)

假设你拥有一家酒店。你要确保房间空着的时候没有灯亮着。你告诉你的客人:请在离开房间时关灯。但是他们总是忘记这一点。

所以，你雇了一个负责这个的经理。一旦这位经理发现这个房间不再被使用，他/她就会关灯…
