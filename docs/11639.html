蕾哈娜的内衣品牌Savage X Fenty考虑上市

原文：https://medium.com/geekculture/rihannas-lingerie-brand-savage-x-fenty-considers-an-ipo-bf7723d6a69c?source=collection_archive---------17-----------------------#2022-04-01

蕾哈娜已经从一名强大的音乐人转变为一股创业力量。她的包容性内衣品牌Savage X Fenty正在考虑进行30亿美元的首次公开募股(IPO)。

该品牌一直在快速发展，旨在颠覆内衣行业，重新定义美容标准。这种商业模式也相当不受疫情的影响。鉴于该公司的受欢迎程度，首次公开募股可能是筹集资本和推动进一步增长的战略举措。

建国故事

2018年，蕾哈娜推出了Savage X Fenty作为线上专属品牌。亲密服装品牌销售内衣，内衣，胸罩，睡衣和家居服。该公司努力设计适合各种背景和身材的包容性服装。它的产品以合理的价格出售。

由于她的商业努力，蕾哈娜已经成为亿万富翁。她的大部分财富都与她在2017年创办的化妆品公司 Fenty Beauty 的成功有关。她的剩余财富来自萨维奇·芬蒂，以及她的音乐和演艺事业。

野蛮X Fenty在时尚界推广包容性

促进信心和包容性是Savage X Fenty价值观的核心。该公司的网站和时装秀以代表不同种族背景和体型的多样化模特为特色。

蕾哈娜在公司网站上的一份声明中指出:“野蛮X意味着为你自己制定规则，表达你的情绪、性格和风格——而不是为别人。”

维多利亚的秘密的衰落为Savage X Fenty抢占市场份额创造了机会。维多利亚的秘密的市场份额从2016年的80%左右下降到2019年的65%。在被批评该品牌宣扬不切实际的美容标准后，这家内衣零售商已采取措施彻底改变其形象。维多利亚的秘密的模特大多属于高瘦型，具有欧洲中心特征。尽管努力重塑品牌，但像Savage X Fenty这样的新玩家有望成为全球领导者。

根据市场估计，该公司的定位是到2025年成为全球内衣行业的领跑者。蕾哈娜的公司处于有利地位，可以在3250亿美元的市场中占据相当大的份额。作为增长战略的一部分，Savage X Fenty计划在美国各地开设实体店。

虽然像维多利亚的秘密这样的老牌内衣公司一直在努力维持消费者基础，蕾哈娜的品牌却实现了200%的年同比增长。此外，其VIP会员计划的增长较前一年激增了150%。

融资轮次

蕾哈娜的内衣品牌迄今为止已经筹集了3.1亿美元的资金。2022年2月，该公司在由纽约投资管理公司Neuberger Berman牵头的C轮融资中筹集了1.25亿美元。C轮的其他投资者包括专注于消费行业的私募股权公司L Catterton和Jay-Z的Marcy Venture Partners。

最新一轮融资将有助于推动该公司的零售主导战略。该公司于2022年1月在拉斯维加斯的时装秀购物中心开设了第一家实体店。新的资金也将支持该公司的国际扩张努力和推出新的产品线。2021年2月，Savage X Fenty在B轮融资中融资1.15亿美元。

野蛮X Fenty 领导团队

两位女性联合总裁娜塔莉·古兹曼(Natalie Guzman)和克里斯蒂娜·潘达维斯(Christiane Pendarvis)领导着这家公司。两位女性都在零售销售、数字商务和产品开发方面拥有丰富的商业经验。

首次公开募股

Savage X Fenty正在与包括高盛(Goldman Sachs)和摩根士丹利(Morgan Stanley)在内的金融顾问就首次公开募股进行谈判，其估值可能在30亿美元以上。

尽管2021年的IPO创纪录，但2022年的IPO活动有所放缓。随着利率上升、通胀压力和地缘政治不稳定，IPO趋势降温。在IPO表现强劲的一年之后，投资者可能也会感到疲劳。

Savage X Fenty将不得不监控市场状况，以确定推进IPO是否有商业意义。目前，该公司正在与顾问进行讨论，并采取准备措施。由于市场波动，Savage X Fenty尚未就IPO时间做出最终决定。
