在艺术收藏品的引领下，NFT 市场在 2021 年出现了意想不到的牵引力。到 2021 年底，由于多种原因，NFT 市场的销量下降。自今年年初以来，NFT 生态系统一直在经历潮起潮落——然而，收藏家们已经向 NFT 市场发送了近[](https://blog.chainalysis.com/reports/chainalysis-web3-report-preview-nfts/)**370 亿美元(截至 2022 年 5 月)——而 2021 年的平均总额为 400 亿美元。**

**全球的创作者/艺术家已经在可收藏的 NFT 上花费了数十亿美元；然而，他们也花了一大笔钱在“汽油费”上，以处理和验证他们的交易。不幸的是，这种气体费惹恼了创作者社区，他们希望摆脱它。因此，一些企业家期待 [**创造一个没有汽油的 NFT 市场**](https://www.appdupe.com/nft-marketplace-development) 。**

# **什么是 NFT 燃气费？**

**用户必须支付燃气费来支付在区块链上进行交易所需的计算资源的成本。这类似于当你想在账户间转账或支付账单时，信用卡可能收取的手续费。**

**将数字文件制作成数字资产并存放在区块链上以供交易使用被称为铸造 NFT。在以太坊上铸造一个 NFT 需要矿工进行资源密集型计算，就像其他交易一样。天然气费是支付矿工协助区块链交易记录工作的一种方式。计算 NFT 值有助于估算天然气成本。**

# **创作者在制作 NFT 系列时面临的挑战**

**虽然已经受欢迎的数字艺术家继续接受非功能性数字技术，但疫情造成的财务枯竭的艺术家群体推动了 NFT 空间。艺术家和创作者在制作他们的 NFT 系列时，意识到他们在汽油费方面的必然处境。但是，并不是每个艺术家都能负担得起“昂贵”的汽油费来创作、买卖 NFT。**

## **不是所有的 NFT 都能卖到六位数！**

**不管公众怎么想，非功能性食物通常不会卖到六位数。大多数以几百美元的适中价格出售，许多甚至可能不会被出售。鉴于你必须支付天然气费来生产和销售你的 NFT，你可能会经历财务损失，而不是收益。**

## **汽油费不决定资产价值**

**天然气价格与数字资产的绝对价值无关，在某些情况下，甚至可能比待售资产更贵。对于那些尚未成名的后起之秀来说，这尤其具有挑战性。**

**想象一下，一位艺术家被迫支付“昂贵”的汽油费来铸造他的第一辆 NFT。这位艺术家只有两个选择:一个是通过给 NFT 定价过高来收回油费，另一个是通过给 NFT 定价过低来牺牲利润——无论哪种方式，艺术家都会遭受损失。**

## **避免汽油费**

**如果你冒险进入 NFTs 的世界，你会很快发现汽油费是不可避免的。但是有没有可能减少你必须付的油费呢？遵循以下步骤来避免支付油费或至少尽量减少油费:**

*   **如果你是艺术家/创作者，明智地选择区块链网络。以以太坊为例；除了是一个受欢迎的连锁店，它还收取“昂贵”的汽油费。总是列出低燃气费网络，并从中选择理想的。**
*   **确保你选择的 NFT 市场致力于将天然气费用降至最低——就像 OpenSea 如何将其 [**迁移到海港协议**](https://www.theverge.com/2022/6/14/23166774/opensea-seaport-portocol-update-switch-gas-fees) 。**
*   **计划你的造币时间——因为汽油费可能会高得不合理——导致汽油价格波动。延迟铸造是一个很好的选择，它允许创造者在购买时铸造他们的 NFT。**
