# 结论

以下是我们之前所学内容的回顾

*   `*args`和`**kwargs`是一种奇怪的语法，可以用来将参数输入更改为我们想要的任何内容，而无需声明它
*   `*args`为非字典变量
*   `**kwargs`是字典变量
*   要点在单星(*)或双星(**)内，不在名称 args 或 kwargs 内
