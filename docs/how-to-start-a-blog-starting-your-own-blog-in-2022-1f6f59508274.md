# 决定你的博客利基和博客名称。

这是开始写博客的第一步，也是非常重要的一步。决定博客的利基是非常重要的一点，它可能会决定你的博客的成功。

然而，请记住，有些人写博客是作为一种爱好，没有赚钱的意图，而有些人写博客的唯一目的是赚钱。

如果你真的想用你的博客赚钱，我强烈建议你在你的博客利基上认真思考。

# 选择博客利基的小贴士

*   想想与你的兴趣或爱好相关的小众。你会更容易写出和自己兴趣相关的文章。
*   不要选择太具体或太广泛的利基。过于具体的小众只会针对一小部分受众。一个太广泛的利基将导致网站排名困难。
*   我建议你研究一下博客的最佳位置。
*   一些顶尖的小生分别是，“**金融**”、“**赚钱**”、“**美食和食谱**”、“**自我发展**”、“**投资**”、“**健康**”等等。
*   当然，你可以选择一个你一无所知的利基市场，因为你总能了解它。走出你的舒适区。

最后一个重要的提示是，不要急于选择你的定位。慢慢来。

# 选择博客名字的小贴士

选择一个博客名字很简单，要么选择一个通用的名字，要么选择一个个性化的名字。如果你喜欢隐私，不想让别人知道你，那就用一个普通的名字。

否则，选择一个人名，它不必是你的真名，选择一个介于两者之间的名字。然而，一个个人博客名字因为真实性而吸引更多的读者。

# 如何开始一个博客:为你的博客选择虚拟主机和域名。

当你决定了你的博客利基和博客名称。是时候为你的博客选择虚拟主机了。

如果你以前从未听说过虚拟主机，请允许我解释一下。

# 什么是虚拟主机？

**网络托管**是一种允许在计算机系统上托管/发布网络服务器应用程序(网站或网页)的服务，通过该服务，网络浏览器客户端可以轻松访问互联网上的电子内容。([【geeksforgeeks.org】T4](https://www.geeksforgeeks.org/))

简单来说，虚拟主机就是你从一家公司租了一个空间，然后你用这个空间来存储你的博客/网站，然后全世界的人都可以看到或阅读。

如果没有虚拟主机，你将无法轻松地向全世界发布你的博客。

那么 domain 呢？

# 那么域是什么呢？

好了，现在你已经从虚拟主机公司租了一个空间，用户可以通过访问它的 IP 地址来浏览你的网站，就像 66.249.64.25 那样。请记住，你不是唯一一个租房的人，还有数百万人也在这样做。因此，每个空间通过 IP 地址作为唯一标识来标识是合乎逻辑的。

但是，想象一下，像 66.249.64.25 一样把你的 IP 地址分享给你的读者，你的读者永远不会记得它，而且它看起来是假的，不值得信任。

这就是为什么我们有一个东西叫做领域。“域名是一个你可以购买的名字，它可以作为你网站的地址”。

例如，当你的浏览者想去你的博客，而不是在浏览器的地址栏写**66.249.64.25**，用户可以写`**rasyue.com**`，然后就会去你的博客。

注意，你通常会买一个和你的博客名一模一样的域名。

我会推荐浏览 [**NameCheap**](https://rasyue.com/namecheap-review/) (联盟链接)，因为他们为域名提供最便宜的价格。
