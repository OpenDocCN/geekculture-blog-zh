变量:

变量是从这里到那里变化的值。这是我能给出的最简单的定义。✊在编码中，程序员有时需要一些可变的值，他们称之为变量。但它还有另一个优点。一个变量也可以存储一个值，改变这个值&再次存储改变后的值。

JavaScript中的变量是用let, const , var这些关键字声明的。let 用于声明块级变量，这意味着该变量只能在块内被访问&改变，但var 的作用就像一个全局变量，这意味着任何人都可以从任何地方访问该值。const是常量的简称，意思是不可改变。有些值在数学中是常数(比如PI = 3.1416)。在JavaScript中，编码人员使用const 来声明一个值常量&，这意味着该值不可更改。示例:

let a = 5;

a = 15;

console.log(a)给出15，然而const a = 5; a = 10;会给你一个错误，告诉你不能给const操作数赋值。为了解释let & var的区别，我需要使用函数，所以请配合我，好吗？

var a = 5; function timeTable(){a = 10;} a = 35这里我们已经声明&初始化了我们的变量a为var，所以在函数外的最后一个语句后&的值会改变。

在同样的例子中，如果我们使用let而不是var，我们会得到同样的结果，但是如果我们像这样在函数中使用let

function timeTable(){ let a = 10;} a = 100;那么该值将不会改变，相反它会给出一个错误，导致在函数a之外不被识别

循环:

名字本身就说明了它的作用。有时程序员需要重复做一些改变&循环是他们做的过程。有不同种类的循环，如for while & do-while但同样也有一些for循环的变体，如forEach() for()Of for(In )我将只给出for循环的例子。

假设你要打印出my name is Bill Gates 1000次，你会怎么做？写console.log() 1000遍？绝对不行！因此，您使用一个简单的for循环来重复console.log() 1000次。

for(let i = 0; i < 1000; i++) {console.log(‘my name is Bill Gates')}

看到了吗？你就写一行，打印1000行！神奇！😶
