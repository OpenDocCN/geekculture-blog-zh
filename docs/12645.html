3。DESO

DESO以前被称为Bitclout。它不同于其他选择。

一旦你的股票上线，用户就可以投资你。

如果他们这样做，你可以奖励他们额外的好东西。你也可以向用户提供你未来收入的一部分。每个用户头上有一个价格，显示他们的美元价值。

结论

长久以来，像谷歌和脸书这样的大企业一直主宰着互联网，从我们每次上网时自由放弃的数据中获利巨大——这是一种被称为web 2.0的商业模式。

Web 3.0，通常被称为Web 3，基于加密货币、NFT和区块链的透明性质，通过再次民主化互联网和相关的赚钱潜力来挑战保守派的统治地位。
