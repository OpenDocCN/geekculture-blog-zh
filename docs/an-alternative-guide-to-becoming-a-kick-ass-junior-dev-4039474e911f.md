## 找到一份你想要的工作其实很难

![](img/e6f074826295de226333b4128a1a3f80.png)

Photo by [Hello I'm Nik](https://unsplash.com/@helloimnik?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

进入科技行业有太多惊人的理由，而科技吸引了太多不同类型的人。但是，大多数人有一个共同点，就是从底层做起。我是说入门级的。对一些人来说，这可能是质量保证或测试；对其他人来说，这可能意味着实习变成了工作。不管怎样，我们大多数人都不是从埃隆·马斯克的得力助手开始的，这没关系。

或者，如果你有幸阅读了一些入门级的工作描述，你可能会在他们的要求列表中遇到一些:

*   多任务独角兽
*   开发者忍者
*   愿意努力工作(曾经有一个选择是柔和地工作？)
*   29 年的颤振经验(我也要有时光机吗？)
*   以前的 CTO 经验是一个加号(好吧 jk 这不是一个入门级的工作，但这个角色肯定不是一个高级角色)
*   对区块链的理解(你的商业模式甚至不会从区块链模式中受益。)

这个清单可能是无穷无尽的。众所周知，这些要求只是招聘经理的“愿望清单”或梦想。但为什么他们梦想着神奇的可能性，并以奶酪棒和经验作为回报呢？这里有一个断开点。与其告诉你“更新你的简历”或“人际关系网”，不如让我提供一些可供选择的、可能令人痛苦的建议。我们开始吧。

# 1.)你还不特别……

我们大多数人都没有超级富有的父母或有独特关系的家庭成员，这意味着大多数人从未听说过你，不知道你的存在。如果你把这一点和作为一个新手和缺乏技能结合起来，这是一个灾难的配方，让你的简历被扔进垃圾桶。

然而，并不是所有的希望都落空了；事实上，你可以利用这一点。每个人都是从初学者开始，随着你的学习，你将有机会向其他初学者教授和讲话。你不必做辅导或演讲，而是传播你获得的知识。让内容，甚至是不好的内容，关于你正在学习的东西。每次你传播你的知识，你都有机会认识新的人并影响他人。通过这种影响，你开始积累一些有趣的技能。

# 2.)你以前做过这个…对吧？

当你申请入门工作时，你通常会和几十个甚至几百个申请人竞争同一个职位。是什么让你与众不同？就像重新申请大学一样。如果你没有一份拥有大量工作经验的出色简历来支持自己，你能拿出什么社会证明呢？你的 GitHub 上有项目吗？你参加过黑客马拉松吗？你是本地编码社区的积极参与者吗？

忽略编码面试的所有污点，很难确定一个候选人是否有编程能力。拥有社会证明对提升自己大有帮助。无论社交证明是人、GitHub、机器人竞赛的奖杯，还是你妈妈的食谱组织者，拥有一些你可以详细谈论的东西，而不是计算器，都会有很大帮助。

# 3.)你必须为…免费工作

我不是告诉你去实习或找一份没有报酬的工作。我告诉你，如果有证据证明你能做某事，人们会更容易相信你能做到。这可能意味着周末免费工作，创建项目来充实你的投资组合。它可能是与一个真正需要一个网站的当地慈善机构合作。也许你的弟弟需要帮助破解他的游戏来打败他的朋友。不管你选择什么样的道路，成为一名开发者都需要你学习很多东西。学习的每一刻并不总是有回报的。如果你能接受你需要免费工作，建立社会证明是轻而易举的事。

# 4.)你确实知道如何投球…对吗？

说说你对鲨鱼池和风险投资家的看法；他们知道如何抓住要点。没有人说没有比一个小的孩子和风险投资加速器基金更快。拒绝的主要原因之一，没有独特的价值主张。如果我面前有 100 个应聘者，他们都一样牛逼，我为什么要选你？说“*因为我需要一份工作*”不是一个可以接受的答案。虽然这可能是真的，但它并没有赢得任何奖项。

在这一部分，我变得非常敏感，让你想想你想去哪里工作，以及你为什么想去那里工作。科技是一个艰难的领域。你会有一些漫漫长夜；你可能会被要求随叫随到。你可能会让实习生删除你的整个代码库。知道你为什么想去某个地方有助于你度过那些艰难的时光；如果你在某个地方工作的原因纯粹是自私的，那也没关系；拥有它。

另一方面，从战略角度考虑你想在哪里工作有助于你打造独特的价值主张。如果你是一名前教师，你可能会成为一名优秀的技术培训师或团队领导。你也可以利用这段经历，寻找专注于教育的科技公司。哲学专业的学生可以成为伟大的软件架构师，因为他们习惯于非常抽象和逻辑的思考。发挥创造力。这是学习如何投球的一部分。

# 5.)你已经在…底层了

![](img/2cdcc89d2ceda6c95ae54c7c0086cd69.png)

Photo by [Artem Kniaz](https://unsplash.com/@artem_kniaz?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

这可能会让你感到震惊，但就你的技术生涯而言，你已经处于谷底。你不可能成为比现在更大的 noob，这很好。所以在你的学习和成长中要 100%不要脸。你会失去什么？所有人都知道你很烂。放下自我，接受失败，你会学到更多。老实说，假装你知道事情是很累的。我向你挑战，这周就一次，说，“我不知道。”盯着某人的眼睛说出来。它是自由的。

# 最后一句话&额外提示

成为海扁王并不容易。这将需要漫长的夜晚，巨大的压力，甚至一些崩溃，但我保证这是值得的。你不会因为赢得一个奖杯或别人授予你头衔而变得很了不起。你变得很厉害是因为你在游戏中加入了一些东西，并且相信你是。

最后一点:成为一名开发人员是一份工作，但工作并不代表你是谁。
