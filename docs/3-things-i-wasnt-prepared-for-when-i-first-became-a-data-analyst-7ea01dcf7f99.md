## 关于如何最好地准备数据工作的重要提示

![](img/2310dd7b294de5826b796859dd146c5b.png)

Photo by [Ben White](https://unsplash.com/@benwhitephotography?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

在开始从事数据分析师职业之前，我看了很多关于**数据分析师整天做什么**的文章。我的结论？数据分析师是那些在暗室中工作的人，他们试图理解不断增长的数据量，以解决业务问题。他们拥有出色的数据分析技能，非常注重细节。

一旦我开始在这个领域工作，我的看法就改变了。我意识到，优秀的数据分析师应该具备更广泛的技能，而这些技能在行业外很难培养。这就是为什么我决定分享**关于如何最好地准备数据分析师角色的 3 个技巧**。我想强调一下这份工作中其他文章似乎忽略了的方面。所以，事不宜迟，这里有 3 个技巧可以帮你找到一份数据方面的工作。

## **1。** **尽力收集你需要的数据**

![](img/42ae14559f2436f72d51206a673abf20.png)

Photo by [Agence Olloweb](https://unsplash.com/es/@olloweb?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

收集数据来解决业务问题并不容易。事实上，作为一名数据分析师，您可能会花大部分时间与利益相关者接触，找出他们面临的问题，并探索哪种数据可用于解决它。

如果您曾经参加过数据分析师的在线课程，您可能已经注意到，开始分析所需要的只是下载一个数据集。在现实生活中，很难获得你需要的数据。数据可能不可用，或者不适合您的需要。即使数据*可用并且符合目的，你也必须努力说服数据所有者允许你访问。当公司不是以促进安全数据共享的方式建立时，尤其如此。数据被**【孤岛】**在不同的部门。*

知道如何找到你需要的数据是一种来自实践的技能。收集正确的数据并把它们结合在一起并不总是可能的，但是在大多数情况下**这是可以管理的**。所以下一次，在你进入一个新的数据集之前，问问你自己:“*在现实生活中，我如何获得这些数据？我能从知名组织或开源数据库中获得它吗？数据的可靠程度如何？与我的任务相关吗？*”。通过这样做，您将获得数据分析师每天面临的一些挑战的第一手经验。

💡**专业提示**:如果你正在为你的下一个项目寻找开放数据集，我建议使用免费资源，如[谷歌数据集搜索](https://datasetsearch.research.google.com)、[全球健康观察](https://www.who.int/data/gho)、[地球数据](https://earthdata.nasa.gov)、[皮尤研究](https://www.pewresearch.org/internet/datasets)和[美国人口普查](https://www.census.gov/data.html)。你将能够找到从商业到金融、健康和科学的一切数据。或者，您可以使用 API 或 web 抓取工具从任何网站提取数据。

## **2。** **与利益相关者一起回答业务问题**

![](img/d1488b9aa12b97188d55c583f4d3a19f.png)

Photo by [Dylan Gillis](https://unsplash.com/@dylandgillis?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

数据分析师最重要的技能之一是能够与利益相关者合作，并利用他们的**领域知识**来解决业务问题。大多数时候，你的利益相关者没有数据或分析背景。然而，他们将非常熟悉其业务的复杂性和细微差别。他们将能够提供关于你试图解决的问题的有用信息。

我记得曾为一位客户工作，他试图理解客户流失背后的原因。我们同意使用 NPS(或净启动子得分)作为我们分析的一部分。NPS 是一项常用于衡量客户忠诚度的指标，通过向客户提出一个问题来计算:“*您向朋友推荐我们的产品/服务的可能性有多大？*”。

我后来发现，NPS 调查是在购买完成后不久发出的。换句话说，在收集反馈时，我们正在调查一组很少或根本没有与品牌进行有意义互动的客户。这是一种被称为“抽样偏差”的偏差，可能会扭曲我们的结果。如果我没有在整个项目中与我的客户进行持续的讨论，我们的分析会导致一个非常不同的结果。

数据存在于收集和使用它的环境中。它抓住了企业的细节。这就是为什么你应该和你的利益相关者坐下来一起讨论这个问题。您将了解如何利用他们的领域专业知识从数据中获得洞察力。现在，这是*而不是*说你的技术能力不重要。然而，知道如何操作数据只是数据分析师的一小部分。真正的价值是你对商业问题产生的背景的理解。

💡为了让你养成团队解决问题的习惯，我建议你每两到三个月参加一次黑客马拉松。黑客马拉松是让人们一起解决现实生活中的商业问题的好方法。你将学会如何从不同的角度看待问题，创造性地思考潜在的解决方案。

## **3。** **掌握你的沟通技巧**

![](img/60f919df698f78c58881b72e089d78b8.png)

Photo by [Brooke Cagle](https://unsplash.com/@brookecagle?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

你可能会认为数据分析师是独立于公司其他部门工作的人。那与事实相差甚远。如前所述，一个优秀的数据分析师会在早期与利益相关者接触，了解他们的目标，并推荐实现目标的最佳方式。知道如何与利益相关者**交谈和** **有效沟通**将为你在职业生涯中取得成功奠定基础。

好的沟通者首先是好的倾听者。每个利益相关者都有不同的兴趣、观点和你需要解决的问题。能够就特定的行动方针向利益相关者提供建议，并清楚地展示对他们有什么好处，这是这项工作的一个关键方面。如果你的见解不可操作或不被理解，你的分析将是浪费每个人的时间。

你并不总是能够交付你的利益相关者所追求的一切。你需要知道什么是可能的，什么是不可能的。我们往往会系统性地低估完成一项任务所需的时间，以及与该任务相关的风险。这就是所谓的“规划谬误”。能够向利益相关者指出潜在的陷阱将有助于你设定现实的目标和管理期望。

在某种程度上，数据分析师的角色是一个客户服务工作。花时间了解利益相关者的需求和关注。然后在分析过程中不断迭代。确保你的发现在项目的每个阶段都能被你的听众理解并与之相关。您与利益相关者建立的关系类型可以成就或破坏一个项目。

💡为了提高你的沟通技巧，我建议你每天至少花 15 分钟阅读或写作。你会提高你的知识基础、清晰思考的能力和词汇量。所有这些方面都会让你在工作中成为一个更好的沟通者。

## 结论

作为一名数据分析师开始可能会令人生畏，但这是一个令人着迷的职业。数据分析师的大部分工作是在数据的不确定性中导航，并与利益相关者合作解决业务需求。为了增加你获得数据分析师工作的机会，遵循 3 个简单的提示:

1.  使用开放数据集为您的分析查找正确的数据
2.  参加黑客马拉松，学习如何团队解决问题
3.  练习阅读和写作来提高你的沟通技巧
