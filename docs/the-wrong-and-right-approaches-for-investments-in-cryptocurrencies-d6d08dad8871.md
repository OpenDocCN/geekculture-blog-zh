# 错误的方式(不耐烦)

## 加密货币投资不是赌博

我们以这种情况为例。你会在社交媒体上看到一个关于特定硬币的巨大趋势。

你盲目地进入是因为大肆宣传，内心强烈地害怕错过。你甚至没有注意到，仅仅一周前，硬币的价格比现在低得多，你买得太高了。

下周，价格会下降一点，第二周，价格会更低。你想卖，但你是负的。

这是你能做的最糟糕的事情。现在不卖！你需要耐心。你最好坚持住！反正也没什么损失。长期持有！也许有一天价格会再次上涨，你就有机会获利了。

# 短期投资和交易(例外)

在我看来，只有两种情况下你应该进行短期投资:

*   当牛市接近顶峰时
*   当你交易时

当你接近牛市的顶峰时，你知道价格会继续增长，直到到达顶峰，你有时间平仓。当你交易时，它是类似的东西，但在多个周期中，所以你可以重复这样做。

在任何其他情况下，急躁只能让你赔钱，我会解释为什么。

加密货币的走势图有起伏。有时它像过山车一样快速下降，有时又缓慢但稳定地上升。这只是两种可能的情况。

我的意思是，在某些时期，某种加密货币的大持有者正在购买，在这种情况下，价格上涨，这是他们出售的时候，因为从他们购买的那一刻到现在，价格已经上涨到足以获得丰厚的利润。

无论你是投资还是交易，你都需要意识到这些周期。否则，你会做错的。

# 正确的方法(耐心)

## 买入并持有

为了做好这件事，你需要在每次价格大幅下跌时尽可能多的买入，并长期持有，我指的是长期持有。

不要每次看到价格下降就恐慌卖出。聪明点，坚持几个月，或者几年。那时候你就会看到结果了！

我在这里告诉你的一切纯粹是个人意见，而不是财务建议。然而，我有一些投资加密和交易加密资产的经验。

你可以听我的，也可以不听。这是你的决定。这是一个通常奏效的好策略。

请记住，将你的钱投入任何加密货币都是一项高风险的活动，你不应该花费超过你承受得起的损失。
