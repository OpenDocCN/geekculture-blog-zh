## 深度学习:实现多标签文本分类器

## 训练多标签分类器时要避免的陷阱。

*   在传统的分类问题公式中，类是互斥的，即每个训练样本只属于**到**一个**类。**
*   一个**二元分类**问题涉及将数据集分类为**两个**可能的类别，例如:垃圾邮件对非垃圾邮件、猫对狗、流失预测等。
