# 关于 Neuralink Brainchip，我们现在需要知道的就是！

> 原文：<https://medium.com/geekculture/all-we-need-to-know-about-neuralink-brainchip-till-now-57f8b68dfc6?source=collection_archive---------14----------------------->

Neuralink 实验涉及两个植入猴子运动皮层的大脑芯片。运动皮层负责手和手臂的运动，Neuralink 大脑芯片记录了猴子大脑中神经元的活动。然后，研究人员使用计算机算法来预测手和手臂的运动，猴子最终学会了在不使用操纵杆的情况下玩视频游戏。在人类身上进行类似的实验可能是必要的。但是程序是什么呢？
