数据科学中的变量类型！

总体而言，数据科学领域存在4种类型的变量，具体如下:

  1. 数字的
  2. 绝对的
  3. 日期时间
  4. 混合的

在数据科学中，了解变量的类型对于选择相应的处理技术非常重要。

现在，也就是说，让我们继续解释每一种类型的变量。

数字变量

这是一类只处理数字的变量。该类别可进一步分为2个子类别。

  1. 分离的
  2. 连续的

根据数值变量的值，它可以细分为离散变量或连续变量。

离散数值变量

这是只包含一个离散量的变量类别，也就是整数。此类别中不允许有分数。

这一类别的几个例子有:

  1. 整数或自然数
  2. 你拥有的笔记本电脑的数量，(这里你肯定会有一个整数作为答案，没有人能说它有1.1台笔记本电脑，它明显会是1、2、3等。)
  3. 你有多少件衬衫。
  4. 你拥有的房子数量。
  5. 你拥有的车辆数量。
  6. 家庭中孩子的数量。
  7. 家里宠物的数量。
  8. 一个人拥有的银行账户数量。💰
  9. 一个人拥有的关系的数量。😳💕

还有更多…

连续数值变量

这是处理 连续量或分数量的数值变量的类别。

这一类的几个例子有:

  1. 您的后付费手机账单金额。(一般不是整数，像500.50等)
  2. 观看一部网络连续剧的总时间(120.5秒等)
  3. 网上订餐的总花费(1500.57印度卢比等)
  4. 贷款利率。
  5. 个人体重。(它并不总是一个整数，通常是90.5公斤，等等)

还有更多…

这些变量属于数值变量的范畴。

分类变量

这是处理类别的变量的类别。又分为2个子类。

  1. 序数
  2. 名义上的

根据分类变量的值，它可以细分为名词性变量或序数变量。

有序分类变量

那些与有关联顺序 的变量被称为有序分类变量或归类为有序分类变量。这个顺序是在一个特定变量的值中被观察到的。

这一类别的几个例子有:

  1. 一周中的几天(这里有一个特定的顺序= >星期一总是在星期二之前，以此类推……)
  2. 任何最受欢迎的公司的特定汽车系列，如奥迪等。在这里，汽车型号与特定的价格相关联，因此基于汽车的型号，我们可以按照价格等级的升序或降序对汽车系列进行分类。
  3. 教育学位(博士学位总是在硕士学位之后，等等……)
  4. 大学取得的成绩。
  5. 创建移动电话的步骤。(这个过程肯定有不能改变的有序步骤。)

还有更多…

名义分类变量

那些与没有任何关联顺序的变量被称为名义分类变量或被归类为名义分类变量。

这类的几个例子有:

  1. 国家名称
  2. 城市名称
  3. 性/性别
  4. 操作系统
  5. 显示器类型(视网膜、IPS、AMOLED等)

还有更多…

这些变量属于分类变量的范畴。

注意(洞察):很少有人知道分类变量也可以包含数字，它们并不总是包含字符串。例如，你乘坐的飞机或火车的等级。可以有头等舱，二等舱等。AC在列车中的位置，而不同的类别，如普通的经济、商业等可以首先被编码&，然后它们将按数字顺序排列&仍然是分类变量(准确地说是有序分类变量)。

日期时间变量

这一类变量处理日期&时间方面。此类别可以包含下面提到的值类型:

  1. 只是在约会。
  2. 只有时间。
  3. 有日期和时间的。

这一类的几个例子有:

  1. 出生年月日
  2. 登机时间
  3. 时间戳
  4. 生成系统日志的时间。
  5. 申请日期。
  6. 毕业日期。
  7. 在线订购产品的日期。

还有更多…

这些变量属于日期时间变量的类别。

混合变量

这类变量处理一个特定变量的多个观察值的集合。该类别还可分为以下两个不同的类别:

  1. 不同观测中的数字或标签/字符串。
  2. 同一观察中的数字&标签/字符串。

类别1的例子:

  • 一个人拥有的信用卡数量(这个变量的值对于某些观察可以是一个数字，对于某些其他观察由于任何原因也可以是“未知的”，因此，它包括不同观察的数字和字符串)。
  • 学生的表现(这里有人可以提到CGPA、百分比或分数，包括数字和标签/字符串)。

还有更多…

类别2的例子:

  • 客舱号(A1、C3、E5等)
  • 车辆登记号码

还有更多…

这些变量属于混合变量的范畴。

很好地理解这些变量的类型和子类型，可以为这些变量提供选择处理技术的选项，并且基于不同类型的变量，可以从中获得不同类型的见解。
