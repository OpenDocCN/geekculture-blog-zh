# 黑客的搜索引擎——OSINT-SPY

OSINT-SPY 是一个工具，帮助执行 OSINT 扫描几个在线资源，并检查电子邮件，域，IP 地址和组织的信息。Infosec 研究人员、数据挖掘人员、渗透测试人员和网络犯罪调查人员可以使用它来查找有关其目标的深入信息。
