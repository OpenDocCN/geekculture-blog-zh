# WhatsApp 聊天机器人在帮助非政府组织方面的力量

根据非政府组织运作的国家，WhatsApp 可能是一个尚未开发的通信和数据来源。在发展中国家，与关键行为者沟通可能很困难。存在技术和语言障碍，收集信息可能很困难。但除此之外，收集到的数据经常被闲置，而这些数据本可以用于改善收集数据的国家的状况。以下是聊天机器人如何被用来解决这两个问题。

在地面上收集信息需要大量资源。部署聊天机器人是非政府组织通过调查、回答问题和与当地居民交谈来收集信息的一种经济有效的方式。聊天机器人甚至可以被开发成能够识别人的语言并自动以同样的方式回应。想象一下现场收集信息的翻译和人员所节省的成本。

一旦聊天机器人收集了数据，挖掘的时机就成熟了。事实上，[最近的一项研究](https://health-policy-systems.biomedcentral.com/articles/10.1186/s12961-020-00577-x)深入研究了非政府组织在卫生政策和系统中收集的数据，发现这些数据可以用作主要的数据来源，以便对研究不足和边缘化的群体进行研究。在受冲突影响的地区和发展中国家，学术界很难开展研究，但通过与非政府组织发展伙伴关系来改善数据收集，研究工作可以开始对边缘化人口产生真正的影响。

如果在研究和数据收集中使用聊天机器人，对来自发展中国家的数据进行分析将会容易得多。聊天机器人可以被设计成结合数据分析和信息仪表板，以定制所需的研究。在 WhatsApp 上建立聊天机器人，这在一些发展中国家被大量使用，范围甚至会更大，允许研究工作和卫生政策的改进被导向真正需要的地方。
