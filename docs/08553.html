你能解决这个真正的谷歌面试问题吗？

设计一个迭代器，除了支持hasNext和next操作之外，还支持现有迭代器上的peek操作。

实现PeekingIterator类:

  • PeekingIterator(Iterator<int> nums)用给定的整数迭代器iterator初始化对象。
  • int next()返回数组中的下一个元素，并将指针移动到下一个元素。
  • boolean hasNext()如果数组中还有元素，返回true。
