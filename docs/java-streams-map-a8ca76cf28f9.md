## Java 中流和函数式编程的第一步

从 Java 8 开始，我们获得了一个非凡的工具，它允许我们将**函数式编程**引入 Java。

函数式编程是一种基于应用和组合**函数**的**范式**，被视为一种数据类型:它们可以被赋予名称(**标识符)**并作为**参数**传递，甚至以模块化的方式组合。
