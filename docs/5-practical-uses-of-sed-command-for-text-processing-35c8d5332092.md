## 如何在 Linux 或 Unix 中轻松使用 SED

![](img/14784fdff87cf7fe6ee1a5b36292c435.png)

Photo by: Torsten Dederichs on unsplash.com

SED 代表流编辑器。这是一个 Unix 文本处理实用程序，通常用于解析和文本替换。SED 被认为是基于命令的、面向行的编辑器，它接受文件输入，也支持正则表达式。
