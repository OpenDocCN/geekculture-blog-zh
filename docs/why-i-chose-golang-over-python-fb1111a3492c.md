# 为什么我选择 Golang 而不是 Python？

> 原文：<https://medium.com/geekculture/why-i-chose-golang-over-python-fb1111a3492c?source=collection_archive---------6----------------------->

我使用 Python 已经有 4 年了，虽然不是日常使用，但是 Python 已经帮助我创建了很多网站、后端应用程序、web 服务甚至微服务。

Python 实际上是我学习的第一门编程语言。到目前为止，我可以诚实地说 Python 是我学过的最简单的编程语言。

你可以用 Python 构建几乎任何类型的软件或应用程序，而且永远不会出错。互联网上有大量关于 Python 的文章可以帮助你。
