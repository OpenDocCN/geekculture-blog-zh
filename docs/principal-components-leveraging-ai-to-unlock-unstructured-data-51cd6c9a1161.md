## 视频、音频、图像和文本都可以成为丰富的见解来源，但它们也很难分析。新工具提供了对其中信息的更容易的访问。

你知道在电视犯罪节目中，从非常模糊的图像或视频中提取清晰易读的信息似乎是一件神奇的事情吗？即使对于困难的数据形式，我们也越来越接近这种理解水平。在本周的 Data Science Mixer 播客中，我们与商业运营副总裁崔佛·琼斯和人工智能认知引擎高级总监罗比布斯在 [Veritone](https://www.veritone.com/) 进行了交谈，Veritone 的 aiWARE 为处理音频、图像和视频数据提供了“分层认知”。

以下是我们对话中的三个“主要组成部分”，它们提供了思考和分析非结构化数据的新方法。

# **交互记录为丰富的分析提供了多层次的潜在洞察力。**

**Trevor:** 我们最近探索的一个用例是关于交互分析的。如今，通过 Zoom 和 Hangouts 以及所有这些东西产生了如此多的内容，尤其是自从 COVID 以来。有很多数据没有被挖掘出来。

我们真正看到的用例之一是交互分析。这真正的意思是，如果你有一个人与人之间的对话，我如何能查询这些录音？一些应用程序可能是销售支持培训的销售电话，或者询问我的销售代表是否有效。它可能包括远程医疗:我的医生是否提供了良好的体验？或者呼叫中心:我的呼叫中心代理是否遵守我们的政策，他们是否提供了良好的体验？无论是原生使用我们的平台，还是与 Alteryx 一起工作，您都可以获得仪表板，向您显示:他们在谈论什么？有什么感悟？有计算机视觉方面要搞清楚:面部表情是什么？用面部识别打电话的人是谁？有内容、实体抽取和主题抽取。我们可以在一个解决方案中使用如此多样的人工智能引擎。

# **计算机视觉工具可用于从电力到广告的各种应用。**

**Robbie:** 在电力行业，他们正在使用无人机——例如，像 DJI 无人机那样，机载有 NVIDIA Jetson 设备——以及真正强大的计算机视觉和物体检测技术和模型。暴风雨过后，你可以用它们来检查电线，这样工作人员就不必手动爬上每一根电线杆来确认一切正常。这些类型的场景——能够获取大量的视觉数据，并对哪些数据存在需要人工检查的异常进行分析——可以节省大量的时间。

**特雷弗:**我们与在体育赛事直播上投放媒体的广告商合作。你并不总是知道摄像机会在哪里，或者它们朝向哪个方向。你想试着衡量你赞助的广告价值。我们为体育赛事摄取媒体，并通过徽标检测来运行它。它识别它在屏幕上的位置，它的角度，以及它在屏幕上的突出程度。我们编制了一个综合指数，基本上等同于价值，然后我们将它与[其他指标]进行匹配，无论是谷歌分析的印象还是订单。我们把这归因于实际的印象。

# **不同种类的数据科学工具将继续发展，并赋予不同用户应对数据问题的能力。**

**Robbie:** 看看其他行业，在这些行业中，人们赋予科学家或其他人不需要高水平切入点的工具。如果你想到视频游戏的发展和游戏引擎的兴起，像 Unity，Unreal Engine，以及诸如此类的东西，这种差异真的令人惊讶。我开始做电子游戏生意，我们有 150 个人，花了 5 年时间。很痛苦。然后，我们开始玩转通用引擎的概念。

当我在视频游戏领域的任期结束时，你许可了你的引擎，并且你的团队的大部分人真的在那个脚本层或者拖放层工作。所以突然之间，你的团队没有了 150 名工程师和几个象征性的艺术家。你基本上有设计师。你有创造体验的人。你在授权给创意人员。我认为我们的工具和其他类似工具的目标，不可避免地会出现，是让你可以运行任何类型的认知，而不必编写任何代码来支持它。你可以在云中这样做，并选择你的环境看起来像什么，所以如果你需要在 GPU 上运行它，你可以，只是让它超级简单。我认为五年后的事情看起来真的很令人兴奋。
