在本文中，我们将了解如何在[散点图](https://pythoninformer.com/python-libraries/matplotlib/scatter-plots/)中对数据进行直线拟合。

作为散点图的一个简单例子，对于一组人，我们可以记录他们的身高和鞋码。散点图将由每个人的一个点组成，表示他们的身高(x)和鞋码(y)。
