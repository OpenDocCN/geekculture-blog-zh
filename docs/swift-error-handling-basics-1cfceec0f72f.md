## 小心对待错误。

![](img/7e2defaf876b82d18b278e36d6b61d28.png)

在 Swift 中，错误处理意味着对程序中的错误情况做出反应。

函数既可以自己解决错误，也可以将错误返回给调用者。在后一种情况下，调用者必须采取措施正确处理错误。
