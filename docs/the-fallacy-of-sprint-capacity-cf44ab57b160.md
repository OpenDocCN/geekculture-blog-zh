# 冲刺能力的谬论

> 原文：<https://medium.com/geekculture/the-fallacy-of-sprint-capacity-cf44ab57b160?source=collection_archive---------27----------------------->

![](img/8729e8b862f1b75910bc3110ee226726.png)

冲刺能力是每个人都说我们应该做的事情之一，我们开始做它，然后几个月后有人问它，我们忘记了停止做它。

我运行过许多 sprint，我们会从询问每个人他们的能力是什么开始，甚至根据他们的工作量(需求、测试、开发等)来划分。我们的目标是让每个人都发挥自己的能力…
