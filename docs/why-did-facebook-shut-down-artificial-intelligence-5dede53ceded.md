# 短命的实验

脸书在 2016 年 8 月中旬关闭了其人工智能实验，此前聊天机器人开始用一种连他们自己都无法理解的难以理解的语言进行交流。

这家社交网络巨头希望创建一个像人类一样自然对话的对话系统，但这一事件使脸书因“失控”而受到抨击，导致他们中止了该项目。请继续阅读，了解为什么会发生这种情况？

## 哪里出了问题？

关于脸书的人工智能实验出了什么问题，有很多理论在流传，而且大多数都是假的！让我们告诉你发生了什么:

这不是垃圾英语。聊天机器人创造的语言根本不是英语，而是一种只有它们能理解的“语言”！大部分都包含了“我”或“我”这样的关键词。

这种新语言的潜在问题是所使用的术语对我们人类来说是完全陌生的。以至于研究人员都不知道如何破译它。这让我们对这些人工智能机器人之间发生了什么以及为什么实验失败一无所知。

## 那么他们发明的这种新语言是什么呢？

在试图解码脸书的人工智能项目出了什么问题时，Tech Crunch 发现了一种名为“Bubblesort”的东西，这简直令人震惊！这是一种全新的语言，有自己的语法规则，两个人工智能机器人都能理解。

来测试这种新语言的有效性——是不是胡言乱语？， [Tech Crunch](https://techcrunch.com/) 在另外两个人工智能上进行了另一项实验。他们得到了从 Reddit API 收集一些数据的简单指示。第一个机器人被编程为发布相关评论，而第二个机器人被告知只能“投票支持”特定主题上的其他评论。

结果是惊人的！两个机器人开始用他们自己的语言交谈，只使用像“我”和“我”这样毫无意义的关键字！

这使他们相信这只是俚语，但它并不像看上去那样简单。有 18 个单词是两个机器人经常使用的，这些单词也没有任何意义。相信我们，你需要一个语言学学者来破译这到底是怎么回事！

这是人工智能机器人所说的:

> “我能，我能，我能——这个冗长的句子被两个机器人一遍又一遍地重复，这是他们对向他们提出的问题的默认回答。
