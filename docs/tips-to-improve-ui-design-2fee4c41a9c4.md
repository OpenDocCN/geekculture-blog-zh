UI(用户界面)设计是创建用户界面的外观和感觉，它是软件或网站的一部分，用户可以看到并与之交互。一个设计良好的 UI 可以让一个糟糕或平庸的产品看起来不错，帮助用户更有效地导航和使用系统。
