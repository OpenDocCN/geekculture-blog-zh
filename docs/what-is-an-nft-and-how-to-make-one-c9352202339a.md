## 2021 年 NFTs 的创造者指南

NFT 是真实世界资产的数字表示。NFT 可以代表任何东西，比如一条推文、一张照片、一件艺术品或一件实物资产。NFT 可以是图像、GIF、视频、音乐、一段文字、一条推文或任何类型的文件。NFT 还可以代表所有权，同时也可以像其他类型的所有权记录(如护照或驾驶执照)一样作为所有权的证明。
