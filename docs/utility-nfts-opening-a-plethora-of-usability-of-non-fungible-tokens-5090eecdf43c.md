# NFT 仅仅是装饰吗？

除非你一直生活在岩石和洞穴下，否则你应该对 NFTs 了如指掌。但是，对于所有的新手来说，不可替代的代币给了区块链上的资产一个独占的所有权(也许就像证书一样)。

所以你有一个数字资产，这是数字艺术，摄影，视频音乐，甚至是一条推文。你可以把这个文件包装成区块链的 NFT，炫耀一番。但是如果同一个数字文件有多个副本呢？网上流传着许多蒙娜丽莎的复制品，但只有一幅原画。你可能会说这是一幅画——一种实物资产，但当涉及到数字文件时，你甚至不能拿着它。对吗？

NFTs 确实给了艺术家、独立内容创作者和自由职业者一个很好的机会来将他们的资产货币化。但是，除了在社交媒体上炫耀之外，它给那些非功能性交易的持有者或所有者提供了什么呢？事实上，它给了品牌和企业使用这些内容的权利。还有什么？

# **NFT 公用事业的类型**

如前所述，NFT 不仅仅是为了艺术家、设计师和音乐家的工作赚钱而设计的。自从 NFTs 开始得到公众的认可，用例开始渗透，拉动世界上每一个可能的行业。从体育到音乐，每个行业都开始以自己的方式使用 NFTs。下面给出了一些可以在 NFTs 中实现的实用程序，还有一些正在实现中。

NFT 房地产- 伦敦有 11 间卧室的豪宅汉普顿大厅有一个 NFT 的数字复制品。这座价值 5500 万美元的豪宅有一个 NFT 复制品，可以给拥有它的买家一个虚拟之旅。这为 NFT 房地产打开了大门。

**赛事门票-** 每张赛事门票都必须独一无二，绝不可伪造。所以 NFT 进入这里，不给偷窃和伪造门票留下空间。这已经进入市场，为公众提供了享受无缝售票体验的机会。

**身份识别和认证-** 身份盗窃一直是世界各国政府面临的一个问题。尽管在不同的国家有各种不同的唯一身份系统，但是通过 NFTs 在区块链上拥有一个 ID 可以是全球性的。在组织中，它仍然可以在小范围内替代员工 ID 或在区块链上证明他们。

**供应链和物流-** 区块链和 NFTs 开放透明。将 NFTs 作为供应链和物流中商品的唯一标识，将有助于提高产品在哪里以及存放了多长时间的透明度。这将进一步提高产品的可靠性和价值。

不仅仅是以上四种，区块链的爱好者们已经为 NFTs 提出了各种各样的实用程序。因此，总之，非功能性测试不仅仅是装饰性的；他们在未来拥有大量的用例，这使他们成为区块链的无敌技术。
