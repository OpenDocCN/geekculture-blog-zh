# 商业和数据分析远程学习理学硕士的价值？

> 原文：<https://medium.com/geekculture/the-value-of-a-distance-remote-learning-msc-in-business-and-data-analytics-9e8ecf99336b?source=collection_archive---------17----------------------->

## 鉴于世界的现状——既关注数据分析，又因不可预见的 COVID 锁定而无法旅行——能够在没有这些阻碍的情况下继续寻求理学硕士学位具有巨大的价值。

![](img/46e381e5af91596ccf6a0bca9075f60d.png)

[Uni of Bath Program](https://www.bath.ac.uk/courses/postgraduate-2021/taught-postgraduate-courses/msc-business-analytics/)

数据分析的研究是近来的热门话题。我们的世界已经经历了多轮工业进化。最近，为了保持竞争力，每个企业都需要在日常工作中采用技术。今天，由于每个企业都是技术生态系统的一部分，公司可以从内部和外部获得大量数据来提取对自己有利的信息。由于这种转变，构建数据处理管道和利用可视化工具所需的技能正迅速成为每个人工具包中的一部分。此外，随着使技术方面变得更容易的平台变得更加广泛，越来越多的非工程角色专注于数据收集和分析。

这就是理学硕士派上用场的地方，但在做全职工作的同时攻读理学硕士值得吗？

学习期间继续全职工作有很多好处。它让你有一个更经济的方式来支付你的成本，而不需要贷款，并且对你的职业发展有很大的帮助，因为你不需要牺牲任何工作成果来追求你的学术目标。你可以通过在理学硕士课程中学到的技能以及在当前工作中取得的成就来提升你的职业生涯。

# 我的故事

在过去的两年里，我一直在攻读巴斯大学(University of Bath)的商业分析硕士学位(Masters of Business Analytics)。虽然还有一年的时间，但这个项目已经成为一次有用且富有成效的学习经历，在我的日常工作中获得了回报，并且没有对我在工作中为自己设定的目标产生负面影响。这在一定程度上要归功于课程的延长，这使得教学模块不会重叠。

当我在寻找其他人经历类似项目的信息时，我找不到许多具体的答案来回答我的问题。我将尝试在这里提供这些问题的答案。我希望这将让其他人在决定是否继续这一旅程或更加重视备选方案的决策过程中有更顺畅的体验。

# 谁应该做一个远程理学硕士？

那些:

*   无法负担一年或更长时间的学习，或者不想/负担不起贷款或首付的人
*   不想暂停自己的职业生涯去提高技能，或者对学术经验不感兴趣(亲临大学)

甚至通过免费的在线课程(或者更便宜的课程)也有可能提高技能。因此，如果你对获得资格本身感兴趣，就去攻读硕士学位吧(也许你是为了公司的标准而攻读硕士学位，或者是为了让自己符合未来博士学位的标准)。

我渴望攻读正式硕士学位的一个重要原因不仅仅是知识和资格，还因为我把自己锁在一个很难打破的承诺中。一旦这个项目开始了，我欠我自己和我花的钱去充分利用它。

这种心理方面有助于学习可能容易被忽视的新技能。是的，知识已经在互联网上了，学术界之外也有顾问(免费的和付费的)。但是不管你是否对这个主题有热情，如果没有什么东西推动你去做的话，你很难开始接触新的主题。

# 如果我有家庭或其他义务，甚至超出了日常工作，该怎么办？

如果你有家庭，你最好去参加一个兼职项目，这样你可以从工作中抽出时间来上课、做作业和准备考试。这样，你就不必在日常工作、与家人和朋友互动以及攻读硕士学位之间疲于奔命。

在可能的情况下，将理学硕士学位延长两年以上可以让你一次花时间在一个课程/模块上。这使得事情变得非常容易处理，取决于你个人生活的约束。

# 它和学徒期相比如何？

[学徒制](https://www.gov.uk/topic/further-education-skills/apprenticeships)通常是为资历较浅的人设计的，或者是为那些已经加入这类项目的公司工作的人设计的。这些公司通常将学徒融入到日常工作中，这样他们就可以利用正在学习的人(并且[可以获得税收减免](https://www.gov.uk/guidance/pay-apprenticeship-levy))。这可能是进入一家公司的好方法，但也可以从完全不相关的背景中提升技能。

在某些情况下，学徒可能比理学硕士更容易获得，但不一定是同一级别的资格。

这就是你要做出决定的地方，是获得硕士资格重要还是仅仅获得知识重要。还有其他获取知识的方式，有些是免费的，有些是额外收费的，但仍然更加灵活，成本也更低。

# 我应该什么时候开始？

如果你问自己这个问题，答案很可能是:现在。

你越年轻，承担的责任越少，我相信你就越适合在线课程，因为它们会在你的日常工作之外给你增加更多的负担。

此外，考虑你是否对你的默认生产力和工作稳定性有信心。虽然在线课程非常灵活，可以延长很多年，但有时会感觉像一种负担，干扰你的个人生活。

如果兼职和全职硕士适合你的生活方式，请充分考虑这些选择。在线选择并不意味着更容易。事实上，这可能会持续很多年，可能会变得相当有压力。

# 工作量会对我的职业产生负面影响吗？

![](img/1eeb1d3004b9ea58693fdf0ee716d65e.png)

正如前面提到的，仔细考虑你注册的在线项目可能会持续几年。这种类型的在线课程的目的是让你不必请假就能获得资格证书，并分摊费用。虽然这些好处似乎是显而易见的，但即使在最初几个月之后，你可能会感到压力和疲惫。

例如，考虑在这个项目中呆了几个月，感觉已经超出了你的日常工作，然后意识到你还有一年多的时间。

自我激励和计划你的作业和考试与全日制项目一样重要，从长远来看，懈怠会让你筋疲力尽。

# 申请流程是怎样的？

这通常在你申请的学校的网站上有详细的记录。在我看来，申请中最重要的部分是推荐信和求职信。我相信，这完全是为了给人一种有前途的学生的印象，让他们有能力完成这个项目，并在全职工作的同时应对完成这个项目的压力。

与面对面项目相比，竞争更少，而且你最有可能被录取，因为与面对面项目相比，大学也更容易提供项目。

# 我会和其他学生互动吗？

这可能因程序而异。在我的情况下，与其他学生有预期的互动，其中一些甚至是分级的。这些互动从内部论坛上的小组练习到作为小组项目提交的作业，每个人都应该有所贡献。

# 它和面对面的理学硕士的标准一样吗(从雇主的角度来看)？

据我观察，是的。你可以选择如何向未来的雇主展示你的理学硕士学位。你从课程中获得的知识与亲身学习处于同一水平。如果你亲身参与，你肯定能从老师和同龄人的知识中受益。

# 概括起来

如果您符合以下条件，请参加在线硕士课程:

*   可以利用工作之外的空闲时间
*   想要硕士资格
*   相信它会增强你未来的职业或学术目标。

相反，如果你有能力休假一年，在一所高级大学投资一个面对面的项目，那么无论如何都要考虑这种方法是你最好的选择。

最后，一定要对程序本身做一些研究。即使是一个在线项目，确保这个项目值得你花时间和金钱也是很重要的。
