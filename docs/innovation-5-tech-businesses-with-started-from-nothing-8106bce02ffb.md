# 快门架

一个业余摄影爱好者需要一点额外的钱，创办了 Shutterstock。该公司的创始人乔恩·奥林格(Jon Oringer)看到了市场的一个漏洞，并通过创建一个在线网站来填补这个漏洞，他可以在网上出售自己照片库中的照片。
