# 领域驱动设计简介

> 原文：<https://medium.com/geekculture/an-introduction-to-domain-driven-design-f29fc1877e2a?source=collection_archive---------3----------------------->

领域驱动设计很难，它要求我们理解业务逻辑的复杂性。随着领域建模的建立，我们必须经历领域的陡峭的学习曲线。互联网上有大量的框架可以用 DDD 方法来解释。但每次我们翻阅那些例子，都会觉得不太符合公司的业务性质。
