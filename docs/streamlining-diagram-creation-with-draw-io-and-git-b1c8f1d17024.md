## 以及如何管理受控的人工制品

你是否厌倦了使用过时的图表工具？您的组织是否强迫您使用付费产品，而他们只愿意购买少量许可证？

这并不是说这些工具太旧了，不酷，而是它们不适合使用。随着组织继续使用多个交付伙伴，工具和源文件对所有人来说都是可访问的。
