## 如何使用 github 操作构建 Angular 库并将其发布到 npmjs

# 从 ng-snotify 到 ngx-snotify

在我的项目中，我一直在使用来自 artemsky 的 https://www.npmjs.com/package/ng-snotify。我很了解异步通知是如何在这个组件中实现的，但也了解其他通知类型。该库在通知和用户交互方面提供了高级功能，但是它已经有三、四年没有维护了。
