React 钩子是从 React 16.8 开始引入的。与类的对应物相比，使用钩子的代码通常可读性更好，也更短。React out of the box 提供的一些[基本挂钩](https://reactjs.org/docs/hooks-reference.html)是`useState`和`useEffect`，它们构成了具有状态和副作用的组件的构建块。
