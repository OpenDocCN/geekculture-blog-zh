导入Fuse.js进行模糊搜索

这是WhatsApp克隆项目的一个额外的部分，当你搜索你的朋友时，而不是只显示所有的朋友。你想根据你的输入显示，我们可以通过使用一个名为fuse.js的npm包来做一些模糊搜索。

我们可以通过打字来添加

yarn add fuse.js

转到顶部并导入保险丝

import Fuse from 'fuse.js'
