# 织机网络(Loom)能从多链互操作和 Dapps 中赚钱吗？

> 原文：<https://medium.com/geekculture/can-the-loom-network-loom-make-money-from-a-multichain-interop-and-dapps-66b5c7c84200?source=collection_archive---------14----------------------->

**织机网(LOOM)** 希望通过将不同的区块链编织在一起来赚钱。特别是，Loom 帮助分散式应用程序(dapps)在不同的区块链上运行。

具体来说，Loom Network 是用于 dapp 开发的[生产就绪多链互操作平台](https://loomx.io/)。解释一下，interop 平台允许 dapps 同时在几个区块链上运行，比如比特币、以太坊、创和币安链。Interop 是互操作性的缩写:在不同的链上操作的能力…
