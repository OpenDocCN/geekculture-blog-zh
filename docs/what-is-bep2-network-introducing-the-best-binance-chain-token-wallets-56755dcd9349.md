加密货币的一个主要特点是各种区块链网络更新，从而导致更多的用户和更多的普及。开发者总是试图通过改变加密货币传输网络来改善区块链的情况，币安交易所在其中拥有最大的份额。

在这篇文章中，我们将了解一个新的和流行的网络在区块链。

## BEP-2 网络的定义

可以肯定地说，币安交易所是区块链提供服务最好的交易所之一。这种交易所每天进行的创新在加密货币中创造了不同的条件。为了提供更好的服务，交易所最近推出了新的 BEP2 网络。

BEP2 代表币安链演进提案 2，这是一种在币安链中创建和使用新令牌的技术标准。BEP2 的目的是为币安链平台的票务运营建立一套具体的规则。

**BEP 2 的性能与著名的心房网络 ERC20 非常相似。**

这个网络有几个独特的特点，使它有别于其他区块链网络。在本节中，我们将提到其中的一些功能:

颁发令牌所有者源地址的源地址。
令牌名称
最近在网络上推出的令牌符号。
网络中提供的所有代币的总和
BEP2 网络中支持的查询
由于 BEP 2 网络的高转账速度和合理的费用，这个网络中的很多热门币都可以进行交易。几乎所有世界著名的加密货币，包括比特币、Atrium，甚至稳定币 Tetra，都可以在这个网络上转移，以较低的费用买卖数字货币。这证明了这个网络在加密货币领域进行交易的信任。其中最受欢迎的是 Ramzarz Cardano。除了 ADA 独立的区块链网络，Cardano 还可以通过 BEP2 网络移植。

BNB 密码也是这个网络中的主要传输枢纽之一。

在硬件钱包中，目前有两个版本的 Ledger Nano X 和 Ledger Nano S 支持这个网络。

市面上几乎所有流行的软件钱包也都支持这个网络，比如 Trust Walt wallet 和 Atomic wallet。

## BEP2 网络中的存取

如前所述，BEP-2 和 ERC-20 有许多相似之处。其中一个相似之处就是如何通过这个网络存取款。和 ERC20 一样，你的钱包里必须有一些 BNB 密码才能在 BEP2 上存取。

该网络的本地资产是 BNB(币安链)加密货币，用于支付网络费用。您可以通过 BEP2 快速转移 BNB 网络支持的令牌。

## ERC20 和 BEP2 的区别

这两种网络的主要区别在于地址的类型。

ERC20 网络中使用的地址以前缀 0x 开始，但是在 BEP2 网络中，如上所述，以前缀 BNB 开始。

另一个区别是，在 BEP2 网络上存取款往往需要钱包里有标签或备忘录，而对于 ERC20 令牌，只要有钱包地址就够了。

## BEP2 和 BEP20 网络的区别

币安交易所的另一项创新是引入了币安智能链(BSC)。该网络的目的是在区块链网络中实施智能合同并分散活动。

事实上，BSC 网络是 BNB 网络的补充，两者都在币安生态系统中使用，区别在于它们的功能互不相同。

因此，您不能使用 BEP2 网络来传输智能合约的密码，但在相同条件下，即传输到钱包，您可以使用这两个网络。

与 ERC20 类似的另一个区别是 BEP20 地址中不需要标签或备忘录。
