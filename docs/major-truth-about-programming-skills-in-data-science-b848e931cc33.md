# 数据科学需要多少编程？

现在，你急切等待的问题有了答案

所以，伙计们，如果你想成为一名数据科学家，或者如果你想进入数据科学领域，如果你想在这个领域找到任何工作，那么你不需要很强的编程或编码知识

因为有很多人认为，如果他们有编程背景，也就是说他们是计算机工程师，或者他们有很强的编码知识，那么只有他们才能进入这个领域

所以事实并非如此，伙计们，如果你知道编码或编程的基础，那么你肯定可以进入这个领域

你不需要很强的知识，也不需要像软件开发人员或软件工程师那样成为编码专家

因为如果你是软件开发人员或软件工程师，那么只有你需要强大的编码技能，你必须擅长编程

但在这里，如果你正在考虑数据科学领域，如果你想成为一名数据科学家或数据分析师，那么你真的不需要强大的知识

要进入这个领域，你只需要数据结构的基础知识和一些编程语言中的基本算法

> 如果你知道什么是字符串，什么是元组，什么是列表，while 循环，for 循环，if-else 语句，以及编程语言中的一些基本术语，那么你就可以开始了

如果你了解面向对象编程(oops ),那么它将对你有好处

因为你将在这个领域中使用所有这些概念来分析数据和编写代码，这样你将从数据中获得完美和准确的见解

不是那样的，你需要学习互联网上或市场上的所有编程语言

你只需要擅长一种或者两种编程语言，但是大多数时候一种编程语言就可以了

在数据科学领域，R 和 Python 编程语言非常重要

所以试着学习其中一种编程语言，如果可以的话，你可以同时学习这两种语言

但是最有可能的是 python 在大部分时间里被公司使用，所以只要学习 python 就好了

你不需要成为 Python 的专家，但你只需要 Python 的基本知识，比如字符串、元组、列表、if-else 语句、while 循环、for 循环等等

还有函数、类和其他一些基本术语

因为，当您处理大量数据时，您需要编写一些代码来从数据中获得有价值的见解或清理数据

因此，试着专注于任何一种编程语言，并从中学习基础知识，一旦你开始学习，你就会获得更多的编程知识

还有一个非常重要的工具，或者我们可以说是一种你必须学习的语言，那就是 SQL

SQL 只不过是一种结构化的查询语言，在这种语言中，你需要编写一个查询，使用这些查询，你只需要清理数据并从中获得有价值的见解

这并不难，但是如果你不练习，你就不会擅长 SQL 查询

因此，首先只需学习 SQL 的基础知识，一旦你进一步深入，你将在你的旅程中使用一些复杂的查询

但在此之前，只需在数据集上练习更多的查询，这样你就会对它的工作原理以及你需要遵循的概念有一个基本的了解

但在这里我必须告诉你，如果你想进入这个领域，那么 SQL 和 Python 或 R 是必须的，你必须学习这些东西
