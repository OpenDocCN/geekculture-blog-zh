  • 实现‘停用词’
    停用词    是对句子没有太大意义的英语单词。它们可以被安全地忽略，而不会牺牲句子的意义。例如，像the，he，have等这样的词。这样的词已经被捕获到这个名为语料库的语料库中。Stopwords是一个带有NLTK的子库，可以使用from nltk.corpus import stopwords导入
  • 标记化
    标记化是将原始文本分割成小块。记号化将原始文本分解成单词和句子，称为记号。这些标记有助于理解上下文或开发NLP的模型。标记化有助于通过分析单词的顺序来解释文本的意思。
  • 填充序列
