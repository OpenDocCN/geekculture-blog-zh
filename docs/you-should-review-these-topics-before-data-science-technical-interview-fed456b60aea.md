## 在数据科学技术面试中有什么期待？

数据科学面试流程通常包括四轮，首先是行为面试，然后是技术面试，接着是带回家的作业，最后是与未来团队的第二轮技术面试。这一过程可能因公司而异，但是，这通常是招聘过程中遵循的方案。
