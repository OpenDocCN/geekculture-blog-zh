# Nervos 网络(CKB)有什么价值？—市场疯人院

> 原文：<https://medium.com/geekculture/what-value-does-nervos-network-ckb-have-market-mad-house-72b0fef56c68?source=collection_archive---------15----------------------->

Nervos 网络(CKB) 试图在区块链建立一个类似互联网的公共网络。

建设者们希望 [Nervos 网络](https://www.nervos.org/discover)将支持一种新的分散式应用(Dapps)，他们称之为[通用应用](https://www.nervos.org/)。他们声称一个通用的应用程序可以访问任何区块链上的任何资产。
