直入主题，这家公司就是科技巨头微软。我将在本文的剩余部分讲述微软如何定位自己，以利用各个领域的进步。理解这一点的关键是垂直整合的概念。当一家公司负责其运营中的多个步骤时，就会发生纵向整合。下图对于理解这个概念非常有帮助。

# 阶段 0:硬件

MS 在硬件方面绝不是领先者。然而，他们对 Surface 和其他硬件组件的投资非常清楚地表明他们打算改变这种状况。在未来几年看到这种变化并不奇怪。这是他们写作中最薄弱的方面。

# 阶段 1:数据源

说到 MS 拥有的数据来源的数量，他们是首屈一指的。微软拥有世界上最流行的操作系统 Windows。他们也有自己的 Linux 发行版，并大量参与开源。

![](img/655e074d0930023f0317f77b18d0b9ae.png)

Don’t forget this

在职业方面，他们拥有 Github(稍后会详细介绍)和 LinkedIn。这两个平台让他们能够访问大量非常有价值的数据。我合作过的组织中有一半使用 MS office 套件，这为他们提供了另一个非常有价值的数据来源。MS 拥有的**来自不同平台**的数据量不容忽视。这种数据来源的多样性使他们能够专注于当时投资回报率最高的项目，这给了他们极大的灵活性。

![](img/40bf30c492c4bc53a3b9dd9730d6d554.png)

This is a non-trivial piece of code that copilot wrote on its own

看看 Github copilot 吧。如果没有大量高质量的数据，这是不可能的。他们对 Github 的所有权意味着，他们可以访问它。因此，他们能够与 OpenAI 合作，创建一个革命性的产品，获得好评如潮，并将被许多人使用。

# 阶段 2: ML 模型和研究

这是大多数人一想到机器学习就会想到的。微软在这个领域度过了非凡的一年。在许多其他人当中，微软发表了他们的研究，“伪造它，直到你在野外单独使用合成数据进行人脸分析。”他们能够编写一个面部特征检测协议，该协议纯粹在合成人脸(由网络生成的人脸，不属于人类)上进行训练。他们在真人脸上的表现是最先进的。

![](img/723531b5354b611baebf92b3c4af5f99.png)

The agent detects all the features on a face

他们使用合成人脸的表现非常重要。想想像面部畸形这样的案例，在现实世界中我们可能没有足够的样本数据。在这些情况下，传统模式会失效。这就是为什么我们使用合成数据来增加输入，并允许我们的模型在这种情况下表现良好。使用合成数据可以实现一些有趣的应用，我将在另一篇文章中介绍这些应用。就本文而言，可以说这篇论文令人印象深刻。

![](img/900d5fb6fa650900aeaee566472dcca3.png)

MS is on top of some pretty big names, including Google

MS 甚至在自然语言处理领域掀起波澜。在他们的出版物“在 GLUE 和 SuperGLUE 上高效和有效地扩大语言模型预训练以获得最佳语言表示模型”中，他们展示了他们的最新模型，该模型在 NLP 基准 GLUE 和 SuperGLUE 上击败了所有其他人。要了解更多这方面的内容，请查看这篇文章。

> 值得注意的是，T-NLRv5 首先在 MNLI 上实现了人类对等，在 GLUE 基准上实现了 RTE，这是人类对等尚未实现的最后两个 GLUE 任务。此外，T-NLRv5 比最近的预训练模型更有效，在参数和预训练计算成本减少 50%的情况下实现了相当的有效性

可以肯定地说，微软处于机器学习研究和实现方面的最前沿。**别忘了令人印象深刻的 Github 副驾驶 AI** 。

# 阶段 3:模型部署和维护

一旦一个模型被训练好并准备好被使用，它通常被运送到云中。这就是微软革命性的地方。他们在 2010 年投资了 Azure 和云计算。从那以后，他们继续扩大他们的云业务。

![](img/ea93c75535be04ac264ef8274b21ab17.png)

看看他们提供的产品。通过提供他们的高级分析(例如上一节中的 NLP 架构)，他们能够吸引各个领域的客户。然后，这些客户端要么按原样使用这些分析，要么根据它们进行定制。这两个选项都为 MS 提供了大量有价值的信息，让他们可以进一步定制他们的产品。这使得它变得更好，吸引更多的客户，**导致一个积极的反馈循环**。

![](img/f956974a5b7167799545368ee0a64924.png)

MS is dominating the game

拥有自己的基于云的服务也让他们在一定程度上免受外部故障的影响。这是一个小优势，进一步增加了他们的系统。

微软在多个领域处于领先地位，这给了他们一个绝佳的位置。机器学习是一个复杂的领域，依赖于多个不同的领域。在实际执行时更是如此。这就是为什么我们在这个领域看到如此多的利基市场。通过扩展到 ML 的许多方面，微软处于一个可以利用 ML **和相关领域的进步的位置。这就是 MS 最有可能在 2021 年底脱颖而出的原因。**

请注意，这很大程度上是多年投资和努力的累积。Windows 比我老，Github 是 2018 年收购的，Azure 是 2010 年首次开发的，他们的 ML 研究是从多年的合作和学习中建立起来的。冰冻三尺，非一日之寒，微软之所以处于现在的位置，是因为几十年的积累。头条新闻经常让事情看起来像是一夜之间发生的，但最常见的是需要 20 年的努力才能一夜成功。
