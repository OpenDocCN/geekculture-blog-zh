# 什么是房地产 CRM？

> 原文：<https://medium.com/geekculture/what-is-real-estate-crm-695ba42c2445?source=collection_archive---------14----------------------->

如果你曾经雇佣过房地产经纪人，你可能已经有了一个销售漏斗。毕竟，目标是将潜在客户转化为成功交易。一个执行良好的策略包括举行开放日、交换电子邮件、安排会议以及最终达成交易。但并不是所有的潜在客户都会变成成功的交易。有些需要进一步培养和更个性化的方法。房地产客户关系管理软件是一个伟大的工具，通过销售漏斗移动客户，所以你可以增加你的成交机会。
