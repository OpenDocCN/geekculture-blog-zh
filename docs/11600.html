HDFS命令快速指南

HDFS是Hadoop生态系统的主要枢纽，负责跨各种节点存储结构化和非结构化的大型数据集，从而以日志文件的形式维护元数据。因此，要使用这样一个系统，我们需要精通或者至少应该知道常用的命令和过程来简化我们的任务。在这个问题上，我们已经巩固了一些最常用的HDFS命令，你应该知道与HDFS合作。

首先，我们需要检查下面的列表。

1. 安装HadoopT3】

2.运行Hadoop —我们可以使用‘start-all . cmd’命令或者直接从Hadoop目录启动。

3.验证Hadoop服务 —我们可以使用下面的命令检查我们的Hadoop是否启动并运行。

jps

伟大的..！！！现在我们准备好执行和学习命令。

* *注意:-这些命令是特定于大小写的。写命令时，一定要特别注意大写和小写字母。

  1. version — 该命令用于了解Hadoop的版本，以及附加的本地文件系统位置和编译信息。

hadoop版本

2。mkdir — 该命令用于创建一个新目录，如果它不存在的话。如果目录存在，它将给出一个“文件已经存在”的错误。

hadoop fs -mkdir

3。ls — 该命令用于检查HDFS中的文件或目录。它显示指定目录中每个文件或目录的名称、权限、所有者、大小和修改日期。

hadoop fs -ls

4。put — 该命令用于将数据从本地文件系统复制到HDFS。

Hadoop fs-putT24】

我们可以从HDFS网站上验证这一点。

5。get — 该命令用于将数据从HDFS复制到本地文件系统。该命令与“上传”命令相反。

hadoop fs -get

我们可以从本地文件系统中验证这一点。

6。cat — 命令用于查看HDFS文件中的数据

hadoop fs -cat

7。mv — 该命令用于将文件从HDFS的一个位置移动到HDFS的另一个位置。

hadoop fs -mv

我们可以从Web用户界面验证这一点。

8。cp — 该命令用于将文件从HDFS的一个位置复制到HDFS境内的另一个位置。

hadoop fs -cp

我们可以从Web用户界面验证这一点。

9。copyFromLocal — 该命令用于将数据从本地文件系统复制到HDFS。

Hadoop fs-copy from localT3】

我们可以从WebUI验证复制的文件。

10。copyToLocal — 该命令用于将数据从HDFS复制到本地文件系统。

hadoop fs -copyToLocal

我们可以在本地文件系统中检查该文件。

11。movefromlocial—该命令用于将文件或目录从本地文件系统移动到HDFS。

Hadoop fs-movefromlocial

12。moveToLocal — 该命令用于将文件或目录从HDFS移动到本地文件系统。该命令尚未实现，但很快就会实现。

Hadoop fs-moveToLocalT15】

13。rm — 删除，该命令用于从HDFS中删除/移除文件。

hadoop fs -rm

14。tail — 该命令用于从HDFS读取文件的尾部/结尾部分。它有一个额外的参数“[-f]”，用于显示文件的附加数据。

hadoop fs -tail [-f]

15。expunge — 该命令用于清空垃圾箱。

hadoop fs -expunge

16。chown — 当我们想在HDFS改变一个文件或目录的用户时，我们应该使用这个命令。

hadoop fs -chown

我们可以使用hadoop -ls命令或从WebUI验证用户是否进行了更改。

17。chgrp — 当我们想在HDFS改变一个文件或目录的组时，我们应该使用这个命令。

hadoop fs -chgrp

我们可以使用hadoop -ls命令或从WebUI验证用户是否进行了更改。

18。setrep — 该命令用于在HDFS中更改文件的复制因子。

hadoop fs -setrep

我们可以从网络界面查看。

19。du — 该命令用于检查文件或目录的磁盘使用量。

hadoop fs -du

20。df — 此命令用于显示HDFS文件系统的容量、可用空间和大小。它有一个额外的参数“[-h]”，用于将数据转换为人类可读的格式。

hadoop fs -df [-h]

21。fsck — 此命令用于检查HDFS文件系统中文件的健康状况。

hadoop fsck

它还有一些属性/选项来修改命令的用法。

22。touchz — 该命令在指定的目录中创建一个大小为0的新文件。

hadoop fs -touchz

新文件可以在WebUI中看到。

23。test — 该命令回答关于< HDFS路径>的各种问题，并通过退出状态给出结果。

hadoop fs -test

24。text — 这是一个简单的命令，用于在控制台上打印HDFS文件的数据。

hadoop fs -text

25。stat — 该命令提供HDFS中文件或目录的stat。

hadoop fs -stat

它可以提供以下格式的数据。默认情况下，它使用“%y”。

26。用法— 显示给定命令或所有命令(如果未指定)的用法。

Hadoop fs-用法

27。帮助— 显示给定命令或所有命令(如果未指定)的帮助。

hadoop fs -help

28。chmod — 用于更改HDFS文件系统中文件的权限。

hadoop fs -chmod [-r]

旧权限

新权限

29。appendToFile — 该命令用于将本地文件系统中的两个文件合并为HDFS文件中的一个文件。

Hadoop fs-appendToFile

三十岁。校验和— 该命令用于检查HDFS文件系统中文件的校验和。

hadoop文件系统校验和

三十一。count — 计算特定路径下文件、目录的数量和大小。

Hadoop fs-count[选项]

这个函数也有一些函数可以根据需要修改查询。

32。查找— 该命令用于在HDFS文件系统中查找文件。我们需要提供我们正在寻找的表达式，并且如果我们想要在特定的目录中寻找文件，也可以提供一个路径。

hadoop fs -find

33。getmerge — 该命令用于将HDFS目录的内容合并到本地文件系统的一个文件中。

Hadoop fs-get mergeT40】

可以在本地文件系统中看到合并的文件。
