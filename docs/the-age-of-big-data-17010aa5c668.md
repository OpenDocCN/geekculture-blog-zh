# 大数据时代

> 原文：<https://medium.com/geekculture/the-age-of-big-data-17010aa5c668?source=collection_archive---------8----------------------->

鉴于技术的快速发展，大数据正日益融入我们的生活。可用数据量的增加对全球潜在客户和公司有利有弊。本文将介绍大数据的概念及其一些应用，并评估这种颠覆性技术在社会中的优势和劣势。

# 背景:

![](img/a233c8eaee057f431639ee6aa3692ce4.png)

Photo by [Mitchell Luo](https://unsplash.com/@mitchel3uo?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

大数据是一个庞大、多样的信息集，其数量不断增加，速度越来越快(Segal 2021)。大数据可以以结构化形式(数字形式，易于格式化和存储)或非结构化形式(自由形式，不易量化)存在。它通常存储在计算机数据库中，用于处理复杂的数据集(Segal 2021)。今天，科技巨头，如谷歌和脸书，利用这项技术进行广泛的数据分析，解决复杂的问题。大数据是通过社交媒体和网络上的公开内容收集的，并以购买和问卷的形式从个人电子产品中收集(Segal 2021)。与任何具有无限能力的技术一样，大数据会让事情变得更好，或者以我们从未见过的方式颠覆世界。

# 优势:

![](img/218630aae4d0e31cc8eceb642d83c82a.png)

Photo by [Alex Kotliarskyi](https://unsplash.com/@frantic?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

随着可用数据量的增加，世界各地的公司应该能够为潜在客户量身定制产品，同时提高满意度并创造收入。2020 年，市场研究行业 47%的专业人士使用大数据分析作为研究方法(Statistica 2021)，该行业预计到 2027 年将增长到 1030 亿美元。大数据可以创造突破，显著提高劳动力的效率，同时吸引更多的消费者。大数据最重要的好处之一是，它可以帮助公司建立成本优化模型。例如，物流行业使用大数据强调了成本降低的好处，该数据显示退货成本比实际运输成本大 1.5 倍(upGrad 2019)。有了这些知识，公司将评估他们的产品，通过预测退货的可能性来最小化退货率。这使得公司可以采取适当的措施，降低整体成本(upGrad 2019)。除了节省成本的大数据分析之外，大数据还促进有竞争力的定价，帮助公司保持客户忠诚度，并带来创新。这些优势有力证明了大数据作为全球企业强大工具的潜力。

# 缺点:

![](img/b081281ae8eb6054a20ed29ab1ec8e0d.png)

Photo by [imgix](https://unsplash.com/@imgix?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

虽然大数据的优势很吸引人，但也存在许多挑战。首先，存储大数据，特别是敏感数据，使公司成为主要的网络攻击目标(Harvey 2018)。谈到大数据，公司现在不断将安全漏洞列为消费者安全和金融安全方面的最大挑战。随着大数据的运行大大超出预算，企业还面临着主要的硬件、维护和相关活动成本(Harvey 2018)。除了公司的斗争，违反隐私原则是另一个缺点。鉴于脸书-剑桥分析公司丑闻(脸书向剑桥分析公司提供了 8700 万人的个人数据，违反了数据保护法)，大数据被用来影响政治和操纵消费者行为变得显而易见。这一发现令人不安，因为它完美地展示了大数据在落入坏人之手时的颠覆性能力。由于这种技术被纳入政治，并有能力从根本上影响公众舆论，因此监管大数据的使用以确保它不会破坏民主和自由社会的基石至关重要。展望未来，在当今的数字时代，机遇的出现也带来了挑战。对于领导者、学者和高管来说，确保在全球范围内正确实施大数据至关重要。

# 结论:

不可否认的是，随着大数据集成的新应用呈指数级增长，大数据在 21 世纪正在崛起。大数据有可能从根本上改变社会的运作方式，从影响政治行为者到引发隐私和安全领域的担忧。尽管这项技术有其缺陷，但也有显著的好处。突出的好处包括成本优化和为消费者和企业提供更好的服务。问题是:世界各地的公司和政府能够在不损害公民隐私和安全的情况下利用大数据的潜力吗？
