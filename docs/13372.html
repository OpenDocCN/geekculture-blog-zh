Swift中的ExpressibleByArrayLiteral

当使用 struct s时，您可能会在许多情况下遇到。乍一看，这似乎很难理解，但是了解它实际上是如何工作的以及它提供了什么是很有帮助的。

expressablebyarrayliteral，顾名思义，是一个协议，允许使用数组文字初始化它所符合的类型。此外，它可以增加可读性。为了清楚起见，您将把它作为一个简单的例子来研究。

比方说，你有一个类型为 struct 的堆栈。

struct Stack<Element: Equatable>: Equatable {
  /// Holds the stack data.
  private var storage: [Element] = []
  
  init() { }
}

当你初始化堆栈时，你必须这样初始化:

let stack = Stack(["expressible", "by", "array", "literal"]

但是，如果您将ExpressibleByArrayLiteral扩展添加到堆栈中:

extension Stack: ExpressibleByArrayLiteral {
  init(arrayLiteral elements: Element…) {
    storage = elements
  }
}

您可以像这样初始化堆栈:

let stack: Stack = ["expressible", "by", "array", "literal"]
