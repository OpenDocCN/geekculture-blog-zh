# 1.最初，你的数据分析会把你引向错误的方向

数据科学是一个宽泛的术语，用来描述从数据中提取最多信息的科学方法。数据科学是一个过程。要学的东西很多，要做的事情也很多，但很多都是应用你从过去的经验中学到的东西。这不是灵丹妙药——运用一些技巧，然后期待你想要的结果。这是一种思维方式。数据科学是关于对数据的热情和对数据的好奇。这是关于找到模式，并使用它们成为一个更好的分析师。这是关于利用你从错误中学到的东西，并保持开放的心态。这是关于迭代和改进。数据科学是很多东西，但可以概括为一个过程。

数据科学是一个相对较新的领域，但它已经是几乎每个行业的重要组成部分。这个领域正在快速发展，它正在改变我们分析信息和理解世界的方式。事实上，数据科学非常重要，如果你对该领域没有扎实的理解，你可能会在几年后发现自己有点落后于形势。这是因为数据科学不再仅仅是一个职称。这是一个广阔的领域，涵盖了许多不同的工作和技能。为了帮助你更好地理解数据科学，这个博客将是一个简短的系列课程，这些课程是我从艰辛的道路中学到的。这些课程将基于我在该领域的工作经验，应该可以帮助你在数据科学的复杂领域中导航。

# 2.在尝试处理数据之前，简化您的数据分析

数据科学是本世纪最蓬勃发展的领域之一，不仅有大量的公司从数据科学中赚钱，而且还有大量的公司围绕数据科学行业创建。数据科学是一个相对较新的领域，因此，有大量的资源可以帮助您了解它。这方面的问题是，真正了解数据科学真的很难。有数百种不同类型的数据科学算法，数千种不同的数据科学工具，以及数万种不同的教程。每次你学习一个新的算法或工具，你都必须学习它是如何工作的，它是用来做什么的，以及如何使用它。甚至有更多的资源可以帮助您了解数据科学行业，但是您如何在不被淹没的情况下了解数据科学呢？

任何数据分析项目最重要的步骤之一是尽可能简化数据。这意味着尽可能多地删除不相关的数据。这会让你明白什么重要，什么不重要。这种简化必须尽可能不带偏见。许多人试图太聪明或太聪明，结果把事情搞得太复杂，使他们的分析无效。这是最好通过经验来学习的东西，不幸的是，这是需要花很多努力和时间来学习的东西之一。

当我开始学习数据科学时，我甚至不知道它是什么，也不知道它是如何工作的。我记得看到了很多数据科学的工作，我认为这将是一个开始赚钱的好机会。我知道我需要学习它，但我不知道如何着手。我试着从书上学过，但是学的不多。所以我决定找一门课来上。我发现了一个很棒的课程，并决定加入它，几个月后，我对自己正在做的事情很有信心。我们做的第一件事是分析一些数据，做一些数据挖掘，我甚至根据我拥有的数据做了一些预测。我对自己非常有信心，以至于我能够预测我没有理由预测的事情。我基本上是想预测未来，你猜怎么着？我没成功。过了一段时间，我才意识到我在试图学习一些我完全不理解的东西，这让我很难去学习。

# 3.永远小心你的数据

我们首先要说的是数据。数据可以有多种不同的形状和大小。它可以存储在许多数据库中的一个，甚至可以存储在文件中。当我们说数据时，我们指的是存储并用于某种目的的信息。对于数据科学，我们指的是可以用来预测世界的数据。这些数据存储在数据库和文件中。数据科学，或数据挖掘，是基于使用这些数据来根据这些数据做出预测。比如用数据预测明天的天气，可以适当准备。我们用来做这件事的数据存储在数据集中。我们可以使用许多不同的数据集来进行预测，所以我们需要做的是找到正确的数据集和正确的数据类型。我们可以使用许多不同类型的数据，包括文本、图像、音频等等

数据科学是从数据中提取知识和见解，并将其转化为信息的研究。它是许多不同学科的技能和技术的结合，用于解决各种问题。作为一名数据科学新手，我已经学到了很多教训。我通过参加在线课程、阅读书籍和在 Kaggle 上完成项目开始了我的数据科学之旅。这些项目中的许多都有一些杂乱的数据集，我学会了如何处理缺失值和非线性关系。然而，有些项目更复杂，我觉得我在和我的工具和我自己战斗。浪费了很多时间，有些项目不成功。

# 4.警惕你是如何解读数据的

数据科学是一个不断发展的领域，许多数据科学家被雇用来使用他们收集的数据描绘一个公司或产品的图景。虽然这可能是一个非常有利可图的领域，但也可能是一个非常危险的领域。数据科学家，就像其他任何人一样，可能会陷入这样的陷阱:将他们读到的一切都纳入数据，并基于此做出判断。在本文中，我将向您讲述几个我从数据科学中学到的故事。从我在数据科学领域的经历中，我学到的第一课就是要警惕你是如何解读数据的。虽然数据确实是数据科学家工作中最重要的部分，但它也可能是一件非常危险的事情。我们很容易通过先入为主的观念来解读数据并得出结论，从而让你的数据变成谎言。在我以前的一份工作中，我阅读了数据，并以我先入为主的观念形成了一个假设，最终这是一个巨大的错误。我认为，我们的网站没有转换的原因是因为网站的布局方式。我被自己先入为主的观念蒙蔽了双眼，最终，我错了。

数据科学是目前科技行业最热门的领域之一。它不再只是大型科技公司或财富 500 强公司的专利。数据科学是为创业公司准备的。数据科学是为小人物准备的。数据科学是面向所有人的。虽然数据科学是目前最热门的技术领域之一，但它也是最容易被误解的领域之一。有很多关于数据科学的神话。最大的一个就是数据科学很神奇。真的不是。数据科学的核心是分析数据的过程。要获得数据，你需要得到一些数据，这意味着你需要进入数据科学的过程。

# 5.警惕你的数据需要适应不确定性

当你艰难地学习数据科学时，你开始意识到如果你想成功，你必须适应不确定性。你想到达山顶，但你不知道如何到达。你想成为一名数据科学家，但你不知道如何去做。你想开始一个数据科学的职业生涯，但你不知道如何开始。你想得到提升，但你不知道怎么做。你想加薪，但你不知道如何加薪。你想赚更多的钱，但你不知道如何去做。这种程度的不确定性对大多数人来说是不舒服的，但是如果你想成功，你必须适应不确定性。

# 6.当你不确定的时候，总是依靠你的直觉

首先，请记住，数据科学行业仍然非常年轻，并且在不断发展。因此，现有的具体最佳实践并不多。所以，学习数据科学的最好方法就是开始做。但是，如果您没有数据科学方面的经验，请记住这个行业还很年轻，没有多少具体的最佳实践。因此，从各种数据科学主题的基本介绍开始是有用的。

数据科学与其说是关于你如何做，不如说是关于你做什么。工具、方法、数学、统计，所有这些都是次要的。你可以在几周内学会所有这些，但是你在这段时间内获得的将远远超过你从现实世界应用中获得的洞察力。数据科学更多的是观察你周围的世界，看到模式，然后用它来解决现实世界的问题。

数据科学是一门不确定性的科学。这是一个不断调查和发现真相的过程。对于我们这些不习惯不确定性的人来说，这可能是一个非常可怕的概念。事实上，我认为这是我工作中最可怕的部分。但是我明白了我的直觉很重要。我已经学会相信我的直觉，即使它对我尖叫，“这是错误的！”我明白了我的直觉是一个强大的工具，一个可以帮助我解决我正在处理的问题的工具。我明白了我的直觉背后是有原因的。它是多年经验、一点点逻辑和许多感觉的结合。我已经学会了跟随它。
