# 抖音计划与……亚马逊竞争？

> 原文：<https://medium.com/geekculture/tiktok-plans-to-rival-amazon-310bf86549f0?source=collection_archive---------16----------------------->

当你想到抖音接管世界时，你首先会想到什么？我想这就是抖音迄今为止一直在做的事情:短视频、精选提要、内容爆发、可疑的隐私做法、音乐推广。但这份名单可能很快会改变，因为据报道，抖音现在瞄准了一个全新的领域——购物。没错，你孩子最喜欢的应用即将媲美亚马逊。让我们深入了解一下。

# 行动呼吁

![](img/54e81a6f0284dcbeea83486c90a905c9.png)

Solen Feyissa

就在最近，抖音发布了一系列工作清单，为一个“国际电子商务履行系统”寻找工作人员，该系统将设在美国。核心列表是针对洛杉矶和西雅图的员工的，这两个城市都已经是抖音办公室的所在地。此举将扩大这些设施，并为抖音在该国以及全球的业务设定一个新的优先事项。

此前，抖音已经开始尝试与 Shopify 合作伙伴和 T2 抖音商店合作。这两项投资似乎已经取得了回报，因为该公司显然正在为重大转变做准备，并寻求在美国各地建立仓库和物流。

# 抖音商店要来了

![](img/66d5494e93a13301f324b590b71ea61b.png)

Solen Feyissa

[报道](https://www.ft.com/content/479cd8da-c416-456d-a2a2-533af8a5b7bb)称抖音计划在圣诞节前在北美开设抖音商店，之前在英国受到的反响相当冷淡。这一推出是有意义的，因为假日季节可能会提振初期销售，给该公司一个更强劲的开端。抖音对销售的影响不是秘密，因为音乐行业的艺术家们面临着在平台上“爆炸”的先决条件，然后他们才被允许录制专辑或进行巡演，而在抖音病毒式传播的书籍和电影往往会看到兴趣的健康飙升。

然而，假日推动的增长和大量沉迷于内容的用户是否足以维持一个实际的电子商务业务是一个完全不同的问题。当面对亚马逊这样的竞争对手时，任何冒险都是有风险的，但让抖音远离其根基、将该公司推向全球最大商业公司的道路，可能是彻头彻尾的自杀。
