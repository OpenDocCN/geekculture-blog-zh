## **1.我热衷于建设什么项目？**

**并非所有人都对构建 web 应用充满热情。并非所有人都对数据科学充满热情。并不是每个人都对开发游戏充满热情。不是每个人都有耐心学习人工智能。**

**但是这个决定完全取决于你。**

***你热衷于建设什么项目？***

***你最着迷的软件是什么？***

**是的，我知道！不同的领域有不同的编程语言。**

**所以这里有一个你可能要考虑的分类。但是，记住，这不是路的尽头。**

> **网络开发——HTML，CSS，JavaScript，Python。**
> 
> **游戏开发— C++**
> 
> **移动应用开发——Java，或 Swift，或 Kotlin，或 React Native(一个 JavaScript 库)，或 Flutter。**
> 
> **人工智能，机器学习，数据科学——Python，或者 R 编程语言或者 Golang。**

****这里有一个快速提示** : Python 是最适合初学者的编程语言。**

***为什么？***

**它的语法非常容易理解。因此，通过 Python 学习编程的基本概念会容易得多。**
