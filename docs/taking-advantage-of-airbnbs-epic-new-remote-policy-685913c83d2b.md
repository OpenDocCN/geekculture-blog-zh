# 内容创作

不，不是我做的。你可以试试，但那是一个新的难题。随着世界变得越来越遥远，更多的人将与内容而不是人互动。你可以从两个方面利用这一点——学习创建交互式内容或创建使交互式内容成为可能的平台。

前者包括致力于你的沟通，并学习如何使用某些工具。随着远程工作和不同时区变得越来越普遍，人们将减少会议中的演示设计和技术规格，而只是开始转发这些东西的文档。这已经在做了，但是这个过程将会爆炸。

目前的设计文件有很多问题。它们没有互动性，很难理解。我们将看到更多的交互式设计文档，它们将充分利用许多现代技术。人们还没有谈论这个，但 AR/VR 将会改变某些行业的游戏规则。视觉媒体——像摄影、冶金和建筑这样的重型媒体已经广泛使用了。这些行业会为拥有专业知识的人支付大笔费用。如果你能让技术规范文档更容易理解和理解，即使是软件公司也会感激你。你不知道，作为一名技术作家/视频制作人，我得到了多少份工作邀请。学习如何创造这样的内容来展示你的作品将会有很大的回报。
