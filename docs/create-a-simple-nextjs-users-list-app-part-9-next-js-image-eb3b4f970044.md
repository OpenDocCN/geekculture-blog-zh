## 将用户卡图像更改为下一张图像并添加样式

在本文中，我们将讨论 Next.js 图像组件。

使用 image component 的好处是，自动图像优化允许在浏览器支持的情况下，以现代格式(如 [WebP](https://developer.mozilla.org/en-US/docs/Web/Media/Formats/Image_types) )调整大小、优化和提供图像。
