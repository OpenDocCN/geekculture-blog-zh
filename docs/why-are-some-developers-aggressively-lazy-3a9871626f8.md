# 为什么有些开发人员非常懒惰？

> 原文：<https://medium.com/geekculture/why-are-some-developers-aggressively-lazy-3a9871626f8?source=collection_archive---------6----------------------->

![](img/a3e14a008ad2d53cb4ebf9ce94aa9c62.png)

我和各种各样的开发者合作过。大多数软件开发人员热爱编程，会快速高效地完成任务。他们为自己的工作感到自豪，他们喜欢挑战。这些是每个公司都喜欢拥有并与之合作的问题解决者。一旦他们得到它们，他们会确保保存它们。
