# **NFT 是如何认证的？**

由于虚拟收藏品运行在区块链技术上，你不能篡改它们来窃取或修改数据。例如，如果你把你的数据输入一个区块链，你所有的细节都将被储存在里面，并且不可能被侵入和编辑。人们声称即使是最熟练的黑客也无法侵入你的区块链。这样，您在注册过程开始时输入的详细信息会一直保持不变。
