# 介绍

由于当今大量的语言和技术，开始学习编程是一项艰巨的任务。那么，如何开始学习编程呢？我认为这是一个价值百万美元的问题！

几年前就一直想从网上学习编程。但是我所有的方法都失败了。我想也许我并不期望成为一名程序员。后来，我上了一所计算机科学学校，在那里学习编程。令人惊讶的是，我发现编程是如此容易和令人兴奋，如果你遵循正确的道路。

# 编程是什么意思？

对许多人来说，编程的意义可能是模糊的，它可能会使其他人感到困惑。首先，我们需要知道编程意味着什么？它只是简单地给计算机下达指令去完成某项任务，我们使用一种编程语言，这样计算机就能明白我们到底想要什么。

![](img/1a51b2684c061680bc67b7761a7a17df.png)

“Java” is an example of a programming language

电脑可以解决任何问题，你只需要知道如何给它正确的指令。它可以是一个简单的计算器程序，也可以是一个解决你问题的移动应用程序。例如，以前每当我上床睡觉时，我总是把闹钟设在一个完整的睡眠周期*(一个睡眠周期= 90 分钟)*我在计算什么时候醒来最好。然后，我决定开发一个移动应用程序来解决这个问题。我实际上开始编码，它工作了！

![](img/9a1cf97472423723228d72fcc0277b32.png)

My android application “Smart Alarm”

正如你所看到的，这个程序可以计算出在不打断睡眠周期的情况下醒来的最佳时间。现在，每当我上床睡觉时，我只需打开我的应用程序，选择时间，闹钟就会自动设置。我脑子里不再有计算，问题解决了:)

# 编程的应用

在我们这个时代，编程被用于许多领域，我们离不开它。我将提到作为程序员你可以选择的重要途径。了解这些路径非常重要，这样您就可以决定从哪种编程语言开始。

描述每条路径不是本文的主题，但是我会提到它们，您可以在互联网上搜索更多信息。时下的时髦曲目有**网页开发者**、**移动应用开发者**、**游戏开发者**、**数据科学家**、**机器学习**和**人工智能**。如果哪个领域引起了你的注意，去搜索更多的信息吧，那可能是你未来的职业！

# 为什么首先要学习编程？

这是很重要的一点，你要先问问自己为什么要学编程。如果你现在有动力，那很好，但是相信我，这种动力不会永远持续下去！你应该有一个真正的学习理由。

我相信，头脑中有一个项目想法会给你不断学习的动力。这种动力会持续到你完成项目。以我为例，我试着学习了两次**移动开发**。第一次，我不能继续学习，因为我只是为了学习而学习。但是第二次开始学习的时候脑子里有一个项目的想法。每当我感到缺乏动力时，我就会想起我的项目，它给了我继续学习的巨大动力。

因此，我给你的黄金提示是，在开始学习之前，头脑中要有一个项目的想法。这会让你在这个旅程中保持动力。

# 当你刚开始学习时，你会期待什么？

正如我在导言中提到的。当我第一次想学习编程时，我无法继续。主要原因是我有很高的期望。我想仅仅一个星期后，我就可以开始建立自己的网站或应用程序了！

原来你得先学会编程的基础。老实说，学习编程的原理和基础并不是最有趣的事情。另外，你可能一开始会很纠结。但是建立一个强大的编程基础是很重要的。一旦你掌握了基础知识，你就可以开始学习你想要的任何途径。

# 学习编程的最佳方法

选择你的学习方法将决定你旅程的成败。在我看来，我认为你应该学习编程的基础知识，以及如何开始像程序员一样思考。解决问题是程序员的一项关键技能，只有通过实践才能提高。

你可以从 **Java** 或者 **Python** 开始学习基础知识。你应该知道 **Java** 比 **Python** 难学，这就是为什么有些人建议先学。因为如果你先学最难的，之后你就能学到任何东西。最后，这是你的决定，作为第一步，他们都是很好的选择。

在你掌握了编程的基础知识之后，现在你可以开始学习上面提到的任何途径，比如**移动开发**或者 **web 开发**。

# 结论

请记住，任何人都可以成为程序员，尤其是有了丰富的在线教程。此外，你可以在付费在线课程平台 Udemy 上找到许多有价值的课程。

我最后的建议是尽可能多的练习，你不能仅仅通过阅读来提高编程水平，你必须自己努力，尽可能多的写代码。

最后，永远不要停止学习我亲爱的朋友。
