为什么我在大学主修英语时学习HTML & CSS

原文：https://medium.com/geekculture/why-i-learned-html-css-as-an-english-major-in-college-cdd5abd8f5fe?source=collection_archive---------1-----------------------#2021-01-09

以及学习它们的一些好处，即使你并不想成为一名web开发人员。

我刚从大学毕业，获得了英语文学学士学位。在我的最后一个学期，我选了一门叫做专业写作的课程。老实说，我并不完全知道这门课程需要什么，因为在我注册的时候没有课程说明。我决定试一试，看看会有什么结果。

原来，这个课程的重点是HTML，CSS和建立网站。我慌慌张张走进教室，心里想:“这个我一点都不知道！我即将获得英语学位，而不是计算机科学学位！我这门课肯定会不及格。”

两个主要的想法突然出现在我的脑海中，这对我的职业生涯有什么帮助，我如何才能通过这门课程？那时，我决心成为一名全职作家，在那之前，我所学的课程都是围绕写作和文学展开的。我决定至少坚持前两节课，然后我会让自己决定是否放弃这门课。

事实证明，坚持到底是我能做出的最好的决定。我知道了我有多喜欢使用HTML和CSS，我有多喜欢建立网站的过程。我现在参加了两个课程，在这两个课程中，我学到了更多成为前端开发人员所需的知识！

然而，这并不是我坚持这门课程的唯一原因。上了几节课之后，我知道学习HTML和CSS也会给我带来其他类型的好处。以下是我坚持学习它的几个原因，以及为什么即使你不想成为一名开发人员，你也应该考虑学习它。

这是一项宝贵的技能

有时候，作为一名大学生，你可能会因为专业所需的课程而分心，以至于忘记拓展自己的视野。满足你专业的要求很重要，但尝试你可能从未想过要上的其他课程也很重要。

如果你将课程的主题扩展到专业之外，你会发现一些真正有价值的课程。

不管你是否在大学，学习HTML和CSS是你可以学习的有价值的技能和课程之一。你不需要成为互联网方面的专家，你只需要有学习它的意愿和奉献精神。

学习这项技能很有价值的原因之一是因为你可以学习如何建立一个网站。建立一个网站可以在很多方面对你有帮助，无论是在你的职业生活还是个人生活中。如果你乐在其中，它甚至可以成为你赚取收入的一种方式。

它能让你在就业市场脱颖而出

面对现实吧，我们生活在一个数字时代。随着技术统治我们，为什么不学习一些能让你在就业市场脱颖而出的东西呢？技术不可避免地融入到我们的日常生活中，是我们职业生涯、个人生活和世界的重要组成部分。

特别是HTML和CSS，可以成为一项很好的技能，可以在你的简历上展示出来。这表明你不仅在日常生活中了解技术，而且还能把这些知识运用到工作中。

这些技能对雇主来说很有吸引力，可以给你带来优势。即使你对这些概念只有基本的了解，它们也会成为你的优势，尤其是当你申请商业、技术甚至写作方面的工作时！

它会让你更深入地了解互联网和网络是如何工作的

我们都使用互联网，对吗？参加网络开发课程会让你更好地理解这个你每天都在使用的软件。学习web开发可以帮助你更深入地理解互联网和web是如何工作的。

Web开发不仅仅是一条职业道路，它还是一项需要学习的有用技能。当我第一次在大学开始我的web开发课程时，我不知道互联网和web到底有多复杂。通过参加这个课程，我学到了比我想象的更多的关于网页开发、设计等等的知识。再说一遍，这些不仅是有价值的技能，也能让你在就业市场脱颖而出。

允许自己学习新的东西，这可能会让你走出舒适区。令人惊讶的是，你甚至会找到一条新的职业道路！
