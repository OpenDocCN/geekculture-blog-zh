Swift 为一个从 Objective-C days、enumerations 就已经知道的特性带来了更多的功能。

然而，不像 Objective-C enum 只能列举与整数值相关的名称，Swift enums 要灵活得多，并且不像 Objective-C enum 那样要求开发人员为每种情况提供一个值(也称为“原始值”)。
