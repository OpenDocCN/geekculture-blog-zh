面向软件开发人员的5个高质量播客

原文：https://medium.com/geekculture/5-high-quality-podcasts-for-software-developers-2ffb311cea5b?source=collection_archive---------25-----------------------#2021-09-10

播客对于软件开发人员来说是学习和播种新思想的好方法。与阅读文档不同，播客更加非正式，不需要你专注于屏幕，并允许你被动地了解行业的新趋势。

这里有一个高质量播客的非详尽列表，涵盖了广泛的主题，可以帮助您入门。

1.软件规模化

Software at Scale 是一个学习顶尖科技公司如何开发软件的优秀播客。来自Stripe、谷歌和Dropbox等重量级公司的工程师和领导者来到播客，谈论将软件系统扩展到数百万用户的决策和过程。

如果你是一个初级开发人员或开始自己的生意，大规模的软件既鼓舞人心又有教育意义。

2.堆栈溢出播客

是的，你的第一个目的地是任何编程帮助也有播客。栈溢出播客听起来挺好玩的。剧集30分钟以下，风格不拘。通常每集都有一个主题，但主持人不介意跑题或开玩笑。

这个播客不是最技术性的，主题通常很广泛，但是，它绝对很有趣，值得在短途驾车/步行时听。

3.Jeremy Daly和Rebecca Marshburn的无服务器聊天

无服务器聊天，顾名思义，专注于使用AWS Lambda等云服务的无服务器架构和构建应用。如果你有兴趣学习如何为无服务器设计应用程序，以及无服务器的成本和收益，这个播客很适合你。

尽管关注的是无服务器，但剧集的主题各不相同，包括机器学习、安全性、可靠性等。以及它们如何适应无服务器的世界。鉴于目前无服务器非常流行，有必要去看看无服务器聊天。

4.软件日报

软件日报是一个众所周知的技术播客，如果你曾经寻找过软件工程播客，你可能会遇到它。该节目的特点是来自几个不同公司和背景的工程师谈论他们的产品。这个播客有助于学习新技术和新产品，了解行业的发展方向。《软件日报》最精彩的部分就在于它的名字。每天的剧集都很棒，因为总有新的内容可以听。

5.科特兰·艾伦的独立黑客

独立黑客播客是我个人最喜欢的。科特兰邀请企业创始人谈论他们的业务。虽然它专注于建立和发展新的业务，但很大一部分剧集深入研究了软件开发和支持这些业务的技术。节目中有丰富的知识，围绕着作为技术和非技术创始人创办新企业，以及创始人使用什么技术和工具来发展和营销他们的业务。

我之前提到过软件规模化，它的客户来自大型成熟的科技公司。另一方面，独立黑客提供了对刚刚起步的科技和非科技公司以及小规模使用的软件开发实践和工具的看法。

希望现在你对软件开发人员有一个更好的了解，并且你找到了你可能喜欢的东西。
