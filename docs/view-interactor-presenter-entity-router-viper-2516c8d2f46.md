# 查看互动程序演示者实体路由器(VIPER)

> 原文：<https://medium.com/geekculture/view-interactor-presenter-entity-router-viper-2516c8d2f46?source=collection_archive---------13----------------------->

视图交互器表示器实体路由器(VIPER)是一种架构模式，它将对象分为五种不同的类型，主要由视图、交互器、表示器、实体和路由器组成。

![](img/e1f4f6d69d5929db3e1b1cbc384883c7.png)

VIPER 将逻辑结构划分为不同的责任层，然后隔离应用程序的依赖关系。有了你，就有多个组件扮演不同的角色…
