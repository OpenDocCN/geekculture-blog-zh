`+` `—` `*` `/`这些都是我们熟悉的常见运算符，不是吗？😉JavaScript 给了我们更多的操作符来简化我们的计算。

`+=` `-=` `++` `--` `%`这些是更多的操作符，我将用例子简单解释一下。(忽略 console.log 部分)。

`2 + 3`结果`5`、`2 — 3`结果`-1`、`2*3`结果`6,`

再次，`6/2`结果`3`，

`6%3`结果`0`(我知道刚开始有点混乱，但很简单)。

实际上`6`除以`3`就是`2`，除此之外什么都没有，也就是说`0`&`%`运算符得到完全相同的值。我的意思是，首先你除法，然后除法之后你检查余数。运算符给我们余数

现在到了棘手的部分，`5 += 6`结果`11`。怎么会？好吧，让我解释一下。

`+=`操作符在赋值的同时做加法。`a += b`的意思是`a = a + b`就是这个意思。现在你说`6 -= 4`会是什么结果。试试看。它和`+=`做的完全一样，只是先减去&，然后把值赋给左操作数。所以，`a -= b`就是`a = a — b`的意思。`++`操作符将值增加 1，而`—`将值减少 1，因此`5++`表示`6`，而`5--`表示`4`您还可以使用`+`操作符添加两个或更多字符串。

## 条件/控制结构:

这些是 JavaScript 中一些简单的结构。我将直接通过例子。

`if(condition){ statement; }`

`else{ statement; }`

这是基本的 if else 结构。假设你需要从两个条件中选择一个，例如，你不能决定你应该走哪条路，左边还是右边，这是一个双向条件，所以我们在这里应用 if else 条件。

同样，如果你遇到两个以上的条件，你可以使用 if → else if → else 这样的结构

`if(condition1){ statement; }`

`else if(condition2){ statement; }`

`else if(condition3){ statement; }`

`else{ statement; }`。

还有其他一些控制运算符，如

`===``>``<``>=``<=``==``!=``!==`这些都是一些有趣的运算符。

让我们一起来看一些例子。`if(a == b)`这将检查两个值是否相同(可能不是同一类型)

而`if(a === b)`检查完全相同的条件，但这里类型也是一个因素。

假设`a = '4'`一个字符串& `b = 4`一个数字，我们用`==`运算符比较这两个，`if(a == b)`这个条件为真但是另一方面，`===`运算符，`if(a === b)`检查值并返回 false 导致一个字符串类型和一个数字类型不匹配！(喜欢我&我女朋友😔)

由此可见，`'33' == 33`给出真&给出假`'33' === 33`，就这么简单。

同样地`'33' != 33`假&真`'33' !== 33`

## 变量:

变量是从这里到那里变化的值。这是我能给出的最简单的定义。✊在编码中，程序员有时需要一些可变的值，他们称之为变量。但它还有另一个优点。一个变量也可以存储一个值，改变这个值&再次存储改变后的值。

JavaScript 中的变量是用`let, const , var`这些关键字声明的。`let` 用于声明块级变量，这意味着该变量只能在块内被访问&改变，但`var` 的作用就像一个全局变量，这意味着任何人都可以从任何地方访问该值。`const`是常量的简称，意思是不可改变。有些值在数学中是常数(比如 PI = 3.1416)。在 JavaScript 中，编码人员使用`const` 来声明一个值常量&，这意味着该值不可更改。示例:

`let a = 5;`

`a = 15;`

`console.log(a)`给出`15`，然而`const a = 5; a = 10;`会给你一个错误，告诉你不能给`const`操作数赋值。为了解释`let` & `var`的区别，我需要使用函数，所以请配合我，好吗？

`var a = 5; function timeTable(){a = 10;} a = 35`这里我们已经声明&初始化了我们的变量`a`为`var`，所以在函数外的最后一个语句后&的值会改变。

在同样的例子中，如果我们使用`let`而不是`var`，我们会得到同样的结果，但是如果我们像这样在函数中使用`let`

`function timeTable(){ let a = 10;} a = 100;`那么该值将不会改变，相反它会给出一个错误，导致在函数`a`之外不被识别

## 循环:

名字本身就说明了它的作用。有时程序员需要重复做一些改变&循环是他们做的过程。有不同种类的循环，如`for` `while` & `do-while`但同样也有一些 for 循环的变体，如`forEach() for()Of` `for(In )`我将只给出 for 循环的例子。

假设你要打印出`my name is Bill Gates` 1000 次，你会怎么做？写`console.log()` 1000 遍？绝对不行！因此，您使用一个简单的 for 循环来重复`console.log()` 1000 次。

`for(let i = 0; i < 1000; i++) {console.log(‘my name is Bill Gates')}`

看到了吗？你就写一行，打印 1000 行！神奇！😶
