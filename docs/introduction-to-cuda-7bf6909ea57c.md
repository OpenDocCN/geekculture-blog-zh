## 什么是 CUDA？

CUDA 代表计算统一设备架构*。* CUDA 是一种来自 NVIDIA 的异构编程语言，为通用程序提供 GPU。异构编程意味着代码运行在两个不同的平台上:主机(CPU)和设备(NVDIA GPU 设备)。
