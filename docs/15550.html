谁将赢得Web3竞赛——Twitter还是脸书？

在社交媒体的世界里，有两种主要类型的平台:基于 Web2技术的平台和基于 Web3技术的平台。虽然有许多组件组成了这两种不同类型的社交媒体平台，但的关键区别在于 Web 3.0 平台是去中心化的，而 Web 2.0 平台是集中化的。
