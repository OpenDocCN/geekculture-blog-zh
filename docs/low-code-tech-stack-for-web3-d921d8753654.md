## 几分钟内从 0 到 100。

Web3 是下一代互联网。这是一个分散应用的互联网，对开发者来说是一个新的前沿。Web3 将是一个去中心化和点对点的互联网版本，允许人们和他们的数据之间直接、安全和私密的交互。它将提供一种新的方式来构建不受任何单一实体控制的应用程序，而是由其所有用户控制。
