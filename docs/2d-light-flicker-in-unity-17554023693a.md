## 为 Unity 中的 2D 灯光增添了一种酷炫的身临其境的视觉效果

目标:拍一部 2D 轻弹

我爱上了 Unity 的 2D 灯，最后我把它们做得更酷了！

我写了一个脚本，让 2D 灯像蜡烛、坏灯泡或霓虹灯一样在 T2 身上闪烁。是的，它可以模仿很多东西，因为它是高度可定制的。并且可以与所有 2D 灯类型一起使用。
