在过去的六年里，我一直在开发我的梦幻游戏机 [Pixel Vision 8](https://pixelvision8.com) ，主要是靠自己。我为此倾注了数千小时和数百个不眠之夜。我在家里、飞机上、火车上，以及世界各地的酒店里编写代码。有时候我会忽视我的朋友、家人和我的心理健康。没有一天我不是[在修复一个 bug](https://github.com/PixelVision8/PixelVision8/issues) ，[在增加一个功能](https://github.com/PixelVision8/PixelVision8)，[在更新](https://github.com/PixelVision8/PixelVision8/wiki) …
