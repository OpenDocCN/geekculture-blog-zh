# 为什么我对 AR/VR/MR 的态度变了

> 原文：<https://medium.com/geekculture/why-has-my-attitude-towards-ar-vr-mr-changed-f9d9efc5847f?source=collection_archive---------11----------------------->

![](img/77c6672007307ef95e08173c602edff4.png)

我很喜欢 AR/VR/MR(增强/虚拟/混合现实)方向，尤其是全息图。但是五年前我喜欢它。我不记得这种爱开始的具体事件，但我记得为什么感情冷却

2016 年，我参加了 Epam IT cleanup，在那里我有幸试用了微软的眼镜——holo lens。难以形容。蜘蛛(不是真的😁)沿着我所在的房间的墙壁跑，他们必须在远处用我的手的动作杀死。这不是虚拟现实——在虚拟现实中，你画出了边界，然后，对不起各位😋这里不同——你所在的房间被扫描，物体被投射到墙上。它可以是任何东西:例如，你的笔记本电脑和文件夹。但这次有蜘蛛，但它们很有趣，所以即使是蜘蛛恐惧症患者，很可能也不会害怕

Hololens 根本买不到，只有俄罗斯的几家公司有。所以我的经历是独特而难忘的

自然，我想拥有 Hololens。我甚至曾经给自己买了一台特殊的 Windows 游戏笔记本电脑，甚至在模拟器上为这些眼镜开发。但是我找到的所有微软的教程都非常原始，对我来说还不够，我把笔记本电脑卖了

我用这些技术做了很多实验——我甚至烧了我的 iPhone——当我虚拟旅行到其他国家时，它无法承受负载🚀

5 年前，我甚至在《情报回报》上开了一次会。

当你对一件事充满热情的时候，你想用你的热情去感染别人。向该公司证明它不仅酷、有趣、好玩，而且这些技术是未来，值得朝这个方向发展并不容易。我离开几个月后，公司订购了不同的 VR 眼镜，每个人都可以在那里测试😉

一切都需要时间。把一个想法变成更多的东西

我已经习惯了这样一个事实，我的想法被回答“对我们来说太多了”，但几个月后其他人也这样做了——这给他们带来了成功。

一开始，这让我很受伤。但是当你知道最后会发生什么的时候，你就不再那么积极地去尝试证明自己的眼光了。如果你的想法没有实现，肯定会有人为此感到高兴💯

那么为什么我的“爱”变成了“喜欢”？

1.  大肆宣传已经扼杀了这些技术的发展到了它本来可以达到的程度。它们开始主要用于娱乐。在同一个购物中心。为了省钱，他们开始使用廉价眼镜，这种眼镜不能提供高质量的虚拟现实沉浸感。底线是什么？沮丧的客户说和写 VR 是扯淡。当这种想法深入大众时——没有什么好事
2.  很难或完全没有能力得到特定的眼镜。你需要为不同的系统准备几个玻璃杯——这很贵
3.  知识库不足，而且很原始
4.  俄罗斯缺乏 AR/VR 开发人员感兴趣的职位。这是游戏开发领域，工资比网络开发人员还低。如果专家在这个领域得到的报酬更少，他们为什么还要浪费时间和金钱呢？

尽管有这些缺点，我仍然相信在未来几年内会有很多变化，包括在 AR/VR/MR 领域。这是我不会错过的聚会类型😋
