**21.**slice VS splice:** slice 并不影响原始数组，它只是从数组中切出元素。
splice 用于对原始数组进行修改，要么删除一些元素，要么替换它们。**

**22.**forEach VS map:** forEach 一般用在迭代过程中不需要返回什么东西的时候。
当你想在迭代过程中返回一些东西时，可以使用这个映射。**

****如无必要，不要到处使用 map 进行迭代。****

**23. **in 运算符** 您可以使用“in”运算符来检查数组中元素的索引是否存在。**

**24.**Getters&Setters** 在 Javascript 中大多数时候，我们编写自己的 getter 和 setter 方法来对属性进行一些验证，但代替这种做法，我们可以使用 javascript getter 和 setter 方法。**
