# YouTube 真的是一个好的收入来源吗？

真的有可能从中获利吗

如今，我们都能看到制作这些视频的人在 YOUTUBE 上讲述如何赚钱和变得富有。他们还分享了自己如何从 YouTube 上赚了这么多钱的成功故事。但也有很多人在 YouTube 上分享他们为赚钱而奋斗的故事。因此，人们对于是否能从 YouTube 上赚很多钱也很困惑。

有许多人离开他们的工作，成为一个全职的 YouTube，许多青少年和孩子都渴望在长大后把制作 YouTube 视频作为一份工作来挣。

YouTube 无疑是一个谋生的好地方。虽然一些创作者只能触及表面，但也有许多内容创作者能够每年赚取 7 到 8 位数的收入，并创造出令人难以置信的成功的数字房地产资产。但是要真正理解人们如何从 YouTube 上赚钱，我们需要知道它实际上是如何工作的。

首先，你可以创建一个 YouTube 频道。然后你需要加入 YouTube 合作伙伴计划。但问题是，加入这个项目有一些资格要求。

首先，你的频道需要有至少 4000 小时的观看时间，你还需要有超过 1000 个订阅者才能开始从不同的 YouTube 货币化功能中获利。一旦你符合资格要求，你就可以申请一个 AdSense 帐户。

现在，为了开始赚取广告收入，你需要打开广告。Youtube 通过预滚动、展示和其他广告形式盈利。广告商根据点击和印象付费。YouTube 将这部分收入的 55%给内容创作者，自己拿走 45%。

加入合作伙伴计划后，人们可以开始配置新的可用货币化方法，如推出商品、使用超级聊天和超级贴纸等。

所以，这些都是赚钱的步骤，然后真正开始赚一些钱。而且，你会看到这个平台的名字在关于在家赚钱的文章中上升到了顶端。然而，虽然你肯定可以在 YouTube 上赚钱，但这个目标通常不会以每个人都希望你相信的传统方式实现。

因此，YouTube 的神话是这样的:发布一些视频，吸引观众，并从广告收入中获得一些现金。这听起来简单而不费力，所以这是每个人都反复提到的故事，并用来推销快速致富计划。但现实却大相径庭。即使广告商通过视频广告支付较低的费用来推广他们的产品，也只有他们支出的一部分进入内容创作者的口袋。例如，如果广告客户为每 1000 次广告展示支付平均 20 美元，则显示这些广告的视频可能每 1000 次观看仅产生 2 或 3 美元。

换句话说，每 100 万次浏览，你只能赚到几千美元。毫无疑问，获得数百万的浏览量是非常具有挑战性的。嗯，在 YouTube 上赚钱没那么容易。如果有人期望成为百万富翁并在一夜之间获得成功，那几乎是不可能的。

YouTube 上的许多创作者已经奋斗了这么多年，才真正开始赚钱。也有许多 YouTube 用户正在赚取数百万美元，是世界上最富有的 YouTube 用户之一，但这不是一个月甚至一年内发生的，而是花了近十年或二十年才达到今天的水平。

另一件人们在开始自己的频道之前没有真正考虑的事情是制作他们的视频的主题。YouTube 是一个开放的平台，几乎任何人都可以开始制作视频，并在 YouTube 上发布他们选择的几乎任何主题的视频。人们可以制作他们喜欢的视频。人们可以在任何东西上制作视频，从游戏视频到生活视频。

现在出现的主要问题是，开办和运营一个 YouTube 频道需要多少成本。这个事情也取决于一个人决定制作什么样的视频，但仍然需要花费很多。

任何人都可以猜到，相机是制作 YouTube 视频的最重要的设备，但这不是它，还有更多设备和小工具需要这样做。而很多刚入门 YouTube 的新 YouTube，更喜欢用手机拍视频，但大多数时候不值得。即使你的手机有最好的相机，它也不能打败真正的相机。然而，一个便宜的相机可以完成这项工作。

其他设备，如麦克风、灯光、三脚架、SD 卡、编辑应用程序等。所有这些东西都要花费数千美元。

观众和 YouTube 都希望他们的创作者更加关注视频的质量和内容。但这也可能会耗费大量时间。这也是制作 YouTube 视频的一个巨大缺点，它消耗了大量的时间来思考制作视频的不同想法，然后拍摄，最后编辑。这就是为什么大多数 YouTube 用户更愿意成为全职 YouTube 用户的原因，因为很难在处理 YouTube 频道的同时完成一项工作。

那么，成为一名全职优步真的值得吗？答案真的取决于每个人。如果一个人真的对 YouTube 充满热情，并准备投入一些时间和金钱制作 YouTube 视频，并能在其中持之以恒，那么答案是肯定的。但是，从 YouTube 上谋生仍然需要很长时间，没有人能保证需要多少时间才能赚到想要的钱。由于 YouTube 是非常不可预测的，有时你可能从一个视频中赚到数百万美元，有时你甚至可能在数百万次观看后赚不到几千美元。
