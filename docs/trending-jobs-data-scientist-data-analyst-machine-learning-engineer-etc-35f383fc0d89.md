## 你想成为 DS 或 DA 或 ML 工程师的原因？

回答:有人会说，因为它是市场的趋势，有人会说，在未来，它将是最需要的工作，有些人会回答，因为它的工资比其他 it 工作高，有些人对这些领域充满热情。

## 谁真正受益于这种就业趋势？

每个刚毕业的人，都争先恐后地选修市场上大量的课程。一些教育机构非常关注好的主题和职位所需的技能，并将它们传授给参加课程的学生。

虽然有些机构真的是在利用这些课程做生意，并虚假承诺他们也能保证就业。出于困惑，一些学生相信这些，并失去了大约 50k-70k，没有得到安置。我建议你不要陷入一个安置机会的诡计和战术。

**我最近听到的一个关于安置的真实例子:**

一个计算机专业毕业的人想成为一名数据科学家或分析师，但不知道该做什么，也不知道需要什么技能。有人建议成立一个提供实习和培训的机构。所以他付了钱，有人给他材料让他自己准备。研究所的人进行了一些视频电话采访，电话采访。这让他相信了学院，他等了近 8 个月，但没有得到安置。现在该机构改变了规则，告诉他先通过测试，然后他们将参加面试。由于他没有其他选择，所以他同意进行测试。考试是所有有经验的招聘问题，但他试图回答它们。测试结果出来后，管理人员告诉他，他没有通过测试，因为他没有达到截止百分比。他打电话给他们说，测试是为有经验的人，但他们不同意他说的话，他们对他说的话非常粗鲁。

现在，他因为找不到工作而沮丧，浪费了 8 个月的时间去相信研究所。听了他的故事，我感到很难过。

所以不要相信这样的诈骗机构，不要浪费你的金钱和时间。首先，确定你对什么有热情，并就此进行研究。在谷歌上有很多免费的资源，如果你想参加一个学院的课程，寻找内容和实用知识，不要寻找就业援助。如果学院也提供帮助，那么你是非常幸运的。

据报道，提到的工作是趋势性的，它们是未来的一部分。但是有些人不知道成为 DS、DA 或 ML 工程师到底需要哪些技能。

## 那么成为一名 DS、DA、ML 工程师需要具备哪些实际技能呢？

关于成为一名 DS、DA 和 ML 工程师需要哪些技能，现在有很多困惑。无论你去哪里从事规定的工作，技能都应该是一样的。所以经过大量的研究，我明确了到底需要什么样的技能。

—让我们开始描述每一个:

首先，让我们从数据分析师开始:

**数据分析师:**

数据分析师从不同来源收集数据，如财务数据、销售数字、物流成本，并使用统计技术分析结果。他们的目标是为组织提供有见地的报告。

**数据分析师所需技能:**

统计数据

— R & Python 编程语言

—数据争论

—数据预处理

—阿帕奇猪和阿帕奇蜂房

—准确性和对细节的关注

—微软 Excel

—人际交往和团队工作技能

## 数据科学家:

数据科学家通过分析、探索和处理数百万的数据来解决公司带来的问题。

## 数据科学家所需的技能:

—如何编程，尤其是 python 编程

—计算思维

—代数和分析

几何和微积分

概率和统计

—关系数据库，如 SQL

—阿帕奇火花

—机器学习算法

—演示和讲故事的技巧

## 机器学习工程师:

根据 Arthur Samuel 在 1959 年的说法，ML 被定义为一个研究领域，它赋予计算机在没有明确编程的情况下进行学习的能力。

根据 Tom Mitchell 在 1998 年的说法，机器学习被定义为

—如果一个计算机程序在 T 上的性能(由 P 测量)随着经验 E 而提高，则称该程序从关于某个任务 T 和某个性能测量 P 的经验 E 中学习。

## ML 工程师所需的技能:

—数学原理

—像 Python 和 C++这样的编程语言

—统计建模

—数据评估和建模

—对神经网络有深刻的理解

—计算机视觉和图像处理

—自然语言处理

—分析和调试技能
