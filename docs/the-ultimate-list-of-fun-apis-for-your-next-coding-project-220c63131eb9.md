作为程序员，当我们完成工作时，我们喜欢，嗯，编程。从黑客马拉松到兼职项目，我们喜欢在工作之外构建东西。不过，有时候很难想出下一步该做什么的点子。当这种情况发生时，学习新的 API 来获得灵感通常是有帮助的。这就是为什么我列出了你可能没听说过的有趣 API 的终极列表。希望检查这些 API 能给你一个有趣的副业项目的下一个想法。或者谁知道呢，它可能会成为下一个 Instagram！

# 什么是 API？

如果你不熟悉 API，API 代表应用程序编程接口，是两个或多个软件进行通信的一种方式。例如，谷歌有一个 API，可以让你获取地图信息，Twitter 有一个 API，可以让你以编程方式发送推文。通常，一个 API 通常有文档让你知道如何向它请求信息。

有数以千计的 API 用于各种不同的目的。在这个列表中，你会发现 API 可以做任何事情，从根据名字猜测一个人的年龄到让你交易股票。所以，事不宜迟，这里有一些很棒的 API，您可以在下一个编码项目中使用！

# 有趣的数据

## 1.NASA 开放 API

NASA 为公众提供了一堆超级令人兴奋的 API。他们有一个名为“每日天文图片”的 API，每天从太空返回一张新图片，一个用于获取火星天气的 API，以及一个仅用于火星漫游者照片的 API。

[了解更多](https://api.nasa.gov/index.html)。

## 2.SpaceX API

说到太空，你知道埃隆·马斯克的 SpaceX 有 API 吗？您可以使用 SpaceX API 获取发射、火箭、核心、太空舱、Starlink、发射台和着陆台数据。

[了解更多。](https://github.com/r-spacex/SpaceX-API/blob/master/docs/v4/README.md)

## 3.其他国家

使用 REST Countries API，您可以获得世界各国的信息。例如，您可以发送一个查询来获取一个国家的语言、货币和资本 REST Countries API 是免费的，由捐款支持。

[了解更多。](https://restcountries.eu/)

## 4.开放天气 API

如果你想将天气融入你的应用程序，你可以查看开放的天气 API。你可以使用他们的 API 获取超过 20 万个城市的天气信息。您还可以使用 API 来访问历史天气数据。

[了解更多。](https://openweathermap.org/api)

## 5.多边形. io API

你的下一个项目需要财务数据吗？您可以从 Polygon.io API 获得各种财务数据，如股票报价、历史数据等等。他们还有一个获取加密货币信息的 API。

[了解更多。](https://polygon.io/docs/)

## 6.美联社 API

你想把新闻整合到你的下一个应用中吗？看看美联社的 API 就知道了。他们的免费 API 可以让你实时获取新闻。

[了解更多。](https://developer.ap.org/)

## 7.COVID19 API

COVID 19 API 让您可以访问各种冠状病毒相关的数据。也许你可以建立一个项目，这将有助于在疫情拯救生命！

[了解更多。](https://covid19api.com/)

# 有趣的 API

## 8.Agify API

Agify API 是一个有趣的 API，它试图根据你的名字来预测你的年龄。你不需要一个 API 键来开始，所以你只需要发送一个包含你名字的请求，就可以知道他们的算法认为你多大了。

[https://api.agify.io/?name=cory](https://api.agify.io/?name=cory)

[了解更多。](https://api.agify.io./)

## 9.口袋妖怪 API

使用 Pokemon API，您可以发送一个带有 Pokemon 名称的 rest 请求，它将向您发送一个包含它们信息的 JSON 响应。例如，以下是发送 Charizard 请求的方式:

[https://pokeapi.co/api/v2/pokemon/charizard](https://pokeapi.co/api/v2/pokemon/charizard)

像 Agify API 一样，您不需要注册 API 密钥。您可以在浏览器中直接输入请求，然后得到响应。

[了解更多。](https://pokeapi.co/docs/v2)

## 10.Kanye Rest API

Kanye Rest 是一个免费的 REST API，用于随机 Kanye West 报价(也称为 Kanye as a service)。每次你访问端点[https://api.kanye.rest/](https://api.kanye.rest/)，API 都会返回一个新的 Kanye 引用，比如“如果一个杯子满了，你就不能看着它是半满的还是空的。”该 API 可以免费使用。文档建议使用它来增加 g it 提交消息的趣味，但是您可以用这些数据做的事情是无限的。

[了解更多。](https://kanye.rest/)

## 11.无聊的 API

有了百无聊赖的 API，你再也不会无聊了！当你向 Bored API 发送一个请求时，它会返回一个有趣的活动让你去做。

[了解更多。](https://www.boredapi.com/)

## 12.趣味翻译 API

有趣的翻译 API 允许你在一个计划下使用几种语言 API。您可以访问的许多语言 API 包括 Yoda API(将英语翻译成 Yodish)、Pig Latin API 和翻译多斯拉克语(来自《权力的游戏》)的 API。

[了解更多。](https://funtranslations.com/api/)

## 13.Balldontlie API

Balldontlie API 允许您访问 NBA 相关数据。例如，您可以获得所有 NBA 球队的列表、特定比赛的信息、球员统计数据等等。我是一个超级篮球迷，所以我可能要自己用这个 API 开始一个副业项目。

[了解更多。](https://www.balldontlie.io/#introduction)

## 14.Genius API

Genius 是一个允许任何人给歌词和其他文字添加注释的网站。例如，您可以给一首歌曲添加注释，解释为什么一首诗很重要。使用 Genius API，您可以获得关于歌曲的数据，包括歌词和注释。

[了解更多。](https://docs.genius.com/)

# 新技术

## 15.比特币基地数字货币 API

比特币最近创下每枚 5 万美元的历史新高，引发了人们对加密货币的新兴趣，所以现在是开始加密货币相关副业的好时机。您可以使用他们的 API 来检查加密货币的价格，购买它，等等。

[了解更多。](https://developers.coinbase.com/)

## 16.QR 标签 API

需要将二维码整合到您的下一个项目中吗？您可以使用 QR Tag 的 API 快速创建 QR 码。

[了解更多。](https://www.qrtag.net/api/)

## 17.云自然语言 API

谷歌的云自然语言 API 是谷歌机器学习 API 家族中众多 API 之一。“云自然语言 API”向开发者提供自然语言理解技术，包括情感分析、实体分析、实体情感分析、内容分类、语法分析等

[了解更多。](https://cloud.google.com/natural-language/docs/)

## 18.Clarifai API

我想强调的另一个 API 是 Clarifai API，它可以用于机器视觉。开箱即用，API 可以识别超过 11，000 个概念，如狗、猫等。这项技术太棒了，我可以想象你可以用它来创建许多不同的项目。

[了解更多。](https://docs.clarifai.com/)

# 其他的

## 19.Eventbrite API

Eventbrite API 让您可以找到不同的事件和关于它们的信息，比如它们的容量、价格等等。由于我最近成为了一名开发者倡导者，并试图在尽可能多的事件中发言，我开始考虑自己使用这些数据的不同方式。

[了解更多。](https://www.eventbrite.com/platform/api)

## 20.快速图表 API

辅助项目总能从图表中受益。无论你的副业是什么，添加一些图表总是很有趣的。您可以使用 Quickchart API 将条形图、饼图等快速添加到下一个项目中。

[了解更多。](https://quickchart.io/)

## 21.一副纸牌 API

在我的编程入门书和 Udemy 课程中，我依靠教有抱负的程序员如何创建游戏来保持他们的兴趣。构建不同的纸牌游戏是发展你编程技能的好方法，尤其是如果你在下一步做什么上遇到了障碍。卡片组 API 允许您在程序中实现一组卡片，而无需自己编写代码，对于新程序员来说，这是练习使用 API 的一个很好的方式。

[了解更多。](https://deckofcardsapi.com/)

## 22.网站截图 API

网站截图 API 允许你以编程的方式从网站上抓取截图。我不知道你会用它做什么，但我觉得用它来构建一个有趣的兼职项目有很大的潜力。

[了解更多。](https://screenshotapi.net/documentation)

## 23.开放库 API

开放图书馆 API 将为你提供所有你需要的书籍信息。他们有一个用于书籍、封面、书单等等的 API。

[了解更多。](https://openlibrary.org/developers/api)

## 24.GitHub 作业 API

您可以使用 GitHub Jobs API 来获取 GitHub Jobs board 上所有作业的信息。

[了解更多。](https://jobs.github.com/api)

## 25.应用编程接口

有了这个 API，我把最好的留到了最后！我最近成为了 Vonage 的开发者拥护者，我从使用我们的 API 中获得了极大的乐趣。Vonage 有一个 SMS API，可以让你只用几行代码就能发送文本消息，还有一个语音 API，可以让你打电话，还有一堆其他很棒的 API。我最近使用 Vonage 的 API 创建了一个名为 Songagram 的附带项目，你可以选择一首歌并给出一个朋友的电话号码，该程序将抓取歌词，给你的朋友打电话，并给他们读歌词。

[了解更多。](https://www.vonage.com/communications-apis/)

# 最后的想法

希望这个列表能给你下一个编码项目一些启发。在研究了这个列表的 API 之后，我对云自然语言 API、比特币基地数字货币 API、美联社 API、EventBrite API，当然还有 Vonage APIs 感到特别兴奋。我感兴趣的是建立一些结合了它们的东西。也许是一个使用云自然语言 API 分析新闻，并根据其情绪交易加密货币的程序。或者，我可能会使用 Eventbrite API 来编写代码，以找到我可能想在其上发言的技术事件，并在提交我提议的发言时使用 Vonage 的 SMS API 用文本提醒我。

无论您是像我一样对相同的 API 感到兴奋，还是对列表中的其他 API 更感兴趣，有了所有这些优秀的 API 可供选择，对于程序员来说，这是一个令人兴奋的时刻！

祝你的下一个编码项目好运！
