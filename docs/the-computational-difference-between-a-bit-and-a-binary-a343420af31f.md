## 一位和二进制在算术上有什么不同

![](img/b76f23214513651eb598664f4596514c.png)

在计算机科学中，一位是 0 或 1，而二进制是 0 和 1。

# 什么是一点？

一位不是 0 就是 1。

它是二进制通用集{0 和 1}的子集。
