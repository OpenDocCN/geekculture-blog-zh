# 你能用厨房用的油管煮浓缩咖啡吗？

## 咖啡数据科学

## 出于无聊的怪异实验

去年感恩节，当我在火鸡上涂黄油时，我想“也许吧？！?"我觉得用油管煮咖啡会很有趣。你在管子里有压力，也许可以学到一些东西。或者可以找点乐子。

学到的教训是“不”。然而，我提出我失败的尝试，以防你曾经看着一个巴斯特，也说也许。

![](img/93226f57ac11233f0d49c194047238f8.png)

All images by author.

很简单，但是我必须在底部放一个塞子，这样咖啡就不会流出来了。我用了一个小滤纸。

![](img/e6861a8fec31a66e9b7bbb64398ff841.png)

然后我像往常一样在壁龛上的 7 号台磨咖啡。我用了 10g 左右，注定失败。

![](img/c7de954dc1c1163dd8ae094ebd5a0556.png)![](img/3986ed6b03db9680bec21f76717d148b.png)

我把它设置成一个 Kruve EQ 杯，因为当使用一个临时的咖啡机时，你需要杯子。

![](img/838e7ca993899d9e6e34fa315df79f94.png)![](img/21ea0428e0d5657d56ca90d04abee8b8.png)

结果令人惊讶:

![](img/5e4fbadd62dd85ce3dada145ad8690f7.png)

# 开个玩笑！

这是一场灾难。水花了很长时间渗透，并且油管的底部形成了一个阻塞点。

![](img/d4af38c2894bdea33ee0ca52236e9db4.png)![](img/794a0c2485d750b14c52358d66706202.png)

所以让我们再试一次:

![](img/d1b5d98b77b1b13961a108b55a4de965.png)

我用的是 7g 而不是 10，我去小众上设置了 15。这做得稍好，但不多。

因此，我切换到设置 25，这是非常粗糙的，1g 在底部，然后设置 15 为 3g 在顶部。

![](img/1f0cb74cbb798ed95f0b3d659ff4886c.png)![](img/3acb41680ff7f7d3e0a0837564b72cec.png)![](img/35da7a95726b4880e713860705d9e30b.png)![](img/703a69832739cf967ca0e22dd6290f85.png)![](img/25ad6cdd64d3bc2ce781b6769e27eeb5.png)![](img/00b4dbee8d71452755149aac56731bac.png)

咖啡基本上喝完了，但还是很失败。涂油器的压力不能太高。当然，水是热的，但我没有测量它是肯定的。

![](img/f0d34e5c05be3bb62e30f1393ca4fe1f.png)![](img/e0b2fc0758afcc9b60386d69ae9dca26.png)

咖啡很油，最后，里面有很多磨屑。

我没有品尝这个镜头，甚至没有测量 TDS。这肯定会很糟糕。就像科学一样，当一个假设失败时，你仍然会从实验中获得知识，即使这些知识是没有人要求的。
