在概率论和统计学中，参数为 n 和 p 的二项式分布是一系列 n 个独立实验中成功次数的离散概率分布，每个实验都提出一个是或否的问题，并且每个实验都有自己的布尔值结果:成功是概率 p，失败是概率 q = 1-p。
