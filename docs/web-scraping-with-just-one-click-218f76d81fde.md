# 1.抓取 Youtube 视频

假设你是一个 Youtube，你想分析 MrBeast 的 Youtube 视频。

有了 Listly，你可以一键完成。

![](img/abec7ad012aba6ed87664321a0f70a06.png)

点击列表整体

![](img/b0e0cb4418b023b09e9bc97999fc0b6b.png)

然后，您将被带到 Listly 的数据提取页面。

![](img/14313b2b6c0e6d18b825f531768feb73.png)

请注意，提取了几个选项卡，18 张卡选项卡看起来不正确，让我们检查一下有 120 张卡的选项卡。

![](img/9f2f5789d41e261d4adfec60da7d5968.png)

现在你知道了！120 个视频细节全刮了。

不过 MrBeast 绝对不止 120 个视频，还是用 Listly 的自动滚动功能吧！

进入数据板，点击设置。

![](img/d41f36965f1fb2d01ae9f6ac817d3df7.png)

找到自动滚动部分，并将其设置为您想要的设置。让我们试 20 次。

![](img/b0d68b7a1a0e46a5411cddc3415134a6.png)

然后，单击刷新，Listly 将再次抓取网站以获取更多数据。

![](img/d9e75f102162b5523f7be847e9040f33.png)

现在我们等着。

![](img/4b6ad8e352540acf02b6ffa2969f8015.png)

完成后，点击`Latest`

![](img/4dc96b99057b0a3d64fea38e90601372.png)

瞧啊。我们收集了 721 份他的视频资料！

![](img/32714d7556e0d282c45a6b0f9b8375c6.png)

取消选择排列单元格，用 Excel 下载数据。

![](img/017fe84f370d1d68a1f486cfa4e97748.png)

这是我们得到的前 100 行数据！

你也可以选择导出到 Google Sheets，甚至 API(测试版)。

让我们来看看 API！

![](img/f0c653bf13ed07a3249f597f3e65e4c1.png)

下次再想刮 MrBeast 的视频，直接调用 API 就行了！

你可能对某个特定的主题感兴趣，而不是某个特定的频道，例如数据科学。

![](img/ea6a14f6dfbae0e6741a07d05891e01d.png)

想瞬间刮掉所有结果？Listly 支持你！

![](img/4d3979b671e52ac653f4ccd3ea15c2f8.png)

# 3.刮阿迪达斯运动鞋

假设你是个球鞋头，你想把阿迪达斯在卖的球鞋都刮了。

![](img/367ceb81b8d6efef90b2b3950a826a0d.png)

[adidas.com](https://www.adidas.com/us/men-athletic_sneakers)

单击列表整体。

![](img/d346d75c2b6e3f46bfa38abcac56ccd7.png)

我们从阿迪达斯刮了 48 双球鞋！

但是向下滚动，你会看到这个。

![](img/477a81116f46d27750ff8c3b3c7db86b.png)

总共有 18 页，你可能会想，Listly 也能把它们刮下来吗？

答案是肯定的。

以下是方法。

当我们切换到第 2 页时，网址是:

```
[https://www.adidas.com/us/men-athletic_sneakers?start=48](https://www.adidas.com/us/men-athletic_sneakers?start=48)
```

保存这个。

现在单击红色组按钮。

![](img/f4a53c8af9427c4837d7ceb0fe7616b0.png)

我们现在可以做的是将之前的 URL 粘贴到设置中的添加 URL 部分。

注意到你只需要改变起始号码了吗？让我们在接下来的几页中这样做。

![](img/bb497a090acd5c44ebf8a8d2cd7d7ceb.png)

然后，点击提交按钮。

![](img/7c25d6369acd12493302ae581f3a29d4.png)

然后 Listly 会开始刮。

点击它了解更多关于刮擦过程的细节。

![](img/4624646d4ad5af7d211d9707bd98d991.png)

完成后，点击`Group Excel`

![](img/30db5ccbd24de4e43ae097dd5cd0efbd.png)

这里是最终数据！
