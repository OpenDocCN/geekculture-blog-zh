# 一个世界支付之战

> 原文：<https://medium.com/geekculture/the-battle-for-one-world-payments-75f0bb71a5f4?source=collection_archive---------46----------------------->

只用了两年时间，Venmo 就以超过 2500 万美元退出。一年后，它价值 8 亿美元。该公司成立于 2010 年，当时比特币的售价为 0.08 美元。今天，Venmo 是 PayPal 的一部分，比特币徘徊在 40，000 美元左右。

不管 Satoshi 打下的基础如何，crypto 仍未被接受为主流支付方式。反而成了投机的手段。同时，从理发师到律师，每个人都接受 Venmo。包括一群 20 多岁的年轻人分摊餐馆账单。
