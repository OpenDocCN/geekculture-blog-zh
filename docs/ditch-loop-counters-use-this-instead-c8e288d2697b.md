用 enumerate 编写更多 pythonic 代码

![](img/03c92241319b69774e9e04342ea8f530.png)

尽管避免循环计数器通常是最好的，但有时它们还是有用的。

幸运的是，你不必忘记到目前为止所学的一切——你可以使用`enumerate`功能。这使我们能够访问 Pythonic for 循环中的循环计数。
