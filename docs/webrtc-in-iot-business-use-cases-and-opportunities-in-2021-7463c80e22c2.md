# 物联网中 WebRTC 的用例

从音频和视频通话到消息传递， [WebRTC](https://mobidev.biz/blog/webrtc-app-development-challenges-use-cases-future) 在物联网中的使用分为两类:设备到云和设备到人。

被视为设备到人的 WebRTC 的典型用例与监控摄像机、婴儿监视器和视频门铃有关。WebRTC 允许用户看到或听到另一边发生的事情。

设备到云涵盖了自动化案例。例如，WebRTC 和 IoT 的结合适用于无人机识别故障并提供全面控制的任何领域。想象一下，一家发电公司可以使用无人机检查电塔和电线杆的状况，并将视频流发送到云端。在这里，WebRTC 确保了流程的实时流动。

**WebRTC 在物联网中的主要应用领域包括:**

*   监督
*   共享 AR
*   智能交通
*   远程医疗和健康跟踪工具
*   智能环境
*   生产流程

现有的 WebRTC 用例可以变得非常多样化，但我们将概述最有前景的领域——智能家居、医疗保健、可穿戴设备和工业物联网。

# 面向智能家居的 WebRTC

这是 WebRTC 最赚钱的领域之一，同时也是最相关的[物联网趋势](https://mobidev.biz/blog/iot-technology-trends)之一。例如，利用 WebRTC 与 web 和移动应用程序进行音频/视频通信的门对讲设备或智能邮箱。在前一种情况下，智能家居的居民能够与访客交流，确保没有紧急情况被忽视。

我们还可以提到用于安全访问的智能身份验证解决方案，其中涉及 WebRTC 和基于机器学习的生物识别验证技术。

# 医疗保健中的 WebRTC

根据 Grand View Research 的研究，到 2028 年，远程医疗服务市场预计将在未来 7 年内呈现 22.4%的复合年增长率。所以它可能会成为 WebRTC 的一个主要领域。

通信加密使得 WebRTC 对医疗保健提供者很有吸引力，因为他们关心保护病人的个人健康数据。在美国，健康数据在法律层面受到 HIPAA 的保护，任何相应的软件产品都必须符合其标准。

**实时视频可应用于:**

*   电话会议医患预约
*   远程治疗
*   集体治疗电话会议
*   临床会议
*   手术室的远程观察

上述案例已被许多健康保险计划所接受，以降低成本。

71%的美国患者在疫情开始时考虑过[远程医疗](https://mobidev.biz/blog/technology-trends-healthcare-digital-transformation)，50%的患者已经使用了虚拟预约。

移动应用商店已经有大约 500 个使用 WebRTC 的远程医疗应用程序。虽然不完全是物联网案例，但无疑值得一提。

至于更直接的物联网开发案例，我们可以提到智能床边设备，如生命体征监测仪，它们与位于家中的医疗站进行通信，帮助医生不分昼夜随时检查患者的身体状况。由于疫情，这些问题变得更加重要。

# 工业 WebRTC

跟随制造业的物联网趋势，工业企业在技术和创新方面仍然相当保守。然而，不断变化的业务环境决定了对新解决方案的需求，这些新解决方案正由领先的企业缓慢但稳步地引入其核心系统。

例如，我们可以命名触发或增强视频馈送的应用程序。智能工厂可以利用技术通过传感器监控和指导自动化流程。例如，如果恒温器读数显示机器可能过热，它可以触发摄像机远程监控机器，以实时检查其物理状态。另一个例子是远程支持服务，它可以使用基于 WebRTC 的视频呼叫进行紧急设备故障排除。

# 基于 WebRTC 的远程协助

如果你想知道它在现实生活中是如何工作的，就别再找了。这是一个由我们的团队创建的演示，展示了基于 WebRTC 的实时通信的功能，通过共享增强现实的方式得到增强。

# **物联网中有两个 WebRTC 的例子来说明**

第一个项目设想为一家在线购物和物流领域的公司创造物联网产品。它致力于实施一种解决方案，帮助快递公司和郊区居民远程收发快递。该设备安装在用户的室外。由于 MobiDev 提供的移动应用程序，郊区居民每天大部分时间都在通勤或工作，他们可以远程解锁智能邮箱，以便在快递员的帮助下随时收发快递。移动应用程序包含 WebRTC，用于用户和快递员之间的实时通信。

第二个项目与智能对讲机有关。这种对讲机包括屏幕，适用于入口处安装有单一门禁系统的大型办公楼。上班族用它来进入室内。当第三方访客到来时，他们输入橱柜的编号，之后系统与特定橱柜的终端连接。这个柜子的工作人员可以看见访客，打开入口，让客人进来。系统跟踪所有这些事件。WebRTC 在这里的作用变得至关重要，因为它为用户提供了实时交互的可能性。

# 在面向企业的物联网中使用 WebRTC 的优势

*   增加市场份额
*   经常收入的增长
*   服务现代化

尽管有这么多好处，但是 WebRTC 与物联网的集成也有它的挑战。设备具有不同的需求和功能，这使得互操作性变得困难。跨平台通信通常是一项重要的任务；然而，如果没有 WebRTC，这将变得更加不可能。

# 在物联网应用中使用 WebRTC

总而言之，WebRTC 提供了许多增强物联网应用程序开发的功能。

*   高清视频通信:WebRTC 有助于在浏览器之间实现安全的音频/视频流，包括音频和视频。
*   移动到移动的连接:由于移动应用程序可以使用 web 接口，WebRTC 不局限于在桌面 web 浏览器上运行。Android 和 iOS 都有原生库。
*   机器对机器的连接:当谈到物联网时，WebRTC 可以与智能设备集成，以实现所需的功能。
*   消息传递和文件共享:WebRTC 建立安全连接，共享数据，并通过直接交付数据消除了在云中或数据中心存储文件的需要。
*   电话到浏览器的连接:WebRTC 可以在公共交换电话网和浏览器之间建立连接。HTML5 API 和 SIP 网关用于从单一位置进行呼叫。

WebRTC 在物联网中的范围将会扩大。从远程医疗服务到智能家居和智能办公室，某些行业的发展证明了这一点。
