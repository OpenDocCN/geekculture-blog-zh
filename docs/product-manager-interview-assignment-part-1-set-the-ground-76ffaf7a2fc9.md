# 概观

Blueground 的业务是从业主那里租赁房屋，对其进行翻新和装修，然后以更高的价格将其转租一个月以上。一小部分额外收入来自出售家具，用于翻新他们的房子。

## 1.1 SWOT 分析

*(SWOT 是快速提供概述的好工具。下面的 SWOT 分析是部分的，每一节只有几个点。
看公司的资产负债表也可以得到非常有趣的信息；然而，对大多数人来说，阅读并不容易。)*

**优势**

*   安心:顾客得到一个设备齐全、现代化和干净的公寓。
*   比酒店便宜。
*   专有技术来管理一个人的住宿。(我假设所有软件都是内部开发的，没有第三方的标签)。
*   管理运营(翻新、装修、清洁等)的专有技术。)(假设同上。)
*   翻新和装修的规模经济。
*   品牌形象。

**弱点**

*   非常依赖商务客人(这是假设，不一定是真的)。
*   高于全装修房屋的平均租金[1]。
*   商业模式需要高入住率才能盈利。每个新房产的高边际成本。

**机遇**

*   由于 Covid 和在家工作，人们离开了市中心(WFH)。
*   房价上涨可能会提高利润率，因为公司可以提高新租金的价格，同时向房东支付相同的租金。房价下跌可以分别被视为一种威胁。

**威胁**

*   整个酒店行业的主要威胁是 Covid，对商务旅行产生了严重影响，有效地削减了公司收入。
*   关于租房的规定，如最高房租上限(在被高等法院驳回之前，柏林就是这种情况)。
*   脱媒:房东给 Blueground 的原始租金到期后，转租人可以直接联系房东，为双方达成更好的交易，无需 Blueground。

## 1.2 目标客户群

写下你认为这家公司的潜在客户是谁，这一点很重要。它将帮助您思考新功能、创建人物角色和发现客户难题。)

*   外派人员&长期外派的公司高管(一个月以上)。
*   数字游牧者。
*   离异人群(典型的是短时间内需要新房子的男性)。
*   学生(大概就是在贵的院校读的能付得起房租的那种)。

## 1.4 制约因素

*(简要介绍公司面临的主要制约因素。以后可能会用到它们，以便检查各种想法的可行性。你也可以在这里使用假设，因为这只是一个面试任务。)*

**蓝地处理实物:**

*   每一个新房产都有很高的边际成本。我预计一套新房产只有在租了 9-12 个月之后才能盈利。
*   我们需要本地运营和人员来扩大规模。
*   我们需要在一个城市拥有最少数量的房产，以证明当地办公室的成本是合理的。

## 1.5 战略

最后，在这一点上，最好能知道 Blueground 选择了什么样的战略来实现它的使命。由于该策略不是本次作业的一部分，我将做一些假设:

*   使命宣言(即我们的愿景，我们希望该组织成为什么样的组织):创造一个客人喜欢的技术驱动的居住体验。
*   战略(即如何实现愿景的总体计划):
    —进军高端市场。瞄准正在找房子的最富裕的人群，提供利润率较高的额外服务。Blueground 不想与其他带家具的公寓在价格上竞争，因为这将不可避免地缩小利润率。高收入者通常是高消费群体。
    —利用技术最大限度地减少开支，从规模经济中获益，自动化内部流程，改进或外包维护。
    —利用技术提供无与伦比的客户体验。创建最先进的移动应用程序，与供应商集成，使用预测分析来预先发现客户需求。
