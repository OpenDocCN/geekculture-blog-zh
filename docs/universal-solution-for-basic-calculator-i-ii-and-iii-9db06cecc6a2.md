# 基本计算器 I、II 和 III 的通用解决方案

> 原文：<https://medium.com/geekculture/universal-solution-for-basic-calculator-i-ii-and-iii-9db06cecc6a2?source=collection_archive---------24----------------------->

在过去的 6 个月里，脸书问了这个问题 48 次，你准备好了吗？

实现一个基本计算器来计算一个简单的表达式字符串。

表达式字符串只包含非负整数、`'+'`、`'-'`、`'*'`、`'/'`运算符，以及左`'('`和右括号`')'`。整数除法应该**向零方向截断**。

你可以假设给定的表达式总是有效的。所有中间结果将在`[-231, 231 - 1]`范围内。
