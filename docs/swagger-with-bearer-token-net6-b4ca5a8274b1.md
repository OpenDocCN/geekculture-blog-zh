Swagger 是一种描述 REST APIs 的语言无关规范。它允许计算机和人类在不直接访问源代码的情况下理解 REST API 的功能。其主要目标是:

*   最小化连接解耦服务所需的工作量。
*   减少准确记录服务所需的时间。
