## 什么是 Plotly Dash？

Dash 是 Plotly 开发的一个 python 框架，用于创建基于 web 的交互式应用程序。Dash 写在 Flask，Plotly.js，React.js 上面，有了 Dash，我们就不用学 HTML，CSS，Javascript，只需要懂 python 语言就可以了。使用 Dash 构建的应用程序可以在 web 浏览器上查看。
