# 鲍威利农场迎合了对可持续生产的需求

Bowery Farming 因其在 Whole Foods 等商店出售的光滑包装的生菜和混合蔬菜而闻名。随着对可持续农产品的需求激增，该公司已经筹集了 3 亿美元的 C 轮融资。这是室内农业公司有史以来最大的一轮私人融资。由于新的融资，Bowery 的价值接近 23 亿美元。

作为美国最大的垂直农业公司，Bowery Farming 维护着一个智能室内农场网络。Bowery 创新的垂直耕作系统将作物垂直堆放，以最大限度地利用空间。该公司利用城市外的工业空间在室内种植农产品。这使得 Bowery 能够提供当地种植的产品，减少浪费和里程。

Bowery Farming 成立于 2015 年，总部位于纽约，致力于环境可持续发展。使用的每颗种子都不含污染物和杀虫剂。与室外农业不同，Bowery 的现代室内农业不受不可靠天气的影响，作物可以全年生长。该公司依赖于垂直空间的有效利用。一个农场在同样的空间里产出 100 倍多的产品。

## 包厘农场操作系统

人工智能是该公司专有的农场操作系统 BoweryOS 的关键组成部分。一个设施同时种植数千种作物。每一种都有自己的最佳温度点和环境条件。定制对产量很重要。

这就是人工智能发挥关键作用的地方。一个由摄像机和传感器组成的庞大网络监控着农作物。数据点被实时输入到机器学习算法中。这使得 BoweryOS 能够跟踪芝麻菜植物是否没有得到足够的水分。或者香菜作物没有得到足够的光照。

Bowery Farming 的首席商务官 Katie Seawell 描述了能够微调变量以优化生长条件的优势。“让植物承受压力的是让它暴露在更高强度和更长时间的光照下，所以我们能够优化配方，因为我们控制了植物生长的每个方面或输入，”Seawell 说。

## 数据推动增长

BoweryOS 分析其收集的历史数据，以更好地优化未来不同类型作物的生长条件。数据从成千上万个生长周期中积累而来，包括气流、养分、光照强度、二氧化碳水平、湿度和灌溉计划。人工智能的使用也有助于 Bowery 节约资源和减少能源消耗。

Bowery Farming 的创始人兼首席执行官欧文·费恩(Irving Fain)说:“在 Bowery，我们正在重塑一个更简单、更安全、更可持续的新供应链，并最终提供与今天不同的充满活力的美味产品。”在 2015 年创立 Bowery Farming 之前，Fain 是一位连续创业者，创建了许多其他成功的创业公司。他在花旗私人股本集团开始了自己的职业生涯。

## C 轮融资刚刚完成

包括从 C 轮融资中筹集的 3 亿美元，Bowery Farming 现在已经从私人筹款中筹集了 4.72 亿美元。自 2020 年以来，该公司经历了超过 750%的零售增长。Bowery 的包装产品在 850 多家商店出售。其包装产品服务于各种专业杂货店和电子商务平台，如亚马逊生鲜、全食超市和沃尔玛。Bowery 最近宣布与拥有 Safeway 的 Albertsons 建立新的合作关系。

C 轮融资由[富达管理&研究公司](https://www.fidelity.com)牵头。许多风险资本和投资公司参与了这一轮投资，包括 GV(前谷歌风险投资)、General Catalyst、g GV 资本和淡马锡。娜塔莉·波特曼和贾斯汀·汀布莱克等名人投资者也给予了支持，他们都是植物性饮食的倡导者。

“Bowery 的室内农业方法代表了对传统农产品供应链的有意义的破坏，其基于系统的工程和农场设计方法是无与伦比的”，GV(前谷歌风险投资)的普通合伙人安迪·惠勒说。

Bowery 打算利用新筹集的资金来加速创新、雇佣更多员工、支持研发和建设更多室内农场。2021 年 1 月，Bowery 聘请 Injong Rhee 担任公司的首席技术官。李承晚曾任谷歌副总裁和三星首席技术官。

## 宾夕法尼亚州的包厘农场

宾夕法尼亚州伯利恒的一个工业用地目前正在被改造成鲍威利迄今为止最大的垂直农业用地。Bowery 尤其注重在美国东北部和大西洋中部地区扩展其网络。该公司还在新泽西州的卡尼开设了一家名为 Farm X 的高科技研发中心。农场 X 将有一个内部育种项目和研究实验室。通过致力于研发工作，Bowery 希望加快农作物种植的商业化进程。

鲍威利开创的现代农业概念有可能彻底改变农业的未来。随着未来天气条件变得更加不确定，室内农业将因其可靠性而发挥重要作用。
