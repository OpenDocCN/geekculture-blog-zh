# 问题是

一年多前，我获得了生物学学士学位，主修微生物学。在我高三的时候，我在一个潮湿的实验室工作，并对继续在“生物医学科学的最前沿”工作产生了强烈的热情。我想我会在实验室里寻找研究职位，在那里我可以发展我的研究技能，但是在三月…生活停滞了。恐慌像野火一样蔓延，未来是不确定的，疫情让我在充满敌意的就业市场中徘徊，我面试的主要调查人员不确定他们是否有足够的资金维持自己的实验室运行。

# 解决方案

现在放弃我的职业生涯显然还为时过早，所以我试着利用所有空闲时间，努力提升自己的技能。在过去的几年里，世界产生了比以往更多的数据，作为一名生物学家，我能够注意到这种在实验室中使用计算方法的增长趋势。是的，测序技术已经存在多年了，但这种趋势并不局限于任何主题，这种趋势包含了我们观察到的一切，从细菌的阴影到它们的增长率的微妙趋势，甚至是库存是如何保持的，然后我突然想到，如果我想在实验室中取得成功，我绝对需要学习数据科学。

我浏览了几个证书课程，决定从 eCornell 开始 [Python for Data Science](https://ecornell.cornell.edu/certificates/data-science/python-for-data-science/) 证书，ecor nell 是康奈尔大学开发的一个在线学习平台，包括从酒店管理到商业智能的课程。我选择了这个付费课程，因为它教授 Python 的基础知识，并以数据科学项目为基础，最终提供了一个有形的证书。除了付费资源之外，你还可以看看 [Codecademy 免费的](https://www.codecademy.com)，这对建立一个强大的基础很有帮助，Youtubers 喜欢 [Tech with Tim](https://www.techwithtim.net/tutorials/) ，Coursera 上的免费证书，喜欢 [IBM 数据科学](https://www.coursera.org/professional-certificates/ibm-data-science?utm_source=gg&utm_medium=sem&campaignid=8985360908&utm_campaign=10-IBM-Data-Science-IN&&utm_content=10-IBM-Data-Science-IN&adgroupid=86017337010&device=c&keyword=ibm%20python%20for%20data%20science&matchtype=b&network=g&devicemodel=&adpostion=&creativeid=412873341069&hide_mobile_promo&gclid=CjwKCAjwq7aGBhADEiwA6uGZp1vOcMFpUYZDxi0p9wy2lUqskP6lILEIqejhK7Mu_jRkhGfqbPh7yxoCckUQAvD_BwE)项目。

# 该过程

我在大学时尝试过计算机科学，并从 Java 开始，但它并不适合我。我感到格格不入，一切似乎都在我的头上。在经历了一次糟糕的成绩后，我觉得编程可能不适合我，于是继续我的生活。当我得出我需要学习数据科学的结论时，我知道阻碍我的是什么艰巨的任务，编程。这一次我将学习 Python，我不知道我是否会成功实现我的目标，我只知道这将是艰难的，我可能会无聊到死。

课程开始了，尽管仍然很困惑，我还是取得了一些进步。慢慢地，但肯定地，我的观点正在转变，我能够看到编程的辉煌。Python 就像英语一样，你输入一些命令，计算机就会去做。我觉得自己就像一个拥有新玩具的孩子，只不过这个新玩具有无限的可能性。很长一段时间以来，我第一次觉得自己有能力解决任何我想解决的问题。随着课程的进行，我了解了熊猫、Seaborn、Sci-Kit learn，当课程接近尾声时，我意识到我现在已经掌握了使用 Python(和 StackOverflow)解决我能想到的任何问题的技能……算是吧。

这种令人兴奋的经历促使我报名参加了一个为期两个学期的生物信息学研究生证书项目，在这个项目中，我们将学习基于 Python、R 和 bash 的程序。我想学更多，做更多。在我的节目开始时，我得到了另一个启示。我的教授强调说

> “到这个十年结束时，生物医学研究将成为真正的数字化企业，计算科学家将站在前沿，你有很多可以期待的”

我没有完全理解这句话的深度，但随着时间的推移，我能够理解它，并意识到我当初为什么从事编程。生物学不再局限于琼脂平板和移液器，它延伸到已经占领世界的硅片所提供的无限可能性。

# 我对初学者的建议

学习编程就像向一个完全陌生的人打招呼，这是一个可怕的命题，但他们很可能会向你打招呼。帮助我真正钻研这个课题的是用生物信息学来学习它。我能够理解我们需要解决的生物学问题，然后在尝试解决它们的同时学习编程原理。这种联系帮助我在更深层次上理解生物学和编程。最终我对这个主题非常着迷，我花了几个小时观看 youtube 视频，浏览 StackOverflow 线程，并从 medium 文章中学习。实际上，我就是这样找到 medium 的，寻找一种[没有数学解释的马尔可夫链蒙特卡罗(MCMC)方法](https://www.google.com/url?sa=t&rct=j&q=&esrc=s&source=web&cd=&cad=rja&uact=8&ved=2ahUKEwiw2460hcTxAhXgIbcAHef2C9MQFjAEegQIBhAD&url=https%3A%2F%2Ftowardsdatascience.com%2Fa-zero-math-introduction-to-markov-chain-monte-carlo-methods-dcba889e0c50&usg=AOvVaw18svztDzZVxdOe-bz_7PLl)。

如果你想在没有任何经验的情况下开始编程，试着找一个你感兴趣的领域，比如生物、地理、金融、数学甚至艺术，它可以是任何东西。以那个领域为例，将你的编程过程与那些领域中的问题联系起来，并试图解决它们。你很有可能会失败或陷入困境，但没关系，你只需要后退一步，从不同的角度来看待这个问题。这绝对有助于培养你对技能的兴趣，并提供一个坚实的基础。

随着我了解的越来越多，我希望能写一些关于媒体的文章来教育那些可能面临和我刚开始时类似问题的人。像上面提到的 MCMC 方法这样的简单解释确实帮助我理解了复杂的编程模型，我也将尝试这样做，用我的经验提供编程主题的许多观点之一。
