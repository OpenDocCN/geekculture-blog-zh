# 什么是数据科学？

数据科学是信息技术的一个分支，负责分析数据并使其可用。更准确地说，它处理将数据转换成信息。一个公司或个人可能有万亿字节的数据，但没有信息；这是常有的事。举一个实际的例子，读者可以想到所有与消费有关的数据
