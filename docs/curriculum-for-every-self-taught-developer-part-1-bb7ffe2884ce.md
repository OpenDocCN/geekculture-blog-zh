## 成为软件工程师的一步一步的指南

成为一名软件工程师不仅令人满足，还会让你接触到过多的可能性，更不用说这份工作的需求量很大，而且 T2 的薪水非常高。

如果你当时没能赶上这股潮流，不要烦恼；这是**为你在 2022 年成为一名自学成才的开发者提供的一步一步的指导**。

# 概观

这个课程大约需要八个月，取决于你每天花在学习上的时间。

在开始之前，让我们列举一下您将在接下来的几个月中涉及的主题

1.  计算思维，通用编程语言
2.  算法、数据结构
3.  Web 开发
4.  个人项目
5.  系统设计
6.  计算机网络、操作系统、数据库管理
7.  面向对象编程，设计模式，坚实的原则
8.  LeetCode

一开始这看起来令人畏惧，如果不是不可能的话，知道你需要在做好工作准备之前覆盖大量的话题。

然而，坚持下面的课程，到这八个月结束的时候，你会对你已经走了多远印象深刻。

我们开始吧！

# 第一个月:计算思维，通用编程语言

![](img/3e9d8b60aeefb67ef9742efdd010efd9.png)

Photo by [cottonbro](https://www.pexels.com/@cottonbro) on [Pexels](https://www.pexels.com/photo/photo-of-person-s-hands-using-laptop-4065617/)

*   [CS50](https://cs50.harvard.edu/college/2021/fall/) (2 周)
*   [Python 课程](https://www.youtube.com/watch?v=rfscVS0vtbw) (1 周)
*   [Pygame 项目](https://www.youtube.com/watch?v=Q-__8Xw9KTM&t=6052s) (1 周)

第一个月，我假设你根本没有编程**的先验知识**。

如果你已经很好地掌握了一门通用编程语言，例如 Python、JavaScript 等。，直接跳到第二个月。

在 CS50 中，David J Malan 将带你了解计算思维的基础，激发你对编程世界的兴趣。

CS50 中的练习需要巨大的努力。因此，你可以**忽略练习**，现在主要集中在讲课上。

CS50 之后是 Python 课程，时间不超过一周。你的目标是**理解 Python 的基本语法**开始修补项目。

你将奉献最后一周来弄脏你的手。Pygame 让你在获得具体编码经验的同时获得乐趣。你可以建造任何像太空入侵者这样的小 T21 来给自己编码的要点。

第二个月，你会做一些举重运动。

算法和数据结构是决定你成为软件工程师之旅成败的主题。

在我看来，AlgoExpert 是最好的资源之一。它提供

*   高质量的概念概述和编码演练
*   干净优质的样品溶液
*   全面的时间和空间复杂性分析

简而言之，它降低了惰性，提高了学习效率。考虑到投资回报率高，绝对物有所值。

> 稳扎稳打赢得比赛。

一旦你完成了简单和中等的问题，你将在接下来的六个月里分配一部分时间来解决**algo expert 或 LeetCode 上的一个编码问题**。

# 第 4 个月:全栈项目

![](img/d7a02e830d9aae8a680106ddf819eda5.png)

Photo by [Life of Pix](https://www.pexels.com/@life-of-pix) on [Pexels](https://www.pexels.com/photo/person-mixing-dough-9095/)

*   两个完整的项目(4 周)
*   [算法可视化器](https://www.youtube.com/watch?v=n4t_-NjY_Sg&t=330s)(参考)
*   [网飞 App](https://www.youtube.com/watch?v=tsNswx0nRKM) (参考)

> 最好的学习方法是边做边学。

第四个月，你会从网络课程中休息一下，把手弄脏。

你将收集你所学的一切，重新消化这些概念，将它们缝合在一起，然后自己想出两个完整的项目。

你可以考虑任何想法，但是你应该努力在你的项目中包含这些特性

*   登录和注销流程
*   通过 cookie 进行用户认证(JWT/会话 cookie)
*   用于认证和授权的 Google OAuth
*   用户通过 web 套接字推送通知
*   上传和检索图像文件

这些都是重要的概念，经常在面试中被测试，它们最好通过项目来学习。
