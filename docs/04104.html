宝石的核心总是保持在聚光灯下，如机器和深度学习，每个人都试图随时跳跃。因此，来自任何领域的人都可以通过学习建议的课程很容易地了解这个领域的概况。让我们从列出每个新手或中级学习者的常见问题开始，例如:

1.从哪里开始？

2.选哪些课程？

首先，学习编程语言，这是最基本的东西。你可以从广泛使用的语言开始，如 PYTHON 或在一般实践中不常用但在基于计算机视觉的研究项目中使用的语言: MATLAB

不太常见的:C / C++ / JAVA

对于数据准备:JavaScript(但不是一开始就必须的)

研讨会/网络研讨会:

尽可能多地参加研讨会，以了解新的和我不知道的东西。

利益:同一意识形态群体的成长

完成课程后参加研讨会

等课程完成后的项目

MNIST(手写数字)、时尚MNIST、虹膜、股市预测、假新闻检测、推特分析/情绪分析

注意:尽量只用Python做这些项目，不要用任何框架(TensorFlow / PyTorch )

我之所以推荐这些项目，是因为在互联网上有成百上千的参考资料可以帮助你，无论你在哪里遇到困难。然后，您可以进行特定应用项目，如电子元件分类、家用物品分类、分数预测、出勤预测

注意:通过参考以前的数据集制作你自己的数据集，并应用你的知识，参加黑客马拉松(初级)
