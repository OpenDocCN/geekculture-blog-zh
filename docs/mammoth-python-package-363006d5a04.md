# 猛犸蟒蛇皮包

> 原文：<https://medium.com/geekculture/mammoth-python-package-363006d5a04?source=collection_archive---------12----------------------->

将 Microsoft Word 文档转换为干净的 html。

mammoth 包是一个 Python 库，它提供了将 Microsoft Word 文档与 HTML 格式相互转换的工具。它是为了解决不同文字处理软件和平台之间的互操作性问题而创建的，因为 Microsoft Word 文档由于其格式的专有性质而很难在其他程序中使用。
