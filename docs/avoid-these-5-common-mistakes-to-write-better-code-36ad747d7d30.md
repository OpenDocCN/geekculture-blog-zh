专业的软件开发人员不只是写代码让它工作，他们也写代码让其他人理解。

在现实世界的项目中，开发人员大约花 60-80%的时间阅读代码。因此，为了节省团队中每个人的时间以及避免错误，产生可读的代码是非常重要的。
