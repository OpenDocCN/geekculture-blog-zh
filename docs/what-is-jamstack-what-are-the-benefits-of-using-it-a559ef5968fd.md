# 什么是 Jamstack？用了有什么好处？

> 原文：<https://medium.com/geekculture/what-is-jamstack-what-are-the-benefits-of-using-it-a559ef5968fd?source=collection_archive---------10----------------------->

作为一名开发人员，您可能会问自己:什么是 Jamstack？用了有什么好处？有很多。您可以轻松扩展 Jamstack 站点。这种架构使团队能够快速构建 MVP，并随着站点的扩展继续添加功能。因为文件存储在 cdn 上，所以您的站点很容易扩展。Jamstack 还有一个活跃的社区，定期创建新的工具。
