保持代码整洁高效并不容易。保持轻松愉快也不容易。作为一名开发人员，你会有这样的一天，你正在做的项目会让你沮丧到胎儿的位置。你会有这样的日子，你只想复制粘贴所有的东西，写出你所写过的最程序化、最重复的垃圾，然后乘公交车直接回家。
