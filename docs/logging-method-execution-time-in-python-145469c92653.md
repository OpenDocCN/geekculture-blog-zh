## 使用 Python decorators 记录 Python 中方法的执行时间

Python decorator 是一种标准的设计模式，允许您在不修改现有方法结构的情况下向其添加额外的功能。例如，我们可以使用 decorator 来完成日志、缓存、分析、认证等任务。
