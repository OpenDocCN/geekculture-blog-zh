随着技术的进步，编程不再被认为是一种利基技能。无论哪个行业，向数字时代的逐渐转变已经开始。

这种数字化转变的基础是编程。虽然对许多人来说，这不是最简单的技能，但在 21 世纪，拥有一门编程语言将是一项有价值且适用的技能。
