## 什么是 NFT 市场？

在建立市场之前，了解 NFT 市场是至关重要的。这是一个在线市场，创作者出售他们不可替代的虚拟产品，买家使用比特币或以太网等众所周知的加密货币购买这些产品。每次购买 NFT，市场都会收取交易费。

付款后，买家会收到一个令牌，这是一种特殊的标识符，包含所有权证明以及作者、出处和真实性的详细信息。

这些 NFT 市场是建立在各种区块链之上的，这些连锁店有他们所坚持的一些标准。然而，最常用的链条是以太坊，创建 NFT 最常用的标准如下。

有两个令人难以置信的标准经常用于 NFT 的创作:

**ERC721:**

根据该标准，唯一标识符被映射到代表标识符所有者的地址。“TransferFrom”方法提供了转移资产的权限方法。

**ERC1155:**

根据这个标准，所采用的 id 代表一类资产，而不仅仅是单个资产。它比 ERC721 标准有效得多。例如，如果你是一名游戏开发人员，想要销售 100 套盔甲，你必须为 ERC721 写 100 份智能合同，但为 ERC1155 只写一份有一个数量的合同。

## **最不可思议平台的运作**

在 NFT 市场工作包括以下几个简单的步骤。

*   用户首先向 NFT 平台注册。
*   接下来就是在平台上设置一个加密货币钱包，用来存放不可替代的代币和数字货币。
*   然后，用户建立一个 NFT，并适当地设置拍卖的参数。
*   目前，这些收藏品正在拍卖。
*   之后，项目由选择的平台进行审核。
*   然后，不可替换的代币出现在拍卖列表上以供购买。
*   拍卖中的买家出价。
*   拍卖结束后，买方的最高出价被传达给卖方。
*   然后，该平台处理交易的结束以及商品和货币的转移。
*   NFT 市场对销售额征收一些费用，以换取平台转移资金和不可替代代币的劳动。
