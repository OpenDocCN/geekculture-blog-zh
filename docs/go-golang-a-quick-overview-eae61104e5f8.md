在回顾一份工作描述时，我发现了一种引起我注意的编码语言。Go——我知道游戏 Go 的名字，但从未听说过任何关于这个名字的编码语言。我知道围棋非常复杂，需要战略思维和高度专注。

![](img/91ac47f8b6216d3f3497c6be22bbf7be.png)

嗯，原来谷歌在 2007 年设计了 Go(编程语言)，2012 年发布。我很想了解更多关于这种语言的知识，所以这里是我的发现的一个快速概述。它也被称为 Golang，因为它的网站是 https://golang.org/的 T2。这是了解这种语言的一个很好的资源，因为他们有:

*   互动游—【https://tour.golang.org 
*   视频游览—[https://research.swtch.com/gotour](https://research.swtch.com/gotour)
*   教程—[https://golang.org/doc/tutorial/](https://golang.org/doc/tutorial/)
*   还有更多。

它是一种静态类型的编译语言。所以变量类型必须在编译时显式声明。它的语法类似于 C 语言(他们是这么说的)，但是它变得更具可读性和可用性。Golang 的一个关键特性是它的并发性，这赋予了它独立运行函数的能力。它还可以与多线程一起工作，因此使编程、编译和执行变得快速而简单。与 Javascript 相比，它有自己的测试工具，这使得测试更加容易和有效。

正如我之前提到的,[golang.org](https://tour.golang.org/welcome/1)有很多信息，但许多人声称这是一种容易学习的语言。我已经在 Linkedin 上找到了一些课程。
