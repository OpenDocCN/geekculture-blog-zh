> 没有工作经验如何获得商业分析工作

在商业分析领域开始并发展自己的职业生涯是极其艰难的。然而，有几种方法可以让你自己获得现实世界的商业分析经验。

有几个教程涵盖了成为业务分析师必须学习的内容，但是没有多少工具可以让你将这些知识付诸实践。出于这个原因，我收集了大量的资料，可以帮助你获得商业分析师应该具备的知识和能力。

1.  **解决 SQL 问题**

SQL 是可以帮助你在不同岗位找到工作的技能之一，商业分析角色也不例外。

**SQL 练习**

不同的网站提供不同数据库管理系统的免费 SQL 练习。通过订阅计划，他们将锁定解决方案，但也可以选择查看其他人的答案。

*   [**黑客等级**](https://www.hackerrank.com/domains/sql) :提供一系列按难度划分的挑战。
*   [**Stratascratch**](https://platform.stratascratch.com/coding?code_type=1) :提供各种最大的科技公司正在询问的 SQL 问题。提供的数据库是 MySQL 或 PostgreSQL。
*   [**Mode**](https://mode.com/sql-tutorial/):Mode 上有成千上万的教程可以用来学习像什么时候使用 SELECT、CASE、WHERE 等基础知识。它有一个名为“SQL 分析培训”的部分，包含实际的 SQL 问题，并有一系列真实的 SQL 问题。
*   [**Leetcode**](https://leetcode.com/problemset/database/) :根据公司询问的难度和技术问题提供各种 SQL 问题。
*   [**SQLPad**](https://sqlpad.io/) :还提供了各种真实的 SQL 面试问题。每个代码问题都有一个视频讲座，解释正在测试的想法(或概念)。此外，您可以在这里检查数据集并输入 PostgreSQL 和 MySQL 代码。

**2。数据可视化工具**:

数据可视化是一种更快、更简单的方式来呈现业务运营和识别机会领域，使决策者能够快速检查大量数据，有效地了解趋势和问题，并利用数据做出更准确的行动。

有几个数据可视化工具，如 Power BI、Tableau、Qlikview、Sisense 等。Tableau 和 Power BI 是最常用的数据可视化工具。您可以从学习任何一种工具开始。

以下是免费的 Power BI 资源列表，您可以使用这些资源来学习 Power BI:

*   [**免费学习异能 BI**](https://www.udemy.com/course/learn_power_bi_for_free/):这是初学者学习异能 BI 的最佳课程。本课程让您深入了解 Power BI 的基础知识，并演示如何构建一个交互式仪表盘**。**
*   [**使用 Power BI 分析和可视化数据**](https://www.edx.org/course/data-analysis-in-power-bi) **:** 这是一个为期四周的课程，是在 Power BI 中使用数据的初学者指南，非常适合专业人士。它从如何安装 Power BI 和创建不同类型的可视化开始。最后，您还将学习如何创建功能齐全的仪表板并与您的团队共享。
*   [**一小时内创建 Power BI 仪表板**](https://www.udemy.com/course/dashboardinanhour-powerbi/) :这是一门初级课程，您将学习如何在一小时内创建 HR 分析仪表板。

学习了 Power BI 的基础知识后，您可以通过使用以下网站上的真实数据来练习 Power BI:

*   [**Maven Analytics**](https://www.mavenanalytics.io/):提供电源 BI 的各种挑战。您可以在各种数据集上练习，如零售、运输、旅游、商业等等。
*   [**缟玛瑙数据**](https://onyxdata.co.uk/) :每月提供一次力量匕挑战。顶级赢家赢得亚马逊代金券。
*   [**健身周三**](https://www.workout-wednesday.com/power-bi-challenges/) :这是一次力量 BI 社区挑战赛。WOW 有一个受欢迎的 tableau 每周挑战，看到 Power BI 有类似的东西是如此令人兴奋。您还可以找到数据集的解决方案。

3 **。Excel**

Excel 是开始商业分析之旅的重要工具之一。

以下是您可以参考的免费课程列表:

*   [**对初学者有用的 Excel**](https://www.udemy.com/course/useful-excel-for-beginners/):在本课程中，你将学习 Excel 的基础知识。
*   [**Microsoft Excel 快速提升你的技能**](https://www.udemy.com/course/microsoft-excel-improve-your-skills-quickly/) :在本课程中，你将学习条件格式、筛选器、数据透视表、数据透视表图表、sumif、vlookup、offset 等函数，以及更多。
*   [**Excel(edX)**](https://www.edx.org/course/introduction-to-data-analysis-using-excel-2)数据分析简介:在本课程中，您将学习使用数据透视表创建数据聚合，使用数据透视表可视化表示数据，以及数据透视表的一些计算。

以下是一些可用于练习 Excel 的资源列表:

*   [**Excel 在线练习**](https://excel-practice-online.com/) :在本课程中，您可以练习初级、中级和高级水平的 Excel。
*   [](https://www.automateexcel.com/)**:该网站提供了 Excel 主题的简要描述，然后是一些关于统计查找、文本和日期函数等主题的练习，最后是一个定制的练习测试。**
