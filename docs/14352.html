面向ARM64/AMD64的Python 3.9 docker容器

原文：https://medium.com/geekculture/docker-container-with-python-3-9-for-arm64-amd64-f2cdf167230f?source=collection_archive---------12-----------------------#2022-08-28

这是基于Alpine Linux的Python 3.9，没有Gnome-Keyring。也没有-anaconda版本或者-slim版本，只有一个版本。我稍微改变了操作系统级& python包的顺序。它只包含绝对必要的依赖项，如cffi。它还包含Git & Twine以便于开发开箱即用的Python项目。它还包含lapack 包，Fortran, gcc, etc.来编译numpy/scipy/etc ( pip install numpy scipy)将会工作，所以你可以安装它们。你甚至可以安装木星笔记本，类似于

docker run -i -t -p 8888:8888 alexberkovich/alpine-python39 /bin/bash -c "pip install jupyter -y --quiet && \
mkdir -p /opt/notebooks && \
jupyter notebook \
--notebook-dir=/opt/notebooks --ip='*' --port=8888 \
--no-browser --allow-root"

注意: Gnome-Keyring被移除是因为https://gitlab.gnome.org/GNOME/gnome-keyring/-/issues/77

注意:如果您想要Python 3.8或基于Anaconda的docker容器，请参见我的故事“Docker容器与Python for arm 64/AMD64”
