# 什么是分散金融？

**DeFi 是一种基于区块链的金融形式。它利用智能合约(一种基于区块链的自动执行计算机程序)来执行交易，而不是依赖于中央金融机构。DeFi 的目标是创建一个新的金融生态系统，每个人都可以访问，并且不需要信任第三方**
