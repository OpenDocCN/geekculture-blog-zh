# 智能手表的心理学

> 原文：<https://medium.com/geekculture/the-psychology-of-a-smartwatch-6f983302ea43?source=collection_archive---------23----------------------->

第一款智能手表是在 1998 年推出的，尽管我们已经走过了漫长的道路，大多数人可能会认为这是当今的主流，要么是一些 android 手表，要么是 apple watch。

智能手表已经进化，它们的功能也是如此；十多年过去了，没有人注意到每一个增加的功能是如何将这种时尚作为我们日常生活不可或缺的一部分接受的？

多年来，智能手表的使用越来越多，距离 1998 年第一款智能手表还有一千英里。
