**什么是对象**

把一个对象想象成一个现实生活中的实体，一个足球比赛中的运动员，一个商店的顾客，一个部门的雇员，任何你想用代码表现的“活的”东西，也许是一个关于你的猫的程序😼你需要用代码来表示小猫

在我们的例子中，我们模拟了一个在线游戏的玩家，所以传统的统计数据包括他的名字和年龄等等

理解什么是编程中的对象为你打开了这么多的门，这就是为什么我们需要多检查一下我们的`player`

每个对象都有两个东西，属性和方法，属性是与该对象相关联的特性，方法是对象可以做的“动作”

想象这样一个属性，马克有一头金发，所以是 T1，玛丽有一头红发，所以是 T2，或者一个叫布兰妮的女孩很矮😏所以那就是`isShortie:true`

属性可以是任何可以想象的数据类型，一个`string`表示姓名、头发颜色、眼睛颜色、喜欢的电影等等，或者一个`int`表示年龄、工资、孩子数量、身高、体重等等，或者一个`boolean`表示某人是否单身😏或者有没有工作

**方法**另一方面表示对象所做的并且只与该对象相关联的活动，所以假设我们正在建模一个`user`对象，所以用户很自然地会做`login()`、`logout()`、`addComment()`、`sharePost()`、`addPost()`等等，所以你会看到这些动作总是与用户相关联
