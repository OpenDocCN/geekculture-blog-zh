# 给作者发电子邮件

当你从出版物上购买作者的论文时，他们赚不到钱。你为科学论文支付的费用总是归出版商所有。到目前为止，我已经发表了 3 篇论文，我没有赚到一分钱，但是我所有的出版商都为此收费。
