*什么是 NFT？*

*![](img/d55e233a81f8518a00217881d937b34e.png)*

*Photo by [Markus Spiske](https://unsplash.com/@markusspiske?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) on [Unsplash](https://unsplash.com/s/photos/bored-ape?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText)*

*不可替代令牌(NFT)是一种独一无二的资产，用于验证特定资产的数字所有权。可能是一幅画或一段音乐。一些最著名的 NFT 出现在以太坊区块链，这是最受欢迎的生成和交易 NFT 的平台。*

*NFT 运动对创意人员来说是一种受欢迎的解脱。世界各地的艺术家现在可以消除第三方，从而赚更多的钱。此外，他们现在可以鉴定他们的作品，减少剽窃，并向收藏家和其他艺术家出售他们的原创内容。*

*许多艺术家现在制作自己的 NFT，并在 NFT 市场上出售他们的作品。数码艺术品并不是唯一可以进行非功能性交易的东西。任何实物都可以转换成数字令牌，并保存在加密货币钱包中。*

***哪些特征可以区分不可替换的令牌？***

***1。不可分割性***

*您不能拆分 NFT。不可能拥有它的一部分。在其他加密货币的情况下，您可以轻松地将其拆分。如果你没有足够的资金购买整个比特币，你可以购买一些比特币或以太坊。对于 NFT 来说，这是不可能的，因为每个 NFT 都是唯一的。*

***2。唯一性***

*没有两个 NFT 是相同的。每一个 NFT 都是独一无二的作品。因此，你不能完全把一个换成另一个。*

***3。稀有度***

*NFT 如此受欢迎的原因之一是它们的可用性有限。如果 ERC20 令牌的开发者想要增加总体供应，他们可以在智能合约的上下文中进行，而不受任何其他人的干扰。*

*然而，当涉及到 NFT 代币时，同一个开发者被限制在一定数量的创作中。这就是为什么 NFT 因其稀缺性而备受收藏家追捧的原因。*
