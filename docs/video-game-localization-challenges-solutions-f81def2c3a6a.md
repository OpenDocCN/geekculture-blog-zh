# 什么是本地化？

在谈论游戏本地化之前，首先必须了解什么是本地化。

简而言之，本地化就是让一部分内容或产品更适合特定的地点。

![](img/260d3fa8132537cee57c34c9e4279545.png)

这是一个复杂的过程，通常需要你做好多件事情——比如[翻译](https://www.transifex.com/blog/2021/what-is-localization/#Translation_and_Interpretation)，图像改编，[国际化](https://www.transifex.com/blog/2021/what-is-localization/#Internationalization_and_Globalization)等等。视频游戏本地化也不例外。

你可以本地化几乎任何与内容有关的东西，包括游戏！

# 视频游戏翻译与本地化

既然您已经知道什么是本地化，那么区分本地化和翻译就很重要了。

翻译就是:翻译。你所要做的就是把一个单词或句子翻译成另一种语言。

另一方面，在本地化时，您必须考虑目标地区，然后相应地调整内容。这同样适用于电子游戏。

举个例子，让我们想象一下，你有一个场景，角色正在躲避大雨，说“真的下大雨了”。大多数语言的直译可能会变成，“雨下得真大。”，技术上没问题。

但是，如果你想更进一步，把那句话本地化呢？说到大雨，希腊人往往会说:“下雨的是椅子腿。”表明我们正在应对非常强的降雨。

这就是翻译和本地化的区别。

# 什么是电子游戏本地化？

游戏本地化就是将视频游戏本地化到一个特定的国家或地区。同样，请记住本地化和翻译不是一回事。

当本地化一个游戏时，除了使文本相关的东西对一个地区更有吸引力之外，你还必须处理其他的内容，比如动画、图像、电影等等。

![](img/28bb6d30667b0f76612487e41304ed79.png)

这也是为什么游戏可能是最难本地化的内容。大多数形式的数字内容，如网站或应用程序，只需要您处理文本和图像。另一方面，游戏有各种各样的挑战要应对。

# 视频游戏本地化的流行例子

如上所述，本地化不仅仅是翻译，也是适应。在电子游戏世界里，有时，你必须审查或改变插图，以使游戏更吸引人或不那么令人反感。

例如， [Wolfenstein，新秩序](https://en.wikipedia.org/wiki/Wolfenstein:_The_New_Order)，大量关注纳粹。但是，由于显而易见的原因，德国对此并不满意。因此，在游戏在这个国家发行之前，所有提及纳粹的内容都必须被重新标记。

另一方面，日本虽然不在乎审查纳粹相关内容，但裸体一直是个问题。而且不仅仅是游戏；日本甚至审查色情内容。这就是为什么在游戏正式在日本发行之前，裸体角色必须被遮盖起来。

这并不总是关于审查。为了提供更多的沉浸感或更好的吸引力，许多游戏也被本地化了。例如，日本的攀冰者要把海豹当作敌人来对付。

![](img/73188dacba57c93e4a87cc6acf82733a.png)

问题是，这是对美国海豹俱乐部的引用，这就是为什么任天堂在国际版中用雪人代替了海豹，希望它们对美国观众更有吸引力，更少冒犯。

# 为什么要为游戏本地化而烦恼？

毫无疑问，本地化一款游戏既昂贵又耗时；尤其是当你用电子表格手工操作的时候。所以，为什么还要麻烦呢？反正大部分人都懂英语，对吧？

虽然现在大多数人确实懂英语，但这并不能改变不同地区有不同文化的事实。本地化不仅仅是翻译，还包括帮助您的客户熟悉您的内容。

感觉对产品更熟悉的顾客更有可能购买它。因此，他们获得了更好的体验，而你获得了更高的 ROI(投资回报)。

更具体地说，[研究显示](https://www.slideshare.net/allcorrect/roidriven-localization-from-pets-to-heroes)你可以获得从 50%到数千甚至数百万的投资回报率，这取决于你的游戏排名和你的目标地区。

更不用说，如果你也本地化你的网页内容，[你增加了在多个国家排名的机会](https://developers.google.com/search/docs/beginner/seo-starter-guide)，这将带来更多的客户和更高的总收入。

![](img/7dca4b181d3c43eb79b620836386785e.png)

对大家来说都是双赢！玩家可以用他们的母语享受游戏，你可以获得更大的影响力和更高的收入。

因此，很明显你的游戏本地化实际上是必须的。但是，面对如此巨大的挑战，我们该从哪里着手呢？
