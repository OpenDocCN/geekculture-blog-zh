那么，什么样的技术会成为主导呢？让我们来了解一下！

# 1.3D 建模:

3d 建模已经存在很多年了，但是在不久的将来，这个领域将会蓬勃发展。

我为什么这么说？

答案很简单，随着元宇宙的出现，虚拟世界的热潮正在增加，这些虚拟世界当然将是 3d 的，所以为了制作这些虚拟世界，公司或组织将需要 3D 物品、化身、玩家、建筑，简而言之，一切都将是 3D 的，而为了制作这些 3D 物品，组织将需要一些具有 3D 建模或设计技能的优秀艺术家。

因此，3D 建模将成为主导产业之一，这不仅仅是因为元宇宙，还有许多其他方面，如设计大型机器、飞机、火箭、国防武器等，所有这些都需要 3D 建模。

# 2.人工智能:

因此，几十年来我们一直听到人工智能将成为未来，然后一种被称为机器学习的新技术开始受到关注，它被认为是人工智能的一个子集。

人工智能在过去非常受欢迎，在未来将会更加受欢迎和广泛，人工智能有无数的使用案例。

如今人工智能的一个流行用例是在数据库上建立人工智能模型，如 MySQL 和 model do 预测和分类类型的东西。

我为这类产品中的一个名为 [mindsdb](https://mindsdb.com/) 的产品做了一些开源贡献。对于许多与数据科学和机器学习相关的问题来说，这确实是一个很好的解决方案。

我真的很喜欢这个产品的创新用例，在未来，我们可以获得更多像这样的创新用例。一定要看看它，了解人工智能是如何改变许多事情的。

# 3.虚拟现实(VR):

虚拟现实(VR)自 1962 年就存在了，我们已经看到虚拟现实以游戏的形式变成了很好的动作。

随着元宇宙的出现，虚拟现实(VR)将成为领先的技术之一，你将可以玩游戏，在办公室进行远程工作，练习驾驶，甚至可以使用这项技术练习驾驶飞机！

是的，一些空军使用虚拟现实技术来训练新飞行员如何驾驶飞机。

单说游戏，虚拟现实将会在游戏中占据很大比重，并将彻底改变虚拟现实。

要了解更多关于虚拟现实(VR)的信息，请看这篇文章:[https://nerdsculture.com/what-is-virtual-reality/](https://nerdsculture.com/what-is-virtual-reality/)

我在这篇文章中详细解释了 VR，所以你可以去看看。

# 4.增强现实(AR):

增强现实(AR)是另一种与虚拟现实相关的热门技术，但它与虚拟现实有很大不同，尽管这两种技术都与现实相结合。它在元宇宙之后也变得流行起来，但是它存在了很长时间。

不管怎样，增强现实意味着增强的数字世界和真实世界，就像这样。

希望你玩“口袋妖怪 GO”或者听说过它，它是增强现实的一个很好的例子，你到处移动你的手机，你就可以通过手机看到口袋妖怪。

在那个游戏中，你可以在你的床上、车里、街上或现实世界的任何地方看到口袋妖怪！

这是增强现实，一些高级用例将是全息图，就像在电影《终局》中，漫威队长使用这项技术与史蒂夫交谈。

增强现实(AR)将在许多方面改变未来。
