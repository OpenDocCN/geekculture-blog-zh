蕾哈娜已经从一名强大的音乐人转变为一股创业力量。她的包容性内衣品牌 Savage X Fenty 正在考虑进行 30 亿美元的首次公开募股(IPO)。

该品牌一直在快速发展，旨在颠覆内衣行业，重新定义美容标准。这种商业模式也相当不受疫情的影响。鉴于该公司的受欢迎程度，首次公开募股可能是筹集资本和推动进一步增长的战略举措。

## **建国故事**

2018 年，蕾哈娜推出了 Savage X Fenty 作为线上专属品牌。亲密服装品牌销售内衣，内衣，胸罩，睡衣和家居服。该公司努力设计适合各种背景和身材的包容性服装。它的产品以合理的价格出售。

由于她的商业努力，蕾哈娜已经成为亿万富翁。她的大部分财富都与她在 2017 年创办的化妆品公司 [Fenty Beauty](https://fentybeauty.com/) 的成功有关。她的剩余财富来自萨维奇·芬蒂，以及她的音乐和演艺事业。

## 野蛮 X Fenty 在时尚界推广**包容性**

促进信心和包容性是 Savage X Fenty 价值观的核心。该公司的网站和时装秀以代表不同种族背景和体型的多样化模特为特色。

蕾哈娜在公司网站上的一份声明中指出:“野蛮 X 意味着为你自己制定规则，表达你的情绪、性格和风格——而不是为别人。”

维多利亚的秘密的衰落为 Savage X Fenty 抢占市场份额创造了机会。维多利亚的秘密的市场份额从 2016 年的 80%左右下降到 2019 年的 65%。在被批评该品牌宣扬不切实际的美容标准后，这家内衣零售商已采取措施彻底改变其形象。维多利亚的秘密的模特大多属于高瘦型，具有欧洲中心特征。尽管努力重塑品牌，但像 Savage X Fenty 这样的新玩家有望成为全球领导者。

根据市场估计，该公司的定位是到 2025 年成为全球内衣行业的领跑者。蕾哈娜的公司处于有利地位，可以在 3250 亿美元的市场中占据相当大的份额。作为增长战略的一部分，Savage X Fenty 计划在美国各地开设实体店。

虽然像维多利亚的秘密这样的老牌内衣公司一直在努力维持消费者基础，蕾哈娜的品牌却实现了 200%的年同比增长。此外，其 VIP 会员计划的增长较前一年激增了 150%。

## **融资轮次**

蕾哈娜的内衣品牌迄今为止已经筹集了 3.1 亿美元的资金。2022 年 2 月，该公司在由纽约投资管理公司 Neuberger Berman 牵头的 C 轮融资中筹集了 1.25 亿美元。C 轮的其他投资者包括专注于消费行业的私募股权公司 L Catterton 和 Jay-Z 的 Marcy Venture Partners。

最新一轮融资将有助于推动该公司的零售主导战略。该公司于 2022 年 1 月在拉斯维加斯的时装秀购物中心开设了第一家实体店。新的资金也将支持该公司的国际扩张努力和推出新的产品线。2021 年 2 月，Savage X Fenty 在 B 轮融资中融资 1.15 亿美元。

## 野蛮 X Fenty **领导团队**

两位女性联合总裁娜塔莉·古兹曼(Natalie Guzman)和克里斯蒂娜·潘达维斯(Christiane Pendarvis)领导着这家公司。两位女性都在零售销售、数字商务和产品开发方面拥有丰富的商业经验。

## **首次公开募股**

Savage X Fenty 正在与包括高盛(Goldman Sachs)和摩根士丹利(Morgan Stanley)在内的金融顾问就首次公开募股进行谈判，其估值可能在 30 亿美元以上。

尽管 2021 年的 IPO 创纪录，但 2022 年的 IPO 活动有所放缓。随着利率上升、通胀压力和地缘政治不稳定，IPO 趋势降温。在 IPO 表现强劲的一年之后，投资者可能也会感到疲劳。

Savage X Fenty 将不得不监控市场状况，以确定推进 IPO 是否有商业意义。目前，该公司正在与顾问进行讨论，并采取准备措施。由于市场波动，Savage X Fenty 尚未就 IPO 时间做出最终决定。
