加密货币最初是比特币、莱特币、以太坊和类似的代币。这些在开始时都没有任何奖励，除了采矿，采矿并不是真正被动的，因为它需要采矿设备和随之而来的维护。

这些天来，有大量的代币和交易所提供各种各样的选择来赚取额外的代币。我将回顾一下我过去尝试过的几个选项，以及需要注意的事项。

# 立桩标界

现在很多代币都提供赌注奖励。我不会深入它是如何工作的，我会解释你是如何做到的。每枚硬币的做法不同。

赌注是持有你的加密货币而不做任何事情的行为。对，就是这样。如果您将代币放在钱包或提供代币赌注的交易所中，您将不需要做任何事情。**就像领分红一样**。

最好的例子是 CoinBase 中的 ETH2.0 和他们自己钱包中的 Algorand。不同的代币将提供不同的 apy(年百分比收益率),有些代币要求您将代币锁定一段时间。在某些特殊情况下，这通常在 1%到 8%之间。

# 增加流动性

例如，币安提出为某种代币增加流动性。在金融>流动性互换，您可以找到将代币添加到流动性池的选项。在极端情况下，这也将每年产生一些额外的百分比，从 1%到 15%不等。有时甚至更多，但我会对此保持警惕。

流动性是使代币价格上下波动的因素。没有流动性，它会飙升，波动性更大。每当我使用这个选项时，我喜欢坚持使用众所周知并且已经存在一段时间的令牌。例如，我不会增加 SHIB / BNB 的流动性。

在大多数情况下，这是由交易所而不是令牌提供的选项。奖励来自另一个地方，所以要小心。

# 分散彩票

我没有尝试过这个选项，但是我读过几次。分散型彩票将收取少量的赌注/采矿奖励，并将它们添加到一个池中。这个池将随机分布在参加抽奖的地址上。除了代币之外，他们不必购买任何东西就可以进入，并且**不会因为进入彩票而没有中奖而失去任何代币**。

不利的一面是你不能通过下注赢得任何东西，有利的一面是如果你得到更多的代币，你可以得到更多。我还没有尝试过这个，我会警惕任何看起来粗略的令牌，并提供任何好得令人难以置信的东西。

我不是说你应该加入秘密赌场。认真不要加入任何赌场，你会失去你的代币。一些彩票代币会拿走你和其他人从赌博或采矿中获得的任何东西，而不是你实际的代币。

# 结论

这些只是被动赚取你的密码持有额外百分比的一些方法。小心其中的一些，因为你投资的代币可能会一直贬值。永远做自己的研究，不要投资任何看起来好得不真实的东西。
