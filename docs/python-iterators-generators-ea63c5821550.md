## 了解主要差异

![](img/c4712ff6699f355de0ecd9a9b3d1b2d9.png)

**The Python Logo from** [**Python.org**](https://www.python.org/community/logos/)

在 Python 中我们一直使用迭代器。它们是有趣的对象，整齐地隐藏在循环和列表理解中。给迭代器一个简单的定义:它们是可以被迭代的对象。我们把可以派生迭代器的对象称为 ***可迭代*** 。
