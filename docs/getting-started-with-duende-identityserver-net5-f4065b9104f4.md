IdentityServer 是 ASP.NET 核心的流行 OSS OpenID Connect 和 OAuth 框架。它允许您的 ASP.NET 核心网站充当 OpenID 提供商和 OAuth 授权服务器，提供开箱即用的单点登录(SSO)和 API 保护。IdentityServer 负责协议支持，但用户身份验证取决于您，即开发人员。
