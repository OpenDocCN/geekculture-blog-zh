## 通过示例提供 SQL 中一系列窗口函数的透明度

![](img/c86a88ab9ab06484ad04b21103cc4f38.png)

Photo by the author

SQL 中的窗口函数对特定的行组(窗口)执行操作，并按组返回输出。与聚合函数不同，窗口函数返回查询结果集中每一行的输出，而不是每组返回一个输出行。
