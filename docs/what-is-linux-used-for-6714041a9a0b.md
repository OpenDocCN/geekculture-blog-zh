Linux 带来的灵活性是它被用于许多事情的原因。事实上，有很多人每天都在使用 Linux，甚至没有意识到这一点。在进入之前，我有几个关于 Linux 在一个给定的领域是如何占主导地位的事情。

## 超级计算

在这个领域，Linux 是无可争议的冠军。就 2020 年的市场份额而言，所有超级计算机中至少有 99%运行的是 Linux。此前，第一名的位置由 Unix 占据。当时，没有任何其他平台能与 Unix 或 Linux 相提并论。Linux 是 1991 年出来的，大概过了 8 年才成熟，才开始用于超算。一旦它站稳了脚跟，Unix 就完蛋了。从 2000 年到 2010 年，Linux 从 5%的市场份额到 90%的市场份额。超级计算广泛而迅速地采用 Linux 有几个原因，但这总是归结于平台的开放性。我的意思是，仅仅为超级计算创建定制的 Linux 发行版的能力是非常有益的，这是 Unix 所不能做到的。

## 移动设备

全球最大、部署最广泛的移动平台是 Android。就市场份额而言，Linux 在移动设备中的使用可能更高，达到 85%，如果你考虑其他移动设备平台，如 postmarketOS，这是纯 Linux，而不是基于 Linux 的 Android，这个数字可能超过 90%。也就是说，如果你有一部 Android 手机，你每天都会在不知不觉中从 Linux 中受益。

## 云计算

出于某种原因，我觉得这应该是我的列表的顶部，但我知道也有 Windows 操作的服务器。很难说 Linux 的市场份额到底是多少。考虑到仅谷歌、脸书、IBM 和亚马逊就拥有约 400 万台服务器，这一比例被认为在 70%以上。我是说，连微软的 Azure 都用 Linux。我不得不想象世界上有更多运行 Linux 的服务器。我不是说 Linux 是云计算唯一使用的 OS，但绝对是王者。

## 物联网设备

无论是你的 Google home、你的 Amazon Alexa，还是你墙上的 nest 恒温器，Linux 为大多数物联网设备提供动力，大约 75%。可以肯定地说，Linux 是物联网的合理选择，因为它体积小，而且它已经有很多技术栈可以用来做你在物联网设备中会做的常见事情，如蓝牙、wifi 或与硬件接口。从制造商的角度来看，在设计新的物联网产品并寻求快速廉价地提供产品时，选择 Linux 是有意义的，因为 Linux 已经实现了他们将需要的许多核心功能。

## 桌面

Linux 正在迅速占领市场，甚至当它被用来替代 Windows 或 macOS 作为你的主要桌面操作系统时。事实上，现在，我正在 Linux 机器上写这篇文章。不幸的是，Linux 桌面市场份额低得令人尴尬，这是因为它对用户不友好，尽管这种情况逐年改善。找到一个 Linux 发行版作为桌面并不困难。他们遍布互联网，为你准备好了。

## 无人驾驶汽车

特斯拉和谷歌自动驾驶汽车都是如此。在谷歌和特斯拉以外的绝大多数情况下，如果一家汽车制造商正在制造一个自动驾驶平台，他们将使用 Linux 作为其基础。

## （家用）电器

你知道你可以在洗衣机和咖啡机上找到 Linux 吗？现在你知道了。如果你想知道，是的，你的三星洗衣机可以让你在手机上查看你的衣服在世界任何地方的状态，它运行 Linux。

## 科学研究

老实说，我在这里加了这个，因为我觉得它真的很酷。大型强子对撞机运行在 Linux 上。跨越数十万个内核和数十(如果不是数百)Pb 的碰撞数据，Linux 确实为希格斯玻色子粒子的发现做出了贡献。我相信他们使用了 Cern 开发的科学 Linux 发行版。

说够了。总而言之，Linux 是多产的，它在不同的地方被用于很多不同的事情，并且随着时间的推移，使用的数量将继续增加。如果你想分享我没有包括的其他领域，请在评论中分享你的知识。我希望这篇文章有深刻的见解，并让您更接近尝试 Linux。
