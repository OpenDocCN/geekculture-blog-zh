# 什么是 TAM？

技术客户经理(TAM)是一个面向客户的角色，与客户密切合作，帮助他们使用产品来实现他们的业务目标。这里的关键词是**产品**和**客户**。tam 就像两者之间的桥梁，通常负责保持客户的快乐和参与
