# 介绍

WebSocket 协议支持 web 浏览器(或其他客户端应用程序)和 web 服务器之间的双向通信。它有助于与服务器之间的实时数据传输。

有了长期保持的连接，服务器可以向客户机发送内容，而不需要客户机来回请求它和客户机消息。
