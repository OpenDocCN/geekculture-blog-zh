麦克OS X狮子仍然活着——极客紧缩评论

奇怪的是，尽管苹果公司(纳斯达克股票代码:AAPL)已经停止支持这个标志性的操作系统，Mac OS X狮子仍然活着。

然而，苹果仍然为与最新版本的Mac OS不兼容的旧系统提供了一个可下载的Mac OS X狮子版本。苹果网站称，可下载版本适用于搭载英特尔酷睿2双核、酷睿i3、酷睿i5、酷睿i7或至强处理器的MAC电脑。Mac OS X Mountain Lion附带的Mac不能与Mac OS X Lion配合使用。
