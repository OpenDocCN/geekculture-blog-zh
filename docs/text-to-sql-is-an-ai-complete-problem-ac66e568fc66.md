## 为什么 FactEngine 不会参加蜘蛛文本到 SQL 的挑战

文本到 SQL 是一个已经获得很多关注的领域，我将把它称为人工智能完全问题。无论任何参赛者在任何文本到 SQL 的挑战中表现如何，如[蜘蛛挑战](https://yale-lily.github.io/spider)，总会有一个模型和文本需要人类水平的智能来将相对简单的文本查询转换为相应的结构查询语言(SQL)。
