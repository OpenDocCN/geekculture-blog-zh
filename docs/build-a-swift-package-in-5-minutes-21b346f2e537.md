嘿，iOS 开发者们，

你正在做多个项目吗？你会在不同的项目中编写重复的代码吗？例如，您的每个应用程序可能都有一个登录和注册页面、设置页面或消息页面等。用户界面可能有所不同，但背后的逻辑可能非常相似。为了避免通过每个应用程序更新逻辑或修复错误的痛苦，Swift 软件包是方便的工具。
