## Unity Engine LineRenderer 组件:基本信息和用法

**LineRenderer** 是 **Unity 引擎**中的**组件**，允许在场景中动态绘制**线**。

因为它是一个**组件**，所以它必须被附加到一个游戏对象上，并且可以直接从检查器或者使用它公开的一些公共方法进行调整。
