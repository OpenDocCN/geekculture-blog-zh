# 谷歌从 FLoC 到 Topics API 的转变

![](img/4b11eb2d80c9a80c99857922eadbbe63.png)

Mitchell Luo

大约一年前，我写了一篇关于谷歌 FLoC 的文章，根据谷歌可疑的声明，这种算法应该取代第三方 cookies，并在定向广告时保护用户隐私。如果你读过那篇文章，或者事实上，几乎所有关于 FLoC 的文章，你就会知道，自从它被首次提出以来，它就被作为一个概念而被撕裂了。因此，当谷歌宣布 FLoC“[将去北部的一个农场](https://www.theverge.com/2022/1/25/22900567/google-floc-abandon-topics-api-cookies-tracking)”并且他们将放弃反隐私的想法时，这并不是真正的*巨大的*惊喜。

另一件并不意外的事？事实上，Google [立即启动了一个非常相似的项目](https://9to5google.com/2022/01/25/google-topics-api-floc/)，尽管它的名字并不奇怪，叫做 Topics API。这个看起来怎么样，会不会像它的前辈一样孵化或者煮熟？让我们仔细看看。

# 什么是主题 API？

![](img/2eebd5b0d23593c988c4fc08cde7b9ce.png)

Myriam Jessier

理解主题并不困难，因为一切都在名称中 API 将跟踪用户的网络浏览活动，并根据他们访问的网站以及他们与网站的互动方式对他们的兴趣进行分类。利用这些数据，它给每个用户分配三个主题/兴趣，从“曲棍球”到“动物学”，尽管后者可能更难找到广告。分配的主题决定了用户得到什么样的广告，而且应该只在你的设备上决定，这意味着谷歌的服务器不会看到你在看什么。每次你访问一个网站，这些主题会“显示”广告算法的兴趣，每周使用一个主题。因此，如果在第一周你搜索了打猎、芭蕾舞蹈和煎饼，那么广告客户将只能得到其中之一。希望是煎饼。

兴趣数据应该每三周更新一次，这意味着广告定位会跟上你不断变化的兴趣。谷歌承诺，过去几周的数据将被立即删除，当然，这种说法需要一些证据。这并不是说你不应该相信这个已经收集你的数据几十年的大企业集团，但是……让我们看看它是如何工作的，如果它能进行一般测试的话。

谷歌计划赢回用户信任的方法之一是让 Chrome 用户(只有他们)能够看到他们被分配了哪些主题，并在他们认为合适的时候删除它们。通过这种方式，你可以帮助邪恶的算法为你提供广告，让你买东西并改进他们的方法。该项目目前只是被谈论，因为甚至真正的开发测试还没有开始，尽管谷歌正在宣传其在 2023 年前消除第三方 cookies 的计划。

# 比 FLoC 好吗？

![](img/983b283d49f3909b568bf5387f7516db.png)

Pawel Czerwinski

令人惊讶的是，是的。这是真正的入侵性更小，所以它比 FLoC 更上一层楼，谷歌在这一点上得分。然而，它远非完美，因为 Topics 仍然收集用户数据来描述他们，并可能导致广告商自己识别用户，这意味着有重大的隐私漏洞需要填补。但这可能是一个白日梦，因为谷歌不会完全放弃定向广告，而是依靠这些广告的收入来资助公司的其他业务。

理想情况下，谷歌会信守承诺，让话题由用户控制，这样你就可以随时清除兴趣部分。或者，更理想的是，它会将该功能扩展到所有浏览器，而不仅仅是 Chrome，Chrome 在隐私方面是出了名的[不太好](https://www.forbes.com/sites/zakdoffman/2021/03/20/stop-using-google-chrome-on-apple-iphone-12-pro-max-ipad-and-macbook-pro/?sh=3ab57cd14d08)。如果发生这种情况，主题将仍然是一个有问题的工具，同时在正确的方向上迈出了真正值得注意的一步。事实上,“基于兴趣投放广告”方法的整体思想可以以一种不太侵扰的方式来执行，类似于非定向广告的工作方式。如果一个用户登陆了一个关于曲棍球的网站，他们对曲棍球感兴趣是显而易见的。这可以在没有任何数据收集的情况下，仅凭网站上的数据推断出来。利用这一点，你可以有效地向网站的每个访问者提供广告，而不会限制他们的隐私或收集隐私数据，即使是暂时的。

现在对谷歌取消了一个糟糕的想法，并用一个一般的想法取而代之给予真正的赞扬还为时过早，但我会对该公司至少阅读了房间给予一些赞扬。隐私监管机构和消费者的持续压力已经产生了效果，而且是积极的效果，观察该公司将如何发展很有意思。Topics 为更好的广告算法做了一些基础工作，这种算法不是侵入性的，会被用户接受。现在轮到谷歌在这个基础上构建一个坚实的产品了。
