## 因为 PHP 在正确的时间出现在了正确的地方

![](img/cadfc8ea18257b114bdd45f787ee5bf4.png)

在互联网上找到关于 PHP 将于何时消亡，或者关于 PHP 在 2021 年是否仍然相关的帖子并不罕见。

虽然专家们对 PHP 的看法不一，但有趣的是，互联网上发布的任何随机帖子都有 80%的可能被它处理。
