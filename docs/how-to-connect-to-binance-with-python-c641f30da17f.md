## Python 交易机器人

币安是世界上最大的密码交易所之一。它们几乎涵盖了你想要的一切——赌注、交易、储蓄等等。

对于密码交易机器人开发者来说，一个巨大的胜利是，他们还有一个[优秀的 API](https://github.com/binance/binance-spot-api-docs) 。这个 API 是健壮的，并且有合理的查询限制(特别是考虑到它是免费的)。
