# 我今天和团队分享的是

> 原文：<https://medium.com/geekculture/what-i-shared-with-the-team-today-8dc937479076?source=collection_archive---------45----------------------->

我在一家 IT 创业公司做 CTO 很挣扎，每天都在学习。

不可避免的是，当我们的思维聚焦于工程师就是工程师，前线就是前线的时候，我想你可能会反意识到不要过度解释事情，因为“反正其他人对这个故事不感兴趣”。

我认为这是必要的，但我的印象是，这是一个努力理解其他人工作困难的好组织。
