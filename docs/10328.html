对数学的误解

对许多人来说，他们所消耗的大量数学知识都是以使用复杂的方程来解决复杂问题的形式出现的。

从某种意义上说，数学很像医疗保健，它实际上是一个庞大的总括术语，涵盖了许多不同领域和不同类型的问题。整形外科医生和心脏病专家有非常不同的日常工作，解决非常不同的问题。

数学可能非常抽象。我最喜欢的数学是关于数字的本质。像这样的问题:

我们如何证明有无穷多的质数，并且它们不会在某一点上停止？

如果您正在为企业业务构建web应用程序和web APIs，您将永远不会用到微积分或深入思考数字的本质。那你会怎么做？

数据结构

数据结构是用于存储、访问和管理数据的格式。软件工程中有很多常见的数据结构。您可能会遇到:

  • 链接列表
  • 二叉树
  • 大量
  • 行列
  • 很
  • 哈希表

还有很多。但通常情况下，你不会。在您的日常工作中，有两种数据结构更容易找到。

设置

有一个数据结构在其他数据结构中脱颖而出— 集合。

来自维基百科，

在数学中，一个集合是元素的集合。【1】【2】【3】组成集合的元素可以是任何种类的数学对象:数字、符号、空间中的点、直线、其他几何形状、变量，甚至其他集合。【4】

软件工程师总是和器械打交道。他们可能知道它们是数组或列表。根据我的经验，这是软件工程师最常用的数据结构。

软件工程师应该了解什么是集合，如何创建集合，如何向集合中添加元素，如何从集合中删除元素，等等。

你自己的

创建一个数据结构来满足你的组织的问题和数据存储需求是软件工程中一个强大的工具。

你将如何存储和管理关于人的信息？您可以创建这样的数据结构:

class Person {
    firstName: string;
    middleName: string;
    lastName: string;
    age: number;
}

算法

你可以把算法看作是一个问题的解决方案，它涵盖了所有可能的情况或场景。

给定两组数字A和B，

let A = [3, 1, 4, 2, 0, 5];let B = [4, 7, 6, 5, 6, 8];

您可能需要执行如下操作:

  • 对A和/或B进行排序，使数字按升序或降序排列。
  • 创建一个新的集合C来组合这两个集合。
  • 创建一个新的集合C，它包含A和b中的数。
  • 创建一个新的集合C，它包含只存在于A或b中的数。
  • 确定A和/或b中有多少独特的元素。

诸如此类。有算法可以完成这些事情。

复杂性

来自维基百科，

在计算机科学中，大O符号用于根据算法的运行时间或空间需求如何随着输入大小的增长而增长来对算法进行分类。

这是外行人眼中的软件工程师的样子。

假设您有一个唯一数字的有序列表C，下面是C的一个示例:

let C = [
    1, 2, 3, 5, 6, 
    7, 9, 10, 11, 13,
    14, 15, 16, 17, 19, 
    21, 23, 24, 25, 26,
    27, 29, 30, 32, 33,
    34, 36, 37, 38, 40
];

给定一个数 n ，如何在列表C中找到n的索引或位置？那很简单。你要遍历数组的每个元素，一旦找到n就停下来。

这是可行的，但是对于一个足够大的列表来说会很慢。如果你一遍又一遍地把列表分成两半，直到找到n，会怎么样？列表是有序的，所以如果你看中间的元素，你可以确定哪一半包含n，这样快多了。该算法将执行较少的运算。

了解算法使用的运算次数以及它使用的内存量如何与输入的大小进行比较和扩展。这就是所谓的时间和空间复杂性，这是作为一名软件工程师的重要组成部分。

结论

您需要了解存储数据的常用方法，并能够创建存储数据的新方法。您需要创建对这些数据执行操作的解决方案，这些解决方案保证在所有可能的情况和场景下都能正常工作，并且您需要知道如何优化这些解决方案。
