# 用 R 和 Python 计算样本均值的分布

> 原文：<https://medium.com/geekculture/calculate-distribution-of-sample-means-in-r-and-python-657d1ef90c6c?source=collection_archive---------12----------------------->

![](img/67b6153c0f47044fe6d94aaa1a075151.png)

样本均值的分布定义为从特定总体中选取的特定大小为 n 的所有可能随机样本的均值集合。样本均值的抽样分布非常有用，因为它可以告诉我们从随机样本中获得任何特定均值的概率。
