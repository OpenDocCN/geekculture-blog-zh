## 简介:什么是甘特图

甘特图是项目管理中最常用的可视化项目进度和状态的方法。
**甘特图**是一种显示活动(任务或事件)相对于时间的条形图。这张图表是以它的发明者[亨利·甘特](https://en.wikipedia.org/wiki/Henry_Gantt)的名字命名的，他在二十世纪早期设计了这张图表。
