每当我使用诸如 NPM 或 Yarn 之类的包管理器用 JavaScript 编程时，我总是使用相当多的可用库，但是有一个库我更喜欢用于我做的每个 web 开发项目。今天我将与你分享这个库，并给出一个关于它的小教程。

# 顺风

Tailwind 是一个非常著名的库，它使网页样式变得更加简单。关于它的评论被广泛传播，但是特别是在项目的原型阶段，它比 Bootstrap 或 Materialize 更容易使用。

## 什么是顺风？

> TailwindCSS 是一个实用工具优先的框架，包含 flex、pt-4、text-center 和 rotate-90 等类，可以直接在标记中构建任何设计。

本质上，它与 Bootstrap 或 Materialize 非常相似。但是有一些关键的区别。最重要的是，Tailwind 不包含任何预制组件。

然而，它确实包含了大量的类，使得自己创建漂亮的组件变得非常容易，这对于想要在最终确定应用程序的一部分之前试验许多因素的前端开发人员来说是非常好的。

## 为什么我更喜欢顺风？

我个人更喜欢 Tailwind，而不是 Bootstrap 或其他库，因为它更有助于创建自己的风格。每个人都知道引导按钮和表单的样子，但是 Tailwind 要求我从头开始创建自己的按钮和表单。这不是一个困难的任务，也不耗费时间，但它会确保我的网站在某些方面看起来是独一无二的。

## 如何使用顺风

顺风非常容易安装和使用。我已经写了另一篇关于它的文章，你可以在这里查看，所以我不会进入安装它的细节。它是使用 NPM 安装的，一些功能被添加到诸如 NextJS 这样的库中，使得它更容易使用。

Tailwind 有许多实用程序类。这些增加了文件的大小，但是不要担心！当您构建最终的 CSS 文件时，Tailwind 将只添加您使用的类，而否定所有其他的类。我也写了一篇关于这个话题的文章。你可以在这里查看文章。

像 Bootstrap 或 Foundation 一样，您只需添加类来反映您想要对该元素做什么。如果你想要深绿色，你可以加上`bg-green-900`，如果你想要浅蓝色的边框，你可以加上`border-blue-100`。您可以使用`m-4`、`pt-4`、`px-16`等等来添加边距或填充。

# 结论

如果你从未使用过顺风，并且你想尝试一些新的东西，我真的建议你试一试。学习曲线不是很高，如果你喜欢自己设计网站，这肯定比只使用 CSS 更容易。
