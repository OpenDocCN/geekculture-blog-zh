JoyPy:一个功能脊线绘图包

JoyPy 是一个基于matplotlib + pandas的单功能Python包，只有一个目的:绘制joyplots(也称为山脊线图)。 Joypy 可以非常有用地帮助以一种更形象的方式可视化数据分布。要开始使用这个软件包，您首先需要pip安装，您可以使用下面单元格中的代码。

pip install joypy
