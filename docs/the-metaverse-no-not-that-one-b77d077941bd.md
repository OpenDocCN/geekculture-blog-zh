我看到了区块链游戏的巨大潜力，它以玩家的个人体验为中心。所以是的，你可以赚钱。但是你是通过建立忠诚度来赚钱的，而不是靠短期的诈骗伎俩。

如果这个行业打对了牌，你就能拥有成群结队的忠诚。不像暴雪或 Bungie，他们从他们的玩家群中抽走资金。更像是最终幻想十四，它的政策为他们要求的回报提供了巨大的价值。

想象一下在一个集中的多元宇宙门户创建你的虚拟形象。然后，你开始了一次冒险，在众多的流派中进行选择，进行探索。你参加过马尔康的第三次银河战争并幸存下来了吗？好吧，也许是时候安定下来，建造一个你承诺过自己的安静的地方了，如果你活下来的话。

所以，你去了一个你是农民的宇宙。都在同一个角色身上。让马尔坎代理商出现在你的农场，请你回来再服务一次，这有多酷？但是你必须拒绝他们。你现在有了新的生活。

# 如何做到这一点？

想想单片机。同样的概念。你会有一个创意总监在项目之间进行协调。自然，这将是一项意义重大的任务。所以，我相信你会先建立核心。我知道我的虚拟世界正在这么做。但是他们正在开发现有的游戏。但我说的是创造一个游戏多元宇宙。

开发者需要创建一个中心枢纽。为了解释，他们可以说一些实体或力量已经把你的角色从他们的多元宇宙中拉出，因为它即将毁灭，而你并不知道。

中央枢纽将类似于洛基系列中的 TVA 总部，具有更现代的美感。在那里，实体会告诉你如何维持这个多元宇宙的结构完整性。

你可能会发现自己在第一次世界大战的 FPS 战斗中，或者在比那更久远的年代的战斗中。这种体验是由一个项目团队开发的完整的游戏体验。他们必须遵守的是，他们的目标是用代币奖励玩家，当故事完成时，他们可以返回中心，或者当他们积累了足够的代币时。

然后在多种类型的游戏中重复这个过程。最后，你可以让玩家作为公司的一部分继续玩你的游戏。

# 多元宇宙市场

你可以用代币从 MM 那里购买 NFT 物品。这些项目可以是用户创建的，也可以是公司创建的。它们将跨越你能从多重宇宙中心进入的任何宇宙。此外，某些项目的数量也是有限的。并且它可以根据项目而变化。

会有一个由来已久的机器人问题。这就是为什么公司可以实现具有唯一 id 的 NPC 来占领世界，即使他们是不可见的。所以，不可能有 51%的攻击是由想要淹没 MM 或网络的创造型机器人发起的。

MM 的另一个特征可以是外部功能，例如去分辨率。解除将是出售你的真实世界钱包的一些或全部代币的过程。

# 至于宇宙

任何宇宙都可以是现在市场上的任何东西。或者更好，一些新鲜的东西。想象一下，在一个恐怖的宇宙中幸存下来，却发现在一个反乌托邦的未来打一场战争是有利可图的。我相信那会很棒。

然后，新游戏的发布会是如此令人兴奋。每个人都在入口处等着看新宇宙是什么——大量的宣传。你在同一个角色身上做了这一切。此外，你可以奖励那些帮助宇宙新探险者的宇宙老手。

有很多可能性。
