# 为什么是 GitHub API？

GitHub API 是必不可少的，因为如果您正在构建软件，您可能会使用 GitHub 来管理您的软件代码。没有 GitHub 的人也可以使用 Git。理解 GitHub API 也很有帮助，因为它将 Git 功能覆盖到了网络编程接口中。GitHub API 可能是设计最好的 API。

它是超媒体 API，可以是 API 客户端适应 API 变化的成功尝试；这是一个棘手的问题。API 版本控制合理。它很全面，并且与大多数 Git 特性密切相关。它始终如一，组织有序。
