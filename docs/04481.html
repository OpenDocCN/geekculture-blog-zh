卡珊德拉是什么？

Apache Cassandra是一个分布式NoSQL数据库，提供成功的应用程序所需的连续可用性、高性能和线性可伸缩性。

Apache Cassandra是一个开源的分布式NoSQL数据库，始于脸书内部，于2008年7月作为开源项目发布。Cassandra提供了现代应用程序所需的连续可用性(零宕机)、高性能和线性可扩展性，同时还提供了操作简单性和跨数据中心和地理位置的轻松复制。Cassandra每秒可以处理数Pb的信息和数千个并发操作，使组织能够跨混合云和多云环境管理大量数据。
