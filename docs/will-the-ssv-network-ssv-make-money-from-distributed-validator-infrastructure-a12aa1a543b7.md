# ssv.network (SSV)会从分布式验证基础设施中赚钱吗？

> 原文：<https://medium.com/geekculture/will-the-ssv-network-ssv-make-money-from-distributed-validator-infrastructure-a12aa1a543b7?source=collection_archive---------13----------------------->

对 **ssv.network (SSV)** 感兴趣是因为这个协议解决了主要的区块链问题。例如，2022 年 8 月 22 日，ssv.network 是 CoinMarketCap 第二大最受欢迎的稳定币。

问题在于验证，证明协议、节点和用户是真实的，并且是有效的。ssv.network 试图通过用许多验证器替换单个验证器来[分散验证](https://ssv.network/)。理论上，分散验证降低了验证节点失败的可能性。
