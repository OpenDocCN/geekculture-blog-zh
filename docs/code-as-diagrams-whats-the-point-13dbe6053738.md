## 与其将图作为代码，不如将代码作为图提供什么？

## 活文档—第 3 部分

您经常听到短语“**diagrams as code”**指的是使用文本编辑器编写一些可以被解析的代码来创建一个图，而不是使用图形编辑器和 UI 来创建一个图。但是，如果您解析真实的源代码来创建一个图，那么您就是在对它进行逆向工程。那么“**编码成图**给我们工程师带来了什么价值呢？
