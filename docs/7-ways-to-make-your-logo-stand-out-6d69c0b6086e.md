你有没有想过在口头语言发明之前，人们是如何相互交流的？如你所料，他们用符号和标志向他人传达信息。即使在今天，一个标志的影响力也远远大于文字。因此，每个公司或品牌都需要一个标志或徽章来识别自己。

有很多工具，比如像 **adobe 的 **logo 设计**，以及 illustrator** ，但是某些东西需要记住。

# 原创内容

每个标志成功的关键秘密是它的原创性。标识印在人们的思想中。如果你抄袭别人的 l **ogo 设计**概念，潜在客户会失去兴趣。一天之内，你为公司付出的努力就会化为乌有。你冒着失去潜在消费者信任的危险。

# 简单和极简

你的标志应该是有吸引力的和简单的。过度使用精致的色调会偏离最初的目标。一个简单明了的标志就足够了。过于复杂的标志会分散顾客的注意力，并很快忘记它的样子。

# 有意义的

你的标志应该是令人难忘的，并反映出你想传达的信息。即使有人在千里之外盯着广告牌上你公司的标志，他或她也应该能分辨出是哪一个。传达的信息是什么？如果你的标志没有任何意义，那就很难把它和其他标志区分开来。最佳实践是在制作徽标之前进行受众研究。

# 品牌一致性

出于多种原因，品牌一致性至关重要。它让你在竞争中脱颖而出，提升你的公众形象。目标是在各种环境中长时间保持一致性。当你扩展产品线时，这可能是一项艰巨的任务。

# 正确的颜色

始终为您的公司选择合适的调色板。选择互补的颜色，帮助你的商标在观众面前脱颖而出。色彩心理学的色彩理论也可以研究和学习。对颜色及其对人类行为的影响的研究被称为颜色心理学。它将帮助你为你的公司做出最好的、最明智的决定。

# 多才多艺的

你的标志应该适应所有这些广告媒体。企业不能为不同的广告平台创建多个徽标。这会让你的买家对几个商标感到困惑。一个优秀的标志可以在任何表面上看起来都很棒。在所有媒体和广告中使用同一个灵活的标志体现了专业性。结果，它发展了。

# 全球理解

普遍理解是使标志脱颖而出的最关键的方面之一。标志应该表达超越语言和文化的信息。无论是在亚洲、美洲、非洲、欧洲还是澳大利亚使用，一个标志都应该被全球所理解。该公司的业务将因此变得全球化。这个品牌将会在全世界立即被识别出来。
