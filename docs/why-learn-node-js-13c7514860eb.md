# Node.js 的优势

**高效的** **后端运行时环境。**非常适合应用程序的后端，因为它是基于事件驱动的单线程&非阻塞 I/O 模型，这意味着它非常轻量且高效。

**快速&可扩展。** *构建超快速、可伸缩、数据驱动的 web 应用的完美选择*

**非关系数据库的好选择。非常适合构建背后有数据库的 API。最好是 NoSQL(又名非关系型、非 SQL 数据库，如 MongoDB)**

**使用 Node 构建的应用类型，*提示*考虑流。**适用于数据流应用(比如 YouTube)、实时聊天应用(比如服务器端网络应用)

**JavaScript 发电站。**允许我们更深入地使用 Javascript。在 React 应用中，我们使用大量普通的 Javascript，但是如果我们使用 Node.js 作为后端运行时环境，它允许我们在浏览器之外使用 Javascript。这意味着我们可以在 web 开发的服务器端使用 Javascript，并允许更好的工作能力。
