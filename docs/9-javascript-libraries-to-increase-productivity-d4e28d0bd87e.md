## *代码更聪明，而不是更难。*

![](img/5719169d2b3619ff18fc60dac0f24177.png)

Photo by [Adeolu Eletu](https://unsplash.com/@adeolueletu?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) on [Unsplash](/s/photos/javascript?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText)

我们很多开发人员会写很多代码，这些代码可能根本不需要写。简单的数组函数，展平数组，甚至可能用于 HTTP 请求。但是，虽然您可以为每个项目每次都编写这些代码，但是您也可以安装完全符合您需要的 NPM 软件包。

这是我经常使用的库的一个小列表，用来提高我写代码的效率。我希望它也会增加你的。

# Axios

[Axios](https://github.com/axios/axios) 是一个让 HTTP 请求变得更加容易的库。它让我们用尽可能少的代码创建一个简单的基于承诺的 HTTP 请求。所有默认头文件和其他代码不必每次都写。

# 洛达什

Lodash 是一个非常受欢迎的实用程序库，有大量的函数供您使用。从获取数组的一部分、过滤、展平到适合特定任务的复杂功能。洛达什会掩护你的。 [UnderscoreJS](https://underscorejs.org/) 也很像。

# 国际光子

一个非常流行的日期/时间操作库是 Luxon。它使用相当新的`Intl`对象，而不是`Date`对象，这使得它成为新项目中的最佳选择。它具有解析、操作和创建与时间相关的一切的功能。另一个不太受欢迎的库是 [Moment](https://momentjs.com/) 。

# 护照

身份验证可能很难实现，或者如果您需要为您的每个项目都进行身份验证的话，这简直是太烦人了。[护照](http://www.passportjs.org/)有帮助。Passport 是用于节点服务器的认证中间件。它可以用于所有主要类型的身份验证，如 OAuth 和社交登录。

# Nodemon

一些框架和库具有快速刷新的特性，但是如果它们没有，你可以安装 [Nodemon](https://www.npmjs.com/package/nodemon) 来帮你完成。Nodemon 不需要任何代码，只要它注意到服务器文件发生变化，就会重启服务器。

# 骗子

一个可爱的创建假数据的小软件包，Faker，可以用于任何东西，从假的电子邮件地址到假的名字。这可用于演示、原型和营销目的。它易于使用，易于安装。

# 同时地

为了同时或几乎同时做某事，你可以同时使用[和](https://www.npmjs.com/package/concurrently)。像 Nodemon 一样，除了 CLI 命令之外，这个库不需要任何代码就可以在您的`package.json`文件中运行它。

# 使更多

我们都知道人类可读的格式是最好的。但是我们的语言很奇怪。一辆车，两辆车，一个库，两个库，[复数](https://www.npmjs.com/package/pluralize)可以帮这个忙。它可以灵活地添加新的不规则单词或其他问题的复数规则。

# UUID

[UUID](https://www.npmjs.com/package/uuid) ，或通用唯一标识符，是一种存储 ID 和唯一值的通用方式。它符合 UUIDs 或 GUIDs 的'[规则](https://en.wikipedia.org/wiki/Universally_unique_identifier)，可以在数据库中使用。它比自动增加的 ID 更安全，因为用户不能轻易猜出 ID。

# 结论

更聪明地编码，而不是更难。少编码，不要多编码，不要每次编码都重新发明轮子。我希望这篇文章对一些开发人员有用。所有的库都是众所周知的，被认为是很好的选择。
