“crypto”一词意味着数据加密，“currency”是交换媒介。因此，加密货币是一种像普通货币一样的交易媒介，存在于数字世界中，并使用加密来确保交易的安全性。一种[加密货币](https://en.wikipedia.org/wiki/Cryptocurrency)是现金和信用卡支付的替代形式。

比特币刚推出时，获得比特币的唯一途径是使用点对点网络。你可以匿名操作，不需要任何人的许可就可以获得比特币。随着比特币越来越受欢迎，现在有几种方法可以获得它们。

# **加密货币兑换**

你可以在 Coinbase.com 等加密货币交易所注册，用本国货币购买比特币。一些国家不允许购买比特币，所以你需要先检查一下。

当你在像 Coinbase.com 这样的知名交易所注册时，你需要提供一些个人信息。这当然包括你的姓名和地址以及其他财务信息。这是为了遵守许多国家的了解你的客户(KYC)法律。

这击败了比特币的关键基石之一，即匿名性。该网络的建立是为了让你可以进行比特币交易，而没有人知道你是谁或你来自哪里。

有一些加密货币交易所允许你使用法定货币购买比特币，还有一些地方你只能用加密货币购买。你需要检查一个交换提供你需要的服务。

加密货币交易所对每次购买比特币收取交易费。好消息是，这些费用往往比传统金融机构收取的费用低很多。

不要上当受骗。互联网上有许多加密货币骗局，如要求您加入电报组，要求通过网站支付等。使用合法网站进行投资。以下是如何找到它们的方法。

# **比特币 ATM**

在过去的几年里，比特币自动取款机已经在北美和西欧出现。他们现在也在其他国家。在这里，你可以使用银行账户或信用卡中的资金购买比特币。

大多数自动取款机也允许你进行交易。因此，如果你想向另一个人发送或接收比特币，你可以这样做。比特币自动取款机数量的上升证明了数字货币的日益普及。
