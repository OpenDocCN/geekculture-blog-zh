每当你点击社交媒体页面或电子邮件收件箱中的网站链接时，你可能会注意到它异常地长。乍一看，URL 后面似乎有很多胡言乱语。

这是我点击的一个这样的网址:

[https://livestorm.co/powered-by-livestorm?UTM _ source = powered-by-live storm&UTM _ medium = company-page&UTM _ campaign = 360 learn](https://livestorm.co/powered-by-livestorm?utm_source=powered-by-livestorm&utm_medium=company-page&utm_campaign=360Learning)

注意末尾的三个字: **utm_source、utm_medium 和 utm_campaign** 。它们中的每一个都被赋予了特定的值。这些统称为 **UTM 参数**。这些有助于网站所有者跟踪网站流量的来源，从而提供有价值的见解，什么是可行的，什么是不可行的。

# UTM 历史

*UTM 代表海胆追踪参数。它们是我们添加到 URL 中的小块数据，以便查看不同的流量来自哪里。他们是通过一个叫做海胆的分析工具被引入的，这个工具被谷歌收购，并发展成为谷歌分析。在此过程中，它们成为了跨工具跟踪营销活动的行业标准。——*[*crazyegg.com*](https://www.crazyegg.com/blog/utm-codes-guide-with-examples/)

# 了解不同的 UTM 参数

总共有五个 UTM 参数— *utm_source、utm_medium、utm_campaign、utm_content 和 utm_term* 。

前三个 UTM 参数是最受欢迎的，你通常会发现它们附加在每个网站链接上。这里有一点关于它们的解释:

*   **UTM 介质**:指流量来自的通道。例如，媒体可以是社交、电子邮件、搜索、付费、公司网站等。
*   **UTM 来源**:渠道内具体站点。例如，如果流量来自 LinkedIn 上发布的博客链接。那么来源是“linkedIn”，媒介是“social”。或者，如果流量来自点击脸书的付费广告。那么来源就是“脸谱”，媒介就是“付费”。
*   **UTM 战役**:这是指你正在进行的特定战役。该名称可用于表示产品名称、特定促销活动、影响者 ID 等。(*例如:utm_campaign = winter_sale* )

# 结论

追踪有效和无效的流量来源，有助于网站所有者规划他们的营销和内容策略。

所以，下一次你点击一个网站链接时，一定要看看最后附加的参数。你一定能找到一些有趣的见解！
