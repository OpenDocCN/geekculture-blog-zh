## 1.米托

几个月前我就知道了 MitoSheet，我真的很喜欢这个工具。

用你知道的工具编辑你的数据。可视化你的数据，创建数据透视表，编写过滤器等等。
米托电子表格的每次修改都会生成 Python 代码。

我有机会见到[亚伦·戴蒙德-雷维奇](https://medium.com/u/b8fe6cc59d81?source=post_page-----66d28ad6b437--------------------------------)，他的工作和团队真的很棒。

## 3.KNIME

[KNIME ANALYTICS PLATFORM](https://www.knime.com/) 是一款面向高级数据科学项目的开源端到端工具，基于称为节点(它们封装了不同类型的操作或算法)的可视化元素和箭头(流)，这些元素以非常友好和可视化的方式结合在一起，用于数据科学。

这个社区太棒了，而且非常合作，我无法用语言来描述 Knime 团队，特别要提到 [Roberto Cadili](https://medium.com/u/593efa2cf07?source=post_page-----66d28ad6b437--------------------------------) 和 [Rosaria Silipo](https://medium.com/u/fb562cc4a1e2?source=post_page-----66d28ad6b437--------------------------------) 。
