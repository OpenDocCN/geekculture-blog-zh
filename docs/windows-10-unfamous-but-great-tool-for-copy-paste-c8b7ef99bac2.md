## *你希望你以前就知道这些——启用剪贴板历史记录！*

![](img/f0e34b6b85dc2bea3f0974205e90554f.png)

Photo by [Catherine Heath](https://unsplash.com/@catherineheath?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

Windows 自带了一个叫做**剪贴板**的工具，操作系统用它来保存你最近复制的项目。

通过这个工具，**可以从一个地方复制** ( *ctrl-c* )一个文本，然后**粘贴** ( *ctrl-v* )到另一个地方…是的，你已经知道了…
