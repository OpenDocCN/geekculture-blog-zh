# Scrum 是一种软件开发生命周期方法

Scrum 是一种软件开发生命周期方法，它以最基本的形式基于敏捷宣言中阐述的原则。这篇文章描述了使用基本原则如何通过向 backlog 添加零缺陷模型和重构我们如何进行回顾来改进 Scrum。
