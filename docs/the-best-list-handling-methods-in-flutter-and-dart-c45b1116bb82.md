# 还原方法

通过使用给定函数迭代组合集合的元素，将集合缩减为单个值。这个方法从迭代器中的第一个元素开始，然后按照迭代顺序将它与剩余的元素组合起来，

在这种情况下，我们向匿名函数传递两个参数:一个是“值”，另一个是“元素”。单个变量值的集合是参数值。元素表示当前的迭代，我们可以用它执行任何函数，最后返回值(参数值)
