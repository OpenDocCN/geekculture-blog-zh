We’re talking about //baby

不，我说的不是`/*these comments*/`，我说的是`//these comments`，其他语言都支持的。

(HTML 除外。)

你可能被告知不，CSS 不支持单行注释。在 CSS 中留下注释的唯一方法是使用斜杠-星号-星号-斜杠注释，就像这样:

```
div {
background-color: red…
```
