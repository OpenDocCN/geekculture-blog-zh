## 自动处理离线到在线远程同步

![](img/af81b64eb35a084e106808ef89ba5fb7.png)

Source [AWS Amplify](https://docs.amplify.aws/)

AWS Amplify 有一项名为 Datastore 的服务，通过提供远程数据库的本地备份数据库 **Datastore** ，用户可以在离线时继续使用你的应用或网站。
