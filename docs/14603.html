Spring数据JPA、Spring数据R2DBC和Hibernate反应式

原文：https://medium.com/geekculture/spring-data-jpa-spring-data-r2dbc-hibernate-reactive-bcc43e321566?source=collection_archive---------1-----------------------#2022-09-11

数据持久性允许应用程序从非易失性存储中保存和检索信息。在这种环境下，应用程序要么自己管理持久性，要么使用第三方解决方案来处理数据库更新和检索。

在本文中，我们将探讨Spring Data JPA、Spring Data R2DBC和Hibernate Reactive项目。这个想法是比较每个项目提供和缺少什么，并深入了解在引擎盖下发生了什么。
