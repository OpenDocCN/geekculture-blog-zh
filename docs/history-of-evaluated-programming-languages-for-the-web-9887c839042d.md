## **Python、PHP 和 Javascript**

![](img/71b440da0e10851677287a802c64fce1.png)

Photo by [Joshua Aragon](https://unsplash.com/@goshua13?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

自 1989 年以来，我们都在发展服装，以便能够通过数字手段与他人联系。实体店现在都在网上，人们通过在线平台交流，我们甚至现在通过网络工作。但是你知道是什么技术在幕后运行，让你可以享受万维网吗？

有许多类型的编程语言运行在 web 后端，如 Java、C#、JavaScript、Ruby、Python 和 PHP 等。今天我们将只讨论其中的三种语言。历史和他们如何工作，以及其他语言的优点和缺点。

本文先说 Python，PHP，再说 Javascript。

**巨蟒**

![](img/07b99fdde0511d6ff3a0a4dab9655379.png)

Photo by [Hitesh Choudhary](https://unsplash.com/@hiteshchoudhary?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

Python 是一种广泛使用的通用高级编程语言。它最初是由吉多·范·罗苏姆在 1991 年设计的，由 Python 软件基金会开发。它主要是为了强调代码的可读性而开发的，它的语法允许程序员用更少的代码行来表达概念。

有了 Django 和 Flask 这样的框架，这是快速简单地开发 web 的一个很好的选择。Python 在所有编程语言中对 ML 和 AI 的支持是最好的，因为有像 TensorFlow 和 Keras 这样的重量级库。此外，Python 的简单语法和简单性使它成为 ML 专家和数据科学家的编程语言选择。据 Indeed 报道，Python 程序员在美国的平均工资是 **$110，840/年**，这使得 Python 成为收入第二高的编程语言，仅领先于 Ruby。

**PHP**

![](img/9ce383c9628f6f72ce665821d57d2c3c.png)

Photo by [Ben](https://unsplash.com/@benofthenorth?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

PHP 是由拉斯马斯·勒德尔夫在 1994 年秋天构思的。早期的非公开版本被用在他的主页上，以跟踪谁在看他的在线简历。其他人使用的第一个版本是在 1995 年初的某个时候推出的，被称为个人主页工具。

PHP 被认为是后端脚本语言。PHP 脚本通常写在服务器上，前端或客户端代码会与之交互。

一般来说，对于刚开始学习编程的人来说，PHP 被认为是一种容易掌握的编程语言。和任何编程语言一样，PHP 有编码、缩写和算法的规则。但是，如果您完全是编程新手，可以使用许多资源。

PHP 仍然是世界上许多开发者使用的最流行的服务器端脚本语言之一。如果你精通 PHP，就此而言，你不必太担心你的职业或薪水。PHP 开发人员的平均工资。glass door:81k 美元/年，最低 55k 美元，最高 100k 美元。zip recruiter:86k/年，低的 16k，高的**137k**。

**Javascript**

![](img/3806d1ccbb9f13f46512ba9c07932fb0.png)

Photo by [Claudio Schwarz | @purzlbaum](https://unsplash.com/@purzlbaum?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

1995 年 9 月，一位名叫布兰登·艾希的网景程序员仅用 10 天时间就开发出了一种新的脚本语言。它最初被命名为 Mocha，但很快被称为 LiveScript，后来又被称为 JavaScript。

JavaScript 依赖于主机环境，其中最常见的是浏览器。这样的话，就是前端了。JavaScript 也可以在后端运行，因为有几个服务器端主机环境。JavaScript 并不难学，但是如果它是你的第一门编程语言，适应编程所需的思维方式可能会花很多时间。JavaScript 实际上是最容易开始的编程语言之一。事实上，有几种资源可以帮助你轻松地学习它。

根据 Payscale，Javascript 开发人员的平均工资为 83，311 美元。我确实计算过 JavaScript 的年薪为 110，673 美元。据薪酬专家称，JavaScript 的平均薪酬是**99761 美元一年**。

今天我们学习了 3 种不同的网络编程语言。简要的历史，功能，易用性，以及你能从中获得多少。
