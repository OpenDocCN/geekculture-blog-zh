# 什么是时尚搜索，为什么它很重要？

[时尚搜索](https://yesplz.ai/resource/fashion-search.html)是一个包罗万象的术语，包括不同类型的产品过滤和搜索、图像标记以及伴随搜索的 UX。这些组件中的每一个都建立在下一个之上，如果没有特定类型的产品过滤器，用户可能会不满意。或者，如果没有强大的图像标签，用户将很难找到他们想要的东西。这里有一个定义的快速列表，可以帮助你的时尚电子商务优化其时尚搜索:

**时尚搜索:**时尚搜索包括产品过滤器、图片标签和伴随搜索而来的用户体验

**产品过滤器:**产品过滤器是允许购物者根据产品属性缩小搜索范围的工具

**视觉搜索:**[视觉搜索](https://yesplz.ai/resource/ecommerce-guide-visual-search-and-search-clothes-by-image.html)是基于文本的搜索的替代方式，它允许购物者使用图片搜索产品。

**产品属性:**产品属性是产品的定义特征，如价格、尺寸、颜色、合身程度或轮廓。它们也可以被称为产品规格。

**图片标注:**图片标注(或产品标注)是标注产品属性的过程。它可以手动或自动完成，使用人工智能。

以上几个部分结合在一起，为购物者创造了一个强大的搜索体验。

# 传统时尚搜索的问题是

回想一下你最近一次在网上搜索衣服的情形。您可能不得不导航到搜索栏，尽可能输入产品描述，并希望搜索引擎能够产生相关的搜索结果。搜索结果的准确性取决于您决定在搜索中键入的文本，这将导致不同措辞的搜索结果大相径庭。

或者，也许你想探索一个特殊场合的特定产品类别(工作装)，为了探索，需要猜测哪些产品过滤器会引导你选择工作装，因为缺少可用的场合过滤器。

[在 YesPlz，我们发现了传统时尚搜索的 7 个问题](https://yesplz.ai/resource/fashion-search.html):

**1。网站上缺少关键的产品规格:**由于电子商务产品目录的庞大，有数千种产品规格没有包含在过滤器中。

**2。产品规格令人困惑:**不是每个购物者都了解最新的时尚术语。那么，为什么传统的搜索过滤器如此难以解释呢？

**3。缺乏多选项支持:**我们生活在一个复杂的世界，有着复杂的时尚需求——一些购物者想要隐藏身体的某些部分，而炫耀其他部分。这需要多选项过滤器的支持，这样购物者可以准确地选择他们想要的。

**4。搜索结果不是即时的:**令人惊讶的是，许多时尚电子商务页面在选择新产品过滤器时会重新加载搜索结果。

**5。用户会看到长长的文本过滤器列表:**长文本过滤器意味着用户不能完全掌握他们的搜索选项，因为文本太长了，无法阅读。

**6。移动体验弱:**移动端设计不佳的产品过滤器赶走用户。

**7。因为上述问题，时尚搜索结果不准确。**

# 你的时尚搜索有多直观？

直觉是衡量你的时尚搜索有效性的一种方式——用户是否可以在不需要外部工具的情况下轻松导航和理解你的时尚搜索。直觉和时尚搜索齐头并进——你的时尚搜索越直观，用户的体验就越愉快，从而带来更多的收入和回头客。

这里有 3 个简单的方法来衡量你的时尚搜索有多直观:

**1。每次搜索的有效点击量**:当用户进行搜索时，从 A 点(初始搜索)到 B 点(购买)需要多少次点击，更重要的是，这些点击中有多少次帮助了用户，又有多少次破坏了用户的旅程？

有效率的点击和无效率的点击是有区别的——无效率的点击可能包括让用户点击后退按钮，或者重新输入搜索信息。

**2。行话测试:**时尚行话很丰富，即使我们不认识。一个快速的解决方案是通过用户测试来运行你的文本描述，以确保它们是可理解的。但是，这只是一个短期的解决方案。从长远来看，你可以在文本语言旁边使用视觉提示，为用户提供另一个参考点。更理想的是，使用可视化搜索工具来消除对基于文本的主要搜索工具的需求，从而为您的电子商务提供长期搜索功能。

**3。问问你自己:你的时尚搜索反映了用户的购物方式吗？**

用反映用户行为的信息层次来设计你的时尚搜索。例如，避免广泛的监管，如“新来者”——用户很少参与广泛的监管——而是基于搜索行为来监管产品。根据你特定的时尚电子商务，用户可能会更频繁地搜索特定的主题类别或风格。

# 图像标签如何影响时尚搜索的质量

时尚搜索和图片标签是相辅相成的——没有高质量的图片标签，就不可能提供卓越的搜索体验。图像标记是在图像上标记产品属性的过程，通常每个图像包含 20 到 60 个产品属性，这是一个耗时费力的过程。

[图片标签非常重要](https://yesplz.ai/resource/image-tagging.html)，因为它为您的时尚电子商务奠定了基础，提供强大的搜索体验、创建产品描述、提供过滤选项，甚至利用个性化工具。

但是，许多产品标签是由第三方供应商提供的，他们可能会在标记时出错或使用不一致的语言。被错误标记的产品将会隐藏在你的时尚搜索中，从而导致收入机会的错失。

许多时尚电子商务不仅容易受到第三方供应商提供的错误信息的误导，而且内部销售团队也会犯错误，即使他们非常了解产品。为什么图像标签如此容易出错？

许多时尚电子商务品牌仍在使用人工标记，他们可能会反对自动图像标记，因为他们担心人工智能图像标记不准确或可能需要太长时间。
