# UX 训练营 VS UX

> 原文：<https://medium.com/geekculture/ux-bootcamp-vs-ux-on-the-job-b79fe544cd46?source=collection_archive---------13----------------------->

作为一名 UX 设计师，我可以帮助企业打造对人们生活有影响的精彩数码产品。这是一个有趣且有回报的角色，所以难怪未来三十年设计师的数量将增长 100 倍。UX 设计也是[一项抢手的技能](https://business.linkedin.com/talent-solutions/blog/trends-and-research/2020/most-in-demand-hard-and-soft-skills)，它为这个领域带来了大量的新人才。

人们很容易被围绕 UX 设计的炒作所吸引。这似乎好得难以置信，对吗？然而，也有不足之处，尤其是对于新设计师来说。
