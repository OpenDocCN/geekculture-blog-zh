## 第一类和第二类错误之间的权衡

![](img/2547a12800a35a178ab343e834482bf0.png)

Photo by [Yves Scheuber](https://unsplash.com/@xgrt51m?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

**第一类**和**第二类错误**是统计学概念，描述基于统计测试做出决策时可能出现的错误。它们很重要，因为它们帮助我们理解做出错误决策的风险和做出正确决策的收益之间的**权衡**。
