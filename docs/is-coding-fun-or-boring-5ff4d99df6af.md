# 编码基础

什么是编码？编码是为计算机编写指令的过程。这有点像用另一种语言写待办事项。事实上，有几十种不同的编程语言。

例如，编码人员使用 HTML 和 JavaScript 来构建网站。Swift 是 iOS 应用程序开发人员的常用语言。Python 对任何事情都很好。

开发人员使用代码创建从计算器应用程序到整个操作系统的一切。智能手机、笔记本电脑、车载信息娱乐系统和许多其他计算机都使用代码。

程序员可以使用代码创建几乎任何东西。一旦你理解了编程是如何工作的，你就能够制作任何种类的软件或应用程序。

# 计算机编程难学吗？

人们经常想知道计算机编程是不是很难学。毕竟问题有道理——看看上图就知道了。如果你没有编程经验，这可能会让人望而生畏。

令人惊讶的事实是。虽然编码看起来晦涩难懂，但大多数人很快就掌握了窍门。一旦你学会了一种语言，学习另一种语言就变得容易了。

> 但是我试着编码，没有成功。为什么编码这么难？

很多人尝试编码，很容易在早期就气馁。也有可能你第一次尝试的时候没有得到正确的引导。但你并不孤单！无数专业程序员做了多次尝试才找到窍门。

# 有人为了好玩而编码吗？

是啊！许多人出于乐趣和许多不同的原因而编码。对一些人来说，这是构建应用程序的乐趣——结果才是最重要的。对其他人来说，这是创造有用的东西的过程。

编码可能非常吸引人。每一行代码，如果有效，都会导致控制台发生一些事情。看着你的想法一步步成形是一种享受。而且发现一个 bug(并修复它)后的感觉很难复制。

# 什么样的人喜欢编码？

任何人都可以享受编码。然而，喜欢解决问题的人通常会成为优秀的程序员。虽然你不一定要有“工程头脑”才能做好，但如果你也喜欢构建东西，你就会喜欢编码。

在编程的世界里，你会发现艺术家、音乐家、工程师、机械师、商人——几乎是各种各样的人。这是编码最棒的事情之一:任何人都可以做，每个人都可以发挥自己独特的优势，创造出伟大的东西。
