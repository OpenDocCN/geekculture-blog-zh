# 为什么每个人都能从发展人际网络中获益

我要你想象你的朋友/社交圈。他们可能是你经常与之相处的人。最有可能的是，他们有和你相似的人口统计信息。他们很可能做相似的事情并有相似的结果。不，我没有跟踪你。这只是人的本性。大多数大企业都以各种方式从中获利。这是数据被认为如此有价值的原因之一，也是大多数大型科技公司向机器学习人员扔钱的原因之一。

![](img/0c7bbc95fda59732058cfc37e295b66a.png)

These ideas are used heavily by big tech companies. Google, Meta, Netflix, and others all use these principles to recommend things to you based on the behavior of people similar to you. This is an idea called collaborative filtering. Let me know if you want to learn more about it.

这就是问题所在。当你想做一些不同的事情时会发生什么？假设你正在找一份新工作。从统计学上来说，比起强关系(你的朋友)，你更有可能通过弱关系(你知道他的存在，但仅此而已)找到工作。请记住，这项研究是在社交媒体出现之前的 1973 年完成的。现在和弱连接保持联系容易多了。这是为什么呢？

> *有一种理论认为，属于这一类的人*尤其*乐于助人，因为他们倾向于穿梭于不同的社交圈，在不同的公司工作(或者从事完全不同的行业)。因此，与你的关系密切的人相比，他们通常能接触到不同的信息和资源。*
> 
> [来源](https://mdash.mmlafleur.com/networking-with-loose-connections)

你的在线网络以非常相似的方式工作。当我在网上与人互动时，有一件事总是让我大吃一惊，那就是机器学习中人们的多样性。这些人带来了有趣而独特的挑战和创新的解决方案。他们也有自己的一套资源，这可以证明是非常有用的。

此外，在线网络更容易大规模利用。想象一下，你的请求能够同时送达 10000 人。在线网络将为你提供这些。

Internet networking is amazing. Done right, you can reach a lot of people. This does require some patience and trial and error, but the results are worth it. I recently crossed 1M impressions per year. If you’re not a creator, you obviously don’t need to reach such numbers.

# 如何以正确的方式建立你的关系网

不幸的是，大多数人不知道如何交际。尤其是网上。你不知道，有多少人读了我的文章，向我求助

> 嘿，德凡什。热爱你的工作。你能给我做这个大学作业吗？给我介绍一份你网络中的工作？把你的肾捐给我？

请注意，这些人在这条消息之前我并不知道他们的存在。在拿走我的肾之前至少带我去吃顿饭。骑士精神已经过时了。

这不是一个好的联系方式。你不仅很可能会被拒绝，而且你还会给人留下一个糟糕的第一印象。不是你的 W，我的超级天才读者。

有更好的方法。这需要耐心和主动性，但保证会奏效。它将促进你的学习之旅。最棒的是，你不需要有任何网络影响力，也不需要达到我这样的名气高度。任何有技术能力的人都可以做到这一点。

如果有人想看这个例子，可以看下面的视频。那是我和扬尼克·基尔彻的对话视频。Yannic 是机器学习研究领域最大、最受推崇的名字之一。我是通过观察他的论文分解开始写作的。对我来说，这显然是一个重要的时刻，向你展示了这种方法是多么有效。

让我们开始吧。

# 第一步:找到酷孩子派对

第一步是找到和你兴趣相投的人聚集的地方。例如，当谈到机器学习时，有几个 LinkedIn 群组(如 [this](https://www.linkedin.com/groups/961087/) )、Yannic Kilcher 的 Discord、ML Subbeddit 和 Twitter creators，在那里你可以找到许多围绕机器学习的讨论。找到这些，成为这些群体的一部分。现在你有了一个可以与对相似想法和领域感兴趣的人互动和相遇的途径。

![](img/0c161bc3dd5144b94d228f66927e53f2.png)

如果你像我一样是一个创造者，那么你的目标就是最终成为一个有其他人参加的很酷的聚会。对于那些对成为创造者不感兴趣的人，你不必担心这个。所以这个过程就更容易了。

# 第二步:开始增值

这听起来很可怕，但实际上很简单。不总是容易，但很简单。我们以机器学习 Twitter 为例。有很多伟大的创作者在那里发帖(包括你的)。如果他们分享他们对一些 ML 概念的想法，你可以随时贡献你的经验。你可能会问一些关于技术细节的问题，或者只是分享你所知道的相关/冲突的想法。如果你认为他们的立场是错误的，不要害怕不同意。大多数创作者都欣赏经过深思熟虑的反驳。你可以在这里看到[这种情况的一个个人例子，我不同意圣地亚哥的建议，他是一个超级 ML Twitter 的人。他欣赏我的反驳，甚至把我的推文加到他的帖子里。你需要一些时间来找出处理这件事的最佳方法。毕竟每个人的附加值都是独一无二的。这也是这种方法对每个人都可行的原因。](https://youtu.be/X_86gqVPG1I)

总之，现在你已经找到了你所有的酷孩子都去哪里玩了，并且你已经开始参与对话，现在我们进入最后一步(也是最重要的一步)。

# 第三步:耐心点

这是一个让很多人跌倒的步骤。人们期望立即得到满足。不幸的是，好事通常需要时间。你需要坚持几个月(每天大约花 5-10 分钟在每个平台上)才能看到结果。但是不要担心。即使你正在参与的过程中，你也会看到很多好处。成为这些社区的一员会让你接触到大量的新思想、概念和实现。这将极大地推动你的学习之旅和职业生涯。

一旦你开始参与社区，你会发现自己自然而然地与有趣的人联系在一起。你经常与之交流的人。这些人可以帮助你。他们有不同的背景、关系网和经历。你可以向他们征求意见，寻求他们的帮助和建议，甚至互相推荐/帮助。你甚至可以告诉他们某个时事通讯，里面有互联网上最好的内容。
