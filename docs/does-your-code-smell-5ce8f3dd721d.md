> 代码气味是一种表面迹象，通常对应于系统中更深层次的问题。
> 
> 马丁·福勒

虽然如果代码味道留在产品中，这最终会发生，但这并不总是意味着还有更深层次的问题。多种代码气味的累积通常会导致系统中更深层次的问题。
