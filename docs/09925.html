我的孩子没有智能手机，他们很好

原文：https://medium.com/geekculture/my-kids-have-no-smartphones-and-they-are-ok-41793f5ce85c?source=collection_archive---------20-----------------------#2022-01-04

邻居们在街上拦住我，评论说我的孩子打招呼，看着他们的眼睛，进行友好的聊天是多么不寻常。 “孩子们再也不这样做了！”——众人惊呼。我想我知道为什么我的邻居如此惊喜。

我的孩子没有智能手机。

由于他们的妈妈对令人上瘾的技术的危险了如指掌，他们从来没有过任何东西。

然而，我们想给我们的孩子可靠的交流方式，以及安排他们自己社交生活的能力。我们想要他们的手机里有什么？

只有这些:

  • 沟通
  • 安全
  • 独立

不幸的是，普通智能手机实际上是对儿童人性的攻击。注意力分散、有毒社交媒体、令人上瘾的电子游戏，以及色情——仅举几例。

因此，我们多年来试验了各种翻盖手机，结果喜忧参半。

你看，翻盖手机对我的孩子来说并不酷。在朋友面前使用会尴尬。谁愿意自己严厉的父母被人取笑、被人取笑？青少年希望融入智能手机人群，而不是鹤立鸡群！

此外，使用旧GSM标准的翻盖手机存在连接问题——它们会随机断开网络，完全停止工作。真扫兴。

另一个选择是禁用功能，并在普通智能手机上安装家长控制。但我们很快排除了它，原因很简单:这些可以而且将会被黑。相信我，我知道。我一直收到孩子们发来的信息:

“你们这些家长太傻了，以为我们不能黑进你们的家长控制系统！”

面对现实吧，孩子们就是比我们这些科技恐龙聪明。上瘾的技术设计师确保他们有动力找到绕过屏幕时间限制的方法。

我们需要的是一部看起来像智能手机的手机。不需要家长控制的手机。一部允许交流而不会上瘾的手机，所以孩子们会限制自己的屏幕时间。 这就是我们决定为我们的两个大孩子——15岁的男孩上高中，12岁的女孩上初中——购买儿童安全的Gabb手机的原因。

这是他们每天使用“非智能”手机的经历。

男孩，15岁

第一天:第一印象。电话来了。有重大刺激: “看起来像iPhone！”他忙于设置联系人、背景图片，并从chromebook上下载他最喜欢的mp3音乐。在这个过程中，他发现没有太多的存储空间，并接受了这个事实:手机的目的是交流，而不是娱乐。

然后他让我联系他朋友的妈妈，要他的电话号码。他从来没有在翻盖手机上表现出这样的主动性。到目前为止一切顺利！

你看，我的儿子非常关心他的公众形象。当他有翻盖手机时，他断然拒绝使用。“这是古老的技术！太傻了！”他不想看起来与众不同。青少年顺从同龄人——他们只是在父母面前叛逆。当他在公共场合使用Gabb手机时，他看起来就像其他有手机的青少年一样，这对他来说很重要。

几天后，他开始定期使用电话，而且他从来不会忘记带着它。他给邻居的朋友发短信说要聚一聚。他发短信给我说他需要从田径训练中被接走。起初，他拒绝给我们打电话，热情地解释说，他们这一代人已经不打电话了！显然，发短信给他时间思考。不管怎样——他必须学会在需要被接走的时候给我打电话。因为我很少把手机放在身边，这是吸引我注意力的唯一方法。

没有互联网，他无法在手机上获得他最喜欢的迷因，对此他并不感到兴奋。在我看来，这是一件好事——许多十几岁的男孩相互分享的模因是非常不恰当的，并且会让他们在学校、大学甚至未来的雇主那里陷入大麻烦。相反，如果我看到一张有趣的照片通过了我的常识父母过滤器，我会把它转发给他。

在查看了关于童子军活动的电子邮件后，我儿子说:“好了，该给爸爸发短信了”。拿起手机，给爸爸发短信，把手机放回充电站，继续写作业。

电话里没什么可做的了。这是件好事。

如果我现在给我的儿子一部智能手机，我就有可能让他在学校的课桌下玩电子游戏，而不是专心听老师讲课。我知道其他男孩也是这样做的——我的孩子、老师和其他父母公开对此表示哀悼。毕竟，《我的世界》比代数更有趣。

女孩，12岁

一拿到手机，我女儿就用我们的猫的照片和特殊图标定制了她的主屏幕，以便给妈妈和她最好的朋友发短信。然后她开始在手机背面贴上猫咪贴纸。

在使用手机的第一天，她安排了一次和朋友的通宵聚会，以及一次和同学在当地节日的聚会。“我喜欢组织我的社交生活！”拍了更多猫咪的照片，添加到她的手机图库中。我把她的作品照片寄给了她的美术老师，并收到了热烈的赞扬。

今天，她经常通过电话和她最好的朋友一起做作业。不像其他孩子在应用程序上做的那样——分享完成作业的截图来作弊。不，她和她的朋友进行了老式的长时间电话交谈，他们一起解决数学问题。我听到他们互相鼓励继续克服困难，与拖延作斗争，想象得到A的感觉有多好！我女儿告诉我，她被朋友的父母认为是一个“好的影响”。

他们像过去的女孩一样在电话上交谈。他们给自己的宠物拍照，然后互相发送。我们的猫的照片会以彩信的形式发给一些选定的朋友，而不是通过社交媒体向全世界传播。我女儿知道不把照片放到网上对保护她的隐私和名誉的重要性。或者我们的猫。:)

她给朋友发短信。她打电话给奶奶。

她在与人对话，而不是与算法对话。

她抱怨校车上的孩子们——当她想社交时，他们在打电话。“被人忽视真是太让人沮丧了！”

如果我现在给我的女儿一部智能手机，她会受到同龄人的压力，使用每个人都在使用的应用程序，并有可能被卷入危险的社交媒体世界。当时，她的学校辅导员正在为中学女生进行超负荷的风险评估。当她这个年纪的女孩住进精神病院的时候。不敢打开Instagram的《虫虫之罐》。

即使有了儿童安全手机，我仍然坚持让我的孩子把手机放在楼下的充电位置过夜。我不希望他们的睡眠被他们朋友的信息打扰。我希望他们一辈子都不要在卧室里使用任何科技产品，因为当他们最终拥有一部智能手机时，能够脱离它是一项重要的幸福技能。

妈妈，中年

首先，这些手机允许我给我的孩子扩展他们的社交生活。没有智能手机通常带来的负面影响——强迫性使用(T1)、分散学习注意力(T3)和暴露在黑暗的网络中(T5)。我一直在等待我的孩子安排他们自己与朋友的“聚会”(过了一定的年龄，你就不能称之为“玩耍日”！).但是如果没有电话，他们会很乐意使用，这是不可能的。我负责给他们朋友的父母发短信，询问他们应该在何时何地见面的所有细节——就像他们还是蹒跚学步的孩子一样。这占用了我很大一部分时间，让人精疲力尽，还有点尴尬。

像迷你版的“发射失败”。

现在他们用手机安排朋友见面。这款手机没有Discord(在线游戏)聚会的选项，也没有Snapchat、Instagram或众多在线平台中的任何一个，如今孩子们在独自坐在房间里的时候假装“社交”。

当我的孩子外出参加社交活动时，他们的手机有GPS跟踪功能，可以减轻我作为父母的焦虑！我已经在我的智能手机上安装了这个应用程序，并将其与我孩子的手机配对。现在我可以实时看到我的孩子在他们应该在的教学楼里。我看到我的儿子在进行田径训练，我的女儿在图书馆。我可以看到他们在附近哪里骑车——或者和爸爸在州的另一边露营。我喜欢总是知道我的孩子在哪里的保证。

我还喜欢的是，广告商、网络掠夺者或有毒的同学不能利用我孩子的位置来伤害或操纵他们。

他们如何与只使用Snapchat这样的应用程序进行交流的朋友保持联系？他们不知道。除非这种交流是面对面进行的，或者是通过普通的短信或电话进行的，否则这种交流根本不会发生。我的孩子错过了吗？是的，他们有。

  • 如果有一个聚会他们没有被邀请，他们也不知道
  • 如果有关于他们的令人讨厌的流言蜚语流传，他们会浑然不觉
  • 如果有戏剧，他们不是其中的一部分。

我有点喜欢这样。

与FOMO(害怕错过)相反的是JOMO(享受错过)。

当我的孩子们放学回家时，他们看起来很平静，不焦虑。他们打开行李，边看书边吃零食，然后开始做作业。在漫长的一天学习后，他们需要时间放松和减压。

他们对24/7社交媒体狂热的无知是他们的福气——也是我的福气。

现实生活是最有趣的事情

我想提出另一个要点。当我跟踪我的孩子的位置时，我可以看到他们在哪里，但不知道他们在做什么。

然而，我可以假设他们不会盯着手机看。

我不必担心他们是否注意学校里的老师、体育训练时的教练或午餐时的朋友。我知道他们是——因为没有更好的选择了！在Gabb手机上没有太多事情可做，除非他们需要联系一个真实的人。没有游戏。没有社交媒体。没有网络。我想他们可以玩日历或计算器，但这些不是特别有趣。

默认情况下，他们眼前的真实生活成为最有趣的事情。

这就是精心打造他们的环境以防止不良数字习惯生根发芽的神奇之处。智能手机是孩子们手中最迷人、最容易上瘾的毒品。拿掉智能手机——他们开始关注其他事情。改变它们的栖息地，你就改变了它们的生活。为他们在学业、运动和人际关系上的成功做准备。

毫无疑问，我的孩子错过了网上发生的许多好事。这是我们作为一个家庭愿意付出的代价。

  • 他们不会24/7都与他们的同伴保持联系，但他们也不会立即回复信息。
  • 他们错过了通过社交媒体与同龄人联系的机会——但他们乐于与孩子和成年人面对面交流。
  • 没有人在身边的时候，他们很舒服的一个人带着自己的想法和 的书 。
  • 他们什么也不发表，所以他们在网上得不到认可——但是也没有人能用他们的观点来反对他们。巨魔无法伤害他们。

作为父母，我们关注智能手机的危险，但往往忽视了数字媒体对健康人类活动的取代。如果孩子们不使用手机，他们可以做的所有事情。

我们的家庭试图在孩子们的生活中创造这个空间。创造一个空白——这样孩子们就可以用屏幕时间之外的东西来填补它。我站到一边，观察我的孩子如何在没有智能手机或其他技术的情况下打发空闲时间。他们阅读。他们玩。他们互相争斗。健康的无聊真的有用。晚上，他们有时间看电视。

屏幕和现实生活之间是有平衡的。

我认为期望孩子抵制智能手机是不公平的。等到第八次活动鼓励父母签署一份保证书，至少等到八年级再给孩子一部智能手机。我签了。根据最近数字空间的情况，我想我们可能会等到18岁。

我们目前正在讨论的拥有智能手机道路上的一个自然减速带是让孩子们用自己的钱购买他们的第一部智能手机。能够赚钱和存钱需要一定程度的成熟，以及负责任地使用技术。至少这是我们的希望。

我的孩子是否对他们没有智能手机感到不满？一点点。但他们已经接受了一个事实，即他们拥有智能手机的年龄将比同龄人晚得多。

当然，像他们所有的朋友一样，他们宁愿拥有手机。他们相信他们能够处理好这件事。但我们仍在等待。就像电影《勇敢的心》中一样，我们大喊“站住！”推迟不可避免的战斗。令人上瘾的技术很强大——我们想给我们的孩子时间变得更强大。

是的，当他们的父母一遍又一遍地解释智能手机对他们的心理健康、学习成绩和声誉的危险时，他们会翻白眼。正如保护年轻的眼睛的克里斯·麦基纳在童年2.0纪录片 : 中所说，“如果你直到你至少有点讨厌了才跟你的孩子谈论互联网安全，那你做得还不够”。

我的孩子们向我保证我已经够烦人了。

TechDetox妈妈发现自己的孩子在接受令人上瘾的技术并决定反击。

她对技术和心理之间的关系的研究试图揭示数字行为操纵如何影响人类福祉。

她在她的博客TechDetoxBox.com上写道，要找到解决方案来保护我们的家庭，恢复我们的人性。
