Python、PHP和Javascript

自1989年以来，我们都在发展服装，以便能够通过数字手段与他人联系。实体店现在都在网上，人们通过在线平台交流，我们甚至现在通过网络工作。但是你知道是什么技术在幕后运行，让你可以享受万维网吗？

有许多类型的编程语言运行在web后端，如Java、C#、JavaScript、Ruby、Python和PHP等。今天我们将只讨论其中的三种语言。历史和他们如何工作，以及其他语言的优点和缺点。

本文先说Python，PHP，再说Javascript。

巨蟒

Python是一种广泛使用的通用高级编程语言。它最初是由吉多·范·罗苏姆在1991年设计的，由Python软件基金会开发。它主要是为了强调代码的可读性而开发的，它的语法允许程序员用更少的代码行来表达概念。

有了Django和Flask这样的框架，这是快速简单地开发web的一个很好的选择。Python在所有编程语言中对ML和AI的支持是最好的，因为有像TensorFlow和Keras这样的重量级库。此外，Python的简单语法和简单性使它成为ML专家和数据科学家的编程语言选择。据Indeed报道，Python程序员在美国的平均工资是 $110，840/年，这使得Python成为收入第二高的编程语言，仅领先于Ruby。

PHP

PHP是由拉斯马斯·勒德尔夫在1994年秋天构思的。早期的非公开版本被用在他的主页上，以跟踪谁在看他的在线简历。其他人使用的第一个版本是在1995年初的某个时候推出的，被称为个人主页工具。

PHP被认为是后端脚本语言。PHP脚本通常写在服务器上，前端或客户端代码会与之交互。

一般来说，对于刚开始学习编程的人来说，PHP被认为是一种容易掌握的编程语言。和任何编程语言一样，PHP有编码、缩写和算法的规则。但是，如果您完全是编程新手，可以使用许多资源。

PHP仍然是世界上许多开发者使用的最流行的服务器端脚本语言之一。如果你精通PHP，就此而言，你不必太担心你的职业或薪水。PHP开发人员的平均工资。glass door:81k美元/年，最低55k美元，最高100k美元。zip recruiter:86k/年，低的16k，高的137k。

Javascript

1995年9月，一位名叫布兰登·艾希的网景程序员仅用10天时间就开发出了一种新的脚本语言。它最初被命名为Mocha，但很快被称为LiveScript，后来又被称为JavaScript。

JavaScript依赖于主机环境，其中最常见的是浏览器。这样的话，就是前端了。JavaScript也可以在后端运行，因为有几个服务器端主机环境。JavaScript并不难学，但是如果它是你的第一门编程语言，适应编程所需的思维方式可能会花很多时间。JavaScript实际上是最容易开始的编程语言之一。事实上，有几种资源可以帮助你轻松地学习它。

根据Payscale，Javascript开发人员的平均工资为83，311美元。我确实计算过JavaScript的年薪为110，673美元。据薪酬专家称，JavaScript的平均薪酬是99761美元一年。
