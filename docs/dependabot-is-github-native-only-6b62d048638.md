## 告别(谢谢)依赖机器人预览版

2021 年 4 月 29 日令人惊叹的 dependent bot(预览版)关闭了它的大门:所有现有的客户和存储库都将被迁移到 GitHub-Native dependent bot 上，与整个 GitHub 平台无缝集成。
