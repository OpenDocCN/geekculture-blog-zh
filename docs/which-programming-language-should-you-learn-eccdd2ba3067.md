# 为什么选择 Java、Python 或 C++？

在我详细介绍每一种语言之前，我将解释为什么我把范围缩小到这三种。

## 为什么不是 JavaScript？

如果你在谷歌上搜索什么是最流行的编程语言，JavaScript 会排在前 5 名。尽管如此，**我不建议从 JAVASCRIPT 开始。这是为什么呢？首先，如果你已经很好地学习了 Java、Python 或 C++，学习 JavaScript 将会很容易，所以以后需要的时候再学也不成问题。JavaScript 不会教你编程的基础，它是一种非常补丁的语言。其他编程语言让你可以控制你的代码，每一行都可以被跟踪和理解，而 JavaScript 有很多随机的组件，试图解决你眼前的问题。尽管 JavaScript 在多种情况下都很有用，但它主要用于 Web 开发，作为一名程序员，我建议开始时尽可能多才多艺，直到最终发现并强化自己的专长。**

## **基础知识和编程概念**

毫无疑问，每种编程语言都有其优缺点和特定用途。Swift 就是一个很好的例子，因为它仅用于编写 MacOS/iOS 应用程序。但是，即使你的梦想目标是成为一名 iOS 开发者，我还是推荐这几门语言入门。我相信这些语言会教你编程的基础，一旦你熟悉了基本概念，学习其他语言就是小菜一碟，只是语法和具体特性的问题。这 3 种语言非常适合学习在任何类型的编程中广泛使用的概念。

## 面向对象编程

Java、Python 和 C++都是面向对象的语言。面向对象编程(OOP)是学习如何编程的一个非常重要的概念，如果不是最重要的话。还有其他类型的编程，如过程化编程，但这也包括在 OOP 中。在我看来，理解对象是完全理解好的编程的门户。之后，您总是可以退回到低级语言，如 C 或汇编语言。

## 可见且有益的代码

学习像编程这样的新技能可能会令人畏惧，当你能够看到学习中的进步时，这真的很有帮助。是的，对于大多数语言来说，你可以在控制台中看到你的代码运行，但是这些语言设置的简易性让你可以很容易的看到每一行代码到底在做什么。还有许多易于使用的 GUI 库，您可以实现它们来进行下一步并可视化您的工作，而不仅仅是通过控制台。

## 流行

Java、Python 和 C++是最流行的编码语言。这意味着许多程序和公司使用它，所以你的技能可以直接转移到行业。这也意味着网上有丰富的资源。当您的代码不可避免地出现问题时，您无疑会需要这些资源。有些人以前几乎总是遇到同样的问题，所以解决方案很多，也很容易获得。谷歌和 StackOverflow 是你最好的朋友。

由于这些编程语言被广泛使用，因此也有大量的外部库来扩展和简化代码的特性和功能。

# 哪个更适合你？

既然我们已经确定了前三名，下面是我推荐 Java 而不是 Python 或 C++的具体原因。请记住，在某些情况下，一个可能更适合你，但总的来说 Java 是最好的。以下是原因:

## 简单和完美的水平

Python 是最容易掌握和学习的编程语言。C++无疑是三者中最难的。然而，Java 介于两者之间。现在你可能会想，如果 Python 是最简单的，为什么不学 Python 呢？在编程中，有不同的层次。不用太深入，编程中的级别本质上意味着计算机为你做了多少，你必须手动做多少，例如:分配内存和资源。

有了 Python，计算机为你做了大部分工作。它允许您编写更简单的代码，而无需担心底层机制。然而，从这一级别的编程开始也意味着你对较低级别的考虑较少，这导致对代码如何工作的理解较少，以及对安全性或效率等其他因素的关注较少。这种简单性导致 Python 被用于许多原型和小规模应用。对于某些应用程序来说，这可能没问题，但是当您想要处理更大规模或定制的项目时，这种缺乏理解可能会阻碍您

C++是其中最难的。在较低的层次上，您必须担心诸如指针和垃圾收集之类的事情。掌握 C++无疑会让你成为更好的程序员，也会让学习和过渡到其他语言变得更加容易。然而，对于没有编码经验的人来说，这种难度可能会令人困惑和害怕。我开始学习 C++，我已经学了 2 年了；我知道如何编写这样那样的代码，但是直到我学习了 Java，我才完全理解并自信地编写代码。此外，许多现代编程语言现在都有自动垃圾收集功能，并通过指针来帮助您(尽管理解这些概念仍然很重要)。还有一些头文件，对于初学者来说很难建立和理解。

Java 处于这两者之间的完美水平。在编码时，您不必担心垃圾收集。这里也没有指针，但是仍然有参考来帮助你理解一个程序是如何根据它与计算机的内存分配和资源的物理关联来工作的。Java 允许你控制计算机的关键功能，如文件系统、图形或声音，同时保持相对简单。学习 Java 之后，我能够理解其他编程语言，适应 Python 和 Swift 等其他语言也是轻而易举的事情。

## 轻便

Java 具有极强的可移植性。这意味着它几乎可以在任何环境下轻松运行。它可以在网络、桌面和移动设备上运行。尽管其他语言也可以移植到不同的平台上，但这样做可能需要更多的努力。

## 工作机会和雇佣能力

最近一项 2020 年的研究显示，Python 是市场上最受欢迎的编程语言。这是意料之中的，因为 Python 使用起来最简单，这意味着它的生产时间将是最快的。大多数公司，尤其是大公司，重视效率和可伸缩性，Python 不允许最高效或最快的代码，因为它的级别很高。然而，Java 一直是这个行业的主流，并且一直保持着前三名的位置。你必须记住，除了他们的前置网站，公司还有更多的项目。例如，像亚马逊或脸书这样的大公司有成千上万的内部工具和程序来保持公司的运行，通常情况下，这些都是用 Java 编写的。很抱歉打破你的幻想，但大公司的大多数空缺职位都是致力于这些内部工具而不是主要产品，但这也意味着有大量的空缺职位需要填补。

在你的工具箱里有一个相对低级的语言，比如 Java 或 C++，会对你的简历有很大的帮助。精通 Java 和/或 C++向公司和招聘人员表明，你对编程和计算机有很强的理解，而不是编写代码来解决问题。

## 特化作用

如上所述，每种语言都有其优点和用途。这三种语言比其他语言更加通用，但是自然地，有些领域会比其他领域使用更多的语言。以下是这些语言流行的领域:

**Python:** 数据科学、机器学习/AI、网页开发、游戏开发、小规模原型和应用

**Java:** 后端服务、移动应用、物联网、云计算、内部软件、企业架构

C++: 操作系统、系统/应用软件、驱动程序、客户端-服务器应用程序、嵌入式固件

## **学习环境**

在学习如何编码时，你也应该记住你的学习环境。如果你打算自学，python 可能最适合你。如果你有资源和/或指导者/导师/老师，那就瞄准 Java 或 C++。我这么说只是因为学习 python 是直接的和渐进的，而在 Java 和 C++中，你需要在开始时忽略某些概念，因为它们更复杂，以后会教授。Java 和 C++也很难调试，所以有外部的帮助会很有帮助。

# 结论

总之，这三种编程语言都是您编程之旅的良好起点。Java 是最理想的，因为我相信它是复杂性和简单性的完美平衡。尽管如果你已经尝试过 Java 或者觉得你需要一个更短的踏脚石，学习 Python 是完全没问题的。我之所以推荐 Java 而不是 Python，最大的原因是因为大多数学习 Python 的程序员都太安逸了。他们能够让事情运转起来，所以他们不考虑其他任何事情，而 Java 本质上迫使你理解这些概念，以便让事情运转起来。Python 对于初学编程的人来说真的真的很好，但是请不要感到舒服，即使你已经对它感到舒服了，继续学习更多的概念，更深入地学习这门语言。[**这里的**](https://betterprogramming.pub/why-every-developer-should-learn-computer-science-theories-first-bb49781a3432) 是一篇写得非常好的重要文章，讲述了为什么理解编程概念非常重要，以及为什么它不同于运行一个程序。不幸的是，有大量的*程序员*只交付最少的、低效的或者复制粘贴的代码，甚至不理解代码是做什么的。这种*“只要有效”*的心态对程序员来说是有害的。

再次重申，因为这很重要；之后你可以随时学习其他编程语言。仅仅因为你开始学习一种语言并不意味着你不能学习其他语言。这也是我推荐 Java 的主要原因，因为以后学习其他语言会简单得多。

## 再次总结

如果你真的想成为一名伟大的程序员，并且从一开始就理解重要的概念，那么 Java 就是适合你的语言。但是从 Python 开始可以获得更简单、更容易、更吸引人的体验。但是如果你确实从 Python 开始，请超越基础知识，理解 Python 经常帮助你理解的内部机制和基本概念。如果你真的雄心勃勃，你可以从 C++开始。
