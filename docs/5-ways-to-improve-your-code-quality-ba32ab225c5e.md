# 提高代码质量的 5 种方法

> 原文：<https://medium.com/geekculture/5-ways-to-improve-your-code-quality-ba32ab225c5e?source=collection_archive---------9----------------------->

四眼原则是一个非常简单的概念，意味着代码应该由至少两个人来审查。今天，最常见的代码审查方法是通过拉请求。代码的质量是由几个因素决定的，这些因素可以用 linter 自动完成，或者偶尔手工检查。本文讨论了几种提高代码质量的方法。继续阅读以了解更多关于这些技术的信息。
