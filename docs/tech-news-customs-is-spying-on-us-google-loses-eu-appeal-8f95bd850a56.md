# 海关有你的数据

![](img/937ecfe28df46640b2e9581a5885c586.png)

旅行是伟大的，尽管机场通常不是超级令人愉快的，但我们忍受它，并不太想 TSA 检查的不便，行李事故和其他小麻烦。然而，与最近的发现相比，这一切都相形见绌:美国海关和边境保护局(CBP)的雇员一直在从旅行者的手机、平板电脑和笔记本电脑中收集所有数据。重申一下，这是指你。不是联邦调查局名单上的一些人或那些与国土安全部(DHS)有麻烦的人，只是普通的旅行者，他们不幸通过了我们的边境检查站。这意味着通过陆地、空中和海上旅行，所以无论你是飞往坎昆，开车去加拿大还是乘船去欧洲，你的数据现在都存储在一个巨大的数据库中，DHS 可以完全访问。据报道，这些代理每年从多达 10，000 台设备上收集信息。

现在，你可能会说，没关系，这不是一些随机的人，这是*安全*的部门，这很好。但是猜猜谁能进入数据库？2700 名 CBP 警察，他们甚至不需要搜查令就可以查看你的私人信息。雪上加霜的是，这个包含普通人非常隐私的信息的数据库被称为“自动定位系统”，即使它在没有你的情况下没有进行任何定位，也无法区分可能的恐怖嫌疑人和带着家人去温哥华的两个孩子的父亲。

虽然数据收集让我感到困扰，但真正困扰我的是对数据库的无授权访问。轻松获取成千上万美国公民的私人数据并不容易。其实根本不应该是个东西。

# 谷歌上诉失败，面临 41.2 亿美元罚款

![](img/3eba34b6a98a631d778e3e1ad86e4f3a.png)

Christian Lue

[回到 2018 年](https://www.cnbc.com/2018/07/10/eu-hits-alphabet-google-with-android-antitrust-fine.html)，在世界真正出问题的前一年，欧盟以反垄断为由向谷歌开出了 50 亿美元的罚单。监管机构表示，谷歌迫使安卓手机制造商在所有设备上预装谷歌服务和应用，这给了自己不公平的优势。谷歌提起上诉，现在，四年过去了，结果出来了，他们不支持这个科技巨头。

最终的决定是，谷歌仍然需要支付罚款，只是略有减少——41.2 亿美元。尽管判决是一样的，谷歌仍然试图为其强迫安卓用户使用 Chrome 和谷歌搜索进行辩护，称 iPhones 也预装了一些应用程序。这让法院无动于衷，因为最近的判决称罚款“考虑到侵权的重要性是适当的”。也许不在 Android 设备上安装 Chrome 会更便宜，但话说回来，谷歌什么时候变得理性而不是赚钱了？

这是迄今为止对谷歌征收的最大一笔罚款，而且很可能不会是最后一笔。随着企业越来越多地投诉并试图通过法院上诉，时间会证明这些数字是否会继续增加，或者欧盟是否会开出更少的罚款。随着立法者打击谷歌和苹果的垄断行为，我预计未来几年将会有更多类似的新闻。
