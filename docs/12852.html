什么是equals()？

在Object.java中的默认代码中，等于()定义如下:

public boolean equals(Object obj) {
    return (this == obj);
}

该方法使用“==”来比较两个对象。" == "比较Java中的引用地址。如果两个对象引用同一个地址，默认情况下它们是相等的。
