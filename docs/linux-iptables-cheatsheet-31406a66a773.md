## 了解如何使用 linux 的默认防火墙工具 IPTables

![](img/2dcc07a4cca552a6b22bd74d1c9db751.png)

Image by [OpenClipart-Vectors](https://pixabay.com/users/openclipart-vectors-30363/?utm_source=link-attribution&utm_medium=referral&utm_campaign=image&utm_content=151619) from [Pixabay](https://pixabay.com//?utm_source=link-attribution&utm_medium=referral&utm_campaign=image&utm_content=151619)

Iptables 是一个有状态的软件防火墙，默认情况下，大多数 Linux 发行版都附带它。它可以成为包过滤和路由的非常有用的工具。这看起来很复杂，但是一旦你理解了它的工作原理和结构，编写/编辑规则就变得容易了。
