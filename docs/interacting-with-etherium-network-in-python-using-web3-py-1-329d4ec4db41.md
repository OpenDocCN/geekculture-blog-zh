## 如何连接到以太坊区块链并使用 Web3.py 库查看区块和交易的详细信息

Web3.py 是一个 python 库，可以用来与以太坊区块链进行交互。
该库具有修改区块链状态的功能(发送和授权交易)以及不改变状态的功能(查询余额、区块详情、交易等)。).
