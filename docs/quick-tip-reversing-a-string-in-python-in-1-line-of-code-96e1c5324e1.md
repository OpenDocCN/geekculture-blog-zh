该语法通过将字符串转换为数组(在 python 中，字符串实际上是字符数组)然后对其切片来反转字符串。数组切片的语法如下:`[start:end:step]`其中空开始表示 0，空停止表示数组的长度。 **So [::-1]表示从数组的第 1 个元素到最后一个元素按逆序切片。**
