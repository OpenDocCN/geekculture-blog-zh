5个方便的扩展，帮助您入门，让您的编码生活更加轻松有趣

为什么是Visual Studio代码(VS代码)？

作为开发人员，你为什么会选择使用VS代码？微软的任何操作系统、任何编程语言、任何软件的方法确实为他们以及超过1400万采用文本编辑器和集成开发环境的用户带来了回报。这确实是一个一体化的解决方案，但是并不是万能的，万能的，功能是真正深入的，当你考虑到扩展市场中数以千计的可用扩展时，几乎没有什么是这个解决方案做不到的。如果您正在学习编码，开始了编码训练营或其他学术项目，或者已经是一名经验丰富的开发人员，VS Code是一个很好的工具。

扩展提供什么？

管理扩展很容易，它们允许你添加语言、主题、调试器、特定于语言的功能、可访问性和易用性工具、协作工具等等。让我们来看看5，让你开始，我鼓励你探索你的特定需求的市场。如果你对VS代码中的某些东西感到沮丧，或者希望有一个“做那件事”的特性，很可能已经有一个扩展了！

Live Share :微软自己的扩展之一，允许你在代码上与其他人协作，包括共享你的编辑器、调试会话、终端会话、本地主机实例等等，同时保留你的VS代码首选项，使用你自己的光标，以及其他本地特性。这对于结对或分组编程来说是非常棒的！

Live Share扩展包 :这个微软扩展包为Live Share增加了一些附加功能，比如集成的音频和文本聊天。

括号对上色器2 :一个UI增强扩展，允许按颜色匹配括号对，使得跟踪括号对的开始和结束更加容易，特别是当你的代码变得越来越复杂或者有嵌套层的时候。这是我个人最喜欢的，也是我认为不可或缺的。

vscode-icons :另一个UI增强扩展，在浏览器树和编辑器标签中的文件和文件夹旁边添加图标，使导航更容易。

Live Server :在localhost上为您的前端代码(HTML、CSS、JavaScript)启动一个本地的、实时的开发服务器，该服务器具有实时重新加载功能，允许您在编写代码时在浏览器中动态查看更改。

摘要

探索市场以找到适合您的解决方案，并查看顶级扩展以及特定于您的编程语言的扩展。有时你甚至不知道你需要这个扩展，直到你读到它并有了那个“尤里卡！”瞬间。感谢您的阅读和快乐的黑客！
