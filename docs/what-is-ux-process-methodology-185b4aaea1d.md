# 什么是 UX 过程方法论？

> 原文：<https://medium.com/geekculture/what-is-ux-process-methodology-185b4aaea1d?source=collection_archive---------14----------------------->

好的 UX 过程方法应该从定义产品开始。确定用户概况后，UX 设计师应该进行研究和采访主要利益相关者。最终目标是创造一个直观、有用和吸引人的设计。
