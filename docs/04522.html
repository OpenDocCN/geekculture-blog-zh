图像处理OpenCv简介

OpenCv是实时计算机视觉、图像处理和机器学习库的集合。它支持许多编程语言，如c++，java，python。在本文中，我们将通过使用open cv进行图像处理来学习它的基础知识。

装置

让我们从创建虚拟环境开始。

$ virtualenv venv
$ source venv/bin/activate

现在我们将安装opencv。

$ pip install opencv-python
