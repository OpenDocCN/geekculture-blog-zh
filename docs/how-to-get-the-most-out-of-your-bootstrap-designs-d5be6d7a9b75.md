# 为什么 Bootstrap 是开发移动网站的优秀框架？

毫不奇怪，Bootstrap 是从事各种动态 web 应用程序的前端开发人员的最爱。这是因为它遵循了移动优先的概念，强大的工具和丰富的功能支持独立的主题。

自定义模板使开发人员能够构建开源库，通过将内容打包到组织有序的块中来自动上传内容。

可以肯定地说，即使是具备 HTML、CSS 和 JavaScript 基础知识的人也可以在几分钟内建立一个图片完美的网站。

Bootstrap 知道用户在什么设备上浏览，它可以快速适应屏幕大小，但不需要你提前准备多个布局。相反，它以基于网格的格式定位元素。

引导布局是对传统 HTML/CSS 块编辑工具的一个升级。它们将媒体内容缩放到相对单位，防止它们超出页面边界。

作为一个额外的好处，Bootstrap 通过采用 CSS 样式并仅在它们满足正确的条件时包含它们来解决媒体查询的问题。

引导框架通过可重用的代码片段消除了重复代码，为您提供了遵循移动设计最佳实践的预构建布局。

通过将图形设计与网页编码结合起来，Bootstrap 可以节省您测试网站性能的时间和精力。换句话说，Bootstrap 作为一种 web 开发捷径的成功有很多关键因素。

Bootstrap 在几个方面优于传统框架:它保留了一致的环境，并确保您的代码在其他标记上可重用。因此，在 WordPress 或 Magento 这样的 CMS 上实现起来很容易。
