大多数人认为内幕交易是基于机密或非公开信息交易公司股票。然而，内幕交易也延伸到了不可替代代币(NFT)的领域。不幸的是，OpenSea 发现了一个其员工进行内幕交易的案例。

OpenSea 是 NFT 交易的一个流行的点对点市场。前产品负责人 Nate Chastain 在知道 OpenSea 将在网站首页展示这些艺术品之前不久购买了这些艺术品。这一行为构成了内幕交易的典型案例。

作为回应，OpenSea 实施了防止未来内幕交易事件的政策。新政策禁止员工利用非公开信息进行交易。他们还禁止员工买卖 OpenSea 推广的特色产品。

“这令人非常失望，”联合创始人兼首席执行官德文·芬泽说。“我们要明确的是，这种行为并不代表我们团队的价值观。我们对此非常重视，正在对这一事件进行立即和彻底的第三方审查，以便我们对事实和我们需要采取的其他步骤有一个全面的了解。”

## 【OpenSea 如何发现内幕交易

以太坊区块链 crypo 交易员揭露内幕交易。NFT 是在区块链上出售和交易的数字资产。大多数 NFT 交易都在以太坊区块链进行。

一些密码交易员注意到一个匿名用户购买了 NFT，然后出现在 OpenSea 的主页上。一旦价格飙升，同一个匿名用户就卖掉了 NFTs。匿名用户通常在几个小时内就获得了可观的利润。

一名密码交易员查看了区块链的交易记录，然后将 NFT 的买卖与内特·查斯坦的公共钱包联系起来。这名交易员注意到，这些可疑的 NFT 交易的利润回到了同一个公共钱包。

一些人将这一事件解读为 NFT 市场倾向于投机，而不是艺术家的支持。软件公司 [Glitch](https://glitch.com/) 的首席执行官阿尼尔·达什(Anil Dash)认为:“人们将此视为内幕交易的事实表明，这是证券交易(或者仅仅是赌博)，而不是旨在支持艺术家的事情。”。“当艺人在平台上被敲诈时，没有类似的公开声明。如果 Etsy 的员工从他们平台上的创作者那里购买特色产品，那就太好了。没有人会犹豫。因为他们会支持他们的目标，”Dash 进一步评论道。

## **创始之旅**

Devin Finzer 和 Alex Atallah 在 2017 年创立了 OpenSea。总部位于纽约的 OpenSea 已经成为最受欢迎的非金融交易平台之一。2022 年 1 月，OpenSea 筹集了 3 亿美元的融资，对这家初创公司的估值为 133 亿美元。2021 年 7 月，NFT 平台的价值相当于 15 亿美元。由于最新的估值，芬泽和阿塔拉成为世界上第一批 NFT 亿万富翁。

许多著名的加密风险基金和交易平台都在支持 OpenSea。支持者包括安德森·霍洛维茨、比特币基地、达珀实验室、Y Combinator、创始人基金和区块链资本。

Fred Ehrsam 是 Paradigm 的管理合伙人，Paradigm 是一家专注于加密和 Web3 公司的投资公司。关于 OpenSea，他说，“Devin 和 Alex 在过去的四年里表现出了真正的勇气，克服了不确定性，并坚持将 NFTs 视为改变互联网和世界的原始工具。”

该公司打算利用最新一轮融资的资金来加速产品开发、壮大团队和改善客户服务。目前，OpenSea 有大约 70 名员工。它还在 NFT 和 Web3 社区寻找投资机会。

## **NFTs 进入主流**

非功能性测试越来越成为主流。许多名人、时尚品牌和其他非技术人士正在涉足 NFT 空间。例如，帕丽斯·希尔顿最近与两位女性数码艺术家合作推出了 NFT 系列。希尔顿也是 NFT 另一个平台 Origin Protocol 的投资者。同样，2021 年末，意大利奢侈时尚品牌杜嘉班纳(Dolce & Gabbana)销售一空其九件套 NFT 系列。这个名为 *Collezione Genesi* 的作品集创收约 600 万美元。

令人瞠目的价格消息在 2021 年初引起了 NFTs 的主流关注。例如，佳士得拍卖行以 6900 万美元的价格出售了 NFT 数字艺术作品“每一天:前 5000 天”。此外，NFT 的平台，如文克莱沃斯双胞胎的 Nifty Gateway，吸引了 NFT 高端艺术品销售的关注

结果，很多人踏入了 NFT 游戏。世界各地的数字艺术家成为收藏家和创造者。一些人甚至改变了他们的职业轨迹，让自己沉浸在非功能性测试中。例如，来自佐治亚州的 27 岁的前时装设计师 Dani 从世界女装系列购买了 NFT。她花了 17，000 美元购买 NFTs，她的投资组合现在价值约 715，000 美元。

这些外部事件为 OpenSea 的成功奠定了基础。OpenSea 拥有先进的搜索功能，使其网站比许多竞争对手更加用户友好。这也使得拥有 NFT 成为一种身份的象征。当用户在以太坊区块链上购买新的 NFT 时，购买历史会显示在 OpenSea 的网站上。

随着 NFT 越来越受欢迎，许多 OpenSea 用户将他们的 Twitter 个人资料图片改为他们拥有的 NFT 图片。“在嫉妒和欲望的驱使下，这变成了一个循环反馈回路。风险投资公司 1Confirmation 的合伙人陈俊澎开始说道。
