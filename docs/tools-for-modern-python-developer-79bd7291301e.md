# **码头工人**

Docker 的技术是独一无二的，因为它专注于开发人员和系统操作员的要求，使用 Docker 容器将应用程序依赖关系与基础架构分离。容器和虚拟机具有相似的资源隔离和分配优势，但功能不同，因为容器虚拟化的是操作系统而不是硬件。集装箱更便携，效率更高。
