# 深入研究数据结构和算法

> 原文：<https://medium.com/geekculture/dive-deeper-into-data-structures-algorithms-through-application-c71d262ceb78?source=collection_archive---------24----------------------->

在这个博客中，我分享了我学习各种数据结构和算法(DS&A)的经验&在花时间创建一个应用程序并利用我所学的知识后，我如何找到了更好的理解。

一个多月前，在二月初，我决定花时间学习 DS&A，希望学习它们能让我在未来的项目、面试和工作中使用它们。我每天会花很多时间学习不同的概念。学习如何遍历二分搜索法树，理解哈希映射和对象之间的差异，记忆不同排序算法的大 O 符号，等等。

然而，在我学习的时候，有一件事困扰着我。我能够记住所有这些概念，然而，这种学习感觉非常肤浅&我会对自己说“我能知道什么时候真正使用这些信息吗？”或者“一两个月后我会记得这些东西吗？”。对这些问题的现实回答是…不算是，至少没有达到我想要的理解水平。

这时，我决定将我的学习提高到一个更深的层次&创建一个利用这些概念的应用程序。一个不仅利用 DS&A 的应用程序，而且允许使用该应用程序的其他人了解他们是如何工作的。因此，我已经开始创建一个 web 应用程序，它采用数据结构或算法&创建一个动画，一步一步地演示它们是如何工作的。

![](img/b198041bd55d11627c26b4fb27bd839c.png)

One of the animations I created on my app. Displays the creation of a binary search tree & the algorithm to find one of the nodes in the tree.

这对我的学习产生了立竿见影的效果。我从动画排序算法开始，合并排序，快速排序和冒泡排序。然后迅速转向遍历二分搜索法树，弄清楚如何动画添加或寻找节点&它们的连接。

花时间思考每一个相关的步骤，这样我就可以突出我的动画中的差异，这让我比简单地记笔记更好地理解细节。我的学习不再是记忆。我现在有了一个概念&利用它的每个方面来创建一个易于理解的可视化。我的理解力有了显著的提高&我发现自己很享受这段时间，全神贯注于学习。

目前，我正在学习和实现模拟寻路算法的动画&可以自信地说，这个过程不仅让我在更深的层次上学习概念，还提高了我作为开发人员的技能。如果你是一个正在学习或计划学习 DS&A 的人，那么我强烈建议你不仅要记笔记，还要找到一种方法来应用你所学的信息。如果你和我一样，你会发现学习变得更加愉快&看到你学到的东西有了显著的提高。
