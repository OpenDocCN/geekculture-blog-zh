让我们面对现实吧，我们热爱数据科学的一个原因是因为它现在正在爆发。从零售到废物管理，每个行业都在期待数据科学家帮助保持相关性。翻译；这是一个赚钱的好方法。

但是，如果你像我一样，你也花了职业生涯的大部分时间来学习数据科学。通过在互联网上搜索代码、博客、免费教程，以及在你的笔记本电脑上修复无数个小时的错误。在数据科学上教育我们自己一直很难，因为这个领域根本不适合大学模式。它发展得太快，要涵盖的内容太多，很少有人能负担得起我们这些有经验的人。

换句话说，我们是自我激励者。我们有纪律、决心和毅力去单干，把事情做好直到成功，并为未来的雇主提供我们自己最好的支持。我们已经习惯了这样的采访:

> “您能使用 NLP 将文档组织成共同的主题吗？当然可以，这里有一个链接，链接到我的博文，展示了我是如何做到的。训练一个模型来检测图像中的复选框怎么样？没问题，只是为它分享了一个 Git 回购。”

本质上，我们是企业家。以破坏著称的企业家。但是打入企业家世界并不是一件容易的事情。因此，在本文中，我描述了我自己用数据科学背景培养企业家精神的经历。开始了。

# 千与千寻

我一直有创业精神。创造一些能为他人创造价值的东西的想法，足够的价值，让他们愿意为此付给你钱，这简直令人兴奋。我很幸运，我的父亲也是一名企业家。我看着他在多年的奉献、创新和对客户需求的关注中发展他的软件公司。所有这些都发生在微软和苹果等科技巨头崛起的时期。当时，世界刚刚意识到企业家精神的承诺和价值。

随着企业家近年来变得越来越像摇滚明星，尤其是那些知道如何赚钱的人，我们的世界也变得越来越数字化。可以说，企业家时代在很大程度上是由正在发生的巨大数字转型推动的。更大程度的数字化让人们更容易分享内容。分享内容使得传播教育年轻企业家利用数字文化所需技能的内容变得更加容易。

但是伴随着巨大的数字化而来的是巨大的数据流入，伴随着巨大的数据流入“而来的是巨大的责任”(彼得·帕克[蜘蛛侠]…我不得不)。作为企业家，我们有责任以负责任、合乎道德的方式利用数据的可用性，帮助我们为理想的客户打造有价值的产品或服务。要做到这一点，我们必须以不同的方式思考数据，有什么比数据科学更好的以不同方式思考数据的技能呢？

# 数据科学帮助我们以不同的方式看待数据

作为一名有一些经验的数据科学家，我喜欢创造性地思考使用这些技能来解决问题。事实上，在过去 15 年的职业生涯中，我大部分时间都在尝试，从企业家的角度来看，用数据科学的不同应用来解决我所面临的几乎所有问题。

这里有一个例子来说明我的意思。不久前，我面临着一个重大挑战。我发现有一周我在工作和生活中的所有责任都到了紧要关头。就像，我真的开始担心我将无法满足我的一些最基本的截止日期(TPS 报告，吃饭，从办公桌前站起来…你明白的)。

更疯狂的是，在我支持的几所大学里，作为贡献教师，教学负担比平时要高。你看，通常我只在这里或那里教一门课，最多教 20 个学生。然而，这个学期没有。不，这学期我面临着一些额外的课程(“嘿，布兰登，你能在这学期多上一两门课吗？我们有一些教师在休假“*额头上开始出现汗珠* *意识到我太好了，不能说不* *在我迄今为止最不明确和最隐晦的电子邮件中*“嗯，当然。”).但是为了不让我的焦虑和压力战胜我，我转向数据科学寻求解决方案。

虽然我的解决方案的细节已经超出了范围，但我还是留给你这个

…我用我多年来教同样的课程，一遍又一遍地批改同样的试卷

…此外，我还一丝不苟地将所有已评分的论文和我的评论保存在云存储桶中

…作为数据，使用我的数据科学工具包构建评论推荐引擎。

该引擎将消耗新的论文，并根据我过去的评论历史和可信度分数为论文中的每一段推荐评论。这是一个完美的解决方案吗？不，但是它确实减少了我的评分时间，从 10 分钟/论文减少到 2-3 分钟/论文，当你在任何给定的一周有 40 多名学生时，那是节省了很多时间。

但是我跑题了。我的观点是，作为数据科学家，我们有工具可以帮助我们看到业务问题(或个人问题😊)不同，因为我们看待数据的方式不同。我们使用的工具为我们提供了一个不同的视角，让我们了解如何利用数据来解决实际的业务问题。那么我有哪些激发创业精神的小技巧呢？

# 1.思想实验和真实实验一样好

学习如何将不断发展的数据科学技能应用到创业努力中的第一步是参与有价值但往往未被认可的真实事物的继子女，即思想实验。思维实验是一种练习，在这种练习中，一个思考者，大概就是你，让大脑去思考和假设一个现实世界的问题是否有一个数据解决方案。随着今天越来越多的数字化，有越来越多的业务问题需要利用数据。目标是思考数据和数据科学决策的后果，以寻求上述问题的解决方案。进行这种类型的练习有助于培养数据科学思维技能，你做得越多，这些技能就会越自然。此外，从任何硬编码需求或数据需求中释放出来，你还可以探索不可能的甚至荒谬的含义，这可能最终导致更大的创新。这种设备已经存在了几个世纪，它们的价值已经得到了公认。

# 2.但仍然是实验

随着你的思维实验技能的提高，你会找到解决问题的真正途径。事实上，路径是如此真实，以至于你甚至可以开始想象你的代码看起来会是什么样子。如果你发现自己处于这样的境地，不要忘记进行一些实际的实验。尽管我们的思想实验很有价值，但事实是它们仍然只存在于我们的头脑中。而且想法和现实很不一样。实现我们的想法有助于为我们的想法提供一个框架，识别潜在的障碍(技术或其他)，并激发我们继续学习的需求。

以下是一些关于实验的额外建议:

> *从小处着手。*
> 
> *不要担心准确性或性能。*
> 
> 不要做一个完美主义者。
> 
> *找一个平台来分享你的所作所为。*

# 3.兴奋起来…

我已经讨论过数据科学家如何在商业中挣扎，因为我们对数据科学本身如此兴奋

> 听着，妈妈，我的代码成功了，模型训练成功了，现在我可以预测一朵花的花瓣长度了😊

我们没有意识到我们的技能是为了解决更大的事情。这个世界不关心数据科学，这个世界关心的是跟踪购物清单或者在寒冷的日子里在交通堵塞的情况下需要多长时间才能到达热瑜伽。因此，你越是像数据科学家一样练习思考(见第一点)，就越容易将这种解决问题的方式应用到真正让你兴奋的事情上。如果它让你兴奋，它可能会让其他人兴奋。

# 4.寻找社区，做出贡献，并与他们建立联系

我知道你已经知道这一点，但我要再说一遍，网络很重要。我喜欢在两个领域思考网络；专业领域和客户领域。两者都要关注，但要从专业网络开始。专业网络可能会让你接触到客户网络，但你需要参与专业网络，而不仅仅是数据科学。

寻找数据科学家和其他企业家的社区。这样做有助于将你和你的技能展示给其他专业人士，他们可能会接触到你以前接触不到的客户网络。

例如，我知道我擅长应用数据科学工具来处理带有一些智能的文档，我甚至就这个主题写了一个[教程](https://towardsdatascience.com/from-outdated-to-updated-2df4b43fcc38)，但我不知道谁需要这样的解决方案。

我的意思是，我知道许多公司可能会感兴趣(毕竟，哪家公司不处理文件？)但是我在这些公司没有任何联系人可以帮助我适当地开发解决方案。

我的解决方案是，与其他面临这种或类似问题的企业家建立联系，他们可以接触到我不知道的客户世界。

我的观点是，社区是强大的(我[创办了](https://www.facebook.com/groups/thinkdatascience/)，部分原因就在于此，尽管有无情的自我推销*)，客户社区(如果你愿意，也可以说是观众)对成功的企业至关重要。因此，学习如何与这些社区互动和建立关系网可以极大地推动你走向成功。

# 5.我说过从小处着手吗？让我再次强调

利用您学习数据科学的方式。我们大多数人最初是在笔记本电脑上学习数据科学的。我们下载 Python 或 R，我们遍历代码示例，使用数据做一些事情，这些数据非常适合我们的操作系统。

毫无疑问，我们都处理过内存错误、缓慢的训练时间，以及当我们惩罚坐在我们腿上的小机器的 CPU 处理器时出现的依赖性冲突。尽管可能令人沮丧，但能够通过向下采样或分布式处理成功地释放内存，通过较浅的网络或较大的卷积滤波器改善训练时间，以及管理与虚拟环境或容器的依赖冲突，使我们更适合以极低的成本开发概念证明(POC ),以帮助未来的客户接受我们的愿景。

解决小问题的能力对潜在的商业伙伴来说是有用的、有价值的和有吸引力的，因为你知道如何保持低成本，同时提供接触技能的机会，否则要花很多钱来利用。
