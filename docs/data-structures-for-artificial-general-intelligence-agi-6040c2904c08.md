符号人工智能(或“手工编码”人工智能)在神经网络和机器学习的时代已经得到了一个坏名声，在这个时代，人们假设所有的知识都可以从大的标签数据集中提取出来(自下而上)。当符号、规则和关系已经存在于数据中时，为什么还要费心手工编码呢？

然而，人工通用智能(AGI)是不同的。如果 AGI 的目标是建造一台能够独立思考并解决复杂问题的计算机，那么象征性人工智能就是一条出路。当然，神经网络仍然在模式识别、预测、奖励寻求和优化中发挥作用。但是人类有复杂的动机和行为——野心、贪婪、关心、同理心、害怕被评判、需要社会接纳——神经网络根本无法处理。在我们的脑海中，我们需要独立于原始数据和感官输入的方法来表示、处理和模拟/建模世界(及其所有对象、演员、空间、时间和因果关系)。

那么，这里有一组潜在的数据结构，大脑可以用它来代表我们内部的想法、感知、计划、预测和动机。对这些心理表征进行操作的是一套心理规则，既有习得的也有先天的:

**物体(事物，东西)**

*   能够表示我们在环境中感觉到的物理事物，以及它们的属性和分类(例如，一辆汽车)。当我闭上眼睛，我前面的车继续存在于我的精神模拟中。
*   对象可以是分层的(例如，汽车的一部分是引擎，引擎的一部分是火花塞，等等)。)
*   我的心理规则从原始的感官知觉中产生这些“对象”数据结构，这些“对象”保存在记忆中，后来用于识别、确定、关联和预测它们自身的实例。我假设每个心理表征都有一个 GUID 或某种独特的编码，以允许它被心理规则引用

**演员(活体)**

*   代表我和其他人(以及动物)以及他们的议程、动机和行为的数据结构。行动者是一种特殊类型的对象，它有自己的内部动机和媒介(自由意志)

**属性**

*   表示对象属性(HAS-PROPERTY)的编码(或数字数组),例如颜色、形状、纹理、声音、气味
*   可以是分层的，例如像素形成线条，线条形成形状等。
*   每个属性都有一套认知/识别/关联/预测属性的心理规则

**关系**

*   心理表征之间的命名关系，例如“是”、“的一部分”等。(属性也是一种关系)
*   动词——例如，拥有、憎恨、出售、逃跑——也是关系。
*   一段关系通常与一个事件/信念相关联——你什么时候以及为什么相信这段关系是真的，以及这段关系的可信度(概率)

**信念/事件**

围绕事件或信仰的元数据，包括:

*   *什么*被相信:对象/演员+关系+对象/演员，例如，“乔拥有一个球”【可选:否定，“乔不拥有一个球。”]
*   信念的概率
*   *谁*(演员/对象)参与了信仰
*   *何时* —事件/信念发生的时间
*   哪里——事件/信仰的地点
*   *多少*——事件/信念中对象的数量
*   *归因* —你是如何(从谁那里)了解到该事件/信念的
*   *为什么* —事件/信念的原因

**因果关系**

*   两个事件之间的关系，其中第一个事件“导致”第二个事件。—如“比尔推了苏，导致苏摔倒”[可选:否定]
*   表明原因是偶然的还是有意的——这在人类事务中非常重要！
*   原因可能是多层次的
*   维护“原因事件”的贝叶斯概率

**时间**

*   事件发生的“时间”。可以是绝对时间(自 1900 年以来的毫秒数)、相对时间或定性时间(今天、昨天、几分钟前、自上次满月以来、走到邻居家所需的时间等)。)
*   时间可以是相对的和组合的，例如，很久以前，但是仅仅在也是很久以前发生的另一事件之后几秒钟
*   时间可以表示特定的时间点或持续时间，例如满月之间的时间

**位置/空间**

*   事件的位置，可以是绝对的(GPS 坐标)，相对的，或定性的(在我的掌握范围内，在一小段步行距离内)
*   位置可以是相对的和组合的，例如，离另一个位置很远，几英尺
*   位置可以表示空间中的点、区域、2D/3D 近似值(x 在 y 后面但在 z 前面)以及空间中的轨迹

**数量**

*   物体的数量，可以是绝对的(0，1，2，3)或定性的(无，一，二，三，几，许多)
    数量可以组合，例如，许多，但比其他数量少

**反/否定/反事实**

*   代表缺少某物(通常是一个事件)，或者反事实。
*   我没有…没有发生…不是这样…如果发生了呢？

计划/行动/程序

*   一系列步骤，或者代表低级动作的编码或数字阵列，例如，弯曲肌肉。
*   按潜在回报或效用排序的实现目标的可能行动列表。可能是分层的(计划中的计划)

**动机/目标**

*   决定奖励或效用计划进展的一套心理规则/功能，包括识别(尽管模糊)期望的最终状态和评估实现过程中的错误的能力。
*   人类的动机包括:野心、贪婪、情欲、生命/自由、仁慈、关爱、社会从众、害怕被评判、进食、躲避捕食者、寻求庇护、繁衍后代等。
*   本能或天生的行为仅仅是一个优先的目标。人们熟悉的“电车问题”表明，人类天生害怕选择伤害他人，即使是出于很好的理由——这也可以解释对权威的尊重(让别人来决定这件事吧！).

**注意**

*   我们“意识”焦点(工作记忆)中当前的一组心理表征

上面的心理表征并不像 C 程序中的指针那样直接链接在一起。他们更像是漂浮在精神汤里的球形磁铁，有时会相互碰撞而附着，有时会分崩离析，甚至迷失方向。

心理规则作用于心理表征，但它们也漂浮在心理汤里，不断寻找相关的心理表征来采取行动(就像 CSS 选择器)。心理规则可以生成新的心理表征，或者定位并重写现有的表征。

心理规则就是懒。它们只在需要时产生新的心理表征。例如，如果苏格拉底是一个人，所有的人都是凡人，我们可以暗示苏格拉底是凡人。但是我们不需要在我们的头脑中有一个“真理维护”系统来产生和维护苏格拉底死亡的所有可能含义的一致性。我们可以在以后需要的时候，根据需要，简单地思考其中的含义。现在，我们只是逐字记录事实，不加考虑。

最后一个想法。为了使*天生的*心理规则成为可能，该规则必须能够引用使用预先确定的、古老的命名惯例的心理表征，在之前*那些表征和技能是习得的(即，“后期绑定”)。例如，渴望-社会接受和恐惧-被判断和寻找潜在伴侣的规则，如果它们存在，必须依赖于我们出生时几乎不知道的概念(社会、文化、人际关系)，但完全“期望”以后学习。不知何故，当我们学习这些概念时，我们的心理表征被贴上了预先定义的、不变的名称(我们的 DNA 已知)，以允许先天的心理规则找到它们，并在以后的生活中对它们采取行动。*
