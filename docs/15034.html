通过示例理解Python中的Lambda函数

众所周知，我们使用函数来自动化软件中的过程。

Lambda函数实际上被简单地称为匿名函数。因此，采用任意数量的参数，我们得到一个表达式作为结果。

基本上，一个lambda函数接受一个x参数并返回如下结果:

func = lambda x : x
