# 我们会有真正的互联网吗？

> 原文：<https://medium.com/geekculture/will-we-ever-get-proper-internet-95646e62bb00?source=collection_archive---------14----------------------->

我关于开放互联网的最后一条建议并不太乐观，但我对我国宽带的未来仍充满希望。现在，在这方面有了一些新的发展，是时候讨论另一个承诺美国在互联网连接方面取得惊人进展的主要参与者——SpaceX 了。

# 你没有补贴

![](img/39036d7dfb24651f8008213e09fc90cf.png)

Jeremy Bezanger

不久前，由于 Ajit Pai 令人难以置信的无能，联邦通信委员会(FCC) [暂时批准](https://www.pcmag.com/news/spacexs-starlink-to-receive-886-million-from-fcc-to-improve-rural-broadband)向 SpaceX 提供 8 . 86 亿美元的补贴，以通过 Starlink 开发横跨美国的宽带。今年八月，在委员会仔细研究了该交易并收到诸如“为什么世界首富之一需要补贴？”之类的问题后，这个决定被[撤销了“为什么一个反对政府补贴和干预的人需要补贴和干预？”以及“你不是一家非常成功的公司，应该可以做得很好吗？”当然，他们的措辞有所不同，更倾向于调查为什么 Starlink 在补贴获得预先批准的两年里没有取得任何成果，以及为什么 Starlink 实际上需要或应该获得补贴，因为它没有达到要求，也不需要依靠自己的低轨道卫星建设新的基础设施。](https://arstechnica.com/tech-policy/2022/08/fcc-rejects-starlinks-886-million-grant-says-spacex-proposal-too-risky/)

现在，SpaceX 很不高兴，称这一拒绝“非常不公平”,并声称正是这一拒绝将使数百万没有卫星互联网的美国人陷入困境。SpaceX 的大卫·戈德曼甚至说，FCC 的决定是“从互联网上的某个地方挑选出来的”，如果他们想获得 FCC 的好感，这肯定不是一个人应该说的话。

# 但是等一下

![](img/7548f59a72b53dac5dd08e79b8f3efec.png)

虽然毫无疑问有很多人对补贴过于乐观，认为 SpaceX 将拯救我们的互联网，但这值得冷静看待。确切地说，为什么 SpaceX 认为它应该得到补贴？它为什么需要它？它实现了什么？

嗯，高盛声称，拒绝补贴让许多美国人“站在数字鸿沟的错误一边”，这是非常华丽的，但它回避了补贴会改变什么的问题。由世界前三大富豪运营的 SpaceX 会突然松一口气，不再以 600 美元的价格购买 Starlink 用户终端吗？也许正是这种补贴将帮助两家科技巨头——T-Mobile 和 SpaceX——最终获得足够的闲钱来“提供超出范围的覆盖范围”？哦，他们已经做了，没有施舍也做得很好。

与 FCC 的争吵可能会分散人们的注意力，但不会太久，不可避免地会出现这样一个问题:为什么由世界上最成功的人之一运营的这样一个大公司，不能做他们一直承诺的一件事？为什么要花时间与一个不欠你任何东西的委员会争论，而不是展示你的先进技术并让人们站在你这边？

# 这可能发生在明天

凭借 Starlink 的巨大技术能力(Starlink 已经在[乌克兰战争](https://www.politico.eu/article/elon-musk-ukraine-starlink/)中表现出了它的品质)，Musk 和 SpaceX 可以在任何他们想要的时候获得全美国高质量的高速互联网。这不会像补贴崩溃那样需要几年时间，而且会立即让人们站在他们一边。相反，我们仍然在对抗康卡斯特，我们的代表正在处理一些法案，这些法案稍微缓解了一些压力，但并没有解决问题，我们的互联网仍然像以前一样不完善。

我们必须停止落后，为此，SpaceX 需要行动，而不是空谈。
