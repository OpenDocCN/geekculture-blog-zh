你知道 Linux 网络进程是如何工作的吗？

![](img/ad824bfbf58b46a063e278bdb040aad8.png)

和 CPU、内存、I/O 一样，网络是 Linux 系统的核心功能。网络是一种将不同的计算机或网络设备连接在一起的技术。

本质上是一种进程间的通信方式，尤其是跨系统的进程间通信，必须通过网络进行。
