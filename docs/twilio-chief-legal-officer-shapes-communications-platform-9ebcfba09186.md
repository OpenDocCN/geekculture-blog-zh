Twilio 通过易于使用的通信应用编程接口(API)使语音、消息和视频通信渠道大众化。Twilio 的首席法律官 Karyn Smith 是这一努力的关键部分。自 2014 年以来，史密斯一直在这家总部位于旧金山的公司成立五年后加入该公司。她是特维利奥的第一个律师。

## **关于 Twilio**

Twilio 是一家云通信平台即服务( [CPaaS](https://www.ringcentral.com/cpaas.html#office) )公司。它的平台是高度可定制的，旨在改善企业与其客户的互动。Twilio 通过提供几个 web 服务 API 来促进客户参与。

Twilio 成立于 2008 年，目前拥有 4500 多名员工。该公司总部位于旧金山，由联合创始人兼首席执行官杰夫·劳森领导。Twilio 主要从平台使用中获得收入，2020 年，它报告全年收入为 17.6 亿美元。

隐私和网络安全法律问题是 Twilio 法律部门的主要关注领域，也是 Smith 女士的首要任务。Twilio 总部位于加利福尼亚州，该州拥有美国最严格的消费者隐私法。该公司最近面临消费者集体诉讼，声称 Twilio 记录了客户的电话和短信。

## **Twilio 文化与法律团队**

史密斯女士管理着一个 60 多人的全球团队。她领导法律部门的方式反映了公司的整体组织文化。“Twilio Magic”一词被用来定义公司的十大指导原则:1)成为所有者，2)赋予他人权力，3)不耍诡计，4)穿顾客的鞋子，5)写下来，6)无情地优先考虑，7)大胆，8)包容，9)画猫头鹰，10)不要停留。

与许多科技初创公司一样，Twilio 的内部律师在重要的商业决策中发挥着不可或缺的作用。作为 Twilio 的法律总顾问，Smith 女士直接向首席执行官汇报，并且是执行领导团队的成员。

## **法律法规合规经验的历史**

在 2014 年加入 Twilio 领导其法律、政府事务和监管合规团队之前，史密斯女士曾在视频游戏公司 Zynga 担任副总法律顾问。2011 年，史密斯帮助该公司顺利完成了 10 亿美元的 IPO。在加入 Zynga 之前，史密斯是国际律师事务所库利 LLP 的合伙人，该事务所以代理一些最热门的科技公司和新兴成长型公司而闻名。

## **合作、扩张和改善客户体验的计划**

史密斯还帮助 Twilio 投资 7.5 亿美元，收购了 Syniverse Technologies 的少数股权。Syniverse 专注于移动和云连接。美国电话电报公司、T-Mobile、威瑞森、沃达丰和中国移动等移动电话运营商依靠 Syniverse 作为跨网络的公共交换中心。Syniverse 通过运营商网络传送文本消息、通话记录和其他客户数据。

通过与 Syniverse 的合作，Twilio 希望将其 API 服务专业知识与 Syniverse 的移动运营商联系相结合，以提高通信效率。

Syniverse 执行主席 James Attwood 表示:“此次合作将为 Syniverse 提供 Twilio 广泛的企业和 API 服务专业知识，创造机会继续发展 Syniverse 高度创新的产品组合，帮助移动网络运营商和企业更好地为客户提供通信服务。

在疫情期间，Twilio 寻找更多途径来满足客户的数字需求。2020 年 10 月，Twilio 以 32 亿美元收购了客户数据管理平台 Segment。Segment 的技术使组织能够使用 API 在不同的应用程序之间移动客户数据。

Twilio 的首席执行官指出了这个机会对公司长期目标的重要性。Jeff Lawson 说:“数据孤岛破坏了卓越的客户体验。“细分市场让开发商和公司打破这些藩篱，构建客户的完整图景。与 Twilio 的客户参与平台相结合，我们可以在客户服务、营销、分析、产品和销售方面创建更加个性化、及时和有效的参与。”

## 到达了许多里程碑，一次深情的告别

在高增长技术公司工作了 25 年后，Karyn Smith 最近宣布她打算离开公司。她已经同意留下，直到 Twilio 找到继任者。

在 Twilio 工作期间，Smith 女士帮助 Twilio 达到了重要的里程碑。随着 Twilio 发展合作伙伴关系并改进其 API 技术，公司的总法律顾问将继续发挥重要作用。史密斯女士为 Twilio 的法律职能打下了坚实的基础。下一任总法律顾问将会有很大的作为。
