如果你刚刚开始你的编程之旅，你可能已经在 YouTube 和其他网站上看到过无数的教程，教你如何构建简单的应用程序。通常，这些是具有基本 UI 的基本 todo 风格的 web 应用程序。有些可能包括也可能不包括连接数据库和用户认证。这些项目在开始时肯定是有帮助的，但在某些时候，你会想要从 todo 应用程序中毕业，并转向更复杂的东西。这就是副业的用武之地。

在我看来，一个好的兼职项目可以总结为任何需要你自己解决大量问题的事情，同时也让你接触到不熟悉的技术。在本文的其余部分，我将介绍一些当您开始一个新项目时应该考虑的问题。希望这些能帮助你保持学习和尝试新事物的动力！

## 增加复杂性和挑战

这个是不言自明的。如果你想成长为一名程序员，你需要承担更多更具挑战性的任务。然而，这个过程是渐进的，以避免阻碍人们学习。

我们如何增加项目的复杂性？我们以一个 todo app 为例。让我们假设现在我们可以登录应用程序，添加和删除待办事项提醒。让这个应用程序变得更复杂的好的渐进步骤可以是:

*   能够将待办事项分组在一起
*   向待办事项添加截止日期
*   能够按截止日期或分组搜索/过滤待办事项

想出新项目的时候呢？你应该尝试做一些你以前从未做过的事情。从未实现注册表单？试着把它融入到你的下一个项目中。

## 考虑您想要构建的功能

当掌握一门新的编程语言或框架时，教程是很好的选择。然而，当你已经掌握了一两种编程语言时，你应该少考虑学习语法和框架，多考虑如何构建功能。不要想“我想在我的下一个项目中学习 React”，而是想“我需要为我的下一个项目构建一个仪表板 UI，我将使用 React 来构建它。”想创业？然后专注于学习如何创建一个有效的登录页面，不管使用什么工具来构建它。

这与现实世界的软件工程工作类似。工程任务通常是产品或管理部门要求的功能。创建该功能所需的语言或工具并不重要，重要的是确保最终产品能够为用户工作。

回到关于增加复杂性的第一点，当选择一个副业时，试着想想你以前从未构建过的功能或软件组件。如果你也决定使用一个不熟悉的框架来构建它，那就更好了。这样你可以一次学习多种东西！

## 使用你喜欢的产品的 API

缺乏完成项目的动力困扰着我们所有人。这也是为什么在选择新的副业项目时，要迎合你的个人兴趣或爱好。让事情变得有趣的一个方法是将你喜欢的产品的 API 整合到你的项目中。这有助于你保持动力，同时也学习如何阅读 API 文档和提出 API 请求。很多著名的服务都为像你这样的开发者提供免费的 API。这里有几个你可以试着建立:

*   [Spotify Web API](https://developer.spotify.com/documentation/web-api/) —浏览艺术家、播放列表、专辑等。
*   [YouTube 数据 API](https://developers.google.com/youtube/v3) —发布视频，获取频道和视频的分析等。
*   [Reddit API](https://www.reddit.com/dev/api/) —获取帖子和评论进行文本分析，获取模因等。
*   [Twitch API](https://dev.twitch.tv/docs/api/) —实时阅读聊天内容，并通过网络挂钩获取频道事件

你可以通过谷歌很容易地找到其他的 API。比如金融？有一堆金融数据 API。做饭？可能有一个配方 API。如果不是，那就是一个新的副业思路:)。

## 添加云服务

似乎最近的每份工作都要求熟悉云服务。将 AWS、GCP 或 Azure 等云服务提供商整合到一个辅助项目中是获得这种体验的好方法。在所有提供商之间，有数百种不同的服务和工具针对从部署到数据科学和机器学习的特定用例。这意味着可能有一种云服务适合你的项目。

如果你的项目类似于一个网站，一个好的起点就是简单地把它部署在云上。回到我们增加复杂性的主题，然后您可以添加部署了应用程序的多台机器，并将它们放在负载平衡器、CDN 等后面。

一定要仔细阅读提供的服务目录。对外面有什么有一个广泛的想法，为你能建造什么打开了大门，让你更有创造力。像 AWS SNS 这样的服务允许你给你的项目添加消息传递功能。现在你可以给你的待办事项应用添加短信提醒了！

## 把所有的放在一起

根据以上原则，我将列出一些你应该尝试和扩展的项目想法。考虑到自己的兴趣，自己做这个头脑风暴练习。

*   一个日本学习资源聚合器，利用 Reddit 和 YouTube API 获取最新的学习资源，并将它们存储在 DynamoDB 表中。(这个可以扩展到你选择的任何话题，我只是把学日语当成一种爱好。)
*   一个 Twitch 股票模拟器，聊天可以在一个共享的模拟账户上输入订单。(好的，我已经在这里建立了这个。可以尝试一下 crypto。)
*   一个全栈网络应用程序，健身教练可以管理他们的时间表，并跟踪他们的客户的进展。它需要被托管在某个地方，需要存储用户数据。
*   基于用户输入/拥有的配料返回食谱或烹饪视频的食谱 API。

通常，我喜欢一次头脑风暴一堆项目想法，然后从中挑选。这有助于留出一些时间来练习，因为这是一项一开始会很难的技能。

希望这篇文章已经激励你开始挑战自己，这样你就可以成长为一名开发人员。快乐大厦！
