在频数统计领域，夏皮罗-维尔克检验是一种正态性检验。该测试通常用于 50 个元素或更少的小样本。

夏皮罗-维尔克检验的零假设是总体呈正态分布。因此，另一个假设是，总体不是正态分布的。
