# 我从西蒙·沃德利的书中学到了什么(第一部分)

> 原文：<https://medium.com/geekculture/what-i-learned-from-simon-wardleys-book-part-1-c27009ca6d44?source=collection_archive---------49----------------------->

我最近读完了西蒙·沃德利的[书](/wardleymaps/on-being-lost-2ef5f05eb1ec)，书中描述了他发明的一种新颖的绘图技术。因为它非常密集，我花了很长时间才看完，所以我将描述一下我从中得到的东西，这既是一种唤起我的记忆的方式，也希望能为他人提供价值。
