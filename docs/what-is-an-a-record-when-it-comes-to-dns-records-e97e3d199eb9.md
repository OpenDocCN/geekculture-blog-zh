# 说到 DNS 记录，什么是 A 记录？

> 原文：<https://medium.com/geekculture/what-is-an-a-record-when-it-comes-to-dns-records-e97e3d199eb9?source=collection_archive---------21----------------------->

![](img/1f8f3cac020bdb93f02615e47d1b8b81.png)

A 记录是一种将主机名指向一个站点的方法。A 记录是标准的 DNS(域名服务)记录，“A”代表“地址”。

简而言之，A 记录表示一个域的 IP 地址。当你得到一个域名的 DNS 记录时，A 记录会显示 IP 地址。
