# 有哪些不同种类的 HTML 编辑器？

你知道什么是 HTML 编辑器。您在编辑器中编写 HTML 代码，保存它，并查看它生成的基本布局。大多数 HTML 编辑器带有工具栏和下拉选项，为您提供不同的开发模式。

他们分为两种。两者没有太大的不同，但是它们提供了截然不同的开发体验。

两种主要的 HTML 编辑器是:

*   **基于文本的 HTML 编辑器:**这些是简单的文本编辑器环境。你可以直接输入你的 HTML 代码，也可以包含更复杂的 HTML 标签。基于文本的编辑器更有可能拥有健壮的文本编辑特性，比如代码完成和错误高亮。然而，大多数没有实时代码预览选项。这意味着你需要有足够的 HTML 经验来避免错误。
*   **所见即所得 HTML 编辑器:**所见即所得 HTML 编辑器给你一个内嵌的可视化编辑环境。你构建你的页面，并以图形方式添加文本和其他内容。设计或手动编码页面决定了浏览器如何生成最终的网页。使用这个 HTML 编辑器，您可以获得一致的实时预览来指导您的编码。当一个网站有大量的视觉元素时，开发者更喜欢它们。

# 这些年来，HTML 编辑器有什么样的创新？

随着互联网越来越受欢迎，人们想要网站，公司发布工具来帮助他们开发网站。最早的 HTML 编辑器是简单的记事本应用程序。

您可以在其中编写 HTML 标记，保存文件，并在浏览器中打开它们。浏览器将呈现标记。从那时起，HTML 编辑器走过了漫长的道路。

最终，公司开发了专用的文本编辑环境来编写和编辑 HTML。然而，这些编辑器仍然是文本驱动的。他们没有提供现场预览。

渐渐地，HTML 编辑器上市了，允许你直接编辑你的网页。他们是“所见即所得”的编辑。你不需要预览，因为你如何编辑你的网页通常是什么浏览器会呈现。

如今，用户有两种类型的 HTML 编辑器可供选择。不幸的是，一些所见即所得编辑器有一个共同的痛点。并非所有的浏览器都以完全相同的方式呈现 HTML。

然而，有一些可以在所有浏览器上流畅运行。它还具有丰富的特性，使其成为开发人员社区中最受欢迎的所见即所得选项。
