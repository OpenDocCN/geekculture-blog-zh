你是否曾经需要检查访问者/用户是否正在使用脸书的应用内浏览器浏览你的应用？这实际上可以用一点 Javascript 很容易地实现。

我们可以使用`navigator.userAgent`来检查用户正在使用哪种类型的用户代理。
