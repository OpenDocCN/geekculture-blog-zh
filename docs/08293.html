当我们需要在R中建立一个二进制模型时，我们通常会使用glm函数。在二项式部分，我们可以选择两种模型。

glm(admit ~ gre + gpa + rank, family = binomial(link = "probit"), data = mydata)glm(admit ~ gre + gpa + rank, family = binomial(link = "logit"), data = mydata)
